　　“聂兄，我们被人盯上了！”韩盈面色凝重的说道。
<br/>　　“什么？”
<br/>　　聂乾闻言立刻心中一惊，随即急忙问道：“是什么人？有多强的实力？”
<br/>　　韩盈微微侧着脑袋，催动自己的战魂，仔细聆听追踪之人的情况。
<br/>　　他的战魂赋予了他超强的听力，可以听出对方气息波动的强弱，从而判断对方的修为。
<br/>　　“两个人，均是武帝境四重巅峰境界！”
<br/>　　少倾之后，韩盈面色沉重的说道，这个结果让她的心沉到了谷底。
<br/>　　她的修为才武帝境一重巅峰，面对武帝境四重的武者，只有被秒杀的份。
<br/>　　聂乾的脸色亦是很不好看，他虽然有武帝境二重境界，但也完全无法抗衡武帝境四重的武者。
<br/>　　“苏兄，我们怎么办？”两人将目光落在了苏莫的身上，不约而同的问道。
<br/>　　苏莫是他们中实力最强的人，自然而然的成为了他们的主心骨。
<br/>　　苏莫亦是面色凝重，听闻两人之言，心中便立刻急思了起来。
<br/>　　两名武帝境四重巅峰武者？
<br/>　　这倒是让他稍稍的松了口气，只要不是武帝境五重以上的武者，想要杀他也不是容易的事情。
<br/>　　他准帝境之时，便能斩杀武帝境三重的冷江，现在突破到了武帝之境，应该不会比武帝境四重武者弱多少。
<br/>　　而且，他的武魂进化为战魂之后，吞噬之力暴涨了一大截，到底达到了多强的地步，他自己都不清楚。
<br/>　　因为，自从他突破到武帝境界，还没有全力催动过。
<br/>　　这一次，倒是可以试一试了。
<br/>　　“你们两人先走，我留下来对付那两人！”沉吟了少倾，苏莫向两人说道。
<br/>　　“不行！”苏莫话音一落，聂乾和韩盈顿时开口，同时否决了苏莫的决定。
<br/>　　“苏兄，你虽然突破到了武帝之境，但也不一定是武帝境四重武者的对手，而且对方有两人！”聂乾沉声说道。
<br/>　　“是啊！这两人看来是有备而来，我们还是想办法逃走吧！”韩盈说道。
<br/>　　“你们有好的办法吗？”苏莫问道，若是能逃掉，他自然不想留下来冒险。
<br/>　　“这……！”两人顿时语塞，他们并没有好的办法，能保证安然逃脱。
<br/>　　“你们到星云城等我！”苏莫向两人说了一声，便立刻停下了身形。
<br/>　　星云城，便是星空传送大阵所在的地方。
<br/>　　“苏兄！”聂乾和韩盈立刻也停下了身形，苏莫留下来的话，实在太危险了。
<br/>　　“你们快走，我有保命的底牌，你们无需担心！”苏莫立刻催促道。
<br/>　　两人无奈，但见苏莫一脸自信的模样，也只能听从苏莫之言了。
<br/>　　“苏兄，我们等你，你一定要回来！”
<br/>　　两人留下一句话，便立刻闪身离去，他们也知道留下来帮不了忙，反而会拖累苏莫。
<br/>　　见聂乾和韩盈离去，苏莫松了口气，便转身望向身后。
<br/>　　他身形挺立于云端，长剑出现在了手中，等待两位追踪之上的到来。
<br/>　　苏莫并没有等太久，几乎是呼吸之间，便有两道流光风驰电逝而来。
<br/>　　嗖！嗖！
<br/>　　流光划破空间，来到了苏莫前方百里之处停了下来，化为了两名青年男子。
<br/>　　这两人均是身穿黑衣，身材高大，其中一人面色黝黑，面容粗犷，而另一人则是面容俊朗，面色冷峻。
<br/>　　两人到来之后，打量了一番苏莫，面容粗犷的青年大笑道：“苏莫，你知道自己逃不了了，所以留下等死吗？”
<br/>　　苏莫同样在打量着这两人，闻言顿时眼眸一凝，这两人认识他，看来是神武学府之人了。
<br/>　　神武学府中，他只得罪过赤霞洞天一脉，和太史世家一脉的人。
<br/>　　赤霞洞天不太可能派人杀他，毕竟冷江只是一个小人物，而他与冷江也是自愿决斗。
<br/>　　最大的可能就是太史世家一脉，因为他没有屈服于太史轩和尤天韩的威胁，对方恼羞成怒了。
<br/>　　“看来你们是太史世家派系的人了！”苏莫冷冷的说道。
<br/>　　“不错！老子赵城，今天就是来要你的命！”面色粗犷的青年满脸杀机的说道，他也没有隐瞒，因为苏莫很快就是一个死人了！
<br/>　　另一名未开口的冷峻青年，则是转头查看四方，发现只有苏莫一人在此，聂乾和韩盈已经逃了。
<br/>　　不过，他也不在乎，他们此行主要的目的，是杀苏莫。
<br/>　　待杀了苏莫之后，再收拾聂乾和韩盈也不迟！
<br/>　　“我正愁没有足够的资源修炼了，你们两人的修为，倒是正合适！”苏莫嘴角一勾，露出了森然的笑容。
<br/>　　若是能吞噬了这两人，庞大的精气说不定能让他一举冲破境界瓶颈。
<br/>　　“怎么？你还想反杀我们，夺取我们的储物戒？”粗犷青年面带讥讽的说道。
<br/>　　“不仅是储物戒，还包括你们身上的一切！”苏莫淡漠的说道。
<br/>　　“和他废什么话？先杀了再说！”
<br/>　　冷峻青年不耐了，手中剑光一闪，出现了一柄金色的长剑。
<br/>　　“杀！”
<br/>　　暴喝声响起，却是粗犷青年率先出手，一拳凌空向苏莫暴轰而来。
<br/>　　刹那之间，灰白色的拳茫凝结成了一个房屋般大小的拳头，破灭空间，镇杀一切。
<br/>　　拳茫威势滔天，恐怖的玄力波动，震散了方圆十万里内的云层，磅礴的威压让得下方的大地，都轰隆隆的崩裂开来。
<br/>　　拳茫凝结成的拳头速度快到极致，瞬间跨越了百里距离，临近了苏莫。
<br/>　　“的确强大！”苏莫面色凝重如水，此人的实力比冷江强大的太多了，完全不可同日而已。
<br/>　　不过想杀他还远远不够。
<br/>　　“碎！”
<br/>　　一声厉喝响彻八方，耀眼的三色剑光暴冲而起，分开天地，瞬间斩中了灰白色的巨大拳头。
<br/>　　这一剑威力同样惊天动地，剑威冲破九霄，无可匹敌。
<br/>　　轰！
<br/>　　一声惊天爆响，巨大的拳头随之炸开，毁灭了一方空间，咆哮的拳劲如同狂涛骇浪，席卷八方。
<br/>　　嗖！
<br/>　　苏莫的身形止不住的暴退，足足飞退了一千余里，才勉强停了下来。
<br/>　　一击硬撼，他明显落于下风，力量比之粗犷青年差了不少。
<br/>　　不过，这一击只是试探而已。
<br/>　　咻！
<br/>　　苏莫刚刚稳住身形，还没来得及压制体内沸腾的气血，一道浩大无比的金色剑气，便向他急速袭杀而来。
<br/>　　是那名冷峻青年出手了！
