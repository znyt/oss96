　　第616章
<br/>　　天下大乱，英雄辈出。其中难免就夹杂一些滥竽充数，想着浑水摸鱼的人物。这些人还不是少数，而是占了绝大部分。终究是一场大绝杀，这场乱世的赌局中混入了太多不该混进来的人，但只有活到最后的少数几个人，才有资格以天下为战场，决最后的胜负。
<br/>　　如今群雄割据，大半江湖人都投靠在各路诸侯门下。但也有些人不甘心寄人篱下，想着自立门户，与天下群雄争一争天下。
<br/>　　这莫将难就是其中之一，他是西北万马堂的首领，也算是绿林道的枭雄。当初正义教起事时，便想着趟一趟浑水，从中捞点油水。但正义教起来的快，败下来得也快，莫将难也没捞到什么好处。
<br/>　　这些年他可没有闲着，暗中招兵买马，招揽绿林道上的英雄。想着当初林少羽揭竿而起，轻易便扯起十万大军，凭什么他能做到的事，自己做不到。
<br/>　　盯着突然落到自己手中的程大雷，在最初的愤怒过后，莫将难很快冷静下来。双目灼灼的盯着程大雷，想着怎样将他身上的所有价值压榨出来。
<br/>　　他脸上的怒容转变成笑容，躬身行了一礼：“原来是程当家到此，刚才在下有些失态，还请程当家勿怪。”
<br/>　　“好说，好说。”程大雷道。
<br/>　　“我们两家素无瓜葛，同在凉州混，我对程当家的名声也一直很是佩服。”莫将难道：“这才是大水冲了龙王庙，一家人不认一家人，我立刻就给程当家松绑。”
<br/>　　程大雷在江湖上的名声是有的，不过都是坏事做尽的恶名，至于威名……程大雷也不知道自己有什么威名。
<br/>　　莫将难装腔作势就要与程大雷松绑，可双手要伸过来时，他突然顿住。
<br/>　　程大雷抬起头，双眼笑盈盈的看着他。
<br/>　　莫将难长叹一口气：“原该立刻给程当家松绑，差人将程大雷安然无恙送回蛤蟆城。可程当家的手下的人的脾气我也有所了解。我这一腔好意别被当成了歹意。不若程当家修书一封，信中写明原委，我差人送到蛤蟆城，也免得中间生了误会。”
<br/>　　程大雷点点头：“你说得有道理，莫兄一番美意，我又怎能辜负。来来来，取纸笔来，信中好好讲个明白。当然，这次事情也不能让莫兄白忙活，只是不知莫兄是要兵马还是要金银？”
<br/>　　“这还真是，这还真是……”
<br/>　　莫将难搓着手，忽然一怔，程大雷又岂是好对付的人物，自己松开他的绳索，他不写信却要砍自己的脑袋怎么办。他那鬼神不测的快剑，凭自己的本事可是挡不住。
<br/>　　“程当家误会啦，误会啦。”莫将难又道：“发生这种误会，我给程当家赔罪都来不及，又怎么会是想勒索财物。实在是在下心中有件大事，要与程当家商量。”
<br/>　　“喔，愿闻其详。”
<br/>　　“如今天下大乱，百姓遭殃，这种事程当家自然也看到了。我辈当做些事情，替天下苍生做些事。只要程当家点头答应，我万马堂的人马愿为程当家所用，揭竿而起，为天下苍生做些事情。”
<br/>　　“呀，你可是与我想到一起去了。”程大雷道：“快把我松开吧，我们立刻就做这件事。”
<br/>　　莫将难本想施展三寸不烂，忽悠得程大雷和自己联手。可程大雷答应得太痛快了，还满脸诚恳，莫将难倒是很想相信他，但程大雷明显不是诚恳的人呐。
<br/>　　“我一片诚心，程当家该不会是将我当傻子耍吧？”莫将难的声音冷下来。
<br/>　　“这个……很明显么？”程大雷一脸无辜道。
<br/>　　“你。”莫将难顿时如受奇耻大辱，手握紧刀柄，真恨不得一刀宰了程大雷。
<br/>　　慢慢，他又将刀柄松开，既然程大雷不肯合作，那自己只能用另外一种方式发挥他的价值。
<br/>　　“很好，很好。”莫将难冷笑道：“希望程当家死到临头，还能这样豪横。”
<br/>　　说着，他迈步走出农神庙，庙内只有周桃看着程大雷，程大雷斜靠在神像下，目光瞧着周桃。
<br/>　　“卿卿家人，只不过这看人的眼光终究差了些呐。”程大雷有些惋惜道。
<br/>　　周桃冷哼一声，道：“你再敢辱莫郎一句，我便一刀宰了你。”
<br/>　　“好，好。”程大雷道：“我怕了你不成，都说英雄难过美人关，本当家偏偏过不了你这一关。”
<br/>　　“那个……”程大雷顿了顿，还是将心中的困惑问出口：“那鬼脸婆婆成名在二十年前，如果活着也该是一把年纪，可看姑娘的年纪，应该不是那鬼脸婆婆吧？”
<br/>　　周桃冷哼一声：“你又不曾见过我的本来面目，你怎知道我多大年纪？”
<br/>　　这……
<br/>　　这件事程大雷当真说不太好，传说那鬼脸婆婆的易容术出神入化，谁知道现在是不是她的真实面目。
<br/>　　“关于鬼脸婆婆，你了解多少？”周桃忽然道。
<br/>　　“倒是知道一些。”程大雷轻轻点点头，他在长安待过一年，还蛮了解京州一地的大贼巨盗。那鬼脸婆婆成名的本事并不是武艺，说白了她就是个骗子。压箱底的本事一是易容术，一是魅惑之术。
<br/>　　靠着美貌不知哄得多少达官贵人妻离子散，家破人亡。最后事情终于搞大了，骗到皇族身上，朝廷派出鱼龙卫捉拿她。但结果却是一无所获，她在天罗地网的追捕下平安脱身，从此一战扬名。
<br/>　　“其实……”周桃缓缓道：“那鬼脸婆婆并非一人。”
<br/>　　“喔？”程大雷睁大眼睛：“还请姑娘赐教。”
<br/>　　周桃道：“鬼脸婆婆一开始的确只有一人，但很多事情一个人做不来。于是她收集无家可归的孤女，传授易容术和房中术，一群人配合行骗，总是容易方便得多。我确实就是鬼脸婆婆，不过却只是其中一人。”
<br/>　　有些事太过肮脏，周桃也不愿意向人说起。鬼脸婆婆在江湖上的名声很大，但说白了就是一个靠女色诱骗男人家财的骗子组织而已。这周桃从小不知父母，很小就靠美色诱惑男人，背后她经历过什么，外人就难以相信了。
<br/>(本章完)
