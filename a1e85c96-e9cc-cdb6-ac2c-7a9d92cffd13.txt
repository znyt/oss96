        傅夜沉顺势将手臂绕过童筱颖的肩膀，搭在她另一侧的肩头上，将自己一半的重量全压在了童筱颖身上。
<br/>
<br/>        童筱颖有点儿吃力地揽着傅夜沉的腰，支撑着他站稳，非常想给他找个地方坐下来就好。
<br/>
<br/>        此时，那边三个室友见状，于是纷纷起身，走过来帮忙。
<br/>
<br/>        傅夜沉穿着一身黑色的西装，身材高大结实，相貌英俊，眉眼英姿飒爽。
<br/>
<br/>        三个女生一见到傅夜沉这张天生长得帅气的脸，即使是醉酒成失魂落魄，也不失俊颜的样子，一个个忍不住犯起了花痴。
<br/>
<br/>        “筱颖，你哥哥啊？长得好帅！”女室友蓝筱蝶是个爱好二次元的女生，此刻的她就像一只爱撒娇的猫咪，扭着身子，两只手都比到了嘴巴下，娇滴滴地说道。
<br/>
<br/>        好在，另一个女室友恭如兰还是一本正经地关问：“筱颖，你哥哥要不要紧啊？”
<br/>
<br/>        童筱颖回答：“他喝多了！”
<br/>
<br/>        “那去给他开个房，让他休息吧！”就连最不爱说话的室友严恩恩也开口说道。
<br/>
<br/>        “没事，我打电话叫辆车，送他回家就行了。”童筱颖微微一笑。
<br/>
<br/>        蓝筱蝶不禁纳闷地反问：“你家住滨江城啊！这么晚了，哪还有车回去。”
<br/>
<br/>        童筱颖赧然一笑：“他、他是我大表哥，就住在临海城里。”
<br/>
<br/>        “那好吧！那我们就先回去了。”蓝筱蝶接着说道。
<br/>
<br/>        童筱颖扶着傅夜沉点了点头。
<br/>
<br/>        蓝筱蝶、恭如兰、严恩恩一同转身离去，蓝筱蝶还不忘跟恭如兰和严恩恩继续花痴：“筱颖的哥哥真的好帅啊！等筱颖回来了，我一定要问问筱颖，看看她哥哥有没有女朋友！”
<br/>
<br/>        “你啊！一见到帅哥，就想追！太没出息了！”恭如兰笑道。
<br/>
<br/>        “……”严恩恩在一旁只是听着，不发表任何意见。
<br/>
<br/>        三个女室友离开后，童筱颖掏出手机，准备给苏漫雪打电话的时候，却被傅夜沉一把夺走了手机。
<br/>
<br/>        “送我去酒店。”傅夜沉冷冷地说。
<br/>
<br/>        童筱颖瞥了傅夜沉一眼，没好气地回答：“我身上没钱了。”
<br/>
<br/>        “我有卡。”傅夜沉接着说道。
<br/>
<br/>        “你还是回家吧！这里离傅家大庄园不远。”童筱颖凝眉道。
<br/>
<br/>        傅夜沉站直了身子，一手擒住童筱颖的胳膊，一手捏住了她的下巴：“这段时间，为什么冷落我？”
<br/>
<br/>        “我……”视野里，傅夜沉的俊脸瞬间被放大，童筱颖欲言又止，微微偏过头去。
<br/>
<br/>        傅夜沉又将她的脸面向自己。
<br/>
<br/>        他随着凄清夜色一同逼近她，身上的气息愈发明显，占据她全部的呼吸。
<br/>
<br/>        她今天新生开学报到，他其实跟踪了她一天。
<br/>
<br/>        自从她跟傅昼景回去后，她又不来公司上班，他见不到她，每天度日如渡年，想她想到心里发疯。
<br/>
<br/>        所以，他今天当了一天的“变态”，跟踪了她整整一天，她和她的室友逛这条街，他便在这条街的酒吧里喝酒。
<br/>
<br/>        他就坐在酒吧里的那边的落地窗前，可以将在这边坐在小吃摊前的她看得一清二楚。
<br/>
<br/>        而且，他喝酒并不是没有钱买单，而只是想在酒吧门外闹事，吸引她的注意。
<br/>
<br/>        傅夜沉突然觉得自己也有如此幼稚的时候，甚至幼稚到可笑。
<br/>
<br/>        童筱颖只觉他身上散发的是一种熟悉的又陌生的气息，在自己眼前流淌。
<br/>
<br/>        她越是近在咫尺，他越是有种想要强吻她的冲动。
<br/>
<br/>        傅夜沉情不自禁地微微俯身，低头凑了过来。
<br/>
<br/>        童筱颖下意识地偏头。
<br/>
<br/>        “筱颖，你有没有对我动过心？”耳边突然传来他的低语，当他温热的薄唇贴近她软凉的耳垂时，童筱颖的心脏猛地揪了起来。
<br/>
<br/>        几乎是一种本能，童筱颖使尽浑身解数地推开了傅夜沉。
<br/>
<br/>        没有吗？
<br/>
<br/>        她真的从来都没有对他动过心吗？
<br/>
<br/>        哪怕他比阿景更先邂逅她，她也没能对他动过心？
<br/>
<br/>        童筱颖双眼直直地看着傅夜沉，发现他压根就没醉后，气呼呼地转身便走。
<br/>
<br/>        大哥果然是个花花肠子，身边有了苏漫雪和季思妍还不够，还要跟她说这种话来调戏她。
<br/>
<br/>        傅夜沉看着童筱颖生气离去的背影，无奈地苦笑。
<br/>
<br/>        当他准备上前去追她的时候，前面突然有人拦住了她的去路。
<br/>
<br/>        童筱颖也没想到，这个拦住自己的人竟然是何明旭和他的现任女友林若琴。
<br/>
<br/>        “筱颖，你还真的是阴魂不散！”何明旭搂着林若琴的腰肢，对童筱颖冷嘲热讽道。
<br/>
<br/>        童筱颖的柳眉微微皱了皱，不想跟他多费口舌，打算绕道离开的时候，却被林若琴给伸手拦住。
