　　龙都城中，有很多小世界，有阵法构建的虚空小世界，也有大量生灵生存在真实小世界。
<br/>　　龙帝所选择的小世界，是一处很大的虚空小世界，乃是由阵法之力构建而成，空间极为稳固，能经得起高强度的战斗。
<br/>　　这处虚空小世界，方圆百万里，尽是一望无际的虚空。
<br/>　　吞虚龙帝端坐在一张一丈宽的龙椅之上，大量的龙族长老、执事、普通族人，聚集在两侧，黑压压一大片。
<br/>　　龙腾和苏莫，龙族大长老，也均是在龙帝的身侧。
<br/>　　另一个方向，是各大势力之人，神王级别强者加上大量的年轻天才，也有六七百人之多。
<br/>　　唰！
<br/>　　苏莫身形一闪，来到了前方的虚空中，目光扫视天尘宇和紫霄等人，冷漠的道：“谁愿意送死，尽管上来吧！”
<br/>　　嗖！嗖！！
<br/>　　苏莫话音一落，便有两人迫不及待的上前，这两人一位是天王宫弟子，一位是地王宫的弟子。
<br/>　　两人分别得到了天尘宇和地藏神王的吩咐，率先上场，想要抢先一步镇压苏莫。
<br/>　　这两人的修为，一人是四品低级，一位是四品中级，全部都高于苏莫。
<br/>　　而紫霄和雷焱神王殿、万化神殿、无双神殿的强者，倒是没有立刻派人出手，毕竟，他们都知道天王宫和地王宫，对于苏莫是志在必得。
<br/>　　“天尘宇，我方弟子先行上场，总有个先来后到吧！”地藏神王面色凝重的说道。
<br/>　　“笑话，就算是先来后到，也是我天王宫弟子先行出手！”天尘宇冷笑一声，互不相让。
<br/>　　“别废话了，两人一起上吧！”苏莫面色冷漠的说道，手中剑光一闪，硕大的终极神剑出现在了手中。
<br/>　　这一战，他只为杀人，想对付他的人，就要付出生命的代价。
<br/>　　“终极神剑！”紫霄见到苏莫手中的宝剑，顿时眸中精光一闪，这可是他终极神王殿的至宝。
<br/>　　“狂妄！”
<br/>　　那两名上场的天王宫和地王宫弟子，面现怒色，他们均是万古天骄，但是居然被苏莫如此的藐视。
<br/>　　“死吧！”苏莫根本懒得废话，瞬间就出手了，手中终极神剑一挥，一道巨大无比的剑影爆袭而出。
<br/>　　吞天噬地！
<br/>　　这一剑，苏莫动用了两种万化元神的力量，并且将混沌源力催动到了极致。
<br/>　　这一剑不可谓不强，长达千丈的巨大剑影，无上的锋芒简直无法直视，浩荡的力量，简直毁天灭地。
<br/>　　更加主要的是，这剑影之上，蕴含巨大的吞噬之力，不仅能吸收虚空中的力量，还能束缚对手。
<br/>　　“什么？”那两名天王宫和地王宫的弟子，没想到苏莫会陡然出手，现在一见苏莫出手的威势，顿时大惊失色。
<br/>　　电光火石之间，两人纷纷出手了，天王宫的那名天才使刀，击出一招冲天的金色刀芒，威势颇为强大。
<br/>　　而地王宫的那名天才，则是极力防御，身前由土属性源力凝聚成了一块长达十丈的巨大盾牌。
<br/>　　两人一攻一防，实力都非常强大，俱是万古天骄级别的天才。
<br/>　　然后，这一击的结果，却是让两人绝望。
<br/>　　只见巨大的混沌剑影斩过，无论是刀芒还是盾牌，尽皆烟消云散，浩大的剑影直接将两人湮灭。
<br/>　　“啊~~！”
<br/>　　“不~~~！”
<br/>　　惊恐的呼声，震动虚空，响彻四野，但是却戛然而止。
<br/>　　从苏莫出手，到天王宫和地王宫的两名天才陨落，一切也就发生在一个眨眼的时间之内，快到极致。
<br/>　　随着浩荡的冲击波散开，现场一片死寂，落针可闻。
<br/>　　强大！
<br/>　　这是所有人心中生出的想法，不愧是轮回之身，不愧是能击败祖龙塔第十层祖龙的人，这种战斗力简直无法想象。
<br/>　　混沌神朝、亘古魔国等势力之中，有些天才人物，眸中光彩夺目，心中亦是战意升腾。
<br/>　　这个苏莫，真是厉害，天赋战力超越普通的万古天骄太多了，让他们都有一战之心了。
<br/>　　而此时此刻，天尘宇和地藏神王等人，脸色瞬间铁青，心脏都忍不住抽搐了起来。
<br/>　　两名万古天骄，就这么彻底的陨落了，连一点渣渣都没有剩下。
<br/>　　他们两大王宫，每一个王宫之中，总共也就只有寥寥几位万古天骄，每一位的陨落，都是巨大的损失。
<br/>　　这一刻，无论是天尘宇还是地藏神王，心中擒拿苏莫的冲动，都迅速的冷却了下来。
<br/>　　之前，因为他们看到了擒拿苏莫的机会，心中有些激动，现在，他们彻底的冷静了下来。
<br/>　　苏莫是轮回之身，天赋绝世，更甚帝一魂和拓跋青云，完全不能用常理来看待。
<br/>　　“五品之下，皆可来战，败了就能取走我的命！”苏莫遥望天尘宇等人，声音冷漠如霜。
<br/>　　天尘宇和紫霄等人面面相觑，面色沉重如铁。
<br/>　　“拓跋坤，你去！”地藏神王看了一眼身边的灰袍青年，声音低沉的吩咐道，拓跋坤是地王宫的嫡系子弟，也是地王宫天赋最强的人之一。
<br/>　　可以说，在地王宫之中，除了拓跋青云之外，也就只有闻人天都和战乱两人，能在天赋之上比肩拓跋坤。
<br/>　　最主要的是，拓跋坤的修为有四品高级，刚好不足五品，算是五品之下最顶尖的存在。
<br/>　　“是！”灰袍青年拓跋坤点头。
<br/>　　拓跋坤的修为不高，但是体型极为壮硕，面色黝黑，给人一种霸道之感。
<br/>　　言罢，拓跋坤大步而出，脚踏虚空，向苏莫走去。
<br/>　　天尘宇见此，并未阻拦，因为他带来的天才之中，并没有四品高级的人，要么是修为不足四品，要么是修为达到了五品以上。
<br/>　　可以说，他天王宫并没有合适的人选，能战之人，只有一个涂赢。
<br/>　　涂赢是古神族，战力强大，这在天王宫不是什么秘密，但是，涂赢的修为毕竟低，和苏莫一样是三品高级，胜算不大。
<br/>　　所以，天尘宇这次，没有急着派人上场，打算观望一番，这苏莫底气十足，看来是有绝对的自信。
<br/>　　现场之人，不少人都知晓拓跋坤的不凡，见到拓跋坤上场，均是暗暗猜测了起来，猜测拓跋坤能否战胜苏莫。
<br/>　　拓跋坤算是顶级的万古天骄了，又有四品高级的修为，不是没有胜算。
<br/>　　“苏莫，你是我囊中之物！”
<br/>　　体型壮硕的拓跋坤，来到了苏莫前方万丈之处，手中出现了一把巨大的战斧，面上战意澎湃。
<br/>　　他知道这一战的重要性，只要他能打败苏莫，就能将苏莫从龙族之中带走，到时候拓跋青云融合苏莫之魂，很可能一举突破到高级神王。
<br/>　　那个时候，灭杀帝一魂，一统轮回神朝六大王宫，指日可待了。
<br/>　　“刚才两人，就是你的下场！”苏莫面色不变，手中终极神剑遥指拓跋坤，他能感觉到，拓跋坤的实力，不是之前两人所能比拟的。
<br/>　　“不知量力！”拓跋坤冷笑一声，身上源力涌动，气息不断的升腾而起，不过，他的眼眸之中充满了凝重，丝毫不敢小觑苏莫。
<br/>　　拓跋坤的源力，呈现土黄之色，不是单属性源力，而是土金双属性源力，气息极为强大，不仅充满了土属性的厚重，也充满了金属性的锋锐。
<br/>　　“斩！”拓跋坤暴喝一声，身形冲天而起，率先出手了，他双手持着战斧，已斧重重的劈斩而下。
