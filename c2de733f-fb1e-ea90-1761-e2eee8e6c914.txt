　　望着真圣碑上自己的名字，苏莫心中暗叹，很快就要离开真圣界了，他将面临九死一生的地步。
<br/>　　用巫殇和巫人王威胁大祭司，他是一点把握都没有，但他只能这么做了，因为他没有任何其它办法。
<br/>　　嗖！嗖！嗖！
<br/>　　就在此时，那在真圣碑上留名的四十多名天才，身形齐齐向真圣碑飞了过去。
<br/>　　苏莫见此，便也立刻跟了过去。
<br/>　　而后，他便见到众人纷纷将手掌按在了真圣碑上。
<br/>　　惊人的一幕出现了，只见真圣碑仿佛不是实体一般，那些人的手掌所按之处，石碑渐渐扭曲，而后将一名名天才都吸了进去。
<br/>　　“嗯？”苏莫见此双眉一挑，这真圣碑居然如此的神奇，获得真圣之气还必须要进入其内？
<br/>　　眼见众人纷纷消失，均是进入了真圣碑之内，苏莫立刻抬手，将手掌按在了真圣碑上。
<br/>　　而后，身前的碑面立刻扭曲了起来，一股强大无比的吸力作用在了他的身上，将他也吸入了进去。
<br/>　　真圣碑内，是一处独立的空间，明亮异常。
<br/>　　而最为显眼的是，在这处空间的尽头，是一处虚幻的空间通道，不知通向何方！
<br/>　　“难道从这里可以离开真圣界？”苏莫见此心中一震，如果能从这里离开真圣界，是不是不用在出口处面对大祭司了！
<br/>　　嗖嗖嗖！！
<br/>　　众人全部向空间通道飞了过去，苏莫也同样立刻飞了过去。
<br/>　　很快，众人便进入了空间通道，不断的向前飞行。
<br/>　　没有人说话，所有人都只是闷头飞行，约莫过去了盏茶的功夫，众人终于到达了空间通道的尽头。
<br/>　　“这是……？”苏莫见到眼前的场景，顿时心中一惊，立刻瞪大了眼眸，是一片他从未想到的场景。
<br/>　　只见在空间通道之外，数万里远处，一片絮乱无比之地，无尽灵气浩荡，不断的涌动，形成了巨大的空间乱流。
<br/>　　而在空间乱流之中，正有一根巨大的黑色神柱屹立其中，这黑色神柱贯穿虚空，不知通向何方。
<br/>　　这黑色神柱粗.大无比，通体闪烁着九彩之光，美轮美奂。
<br/>　　此时此刻，无尽的灵气正如同万流归宗一般，向黑色的神柱中汇聚，无休无止。
<br/>　　“这不是那旗幡吗？”苏莫心中震惊，那闪烁着九彩之光的巨大神柱，不就是他来到真圣界之前，在这颗星辰上看到的那巨大旗幡吗？
<br/>　　这绝对不会有错，就是那旗幡，简直一模一样。
<br/>　　虽然在这里只能看到旗杆，而看不到其上的幡，但苏莫能肯定这就是那旗幡的旗杆部分。
<br/>　　而这旗杆好像是在吸纳灵气，而且这吸纳灵气的速度之快，简直比苏莫全力催动吞噬战魂吸纳灵气的速度还要快。
<br/>　　“这旗幡到底是什么东西？”苏莫目光遥望着巨大的旗杆，心中疑惑不已。
<br/>　　不是要获得真圣之气吗？
<br/>　　这真圣之气在哪里？
<br/>　　难道与这旗杆有关系？
<br/>　　苏莫满头的雾水。
<br/>　　转头看向其他人，其他人也是尽皆面露震撼之色，有惊喜之人、有疑惑之人，也有好奇之人。
<br/>　　就在此时，只见那九彩旗杆之上，突然喷涌出一股明黄.色的气流，这股气流冲开了涌动的灵气，向众人的方向飞射而来。
<br/>　　“哈哈！诸位，在下先得头筹了！”
<br/>　　只见姜神夜突然大笑一声，身形一闪而出，直接凌空向那股气流抓去。
<br/>　　唰！
<br/>　　瞬息之间，那股气流便被姜神夜抓在了手中，而后融入了对方的体内。
<br/>　　随即，便见姜神夜立刻飞到了一边，盘膝而坐，静静的修炼了起来。
<br/>　　“这就是真圣之气？”苏莫见此讶然，原来这股明黄.色的气流，便是所谓的真圣之气。
<br/>　　但是，真圣之气为何会从这九彩旗杆内流出，这九彩旗杆到底有什么秘密？
<br/>　　众人见姜神夜得到了真圣之气，并没有与其抢夺，均是在静静的等待，等待再有真圣之气流出。
<br/>　　没有人敢去靠近旗杆，因为即便相隔数万里，众人也能感觉到恐怖的气息，这股气息让人心悸，任何人都不敢靠近。
<br/>　　唰！
<br/>　　过了约莫小半个时辰之后，九彩旗杆上再次有真圣之气流出，这一次飞长云一马当先，抓取了真圣之气。
<br/>　　随后，飞长云同样走到一边，盘膝静坐了下来。
<br/>　　接下来的时间中，大概每隔小半个时辰，便会有一股真圣之气流出，纷纷被一名名天才抓取。
<br/>　　苏莫也没有等到最后，在出现第十股真圣之气的时候，他一把就凌空抓了过来。
<br/>　　看着悬浮于掌中的真圣之气，苏莫眸中闪亮，他在其中感受到了一股沧桑、亘古、磅礴的气息。
<br/>　　随即，苏莫也跟别人学习，走到一边盘膝坐了下来，随即他意念一动，便将真圣之气吸入了体内。
<br/>　　真圣之气入体，立刻就四散而开，逸散在了他全身的血肉、骨骼、以及玄力之中。
<br/>　　苏莫立刻心中一震，他清晰的感觉到，这真圣之气在融于他全身之后，在缓缓的淬炼他的血肉和骨骼，甚至在淬炼他的玄力和神丹。
<br/>　　也不能说是淬炼，应该说是真圣之气融入全身之后，全身都在缓慢的改变。
<br/>　　“居然如此的玄妙！”苏莫心中惊讶，这真圣之气不能增加多少实力，却是能增强一些天赋，或者说打牢武者的根基。
<br/>　　立刻抛却心中杂念，苏莫静心的感受着融于体内的真圣之气，改变他的身体与玄力。
<br/>　　这个时间很缓慢，苏莫也不急只是静心而坐，仔细的感受。
<br/>　　在苏莫静坐之时，其他四十七名天才，也纷纷抓取了真圣之气。
<br/>　　一众天才，安静无声，体内均是融入了真圣之气。
<br/>　　约莫过去了十几个时辰之后，苏莫感觉体内异状消失，那股真圣之气彻底和他的身体融合在了一起。
<br/>　　这个时候，他的身体变得更为纯净通透，神丹也变的更为稳固，玄力都变得凝练了一分。
<br/>　　睁开眼眸，苏莫发现自己全身皮肤之上，有少量的黑色油污，这是从他体内排出来的。
<br/>　　苏莫暗暗惊讶，他到了如今的修为，本以为身体已经足够纯净，没有丝毫的杂质，没想到还有这么多！
