　　第624章
<br/>　　被翻红浪，枕上联双，等到后半夜，苏樱将程大雷赶了起来，让他去樊梨花房中。
<br/>　　程大雷披了一件袍子起身，走出屋子的时候被凉风一吹，隐隐约约觉得自己疏忽了某件事。他敲了敲脑袋，一时半刻也想不起来。此时已到樊梨花门前，敲了敲门，身子挤了进去。
<br/>　　到次日清晨，程大雷起床洗漱，忽然一激灵，口中下意识道出三个字：“徐神机！”
<br/>　　程大雷一去数月，日子自然称不上有多舒适。而徐神机则比他更惨，他被困在山寨的地牢中，整个人完全是在坐牢。每天除了黑衣老尼，一个人也见不到。程大雷临走前说徐军师在闭关悟道，这可是一件大事，山寨谁也不敢打扰他。
<br/>　　过去大半年时间，整个人倒是胖了一圈，可脸上一点精神都没有，一副生无可恋的表情。
<br/>　　“徐军师在哪里，徐军师在哪里……”
<br/>　　程大雷大步闯进地牢，一见程大雷，徐神机恨得牙都要咬碎了。
<br/>　　“程大雷！”
<br/>　　“军师息怒，军师息怒。”程大雷哈哈笑着：“我这不刚回来，就立刻来见军师，军师，你可是让我好想。来人，快快放军师出来。”
<br/>　　徐神机被关这些日子，确实也没受什么委屈，不过一个人也见不到，总是有些闷得慌。在经过最初的愤怒之后，他所有情绪转变成委屈，拉住程大雷大倒苦水。
<br/>　　程大雷无可奈何的忍受，毕竟徐神机是他关起来的，他这也算是咎由自取。
<br/>　　徐神机唠叨半晌，方才询问程大雷的经历。程大雷述说连日来的经历，徐神机听着也是唏嘘不已，相比自己还是程大雷受得磨难更多。
<br/>　　“大当家，你该让我随你一起去的，有我出谋划策，你也能少受点苦。”
<br/>　　“谁说不是呢，刚走我就后悔了，出门在外，日思夜想的可不就是徐军师么。”程大雷心道：若你跟在身边，说不得我就不能活着回来了。
<br/>　　听罢程大雷的经历，徐神机心中也平衡许多，一腔委屈便渐渐淡了。
<br/>　　程大雷忽然想起一件事，差人将刘发财唤了过来。
<br/>　　“刘发财，你可知道这位是谁？”程大雷指着徐神机。
<br/>　　刘发财眼见徐神机留着三撇胡须，身材猥琐，立刻道：“那能不知道，这就是神机妙算的徐军师么，江湖上鼎鼎大名，在山寨我耳边更是要磨出茧子。只不过听说徐军师在闭关悟道，一直没有机会见到。”
<br/>　　徐神机托着下巴，脸上浮现得意洋洋的笑容。
<br/>　　“这你不就见到了。”程大雷道：“以后还有机会经常见到，你以后就留在徐军师身边做事，徐军师可是山寨的智囊，多随他学学，你造化不小呐。”
<br/>　　刘发财一下睁大眼睛，有些不太敢相信耳边听到的话。他刚才的话也未必是完全奉承徐神机。神机妙算徐神机，乃是蛤蟆城第一军师，程大雷左膀右臂般的人物，江湖上谁人不知谁人不晓。自己刚到蛤蟆城，还没有出过什么力，也没有露过什么本事，程大雷便将自己安排在他身边做事，这是极其的看得起自己呐。
<br/>　　他倒是不知道程大雷如何想法。程大雷觉得这刘发财似乎没什么本事，做武将他的武艺明显不够格，做谋士他也未必有那脑子。干脆将他放在徐神机身边，两个人互相祸害去吧。
<br/>　　“大当家放心，我一定尽心辅佐徐军师，不辜负大当家的信任。”士为知己者死，现在刘发财抱的就是这心思。
<br/>　　“嗯呐，嗯呐。”程大雷随意应承一番，算是揭过了这件事。
<br/>　　相比刘发财，程大雷更加看重华正则，只不过华正则没有做山贼的心思，如何留下他，程大雷却还需想个办法。
<br/>　　程大雷在外奔波一番，过程中遭遇了很多事，整个人也是身心俱疲。所以他回来后并没有着急做事，先歇了几日，等养足精神后，才着手处理山寨的各种事物。
<br/>　　程大雷先从和珅与刘悲处了解山寨目前的状况。
<br/>　　幸运的是，程大雷一开始已经建立好良好的组织架构，刘悲负责军师，和珅处理内政，所以他不在的这些日子，山寨平稳运转，一切如常。
<br/>　　但这也表示，山寨并没有什么发展，只不过是保持现状而已。
<br/>　　目前的山寨是初级城池，拥有技能：诸神的祝福、江湖传言、建筑精通。
<br/>　　士兵三万有余，平民十万余。另外还有木匠、铁匠、裁缝、医者等生活职业若干。
<br/>　　武将有关鱼、张肥、赵子龙、秦蛮、史文恭、单雄信等人。
<br/>　　文有刘悲、和珅、吴用……似乎还有个徐神机。
<br/>　　想到这里，程大雷不是很有自信，徐神机勉强能算个谋士么，或许，说他是个吉祥物更合适些。
<br/>　　另外还有一批神职。
<br/>　　无论如何，蛤蟆城眼下已经是凉州一地不可忽视的一股势力。虽然兵力并不十分多，但这三万人放在帝国战场上，绝对是恐怖的存在。如果不是因为这个原因，宋伯康也不会忍受蛤蟆城存在到现在，早就派兵打过来了。
<br/>　　蛤蟆城没什么新鲜事发生，程大雷只要稍微了解一下，就可以接手继续管理。可如今帝国境内，闹出的新闻却是层出不穷。
<br/>　　阿喜知道程大雷回来后，向程大雷汇报过最近得到的情报。
<br/>　　宋伯康磨刀霍霍，准备向镇守边关的百里胜动手，双方之间的战斗，已经是箭在弦上，不得不发。
<br/>　　而在整个帝国，如这样的事还有很多。有的是已经发生了，有的是将要发生。在经历过最初的招兵买马后，已经有人耐不住性子，想着要扩充地盘了。
<br/>　　帝国就这么大，地盘被分成二十块，你想扩充地盘就总要有人让出来，那就只能是双方互相商量了。而他们商量的方式，往往是战斗。
<br/>　　而如莫将难那样揭竿而起的事情也有许多，大部分都闹不出太大动静，不出十天半个月就被剿灭了。
<br/>　　这中间还有一件有趣的事情，在揭竿而起的这波人中，有七八股都是打着正义教的旗号。
<br/>(本章完)
