　　宏青璇的记忆被抹除，这让苏莫几乎不能接受，因为记忆一旦被抹除，几乎没有恢复的可能了。
<br/>　　少倾，苏莫压下心中的怒气，瞥了眼如同死狗一般的巫天御，便立刻收回了梦魔战魂，撤掉了梦魔空间。
<br/>　　客厅中，收了苏莫搜魂术的巫天御，整个人双眼翻白，面现痴呆之色。
<br/>　　苏莫不敢浪费时间，立刻催动了顿时战魂，将巫天御的战魂吞噬了。
<br/>　　巫天御的战魂，正是当初的飓风武魂进化而来，算是非常强大的战魂。
<br/>　　吞噬了飓风战魂之后，苏莫吞噬战魂魂立刻离开了分身，瞬间进入了巫天御的体内，占据了对方的肉身。
<br/>　　而后，他稍稍整理了一番，将自己的本体和一些物品全部转移到了巫天御的身上。
<br/>　　他身上的储物戒众多，少司令牌所在的储物戒，只是其中之一而已。
<br/>　　少倾之后，一切回归了平静，客厅中的魔气全部消散，仿佛任何事情都没有发生一般。
<br/>　　巫尘依旧坐在主座之上，而‘巫天御’也依旧坐在客座之上，没有丝毫的变化。
<br/>　　“巫天御，想要我原谅你，那就拿出五千下品灵晶吧！”巫尘淡淡的说道。
<br/>　　“五千下品灵晶，巫尘少司，你未免欺人太甚了吧？”‘巫天御’脸色难看的说道。
<br/>　　“不拿也行，以我现在的地位，杀了你也没人敢说什么！”巫尘的脸上露出冷笑之色。
<br/>　　“你……！”
<br/>　　‘巫天御’脸色阴沉如水，沉默了少倾之后，咬了咬牙道：“巫尘少司，我给你五千下品灵晶，以后我们之间再无仇怨！”
<br/>　　言罢，‘巫天御’从储物戒中取出了五千下品灵晶，递给了巫尘。
<br/>　　“很好！”巫尘微微一笑，点头道：“既然你如此的识趣，那我也不为难你了！”
<br/>　　“告辞了！”‘巫天御’长身而起，向巫尘抱了抱拳，随即长身而起，大步离去。
<br/>　　很快，巫天御便离开了巫尘的府苑。
<br/>　　府苑门前，中年下人看着巫天御离去的背影，眸中露出疑惑之色。
<br/>　　之前客厅中巫尘突然发怒，全身魔气汹涌遮盖了大厅两个呼吸的时间，他也不知道发生了什么！
<br/>　　……
<br/>　　苏莫夺舍了巫天御的肉身，很快便走出了宗府，离开了天冥圣城。
<br/>　　离开天冥圣城之后，他故意路经了黑耀城，随后没有停留，又离开了黑耀城，飞天而去。
<br/>　　苏莫一路疾行，爆发出了最快的速度，整整飞了一天一夜，远离了天冥圣城不知道多少万里之后，才停了下来。
<br/>　　随后他立刻找了个隐蔽之地，放出了本体，战魂重新回归了本体。
<br/>　　回归本体之后，苏莫才真正有了一种归属感，夺舍别人的肉身，多多少少有些不自在。
<br/>　　收起巫天御的肉身之后，苏莫稍稍变换容貌，就地隐藏了起来，待得两日之后才离开，前往聂乾和韩盈隐藏之地。
<br/>　　至于他的分身，依旧在巫族之内，继续做少司。
<br/>　　只要他没有彻底暴露，分身的安全就无碍。
<br/>　　再说了，就算分身被杀了，也丝毫无妨，这种分身对他来说不重要。
<br/>　　通过搜魂巫天御，苏莫已经知道了青璇的情况，可以说是误打误撞完成了目的。
<br/>　　所以，接下来他就要大干一场了。
<br/>　　两日之后，苏莫来到了聂乾两人的隐藏之地，大地之下万里，两人依旧没有离开，一直在等他。
<br/>　　“两位，让你们久等了！”见面之后，苏莫略带歉意的说道，让两人干等了近一年，实在了不好意思。
<br/>　　“苏兄，我们也没有傻等，轮流出去历练了一段时间！”聂乾笑着说道。
<br/>　　“是啊！苏兄，难道你没发现我有什么不同了吗？”韩盈浅笑嫣然，娇嫩的脸庞上光彩照人。
<br/>　　“你突破了？”苏莫打量了对方一番，顿时眼前一亮，此刻的韩盈分明达到了武帝境二重境界。
<br/>　　“嗯！”韩盈笑着点了点头，不过很快便俏脸一僵，布满了无奈之色。
<br/>　　因为苏莫同样突破了，离开之时还是武帝境二重境界，现在已经变成了武帝境三重的境界。
<br/>　　“苏兄，和你比，我的自信都快烟消云散了！”韩盈叹了口气说道。
<br/>　　“呵呵！”
<br/>　　苏莫无语的笑了笑，随即略一沉吟，道：“两位，现在我的目的已经完成了，接下来我们洗劫一些资源，就离开天冥星，返回神武学府。”
<br/>　　“如何洗劫？”韩盈问道。
<br/>　　“苏兄，这里可是巫族的地盘，洗劫的话千万不能大意！我们很可能遭到巫族的围剿！”聂乾面色凝重的提醒道。
<br/>　　“放心吧！我自有打算，保证让你们赚个盆体满钵！”苏莫郑重的说道。
<br/>　　随后，苏莫和两人仔细的商议了一番，便离开了藏身之地，往天冥圣城相反的方向而去。
<br/>　　既然要大干一场，自然不能离天冥圣城太近，不然就是找死了。
<br/>　　苏莫三人飞行了整整五天，远离了天冥圣城无数万里之后，便开始了行动。
<br/>　　“韩姑娘，给我查看一番，附近有没有巫族小镇！要那种没有武帝境五重以上强者的小镇！”苏莫向韩盈说道。
<br/>　　“好！”韩盈点了点头，随即立刻催动了她的洞听耳战魂，聆听八方。
<br/>　　很快，韩盈便找到了目标，对苏莫说道：“西北方向，十八万里，有一座小镇，有七、八万巫族之人，修为最高者武帝境三重境界！”
<br/>　　“好，我们走，屠戮了这个小镇！洗劫一空！”苏莫轻笑一声，便立刻身形一动，向西北方向飞去。
<br/>　　大肆屠戮，这在以前苏莫绝对不会做，觉得这种事情太过绝情狠辣、太过残忍血腥、太过没有人性。
<br/>　　但现在，他心中有的只是杀意，无尽的杀意。
<br/>　　就算将整个巫族屠杀殆尽，他也不会有丝毫的怜悯之心，这无关于人性，而是刻骨的仇恨。
<br/>　　虽然普通的巫族之人没有错，但苍穹世界亿万生灵，千亿武者都有错吗？
<br/>　　难道苍穹世界的人都该死？
<br/>　　就因为出生的位面小世界，就因为实力低微，就活该被杀吗？
<br/>　　巫族狠辣残暴，将苍穹世界的人当成猪狗，肆意屠戮，所以苏莫对于巫族之人已经没有了任何怜悯之心。
