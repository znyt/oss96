　　第382章
<br/>　　刘悲是个很好说话的人，这一点在山寨有口皆碑。没有架子，待人随和，只要你愿意和他交流，他总能和你有得聊。
<br/>　　前提是你愿意和他聊。
<br/>　　在茫茫人海中，奥黛琳娜发掘了刘悲。性格随和，文质彬彬，又多愁善感，这简直是天生的信徒呐。在苦海中挣扎奔波，就等着诸神拯救，只要稍微向他照耀些神辉，他就撒丫子扑进诸神的怀抱中。
<br/>　　这一日奥黛琳娜来拜访刘悲，特意收拾了，换过一套新衣服。本就生得漂亮，如此一打扮，更是比花花解语，比玉玉生香，小脸粉白粉白的，如清晨第一颗露珠。
<br/>　　见到漂亮姑娘，谁都愿意多聊几句，就连刘悲也不例外。对于奥黛琳娜的拜访，他感到有些奇怪，还是请到了茶厅，亲自沏了从南方来的新茶。
<br/>　　茶厅单面开窗，窗台摆着几盆兰花，微风轻拂，分外雅致。
<br/>　　“圣女阁下来见我，是有什么事么？”刘悲问。
<br/>　　“我是慕名而来，专程拜访刘悲先生，刘悲先生平时忙些什么啊。”
<br/>　　“我有什么好慕名的，每天无非给人看看病，有事管管事，闲下来就是看看书，散散步，一个人待会儿。”
<br/>　　奥黛琳娜眼前一亮，这简直就是迷途羔羊呐。她道：“这样会很无聊吧，先生怎么不找人说说话？”
<br/>　　“嗳，都没人愿意和我聊，也没什么好聊的。”
<br/>　　“先生可以找我聊啊，诸神……”
<br/>　　“你愿意和我聊？”刘悲眼睛一亮，打断了奥黛琳娜的话。
<br/>　　奥黛琳娜倒是吓了一跳，下意识点点头。
<br/>　　然后，刘悲打开了话匣子，就像大河开闸，滔滔洪水不绝。刘悲从三岁开始说起，痛陈自己的辛酸史。当然，刘悲不至于像祥林嫂一般，鼻涕一把眼泪一把，他依旧苦着一张脸，可每句话都能给人心灵造成暴击。
<br/>　　奥黛琳娜有些撑不住了，当然她也常开导信徒，告诉他们要坚强，要加油……本身奥黛琳娜也是个很强大的人。心灵强大，信仰坚定，可面对刘悲，她还是遭遇了前所未有的挑战。
<br/>　　她打断了刘悲的话，因为再说下去，奥黛琳娜也要怀疑生存的意义了。
<br/>　　“那么刘悲先生，要不要考虑有一个信仰，诸神可以保佑你。”
<br/>　　刘悲眨了眨眼睛，今天还是很少聊得这么痛快，关键是平日也没人愿意和他聊。
<br/>　　“你们的教义我也研究过……”
<br/>　　奥黛琳娜松口气，接下来颇费口舌的向刘悲解释了新教的教义。
<br/>　　总的说来，奥黛琳娜的教派还是蛮自由的，也没有什么苛刻的教规。关键是诸神的数量实在太多了，天空之神、大地之神、火焰之神、死亡之神……，这波人也是被教廷驱逐的，他们也就不好再自相残杀，抨击你信的神不对。简单来说，你就算随便从茅坑里找快石头，说自己信仰茅厕石之神，也是完全没有问题的。
<br/>　　刘悲听得很认真，他是真正对这些事情感兴趣。一见刘悲如此，奥黛琳娜立刻滔滔不绝起来，从世界起源，讲到诸神降世，最后说得口干舌燥，见刘悲依旧专注倾听着。
<br/>　　“刘悲先生，你觉得怎样？”
<br/>　　“这个……有一个问题，我不知当讲不当讲？”刘悲迟疑了一下。
<br/>　　“神可以解答你的任何疑惑。”奥黛琳娜心中欣喜，这完全就是要投入诸神怀抱的意思呐。
<br/>　　“哈，其实大当家说了，山寨里信仰自由，不可随便攻击你们的信仰，但今天你竟然这么说了，也就别怪我不客气了。”刘悲道：“你们这个信仰有问题呐？”
<br/>　　“呃……有什么问题？”奥黛琳娜被兴奋的刘悲吓住。
<br/>　　然后刘悲好好给奥黛琳娜上了一课，攻击其教义中的纰漏处，问题是，你也不能说他说得完全没有道理。
<br/>　　新教和旧教的最大区别是，旧教认为世间只有一个神，就是造物神。新教认为世间有无数神，诸神一起管理世界。
<br/>　　本来这伙人的组成就有问题，无非被旧教不容的异教徒被迫团结在一起。你信仰这个，我信仰那个，最后凑成一个草台班子。
<br/>　　很多教义，他们还没理顺呢，也不怪刘悲一说就能指出其中破绽。
<br/>　　奥黛琳娜是个坚定的信仰者，因为太过坚定，于是就有些迷，很多事情，其实她也说不出来所以然。今日被刘悲一问，顿时头昏脑乱，过往的镇定也失去了。
<br/>　　“不，你说得不对！”
<br/>　　“怎么不对了？”刘悲一拱手道：“还请赐教。”
<br/>　　刘悲脸上认真的表情，告诉奥黛琳娜刘悲没有和他瞎扯，因为真的想不通，所以才会请教，倒不是故意让奥黛琳娜出丑。
<br/>　　可刘悲提出的问题，奥黛琳娜真的回答不上来。
<br/>　　奥黛琳娜当真不了解刘悲，刘悲是个很悲观，很无聊的人。一天有大部分时间都是一个人独处，他又没什么业余爱好，一个人待着就想些：人从哪里来，要到何处去，人活着为了什么等等之类的终极问题。
<br/>　　想来想去，就想出来一件事：万物皆空，没多大意思。
<br/>　　于是越这样想越悲观，越悲观越觉得没多大意思。所以如果真有什么虚空之神的话，刘悲倒也不介意信一信。
<br/>　　时间过得越久，他的理论越扎实，奥黛琳娜想要感化刘悲，却差点被刘悲影响。
<br/>　　悲观是件很痛苦的事情，刘悲也希望奥黛琳娜给他解惑，可奥黛琳娜的确无法回答。
<br/>　　“这个……我回去后好好想想，等我有了答案再来向刘悲先生请教。”
<br/>　　“一定，一定，如果有了答案一定要尽快告诉我。”
<br/>　　奥黛琳娜几乎可以说是落荒而逃，如果她真的有野心，想要通过宗教获得什么，刘悲倒是一个不错的查漏补缺的工具，正好趁这个机会，从上到下把教义理一遍。
<br/>　　可奥黛琳娜的确是一个坚定的信仰者，相信诸神可以给世间带来光明。但因为刘悲的事情，她的信仰遭到很大考验。
<br/>　　而刘悲此刻翘首以盼，还正等着奥黛琳娜拯救自己呢。
<br/>(本章完)
