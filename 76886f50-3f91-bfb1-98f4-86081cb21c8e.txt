        “冒险者集团的八爷、三爷、五爷和六爷，被人抓了送到我们安防局里来了。”夏阳枭言归正传。
<br/>
<br/>        连华生先是一怔，故作淡定地咧嘴一笑：“这不是好事吗？恭喜啊！夏局，您不劳而获。”
<br/>
<br/>        “你觉得，谁有这种本事，给对方下套再安个莫须有的罪名，抓了他们送到我这儿来？”夏阳枭反问。
<br/>
<br/>        安防局抓人也是讲究实实在在的人证物证，但像这样给歹徒下套伪造人证物证抓人的，可不是他们安防局这样的正派所为。
<br/>
<br/>        连华生双手一摊：“那个冒险者集团，曾制造定时炸弹危害公共安全，又制造生化病毒趁机贩卖疫苗来赚黑心钱，现在他们被抓了，不是更好吗？您索性转交给公安，再移交给法院，让法院去判，多省事。再说了，您还可以对外宣传是安防局抓到的人，您可以向您的顶级上司邀功请赏。”
<br/>
<br/>        “我问的是，你觉得这件事情，幕后是谁做的？”夏阳枭一脸不乐意。
<br/>
<br/>        连华生抿了抿唇，打了个响指：“会不会是那个背叛了冒险者集团的十爷？那个十爷曾是冒险者集团的高管。当了叛徒，冒险者集团肯定会对他下追杀令。与其亡命天涯，不如出击反击，来个黑吃黑，黑控黑。您说对不对？”
<br/>
<br/>        “黑吃黑和黑控黑，是需要巨额的资金支撑的。你觉得那个十爷有那么多钱吗？”夏阳枭拧起了眉头。
<br/>
<br/>        连华生汗颜：“你不会是怀疑阿沉吧！”
<br/>
<br/>        “除了他，还有谁能做到！”夏阳枭恼火道。
<br/>
<br/>        傅氏集团没有任何违法违纪行为，所以，他们压根就没法冻结傅氏集团的资金。
<br/>
<br/>        也就是所，傅氏集团一直在正常运作，换句话说，就是一直在替傅夜沉赚钱。
<br/>
<br/>        连华生耸了耸肩：“您对他下的通缉令还未取消，他哪有这空闲去做这种事情。”
<br/>
<br/>        “如果，他主动联系你了。我希望，你能劝他收手。”这才是夏阳枭的重点所在。
<br/>
<br/>        连华生抬起手来捏了捏自己额前的发梢：“那等他联系上我再说吧！”
<br/>
<br/>        顿了顿后，连华生又接着说道：“对了！你把童筱颖埋哪儿呢？作为她的朋友，我觉得我该去吊念一下她，给她送送花什么的。”
<br/>
<br/>        半年前，童筱颖坠崖后，连华生带人寻找童筱颖的同时，夏阳枭的人也来了。
<br/>
<br/>        当他们找到童筱颖的时候，童筱颖被海水冲到了暗礁上，脸色惨白刚断气。
<br/>
<br/>        连华生替童筱颖把过脉，也抚过鼻息，断定她是因为后脑勺撞到了礁石上，造成脑震荡、失血过多而亡。
<br/>
<br/>        此外，因为涯壁过高，在海水与空气落差的冲击力下，震断了童筱颖脊骨。
<br/>
<br/>        这种情况下，就算童筱颖不死，救上岸后十有八九会是植物人。
<br/>
<br/>        夏阳枭下令把童筱颖的遗体带走了，然后就没下文了。
<br/>
<br/>        连华生很是纳闷这件事情。
<br/>
<br/>        夏阳枭说傅夜沉是他干儿子，那么童筱颖就是他的干儿媳妇，他来处理童筱颖的后事是合情合理。
<br/>
<br/>        连华生当时也这么认为，便将童筱颖交给了夏阳枭。
<br/>
<br/>        但是他又不得不对傅夜沉撒谎，说是自己处理的童筱颖的后事。
<br/>
<br/>        夏阳枭沉默了一会儿，像是在寻思着什么，才回答道：“在你们滨江城公墓园第18030号。”
<br/>
<br/>        连华生微微欠身：“多谢告知，我改天去看看她。”
<br/>
<br/>        别说是改天，连华生离开安防局后，当天就去花店买了白色雏菊，去了夏阳枭所说的公墓园。
<br/>
<br/>        夏阳枭没有说谎，第18030号确实是童筱颖的墓。
<br/>
<br/>        只不过，这墓碑不同于其他墓，一没有照片，二没有立碑人，显得也太潦草了点。
<br/>
<br/>        连华生暗自叹了口气，弯腰将手中的白色雏菊放在了墓碑前。
<br/>
<br/>        “童筱颖，不好意思啊！过了这么久才来看你，总得来说，我还是要谢谢你，谢谢你还记得我跟你说过的话。那天，你若是不死，阿沉真的会冒死来救你。你的死虽然保住了阿沉的命，但……唉——算了，不跟你说阿沉的事情了，其实你就不该遇上阿沉。你和傅昼景在澳洲待得好好的，回来做什么？不回来，你现在一定过得很幸福吧！就算不幸福，那也总比丢了命要好。”
<br/>
<br/>        连华生絮絮叨叨地说了许多，更多的是感慨。
<br/>
<br/>        他没待多久，就离开了。
<br/>
<br/>        连华生回到自己的车里后，看着后视镜里车窗外一闪而过的黑衣人，不禁咧嘴一笑。
<br/>
<br/>        这暗中监视他的人，可不止安防局的人，还有傅夜沉的人。
<br/>
<br/>        他把童筱颖墓园的地址，间接地告诉了傅夜沉。
<br/>
<br/>        想必，今晚这墓园是不会太平了。
