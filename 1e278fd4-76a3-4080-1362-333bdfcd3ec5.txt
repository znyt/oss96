“小小姐，真的是你么？咳咳咳咳…”那端的林婶开始猛烈的咳嗽。
<br/>
<br/>        夏夕绾一听就知道林婶这几年的身体亏空的厉害，像是快不行了。
<br/>
<br/>        她拽着手机紧张道，“林婶，林…”
<br/>
<br/>        苏希直接将手机抢了过去，挂断了电话，“确认了么，我没有骗你。”
<br/>
<br/>        说着苏希拔腿，向夏夕绾逼近。
<br/>
<br/>        夏夕绾往后退了一步，“别过来，我们的交易是，你将林婶交给我，我才陪你睡。”
<br/>
<br/>        苏希蹙眉，脸色阴鹜，“夏夕绾，你是不是没有搞清楚状况，现在是你在求我将林婶交给你！”
<br/>
<br/>        “苏希，没有搞清楚状况的人是你，现在是你求我陪你睡！”
<br/>
<br/>        “夏夕绾，你的自信是哪里来的？”
<br/>
<br/>        夏夕绾冷笑，“昨晚你睡了夏小蝶吧，但是你深夜里就给我打电话，让我今晚来陪你，你这张欲求不满的脸就是我最大的自信。”
<br/>
<br/>        苏希迅速将薄唇抿成了森冷的弧线。
<br/>
<br/>        夏夕绾拿着包就走，“今天就这样吧，等你把林婶接过来，我们再交易。”
<br/>
<br/>        “夏夕绾，你不要逼我！”苏希去拽她。
<br/>
<br/>        但是夏夕绾早有防备，她将手里的包包用力的砸到了苏希的俊脸上，“苏希，究竟是谁在逼谁，十年前你指证我，让我被千夫所指，十年后你又用林婶来逼我陪睡，你这样做对得起我妈咪么，你对得起她对你的喜爱，对得起我叫你的一声苏希哥哥么？”
<br/>
<br/>        苏希一下子僵在了原地。
<br/>
<br/>        夏夕绾眼眶通红的盯着他看，像愤怒的小兽。
<br/>
<br/>        许久后，苏希上前，伸手按住了夏夕绾的肩头，“说到底，你还是不愿意陪我睡，我不明白为什么，为什么别的男人可以，就我不可以？”
<br/>
<br/>        看着他偏执的神色，夏夕绾并没有反驳，她只是不动声色的问道，“你是什么时候知道我跟别的男人在一起的？”
<br/>
<br/>        苏希不愿意提这个话题，他用力的捏着夏夕绾的肩，手面青筋暴跳，“呵，你以为自己做的下贱事情没人知道么，那一年冬天我去乡下找你，你一夜未归，我找了整整一夜，都快找发疯了，最后夏小蝶告诉我，你跟一个野男人在山洞里缠绵了一夜，我赶过去的时候还看见你们睡在一起！”
<br/>
<br/>        夏夕绾终于知道原因了，为什么他一口咬定自己水性杨花，有一年冬天她曾经在冰天雪地里救过一个男人。
<br/>
<br/>        那个男人昏迷了，大雪封了路，根本回不了家，她就将那个昏迷的男人带到山洞里过了一夜。
<br/>
<br/>        原来是这样，夏小蝶说她跟那个男人缠绵了一夜，他也就信了。
<br/>
<br/>        “哈，哈哈哈…”夏夕绾突然笑了，她都要将自己笑出眼泪了。
<br/>
<br/>        “你笑什么？”苏希问。
<br/>
<br/>        夏夕绾用力的将他给推开，然后捡起了地毯上的包包，“没有什么，我还是那句话，把林婶给我，我才会跟你交易，你考虑一下。”
<br/>
<br/>        夏夕绾离开了。
