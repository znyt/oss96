　　　　白忧淡淡开口：“我找的谁出手你也不用知道，更别想着去打听，这世上除了几大神秘家族，还有不为人知的厉害存在。”
<br/>　　被如此轻视，罗杰面色瞬变，眼神也变得犀利起来。
<br/>　　白忧丝毫没有注意罗杰的神色变化，哪怕注意到了也不会放在心上，毕竟在他看来的罗家根本不成气候，尤其是不久前与高家爆发的那一战，更加暴露出罗家在各方面都属于神秘家族中的末端。
<br/>　　“不说也罢，总之楚涵那里是不需要再加派人手去关注了吧？”罗杰试探道：“既然你已出手，那他应该这辈子都醒不过来了。”
<br/>　　白忧没有吱声，沉寂的目光中暗藏了一些情绪，让人难以看懂。
<br/>　　看到白忧的沉默，罗杰似乎找回场子了般一笑：“另外令妹白允儿，她既然已是白家嫡系大小姐，做事情能不能不要如此冲动和任性？她这么往狼牙一跑，恐怕四大家族都知道消息了，你我二人先不说，高家那里可不会就这么容易糊弄过去。”
<br/>　　“四大家族？”白忧忽然抓·住了这一关键词，有些嘲讽的说道：“肖家还算什么家族？”
<br/>　　“肖七不死，肖家还是留了后。”罗杰平淡道：“这个女人最近消失的无影无踪，不知道在哪暗中潜伏着呢！她可不是善茬，与以往的那些冲动行·事的肖家人，完全不同。”
<br/>　　白忧没有就着这个问题探讨，而是忽然一笑，看向罗杰：“你们为何会跟高家打起来？”
<br/>　　“是他们挑事在先，原因我哪知道。”罗杰恶狠狠的说道：“而且这帮人竟然找了普通人来支援，实在超乎我的想象，你知道吗，高家甚至给了那些低等的人类千月石武器！”
<br/>　　白忧眉宇轻皱，忽然感觉到了一丝不对劲：“他们哪来的千月石？”
<br/>　　“我还奇怪呢！当年那些千月石不都被我们的祖先抛入深海了吗？”罗杰面上带着煞气下意识回答。
<br/>　　只是忽的他神色一变，似乎想到了什么，而同一时间的白忧，也周身浮现出了淡淡的杀气。
<br/>　　“高家当年……不会私藏了吧？”白忧的声音冷如冰窖。
<br/>　　罗杰的思维迅速运转起来，更是正襟危坐：“如果他们真有私藏，那么当年的事我们恐怕还需要重新审核一下。”
<br/>　　“当年的事情你我两大家族已经有过对证，肖家和高家的内部宗卷怕是根本拿不到，更别说肖七这个人在哪我们都不清楚。”白忧分析着，手指轻敲桌面：“想要知道四大门族里是否有人在当年产生了异心，怕是还需要去一趟阴阳谷。”
<br/>　　罗杰心头一跳，想到了什么问道：“说起阴阳谷，那地方除了白允儿，你们白家可还有人去过？”
<br/>　　白忧摇头：“不曾，怎么了？”
<br/>　　罗杰皱起眉：“罗生怕是死在了那里。”
<br/>　　白忧瞳孔猛然一缩：“他不会起了贪念，去了墓地吧？”
<br/>　　“不可能。”罗杰万分肯定的摇头：“罗生行动之前我交代的很清楚，而且他本身也不是一个冲动行·事的人，够谨慎我才放心让他去，我怀疑是被人杀掉的。”
<br/>　　听到这种猜测白忧笑出声：“阴阳谷内有什么能杀死一个罗家的正统血脉者？你在跟我开玩笑？而且上一代制裁者的身份你我都知道，现在他也死了，死之前也压根不在阴阳谷附近，所以也不会是制裁一门动的手，这么看来，罗生怎么会死？”
<br/>　　罗杰忽然异常认真的抬起头，与白忧直视道：“难道你真的不怀疑，水火两族还尚存于世？”
<br/>　　白忧眉头刹那间紧皱，一股阴沉的气息渐渐涌·出体外，显然罗杰的这个问题，让他非常忌惮也问到了点子上，这才是白忧至始至终最担忧的问题，而就是因为太担忧，才下意识的回避以及用绝对的言辞告诉所有人，水火两族早就灭绝了。
<br/>　　可事实情况却是，白忧不能接受这两族中还有人存活，如果水火尚存，那他们四偏门当年的那件事，岂不是白做了？
<br/>　　“罗梵当时的情报……”罗杰忽然想到了死去的梵，但很快又叹气道：“死无对证。”
<br/>　　“去一趟看看。”白忧忽然下定决心，他的双拳紧握，以掩饰自己身躯的轻微颤抖。
<br/>　　阴阳谷所有神秘家族成员都想要踏足，却又不敢轻易踏足的地方，得知很多信息的白忧更加如此，知道的辛秘越多，越对那个阴阳谷忌惮异常。
<br/>　　所以这些年白家才会致力于寻找叶家后代，白忧甚至一直纵容着叶子博，也就是为了通过他渗透到叶家内部。
<br/>　　而当下叶子博已死，张玄也送了命，好不容易找到的叶永阳又一次与白忧错过，这让准备了多年就为了确保一定安全性的白忧非常恼火，没有叶家人带路，他并不敢轻易进入阴阳谷。
<br/>　　但是此时此刻，太多的线索聚集在一起，问题暴露的太多，已经没有时间再让白忧耽搁下去。
<br/>　　听到白忧竟然说出要去阴阳谷的话，罗杰大为吃惊中不禁兴奋起来，道：“能与你们白家人一同进去，我也放心。”
<br/>　　“那就一周后。”白忧定下了时间后起身，就此告辞。
<br/>　　罗杰也不多说废话，起身便离开前去准备。
<br/>　　只是在双方都不知道的转身走远后不久，两人都做出了一致又相互隐瞒的举动。
<br/>　　……
<br/>　　白忧在一群白家人的拥护中冷静下令：“派人前去狼牙暗杀楚涵，顺便找找，叶永阳在不在那里。”
<br/>　　“是！”一名白家人立即离开前去安排。
<br/>　　叶永阳消失在高原基地之后的去向，经过重重分析虽然没有直接的证据，但白忧依然觉得是狼牙中人半路劫持。
<br/>　　……
<br/>　　同样在另一边，罗杰在结束了与白忧的谈话后，一改之前的沉重冷静，相当焦急健步如飞，更是满脸紧张的到处找人。
<br/>　　“你，你过来，快去本家传话，让我几个叔叔合计合计，派人去狼牙看看情况，楚涵能不能直接杀了？还有那个白允儿，这个坏事的女人，能杀也一起杀了！”
<br/>　　说着罗杰还不放心，又吩咐道：“让研究院准备一下，一周后他们也一起过去。”
