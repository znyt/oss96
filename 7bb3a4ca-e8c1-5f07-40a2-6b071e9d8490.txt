　　神剑山内部，广阔无尽的空间之中。
<br/>　　苏莫如同入定的老僧，盘坐在虚空之中，纹丝不动。
<br/>　　若是仔细的观察，会发现，苏莫的胸口隐隐的光晕闪动。
<br/>　　这是剑心，剑道之心。
<br/>　　“战气……战意……！”
<br/>　　苏莫脑海之中，不停的呢喃，他在研究自己的不败战气。
<br/>　　不败战气的力量，非常的玄妙，应该是一种类似意志的力量。
<br/>　　这种力量，无形无质，但是，却是威力极大。
<br/>　　而苏莫要做的是，将战意融入剑道之中，这非常的困难。
<br/>　　好在，苏莫现在时间还算充足，不受打扰，静心参悟之下，渐渐有了眉目。
<br/>　　又过去了很久，苏莫手掌一番，长剑再次取出，一剑横斩虚空。
<br/>　　一剑出，虚空崩溃，剑威浩荡，恐怖的混沌剑气横贯长空。
<br/>　　剑气的气息奇异无比，可以清晰的看到，剑气方圆千丈之内，形成了一个巨大气场。
<br/>　　这气场，如同一个领域，乍一看去，又似乎是一个轮盘。
<br/>　　这是吞噬的力量、战气的力量，以及剑的力量结合，形成的一种玄妙气场。
<br/>　　这气场的玄妙，苏莫自己都不能完全了解，但是，气场之强，非常的恐怖。
<br/>　　即便苏莫没有催动任何元神，这一剑的威力，也已经快赶上他以前的巅峰一击了。
<br/>　　整个气场之内，都是一片死地，更有无尽的力量，不断的融入气场之中。
<br/>　　“很好！”苏莫眼眸闪亮，心中略显激动，他算是摸索出了自己的剑道，虽然只能算是入门的阶段，但是效果还是不错的。
<br/>　　“这一招就叫你混沌剑轮吧！”苏莫随便给这记剑招取了个名字。
<br/>　　之前，他已经有了一招吞天噬地，现在又有了一招混沌剑轮。
<br/>　　少倾，苏莫离开了这里，返回绫裳剑宫属于他的阁楼之中。
<br/>　　因为，参悟剑道，已经耗费了他很多的时间，接下来他要尽力提升修为了。
<br/>　　苏莫离开神剑山内部之时，却是没有发现，远处有一双眼睛一直在盯着他。
<br/>　　回到阁楼之后，苏莫进入了密室，取出了大量的五行源石，开始了修炼。
<br/>　　修为到了真神之境，哪怕是天赋再高，资源再多，都不可能快速精进了，需要时间的打磨。
<br/>　　苏莫催动吞噬元神，开始吞噬五行源气，凝练混沌源力，壮大修为。
<br/>　　一切，有条不絮，稳扎稳打。
<br/>　　……
<br/>　　紫霄剑宫，紫霄大殿之中。
<br/>　　紫霄高坐于主座之上，面带威严，下方，则是恭敬的站着一名黑衣青年。
<br/>　　这名黑衣青年，个子不高，身形消瘦，皮肤略黑，但一双眼眸却是锐利如鹰。
<br/>　　“许焱，怎么样了？”紫霄眼眸微抬，看向下方的黑衣青年。
<br/>　　“禀宫主，那秦陨具体和绫裳宫主是什么关系还不清楚。”黑衣青年许焱恭敬的说道，他奉紫霄宫主之命，查探那秦陨和绫裳的关系。
<br/>　　只不过，那秦陨一直在神剑山中参悟剑道，今日才离开，他还没有查出眉目。
<br/>　　“嗯，你多多留意此人。”紫霄微微颌首，为了自己的计划不出差池，所以他便名许焱留意苏莫。
<br/>　　“明白！”
<br/>　　许焱点了点头，随即，面色略显迟疑的道：“不过……！”
<br/>　　“不过什么？”紫霄面色一沉，轻喝道。
<br/>　　“宫主，那秦陨非常妖孽，虽然修为只是二品低级，但是，我看他在神剑山内出手，战力之强，估计足有我的三成实力。”许焱面色惊讶的说道。
<br/>　　适才，他看到了苏莫出手，那一剑的威力，差不多有他的三成实力了，这让他简直不敢相信。
<br/>　　要知道，他虽然不是顶级妖孽，但却是三品中级的修为。
<br/>　　“哦？”紫霄闻言，双眉一挑，心中大感诧异，秦陨的实力居然有许焱的三成！
<br/>　　他虽然也看出来了苏莫不凡，倒是没想到如此的不凡。
<br/>　　“说说看！”紫霄抬手，示意许焱继续。
<br/>　　许焱略一沉吟，继续道：“这个秦陨，修炼的是混沌之力，修为之力非常强，他的剑法气势古怪，不曾见过，我估摸着此人的实力，堪比一般的三品低级之人！”
<br/>　　“有趣！”紫霄闻言，嘴角勾起了一丝笑意。
<br/>　　这个秦陨，天赋如此逆天，差不多相当于万古天骄了，和绫裳到底是什么关系呢？
<br/>　　对于苏莫本人，紫霄倒是不太在意，他在意的是，苏莫会不会坏了他的好事。
<br/>　　“许焱，你去想办法，将此子赶出终极神王殿！”沉吟了片刻，紫霄吩咐道。
<br/>　　他觉得，还是将那个秦陨赶出神剑山最好，免得和绫裳生出感情。
<br/>　　至于那位要他救治的女人，三百年时间快到之时，秦陨自然还会回来，和绫裳一起求他。
<br/>　　反正，那个女子一日不救活，秦陨和绫裳都在他的掌控之中。
<br/>　　“这……宫主，那秦陨是绫裳剑宫之人，受到绫裳宫主的庇护，怕是不好赶出去吧？”许焱闻言迟疑，主要是他担心得罪绫裳宫主。
<br/>　　毕竟，紫霄宫主虽然不在意绫裳，但他一个小小的弟子，还是惧怕的。
<br/>　　“宫有宫规，殿有殿规，谁若是违反了殿规，定不饶恕！”紫霄面带威严的说道，他不会亲自动手，免得激怒绫裳。
<br/>　　“宫主，我明白了！”许焱点了点头，他明白紫霄的意思。
<br/>　　对方的意思，是让那秦陨触犯殿规，从而被逐出神剑山。
<br/>　　这个不难，但是，要好好谋划一番。
<br/>　　“嗯，你自信处理吧！”紫霄挥了挥手，示意许焱离开。
<br/>　　“弟子告退！”许焱躬身行礼，随即便退出了大殿。
<br/>　　离开大殿之后，许焱一边行走，一边琢磨着怎么让秦陨触犯殿规。
<br/>　　若是犯了一般的小错误，有绫裳宫主的庇护，秦陨肯定不可能被逐出去。
<br/>　　只有大的错误，才能让秦陨彻底被逐出神剑山，连绫裳都护不住。
<br/>　　而大的错误，无外乎以下犯上、背叛圣王殿、残害同门、奸.污女弟子等情况了。
<br/>　　很快，许焱回到了住处，召集了几个同为紫霄剑宫的好友，传达了紫霄神王的意思。
<br/>　　然后经过一番讨论，便设定好了详细的计划。
