　　对方说着话，看着他，人还在剧烈的咳嗽，显然有些激动。
<br/>　　涂小安见这中年男人骨瘦如柴的模样，生怕他激动的一个咳嗽没提上劲，一命呜呼。
<br/>　　他太瘦了，简直就是皮包骨，深陷的眼窝，干燥的皮肤，手臂瘦的跟竹竿死的，腿看不见，可那有点窄的裤腿，风吹过来，鼓鼓囊囊的，隐隐约约看见一个比麦秆还细的大腿。
<br/>　　简直比吸毒多年的瘾君子还瘦的夸张，能活着，真的就是一个奇迹了。
<br/>　　“请问你是？”涂小安微微的后退一步，跟对方保持一点距离，这咳嗽的，再用力点，估计就把肺咳出来了。
<br/>　　“涂大师，是我，是我，咳咳咳...”
<br/>　　涂小安一头黑线，你到是说自己名字啊，搞的很熟一样的，那么瘦的人，他怎么可能认识。
<br/>　　认识了也不可能忘啊，这形象太立体，太抽象了。
<br/>　　苏羽开口：“小安兄弟，你真的认不出他是谁？”
<br/>　　涂小安看了看苏羽，然后认真看了看这位骨瘦如柴，激烈咳嗽的人，还别说，仔细一看，有点眼熟，忽然，一只威风凛凛的猫从他的脑海中闪过，顿时，他诧异道：“你是余子童？”
<br/>　　白家的一品捉妖师，出门全靠一只猫打天下的余大师。
<br/>　　涂小安认出来之后，更为震惊了，余子童之前是什么样子他可太清楚了，虽然不说是一个胖子，但是跟一个瘦子也八竿子打不着。
<br/>　　而且还是如此的骨瘦如柴，这才几天不见呀。
<br/>　　“涂大师，救我，你一定要救我”
<br/>　　骨瘦如柴的余子童用双手抓在了涂小安的肩膀上摇晃，已表示自己激动的心情跟复杂的情绪。
<br/>　　可摇晃了几下，涂小安楞是站在原地，纹丝不动，他自己却跟着摇晃起来。
<br/>　　涂小安看着这家伙的模样，颇为无语。
<br/>　　余子童现在未免有些，有些太弱鸡了吧，浑身一点力气没有，被一阵风都能吹走。
<br/>　　这浑身骨头架子，看着都瘆人。
<br/>　　忽然，涂小安从他身上闻到了一股莫名的气味。
<br/>　　是妖气，觉醒生物的气息。
<br/>　　涂小安俨然起来，眼睛微微一缩，浑然一眯，这家伙身上竟然弥漫着淡淡的妖气。
<br/>　　这时，苏羽道：“小安兄弟，你看看余大师到底是怎么了，简直跟中邪了一些，我带他去医院检查过，想要查出病因，可是医院的仪器检查下来，根本检查不出任何异常病症”
<br/>　　一个好好的人，没两天下来，就成这样子了，好像被吸血鬼吸过一样，全身只剩皮包骨。
<br/>　　“余大师，你告诉我，你是不是遇到什么事了，还是谁说把你害成这样”涂小安同情的看着他问。
<br/>　　把一个人变成这样，这手段太残忍了，还不如杀了他。
<br/>　　“咳咳...我...咳咳...我”余子童一激动，情况变得越来越糟糕。
<br/>　　“......”涂小安一头黑线，说一个字，咳嗽许久，一句话都讲不下来。
<br/>　　涂小安顿时想到什么问：“你的猫吗，你那只拥有老虎之威，狮子之势的黄金瞳呢”
<br/>　　余子童出门全靠一只猫，可现在这只猫却不在，这不正常。
<br/>　　“我...咳咳...我咳咳...”
<br/>　　我去。
<br/>　　“行了，你先别说话了”涂小安真怕他一口气上不来，就嗝屁了。
<br/>　　“我说你听”
<br/>　　涂小安深深的看着他这具骨瘦如柴的身体，道：“我感觉到你身上有股淡淡的妖气，你变成这样，应该跟身上的妖气有关系把”
<br/>　　余子童边咳嗽边拼命的点头，眼中有喜色。
<br/>　　“你站着别动，让我看看你身体内有什么东西”涂小安眼睛浑然一眯，强大的意识力覆盖到了余子童的身上。
<br/>　　意识力将他每一寸皮肤都笼罩着，涂小安闭上了眼睛，就跟内视一样。
<br/>　　他在搜寻余子童体内的淡淡妖气，很快，他找到了妖气的所在，这妖气就如血液一样的在他的身体内到处流淌，血是红的，这妖气就如一抹黑，分外的明显。
<br/>　　在血液流淌，如同大海上的孤舟。
<br/>　　意识力的内视，并不能让涂小安知道这妖气到底是什么东西形成的。
<br/>　　他的眼睛猛然睁开，蛇王左眼闪烁光亮。
<br/>　　蛇王左眼看到了，透视一般的看到了余子童血液中的妖气，那是什么，一条小泥鳅一样的东西在吸食他的血肉，吸食他的精元，如蛆如骨，如恶魔附体。
<br/>　　再之细看，涂小安顿时大惊，这并不是一条小泥鳅，还是一条小蛇，非常非常小的蛇，已经不能用迷你来形容了。
<br/>　　看清楚之后，涂小安一脸的古怪，半天没说话，仿佛是陷入了某种思忖，身边的苏羽忍不住的问：“小安兄弟，你发现了什么，余大师的身体里面，莫非有什么脏东西”
<br/>　　什么叫脏东西，任何无法用常理去解释的诡异，都为脏东西。
<br/>　　“他身体里确实有点东西，准确的来说，里面有一条蛇”
<br/>　　“蛇？”苏羽闻言便大吃一惊，茫然若失，而余子童本人眼中顿时布满惊恐之色。
<br/>　　“莫非是余大师被人下蛊了”苏羽想了想道：“我听说，被人下蛊的人，身体会跟吸了毒一样的日渐消瘦，神志不清，然后慢慢的走向死亡，但就算是被下蛊，也是慢性毒药，不可能几天，余大师就成这样了”
<br/>　　涂小安想了想道：“蛊为虫，而他体内的应该是蛊蛇，非常致命的那种，里面那条蛇蛰伏在余大师的体内，疯狂的吸食血肉，精元，乃至寿元，所以才导致整个人几乎是已肉眼可见的速度消瘦”
<br/>　　“根据我的判断，按照这个速度，余大师恐怕活不到明天的太阳了”
<br/>　　“什么，那么严重”苏羽震惊。
<br/>　　“咳咳...咳咳...”余子童剧烈的咳嗽，浑身颤抖，就如同骷髅上批了一层皮，双眼深深的陷进去，面无死灰。
<br/>　　苏羽还是有点不敢置信：“怎么可能会有蛇在人的身体里”
<br/>　　“不信，那我就给你看看”涂小安眼睛闪烁，直接将余子童的上衣给扒了下来，露出了皮包骨的上半身，几乎就是骨架，让人看一眼，不寒而栗。
<br/>　　紧接着，涂小安一手按在余子童的肩膀上，意念转动。
