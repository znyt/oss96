　　第143章
<br/>　　“众家兄弟与我共饮此酒，饮罢，我们一起去救大当家。”
<br/>　　徐神机将酒碗高高举起，一饮而尽。
<br/>　　院内诸人同时将手中酒饮尽，浇起一腔杀意，然后听哗地一声，酒碗在地上砸得粉碎。
<br/>　　杀气腾腾！
<br/>　　一道烟花从院子内掀起，直冲云霄，这白日烟花会传到青牛山。然后黄三元带人杀过来，等接近落叶城时，再在城外放一道烟花。
<br/>　　以这道烟花为号，徐神机将会带人去劫方伯山的队伍，提前已经有弟兄安排在街道两侧。城里劫人，城外攻城，里应外合砸开城门，劫走方伯山后，以方伯山去换程大雷。
<br/>　　这计划是提前制定好的，如果方伯山没有防范，有很大把握成功。
<br/>　　现在，大家都在等待城外的烟花信号，便一股脑冲杀出去。
<br/>　　到时候，城内必定大乱，如徐神机所言，此战必是血战，到时候落叶城血流成河，尸体横街也不是夸张。
<br/>　　但这些人都是林中的魔王，山中的恶怪，既然踏上这条路，便早已将脑袋别在裤腰带上。自然没有人不怕死，可想到即将到来的杀戮，这些人体内的血突然变得滚烫起来。
<br/>　　方伯山的大队走在街上，中间一顶软轿，由人抬着，晃晃悠悠向码头行去。方伯山坐在软轿内，已经有了些许困意，忽然一道响声，将他吵醒。
<br/>　　“出什么事了？”
<br/>　　“城里有人放烟花，要不要我让人去看看？”轿外的管家道。
<br/>　　“大白天放什么烟花。”方伯山想了想道：“不必了，让他们加快脚步，咱们走快点。”
<br/>　　“是。”
<br/>　　街道两侧，有人鬼鬼祟祟望这边往，这些自然都是蛤蟆寨的人。以软轿和湖底捞两点画一条直线，有人来回奔波，随时随地将方伯山的位置报告给徐神机。
<br/>　　时间慢慢流逝，距离方伯山抵达码头已经没剩下多少时间，可城外的烟花信号还没出现，难道是山寨出了什么意外。如果等方伯山到了码头，在码头动手要比在街道动手困难许多倍。如果今天无法动手，程大雷未必能活得到下个月。
<br/>　　徐神机心急如焚。
<br/>　　“军师，怎么办？”此刻刘悲也慌了手脚，将希望寄托在『足智多谋』的徐神机身上。
<br/>　　“再等一刻钟。如果再没有信号，我们就杀出去。”
<br/>　　计划一开始便出现偏差，这大概就是谋事在人，成事在天的道理。
<br/>　　时间一点一滴过去，每一刹那都在此刻变得无比漫长。
<br/>　　湖底捞后院，诸人的视线都看着徐神机，等着他的命令。没有城外的援军，自然危险万分，可事到临头，已经是不能不做。
<br/>　　街道两侧，依旧有人来回奔波，随时报告方伯山的位置。
<br/>　　软轿晃晃悠悠，轿内的方伯山又打了一个盹儿，他除了觉得今天天气不太好外，倒不觉得有什么。
<br/>　　随着时间流逝，事态在缓慢却坚定的向前推进，如一张不断被拉开的弓，只等着最后时刻，射出那道致命的羽箭。
<br/>　　“徐军师，一刻钟过了。”刘悲道。
<br/>　　徐神机点点头，真的不能再拖下去，再拖下去便要错过最佳动手时机。
<br/>　　他深吸一口气：“动手！”
<br/>　　那张弓已经拉到最慢，扣弦的手指已经微微扳开，箭在弦上，蠢蠢欲动。
<br/>　　街道上，队伍仍在行进，突然，一匹奔马斜刺里冲出，马上一人高呼。
<br/>　　“停轿！”
<br/>　　酒楼内，大队人马携刀，由徐神机率领，准备杀出大门。
<br/>　　咚咚咚！
<br/>　　敲门声此时突然响起，门外一人狂呼：“快开门，快开门！”
<br/>　　是黄三元的声音，徐神机一惊，他怎么来了，难道众人不在的这段光景，山寨真的出事了！
<br/>　　院门打开，黄三元气喘吁吁的撞进来，徐神机还来不及问，就听黄三元上气不接下气道：“大当家，大当家有救了！”
<br/>　　那张拉到最满的弓，突然被勒住，蠢蠢欲动的箭，终究没有射出。
<br/>　　……
<br/>　　这匹突然闯进队伍中的马，让软轿周围的兵丁有些慌乱，他们之所以没有第一时间动手，是因为骑马之人穿着的是官服。
<br/>　　轿内的正打盹的方伯山被吵醒，他很不耐烦道：“又怎么了，能不能让我好好进行业余爱好！”
<br/>　　“吾乃幽州齐德强，有要事见方伯山城主！”
<br/>　　方伯山顿时睡意全无，立时从轿内走出，挥手将周围的兵丁赶散。
<br/>　　“原来是齐将军大驾光临，不知齐将军有什么事，来得这么急，让我都没时间准备。”
<br/>　　幽州王治辖幽州，但现在皇权衰老，有些城池是不太给幽州王面子的，比如韩虎距管制黑石城的时候。而有些城池直接是幽州城的附庸，比如落叶城。
<br/>　　论官职，方伯山比齐德强高，但齐德强是杨龙停面前的人，如此两人的关系倒是不能以官职论。
<br/>　　齐德强大步过来，着急道：“方城主，你差点坏了主公的大事你知道么！”
<br/>　　酒楼内，黄三元终于喘匀了气，所有人的眼睛都盯着他。
<br/>　　“你的意思是说，幽州王的封赐下来了？”徐神机问。
<br/>　　黄三元点点头：“封游骑左将军，赏白银百两，盔甲一副，锦袍一件，佩刀一柄。”
<br/>　　帝国的军衔制度十分乱，粗略分大概是：什长，百夫长（队长），校尉，将军，元帅。
<br/>　　到将军这一级就特别乱了，各地情况各不相同，牙将、裨将、参将、游击将军……领十万人也可以叫将军，领一百人也能叫将军。
<br/>　　而程大雷这个将军呢……官位不低，权力没有，等于虚衔。
<br/>　　“对咱们各位也有封赏，百夫长校尉什么的，这些幽州王没封，留着给大当家自己决定。”黄三元又喝了一口水：“齐德强亲自来传的令，我一得到消息，就立刻赶来了，现在齐德强已经去方伯山那边了。”
<br/>　　“有了这个左将军的官位，大当家就有救了，咱们也不用动手了？”徐神机。
<br/>　　“这是自然，幽州王绝不会因一时误了长久的计划。”刘悲道了一声，忽然看着徐神机：“看军师的意思好像有点遗憾？”
<br/>　　“也谈不上遗憾啦。”徐神机把此时还握着的刀收起来：“就是刚才酒喝多了，有些上头。”
<br/>　　
<br/>(本章完)
