　　短时间内也许还做不到这一点，但是只要是过了五年，甚至是十年之后。
<br/>　　这些弟子，只会记得剑道门，而不会在记得北邙学院。
<br/>　　一个势力的兴衰，便是如此，楚风眠心中也清楚。
<br/>　　“邓灵，武皇这些财富交给你，未来你就是剑道门北邙堂的长老，你负责去培养他们。”
<br/>　　楚风眠看着邓灵武皇二人，开口道。
<br/>　　楚风眠自己的时间都不够用，当然不可能抽出空来培养弟子。
<br/>　　培养弟子的事，交给邓灵跟武皇去做。
<br/>　　他们二人本来就是北邙学院的高层，对培养弟子的事，比起楚风眠清楚的多。
<br/>　　而且他们二人，也已经是彻底臣服了楚风眠，楚风眠也不担心他们二人背叛的可能。
<br/>　　寒雪阁那边，楚风眠是打算交给弑剑君跟洛河负责，至于北邙学院这一边，就是交给邓灵跟武皇了。
<br/>　　“多谢门主！”
<br/>　　邓灵跟武皇，听到楚风眠的话，急忙跪在地上，高声的喊道。
<br/>　　脸色无比欣喜。
<br/>　　他们二人也是从小在北邙学院之中长大，对北邙学院有着很深的感情。
<br/>　　现在看到楚风眠拿出来的资源他，他们二人心中也明白，得到了如此之多的资源后。
<br/>　　北邙学院绝对是一飞冲天，甚至是未来可能足以比肩七大宗门，让他们二人的心中也是无比欣喜。
<br/>　　至于其他北邙学院，不，现在应该说是剑道门北邙堂的弟子。
<br/>　　如今看到了如此之多的资源，心中本来还有的几分反抗的心思，也都没有了。
<br/>　　顿时之间，北邙学院的变化开始了，一座座牌匾，不断的更换。
<br/>　　所有北邙学院的名字，如今都已经是变成了剑道门。
<br/>　　北邙学院，地处一处单独的空间之中，是曾经剑凌施展出无上手段，缔造出来的一方小世界。
<br/>　　楚风眠建立剑道门，也就以这一方小世界为根本。
<br/>　　嗖！嗖！
<br/>　　在这空间的中心，一座座山峰拔地而起，在这一座座山峰之上，无数宫殿林立。
<br/>　　这些宫殿，正是寒雪阁之内的宫殿，如今北邙学院跟寒雪阁合二为一，楚风眠也将寒雪阁，从武胜国度，整个带到了这一处小世界之中。
<br/>　　在北邙学院里，是有着一座灵泉存在的，所以其中的灵气无比浓郁。
<br/>　　寒雪阁的弟子进入其中，也足以加快修行的速度。
<br/>　　无数忙碌的身影，在不断的穿梭这。
<br/>　　寒雪阁弟子，北邙学院弟子，如今都已经是加入了剑道门，他们之间不少人都有些恩怨。
<br/>　　一开始还有几场内斗，但是都被楚风眠以铁血手段处罚了之后，便是都平静了下来。
<br/>　　毕竟没有几个人，愿意因为意气之争，而最终丢了性命。
<br/>　　剑道门的建立，稳扎不稳的进行着。
<br/>　　而楚风眠，也没有闲着，在这段时间之中，楚风眠一直在这一方小世界之内穿行。
<br/>　　在各个角落之中，布置镇压，同时放下黑玄灵剑。
<br/>　　一千把黑玄灵剑，都被分布在这一方小世界之中。
<br/>　　同时这每一把黑玄灵剑上，都有楚风眠亲手刻画出来的法阵，威力比起之前，强上五成不止。
<br/>　　整个剑道门，都是被一道巨大无比的阵法所笼罩，这阵法比起当初楚风眠在寒雪阁之内设下的，威力还要大的多。
<br/>　　毕竟在寒雪阁内，并没有灵泉，可以源源不断的支撑灵力消耗。
<br/>　　而在这剑道门之内，而是有着一整座灵泉，楚风眠将最终的阵眼，放在了灵泉之下。
<br/>　　如此一来，如果遇到危机，便是足以源源不断的利用灵泉的力量，来进行抵挡。
<br/>　　这护山大阵，让楚风眠亲自出手，全力之下，没有半日的时间也不可能打的破。
<br/>　　楚风眠的实力已经是无限接近九劫古帝了，可以说来一位九劫古帝强攻，这护山大阵也足以拖延足够的时间。
<br/>　　“可惜只有一座灵泉。”
<br/>　　楚风眠布置完这一切，打量了一眼，还是摇了摇头道。
<br/>　　如果说多几座灵泉的话，楚风眠就可以多布置出几个阵眼来，倒时候这护山大阵的威力，还会在强许多。
<br/>　　甚至是足以完美抵挡九劫古帝的轰击，让一位九劫古帝，都要花费数天的时间，在足以打破。
<br/>　　“在归墟天里，倒是有着不少的灵泉，以我现在的实力，摄取几座，并不困难。”
<br/>　　楚风眠心中想到。
<br/>　　他想要突破帝尊境，第一个选择，就是要去归墟天。
<br/>　　归墟天是楚风眠知道的，九域之中宝藏最多的地方，而且楚风眠本身也是去过归墟天。
<br/>　　对归墟天也是熟悉，知晓许多宝藏的所在。
<br/>　　之前因为楚风眠的实力不济，所以许多宝藏，楚风眠发现了，但是都没有实力可以去取。
<br/>　　但是现在楚风眠却是没有这个担心了。
<br/>　　在将剑道门的事安排好之后，楚风眠就准备启程归墟天了。
<br/>　　“这护山大阵，足以比得上圣龙秘境五灵宗的护山大阵了。”
<br/>　　洛汐站在一旁，看着这护山大阵被楚风眠布置出来，感叹了一声道。
<br/>　　足以抵挡九劫古帝的护山大阵，整个九域之中都未必足以找的到第二个。
<br/>　　像是在圣龙秘境，五灵宗虽然是四宗之一，但是布置出的护山大阵，也就是楚风眠这个级别的了。
<br/>　　布置护山大阵，比起想象之中的还要困难的多，一是材料，而是阵法的选择。
<br/>　　楚风眠前世熟读了大量的古籍，所以在阵法的选择上，倒是了然于心，他选择的，是一种古老的剑阵。
<br/>　　以剑为阵眼，一千把黑玄灵剑，就是最好的选择。
<br/>　　而布阵的材料，也决定着护山大阵的强弱，这一千把黑玄灵剑，可全部都是天级灵器。
<br/>　　只怕是大罗天星域的一些大势力，都未必可以一口气拿得出一千把一模一样的灵器来。
<br/>　　这两点，让楚风眠虽然没有圣龙秘境那么雄厚的底蕴，但是这布置出的护山大阵，也是丝毫不弱。
<br/>　　“我也来助你，布置一下这护山大阵吧。”
<br/>　　在打量了一眼后，洛汐突然身躯一动，一步上前，突然运转起灵力来。
<br/>　　在洛汐的背后，混沌的门户，缓缓打开。
<br/>　　混沌之力，当场降临。
