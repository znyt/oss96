　　叶少阳烧了一张萧逸云给的玉符，过了不久，萧逸云上来找他，听他说了这件事之后，表示关于这种封神的事，地府七十二司中，有一个“掌山林鬼神司”，专门负责这个，司主是南宋名臣李纲死后担任（看过岳飞传的都都知道这位爷，北宋末年、南宋开国第一位肱骨之臣），这个山林鬼神司虽然是个清水衙门，一般不参与阴司的具体事务，不过阴阳两界的司主一下的神，都由这位爷来册封。
<br/>　　萧逸云决定去找李纲说下情况，让叶少阳和毛小方在这等着。
<br/>　　叶少阳闲着也是闲着，干脆吐纳起来，吐纳了两个周天，萧逸云姗姗而来，一见面就耸了耸肩，说道：“不好弄。李纲那边帮我查了，说这五通神是大概三十年前封的神，一向在这里镇守，保一方平安，素有功劳，也从来没有伤过人性命，因此不能撤销神位。不给面子，我没办法。”
<br/>　　叶少阳和毛小方沉吟不语。
<br/>　　萧逸云道：“你们看这样行不行，我出面跟他谈，让他以后不要再祸害姑娘，好生镇守一方，大家相安无事，如何？”
<br/>　　叶少阳静静地看了他一会，说道：“这种事他干了几十年，俗话说食髓知味，他有这神通，你怎么保证他以后不会再干？再说，以前那些被他祸害过的姑娘，就白白被祸害了？”
<br/>　　“那……你的意思？”
<br/>　　叶少阳做了一个抹脖子的动作。“我要屠了他。”
<br/>　　萧逸云嘴角抽搐了一下，道：“屠神？”
<br/>　　“屠神怎么了，我也不是第一次干了。”
<br/>　　“不是吧，你以前屠神过？”毛小方和萧逸云都十分震惊。
<br/>　　“两次。”叶少阳杀过十二年蝉七姑和河神。
<br/>　　萧逸云问清楚了详细情况，说道：“这情况不一样，那河伯虽然你没资格杀他，不过也毕竟犯了杀戒。你说的那十二年蝉也一样，而且那十二年蝉跟这阴司册封的正神不同，不过是一个邪神而已。”
<br/>　　七奶奶跟眼下这五通神还是不一样的。叶少阳当然也知道，七奶奶只是受到一方供奉，在犯杀戒之前，被阴司默许了她的存在，并不算是正式册封。这五通神不同，
<br/>　　萧逸云接着说道：“李纲说了，这五通神虽然淫邪，不过毕竟镇守一方，福泽生灵，算是功大于过……”
<br/>　　叶少阳面色一沉，说道：“敢情那被糟蹋的不是李纲的女儿，如果是，我看他还这么说？”
<br/>　　萧逸云和毛小方都不说话。
<br/>　　过了一会，萧逸云道：“你想清楚。”
<br/>　　叶少阳道：“我自己慢慢想，你回去吧。”
<br/>　　萧逸云笑道：“你什么意思，不想让我帮你一把，怕连累我？”
<br/>　　“是啊，你是天子殿押司，如果参与屠神，就算是崔府君也不好保你。我们不一样。”
<br/>　　萧逸云耸了耸肩，“如果你真的要去做……也可以算我一份，我刚才代入了一下，如果被欺负的是我媳妇……这个真的不能忍。”
<br/>　　“要是你媳妇，我早就把这个王八蛋祖坟刨了！”叶少阳瞪眼说道。
<br/>　　萧逸云怔住，喃喃道：“什么情况，我说我媳妇，跟你什么关系？”
<br/>　　“没事没事。”叶少阳不想跟他说出有关橙子的事情，刚才自己也是代入了一下，如果橙子被人这样欺负，自己真的敢屠杀他祖宗十八代。
<br/>　　“不扯这个，我不让你参与，不是怕连累你，而是你在天子殿，可以做更多事情，再说如果我真屠神了，你也可以利用身份帮我斡旋一下，如果你自己也跳进来，那我们就真完了。换句话说，你不是不参与，而是你参与的方式，就是帮我们善后。”萧逸云能提出参与屠神这种事，叶少阳还是很高兴的，虽然这是百年之前，他们还不是兄弟，但是萧逸云的个性还是有的。
<br/>　　叶少阳看到他，仿佛看到了当初——应该说之后，偷取天子持节、为了自己大闹轮回司的萧逸云，忍不住拍了拍他的肩膀，能在这个世界找到他，自己还是很欣慰的。
<br/>　　萧逸云听他这么一说，觉得也的确是，点点头说道：“那什么，你打算怎么办？五通神修为极深，你别硬来。”
<br/>　　“放心吧，我没那么傻，肯定要想个万全的办法，到时候需要你帮忙，我再找你。你先回去吧。”
<br/>　　萧逸云说了几句话，往阴间去了。
<br/>　　毛小方问叶少阳：“你打算怎么办？”
<br/>　　“不知道，我还没想好，先想办法救下那个姑娘再说。”
<br/>　　“怎么救。”
<br/>　　“说了没想好呢，等我们再想想，先找个地方睡觉。”
<br/>　　这里前不着村后不着店的，回镇上是可以，但是有点远，现在去了明天还得来，两人随便找了个避风的地方将就了一个晚上，天亮之后，找了个小溪洗了洗脸，吃了点干粮，接着商量怎么办，叶少阳决定直接上山去把那姑娘救出来。
<br/>　　“白天，五通神应该不在山上，我们可以先把那姑娘救了，然后再想办法对付五通神，你觉得呢？”叶少阳提议道。
<br/>　　毛小方想了想道：“他未必不在山上吧，我们昨晚烧了他的庙，他估计也想到不可能是土匪所为，今天也许正在山上等我们。”
<br/>　　叶少阳道：“那也没关系，我有办法让他不敢把我们怎么样。”
<br/>　　“什么办法？”
<br/>　　叶少阳跟他说了之后，毛小方也觉得有道理，于是两人原路返回，前往山神庙。
<br/>　　在山脚下，看到一伙人扛着铁锹上山，还有几个抬猪头的。毛小方打听了一下得知，这伙人是那庙祝陈老汉从附近镇上找来的，因为昨天庙被火烧了，多少有些受损，特意找人来修。至于那些抬着猪头的，是镇上村民自发前来供奉血食，安抚神灵的。
<br/>　　毛小方别有意味地看了叶少阳一眼，道：“你知道我想说什么？”
<br/>　　叶少阳点点头，他明白毛小方的意思，毛小方是想说：这五通神在这一带十分受到民众的崇拜，想要对付他，这些村民将来可能也是阻力。
<br/>(本章完)
