
<br/>　　<br/>　　    王惊蛰的路子谈不上有多野，也没有多么的离奇，但是他的手段却能够直击人灵魂的最深处。<br/>　　<br/>　　    每个人的心里都有一道防线，每个人都有秘密，这是绝对不会例外的，当你能够击垮这个人的防线时，你自然能够得到他的秘密，某位心里学上的大师曾经讲过，人是最脆弱的一种生物，只要你能够到达他的心灵最底端，你也许只凭一句话就可以俘获了他。<br/>　　<br/>　　    王惊蛰就想要抵达一个叫杨可凡的人心灵的最底端，因为他就是恒晟地产的财务总监，在这个公司里，但凡有关账目上面的问题，肯定得经过他的手，所以徐少中如果涉及到洗钱的话，不管绕过谁都不可能绕得过他。<br/>　　<br/>　　    自然，王惊蛰就找上了他，两人是在一个高档小区里见上面的，当时是早上八点钟，杨可凡从家里出来，领着两个孩子，正要送进车里打算离开，这是他每天的经历，早上送孩子去上学，然后他在九点钟左右抵达公司。<br/>　　<br/>　　    杨可凡刚拉开车门，让孩子坐到后面，他正要发动车子的时候，忽然听到身后有人说了一句话：“你有三个孩子……”<br/>　　<br/>　　    王惊蛰蹲在花坛边，穿着身很普通的道袍，笑吟吟的望着杨可凡，对方听到他的话后就皱着眉头转过脑袋，说道：“你谁啊？”<br/>　　<br/>　　    王惊蛰提了下长袍，站起来走到他身前，淡淡的说道：“我观你，知你家最近多事，施主如果有疑，我能为你解答一二，还请您稍留一点时间，让我说上一说”<br/>　　<br/>　　    王惊蛰此时的做派，很想走街串巷的算命先生，他唯一欠缺的可能就是下巴上的一缕长须和没有扛着的算命招牌了。<br/>　　<br/>　　    “神经病！”杨可凡嘟囔了一句，然后坐到车里，摇头说道：“这小区怎么搞的，什么人都让进了呢”<br/>　　<br/>　　    王惊蛰笑吟吟的在杨可凡升上车窗时，又接着说道：“你有三个孩子……幼子多病，体虚，多日不见好转”<br/>　　<br/>　　    “刷”杨可凡手按在车窗按钮上，刚刚升起来一点就顿住了，他抬头问道：“你怎么知道的，你到底要干什么？”<br/>　　<br/>　　    杨可凡有三个孩子，车后面坐着的是他的两个女儿，一个六岁一个七岁，在连续生了两个孩子之后，他非常想要个儿子，就跟老婆努力了两年，最终在去年的时候生下了一胎男婴，这下子可把杨可凡一家上下老小给乐的够呛了。<br/>　　<br/>　　    但是杨可凡和媳妇可有点闹心了，才一岁多点的小儿子不知道是出了什么问题，按理来讲这个年纪的幼儿应该没事就扑腾着手脚，爬来爬去，再不就是又哭又闹的，但他的孩子却很奇怪，一直都是很安静的躺在床上，从来不哭闹，也没什么反应，除了眨着眼睛和喘气以外，几乎从来不会有任何多余的动作。<br/>　　<br/>　　    说白了，就是跟个白痴似的。<br/>　　<br/>　　    杨可凡和老婆曾经带着孩子去了几次医院检查，但都没得出什么结论来，大夫给出的回答则是这孩子有可能发育的晚，这个解释其实挺让人担忧和无语的。<br/>　　<br/>　　    杨可凡仰着脑袋看着他，王惊蛰徐徐的说道：“你要是信我的话，就在今天十二点一刻的时候，把这张纸贴在孩子的肚脐上，然后你再看看他有什么反应，到时如果你感兴趣的话，可以打这个电话给我……”<br/>　　<br/>　　    王惊蛰递给了杨可凡一张黄色的符纸，和写着他电话号码的纸条，等对方接过来后，他就再也没有说什么，扭头就走了。<br/>　　<br/>　　    杨可凡呆呆的看着手里的符纸，他下意识的就想给团起来扔掉，但是当手攥起来的时候，他的胳膊伸出车窗外面就顿住了。<br/>　　<br/>　　    不是王惊蛰刚才说那番话的时候表现的有多么仙气飘飘，而是他这句话说到了身为人父的杨可凡的心里去了，他这半年多的时间里，为了儿子的问题，已经熬出焦虑症来了，这孩子真要是有什么治不了的病，他的一切希望可能就要落空了。<br/>　　<br/>　　    稍微挣扎了下，杨可凡收起了王惊蛰给他的符纸，然后一路上脑袋都很混乱的把孩子送到了幼儿园，又到了公司上班，上午的时候徐少中主持召开会议，继续研究这两天的问题。<br/>　　<br/>　　    虽然如今闹得风风火火，乱糟糟的，但顾西凤说的话很对，骚乱和负面新闻不过是一场正常的风波而已，几乎很多公司都遇见过，不管用什么方式，都有过去的一天，稍微坚持下就行了，谈不上上纲上线的地步。<br/>　　<br/>　　    但他们可能都没有意识到，这不过是场热身罢了。<br/>　　<br/>　　    会议没到中午就结束了，徐少中和几个高管在离开会议室的时候，问了他一嘴中午要不要一起去食堂吃饭，杨可凡本来想要答应来的，但想了想早上的那一幕，他就说孩子不太舒服，先回家看看。<br/>　　<br/>　　    杨可凡的决定，有点鬼使神差的意思，他自己的思想里对王惊蛰说的话，是不认可的，但你架不住人心理上的琢磨啊，这完全就是一种心里暗示，可条件反射。<br/>　　<br/>　　    杨可凡觉得，试试，就试试，反正也没什么麻烦的。<br/>　　<br/>　　    于是，杨可凡就开车回到了家里，他回来的时候老婆刚给孩子喂完奶，孩子正安安静静的躺在床上眨着眼睛。<br/>　　<br/>　　    “你怎么回来的这么早，有事？”杨可凡老婆诧异的问道。<br/>　　<br/>　　    杨可凡手里握着王惊蛰给他的那张符纸，也不知道怎么解释，就苦笑了两声，然后掀起了孩子的衣服，将符纸贴到了肚脐上。<br/>　　<br/>　　    他老婆顿时不解的问道：“你这是干什么啊？”<br/>　　<br/>　　    杨可凡没有吭声，紧盯着孩子的小脸蛋，见儿子仍旧没啥反应，还是那么安静的时候，心里默默的叹了口气：“自己真是脑子坏了，居然相信个江湖骗子？”<br/>　　<br/>　　    杨可凡寻思了一下后，就弯下腰伸手就要揭开符纸，同时说道：“今天早上碰到个人，他说他知道孩子问题出在了哪，让我在十二点一刻的时候把这张纸贴到儿子的肚脐上，我也是懵了，居然还信了他的话，呵呵”<br/>　　<br/>　　    杨可凡刚说完，他老婆就抬头看了眼墙上挂着的时钟，下意识的说道：“这不正好十二点十五了么？”<br/>　　<br/>　　    杨可凡也抬头看了一眼，然后就低下脑袋，手指碰到了符纸上，还没等他揭起来的时候，小孩子顿时“哇”的一声哭了。
