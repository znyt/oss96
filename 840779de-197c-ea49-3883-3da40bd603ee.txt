　　“前辈，这通道会开启多长时间？”苏莫向秦不死沉声问道。
<br/>　　“具体不清楚，但应该不会超过一刻钟！”秦不死道。
<br/>　　苏莫点了点头，便不再作声，他可不相信那些巫族试炼者不想离开，他只需守株待兔便可。
<br/>　　时间缓缓流逝，约莫过了大半刻钟之后，依旧不见试炼者的踪影。

<br/>　　“前辈莫急，再等等！”苏莫沉声说道，他现在已经知道这些巫族试炼者是什么打算了。
<br/>　　这些人的打算，无非是等待，等他和秦不死进入通道，这些人再随后出去。
<br/>　　就看谁更有耐心了！
<br/>　　苏莫眸光精光闪烁，他丝毫不急，现在他身处空间通道的入口，只要通道有关闭的趋势，他立刻便能离开。
<br/>　　所以，他等得起，那些巫族却等不起。
<br/>　　时间缓缓流逝，很快，又过去了小半刻钟，秦不死彻底的焦急了起来。
<br/>　　“苏莫，我们不能再等了，再晚就来不及了！”秦不死急声说道，已经到了一刻钟，通道可能很快就会关闭，他不愿意等了。
<br/>　　苏莫面色凝重，无奈的点了点头，既然这些试炼者不来，他也只能放弃了，逃出去才是最重要。
<br/>　　嗖嗖嗖！！
<br/>　　就在此时，只见远处浩浩荡荡四百余人，急速向通道飞了过来。
<br/>　　这群人中，为首之人正是巫天御。
<br/>　　“来了！”苏莫眼眸一亮，身上气息骤然狂涌，滔天杀机暴冲而起。
<br/>　　“杀！”
<br/>　　“杀！”
<br/>　　“杀！！！”
<br/>　　惊天的喊杀声骤然响起，四百余名巫族试炼者，还未临近苏莫二人便齐齐出手了。
<br/>　　轰轰轰！！
<br/>　　刹那之间，无数的枪影、拳芒、剑气、刀光纷纷洒洒，遮天蔽日，裹挟毁天灭地之势，向苏莫轰杀而来。
<br/>　　数百道攻击，每一道都强大无比，汇聚在一起简直威不可挡。
<br/>　　恐怖的玄力波动如同怒海狂涛，磅礴的威势震动九天十地，数百道攻击碾碎一切，方圆数万里之内的空间刹那间崩碎。
<br/>　　正面面对这么多攻击，秦不死更是眼眸一瞪，面色有些苍白，就算是苏莫都感觉心中剧震，骇然不已。
<br/>　　不过，苏莫当然不会愚蠢到硬挡这么多攻击，他只是担心，这么多攻击不会把空间通道轰碎了吧！
<br/>　　但他的担心只是一瞬而已，这些人既然敢肆无忌惮的出手，空间通道想必不会有事。
<br/>　　“跟着我！”苏莫大喝一声，随即身形一闪，如同利剑般闪掠而出。
<br/>　　嗖!嗖！
<br/>　　秦不死本想冲进通道，但还是咬了咬牙，紧跟着苏莫飞了出去。
<br/>　　轰隆隆！！
<br/>　　数百道攻击没有击中苏莫，其中一多半都轰在了空间通道中，顿时响起了惊天的爆炸，毁灭般的气浪扫荡通道内外。
<br/>　　不过，这空间通道居然稳固无比，丝毫没有任何不稳的迹象。
<br/>　　苏莫和秦不死两人，轻易的避开了数百道攻击，闪掠到了千里之外。
<br/>　　而后，苏莫立刻开始大开杀戒，他心神一动，二十九柄本命灵剑尽数激.射而出，如同道道极电，穿梭虚空。
<br/>　　众巫族天才见没有击中苏莫，一边向通道冲去，一边再次齐齐出手，又是数百道攻击向苏莫杀来。

<br/>　　所有被本命灵剑击中的人，均是脑袋炸裂，惨死当场。
<br/>　　除了巫天御有本事勉强挡住灵剑的袭杀之外，其他人完全没有抵抗之力，瞬息间便惨死数十人。
<br/>　　“冲过去！”巫天御放声大喝，带领一众巫族天才，仿佛不怕死一般，个个爆发出了最快的速度，瞬息千里，急速向空间通道冲去。
<br/>　　与此同时，他们依旧在不断的出手，漫天攻击浩浩荡荡，不断的向苏莫攻击而去。
<br/>　　他们的目的，就是用洪流般的攻击挡住苏莫，使得苏莫无法靠近，从而冲进通道中。
<br/>　　这是他们没有办法的办法！
<br/>　　咻咻咻咻！！！
<br/>　　剑光闪动，杀机无限，苏莫一边躲避众人的攻击，一边控制灵剑袭杀。
<br/>　　啊啊啊！！
<br/>　　惨叫声不断，巫族天才的尸体，犹如下雨一般不断的向下方坠落。
<br/>　　巫天御自然是苏莫重点‘照顾’的对象，其中有七、八柄灵剑，不断的向对方绞杀。

<br/>　　嗖！
