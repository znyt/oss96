　　就连叶尘看到自己抽取的药方，也不由得哑然失笑。
<br/>　　他自重生以来，并没有怎么潜心研究炼丹术，所依仗的，无非就是前世的炼丹记忆，以及三昧真火和五昧真火这种强大的内火辅助，所以才能无往而不利。
<br/>　　而事实上，他所炼制过的二品丹药很少，如果抽到一个他不熟悉的药方，虽然他自信也不会有什么问题，但毕竟还是要耗费时间研究一番。
<br/>　　可是如今，他抽到的居然是筑基丹，恰好是他那些二品丹药之中，他最熟悉的一种，运气简直不要太好！
<br/>　　因而这一次，叶尘依旧没有动用自己的内火，还是和第一场一样，规规矩矩的来到丹师盟会为他们准备的炼丹炉前，点火、开炉、入药……
<br/>　　叶尘的动作很优雅，无论是火候，还是药材，都拿捏的分毫不差，整个过程犹如行云流水，光是看着，都觉得赏心悦目，即便是台上的那几位长老，也不由得频频点头。
<br/>　　倒是台下的那位蝉衣小姐，看了半响，忽然两眼微微一瞪，眼眸中透着一抹惊色，低声喃喃自语，道：
<br/>　　“奇怪！这人的炼丹的手法，怎么这么像，古籍中记载，早已经失传的，云丹祖师的成名绝技‘绕指柔’？”
<br/>　　“可是这手法，因为过于难练，早已经失传数百年，他是从何处学来的？”
<br/>　　“莫非是我看错了？”
<br/>　　就在蝉衣小姐心中疑窦丛生，百思不得其解的时候。
<br/>　　“轰！”
<br/>　　炼丹台上，再次发出了一道巨响！
<br/>　　而且这一次的威力，比之前还要强大十倍都不止，整个考核大台，都不由得狠狠晃动了一下。
<br/>　　众人先是一惊，等到他们定睛一看，看清楚眼前的情景之后，顿时全都目瞪口呆。
<br/>　　只见，
<br/>　　在那丹炉之上，悬浮着一颗，散发着熠熠光辉的丹药，宛若神丹！
<br/>　　“这、这……他居然又炼制出了一颗完美品阶的丹药？”
<br/>　　“不是吧！不是说完美品阶的丹药，即便等级再低，又是可遇而不可求的吗？这家伙竟然可以一连两次，炼制出完美级的丹药？”
<br/>　　“难道这小子，竟然是一个深藏不露的炼丹大师？可如果他真是炼丹大师的话，怎么会才只是区区二品？”
<br/>　　……
<br/>　　一时间，众人众说纷纭，整个大殿热闹非凡。
<br/>　　至于那屠苏公子，早已经完全傻眼了，
<br/>　　他之前说叶尘是走了狗屎运，所以才侥幸炼制出了完美级丹药，可是眼前的一切，已经足以证明，
<br/>　　一个人就算运气再好，也不可能一连两次，都能炼制出完美级的丹药吧？
<br/>　　而此时，
<br/>　　考核大台之上的那几位长老，也终于坐不住了，纷纷来到考核台前，
<br/>　　“不错！这的确是一枚完美品级的筑基丹！”
<br/>　　那位来自云丹宗的火龙长老，直接一锤定音，向众人宣布道。
<br/>　　轰！
<br/>　　得到了准确答案之后，整个大殿顿时再次躁动起来。
<br/>　　“这位道友，你的这颗筑基丹，能否卖给我，我愿意出一万上品灵石！”
<br/>　　台下一个身材肥胖的中年男子，忽然冲到台前，冲着叶尘高呼道。
<br/>　　很快，
<br/>　　便又有人跳了出来，
<br/>　　“我出两万上品灵石！”
<br/>　　“我出五万！”
<br/>　　“我出八万！”
<br/>　　……
<br/>　　一转眼的功夫，叶尘炼制的这颗筑基丹，已经被哄抬到了十几万。
<br/>　　就连叶尘一时间，也有些愕然，
<br/>　　不过想想也并不奇怪！
<br/>　　和之前那枚完美品阶的一品洗髓丹不同，这枚完美品阶的筑基丹，实际用途可就要大多了！
<br/>　　尤其是云丹城里的这些炼丹师，本身修炼天赋都不怎么高，很多人穷其一生，也不过只能堪堪达到金丹境罢了。
<br/>　　甚至在场的有不少人，修为还一直困在炼气境巅峰，始终不得突破。
<br/>　　修为不得突破，炼丹之术的提升，也同样会大大受限。
<br/>　　而眼前这枚完美级的筑基丹，据说可以将突破筑基境的概率，提升到九成以上！
<br/>　　这让他们如何不眼热？
<br/>　　即便是他们这些人用不到，将来留给他们的子女，也能帮助他们早日突破筑基境，将来赢在起跑线上。
<br/>　　然而，
<br/>　　面对众人的高价疯抢，叶尘十分干脆的拒绝了。
<br/>　　虽然，根据丹师考核的惯例，在考核中炼制的丹药，全都归考生所有，不过叶尘并不想把这枚丹药卖出去，他打算将这枚丹药，还有之前那枚洗髓丹，全都留给灵溪小丫头使用。
<br/>　　说实话，连叶尘自己都没有想到，自己竟然可以一连两次，炼制出完美级的丹药，连他自己都有些意外。
<br/>　　或者说，他今天的运气确实不错！
<br/>　　叶尘接过工作人员颁发给他的二品丹师宝卷，正准备走下台去。
<br/>　　没想到这时，那位云丹宗的火龙长老，突然走上前来，
<br/>　　“这位小友，请留步！”
<br/>　　叶尘转过头，在对方身上扫了一眼，隐约感觉到，对方的身上，已经有了一丝化神之力，不过还不够纯粹，应该只是半步化神而已。
<br/>　　在云丹星域，这个炼丹师云集的地方，以对方的此等修为，应该已经算是一流水平了。
<br/>　　不过，对方既然是炼丹师，战斗力应该不会太高，所以对叶尘构不成丝毫威胁。
<br/>　　“有事？”
<br/>　　叶尘一脸平淡，宠辱不惊。
<br/>　　火龙长老见状，顿时眉头一皱，
<br/>　　他身为云丹宗的十大长老之一，一般小辈见到他，恐怕早就连大气都不敢出，个个毕恭毕敬。
<br/>　　眼前这小子倒好，似乎根本没有把他当回事！
<br/>　　虽然心里有些不爽，不过火龙长老按耐了下去，脸上露出一抹微笑，道：
<br/>　　“老夫乃是云丹宗的十大长老之一，我观小友你的炼丹之术，颇有独到之处，不知道师承何方？”
<br/>　　叶尘眉头一皱，他最讨厌别人打探他的隐私，不过眼下毕竟在云丹宗的地盘上，而且他还想让灵溪拜入云丹宗，不宜节外生枝，于是淡淡道：
<br/>　　“我没有师父，我的炼丹师，全都是从一本古书上习得！”
<br/>　　火龙长老听到这话，眼眸中顿时泛起一抹精芒，不过一闪即逝，脸色的笑容也越发灿烂，笑眯眯道：
<br/>　　“既如此，那小友可愿意拜入我云丹宗门下？准确的说，是老夫想收你为徒！你意下如何？”
<br/>　　轰！
<br/>　　火龙长老这话一出口，整个大殿顿时一片轰然。
