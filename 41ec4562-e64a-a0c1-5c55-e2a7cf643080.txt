　　第75章
<br/>　　“你……怎么过来了？”苏樱声音不自觉低下来。
<br/>　　程大雷振振身上长衫，负手立在苏樱面前：“苏姑娘有事，某焉能不来，莫说小小落叶城，便是刀山火海也来了。”
<br/>　　苏樱垂下头，视线落在那口棺材上，口中慢吞吞道：“活人睡棺，不吉利的。”
<br/>　　“收拾起来。”程大雷挥挥手：“别把棺材摆在当院，给人添晦气。”
<br/>　　众人开始忙碌，从马车上搬出一坛坛酒。在大家忙碌的时候，程大雷被引进屋子大堂。
<br/>　　院子外面一阵阵恶臭传来，苏樱苦笑道：“从那天开始，他们就用粪车堵了门，我们也没有办法，只能一直关门。”
<br/>　　程大雷捏住鼻子，摆摆手道：“不妨事，先让他摆两天，到时候旧账新账一起算。”
<br/>　　“凭你带过来的几个人，怕不是杜茂的对手。”苏樱微微顿顿，满怀期待道：“你敌得过杜茂？”
<br/>　　程大雷有些尴尬，外面以讹传讹，苏樱难免也对程大雷的实力有些误解。说不定真以为程大雷懂什么五雷法，千里眼顺风耳。而事实上，程大雷站在杜茂面前，几乎等于白给。
<br/>　　“苏姑娘勿用担忧，某已有妙计。”
<br/>　　苏樱还想再问，忽然发现程大雷的脸上在挂出一抹狡黠的笑容。
<br/>　　……
<br/>　　暮秋时节，天高气爽，太阳落在人身上懒洋洋的。
<br/>　　大槐树的叶子已快落尽，随风飘到院子里，院内有肉香传来。
<br/>　　几个大汉围坐在一起，中间架着炉子，几个人正在吃肉。黑狗脱毛，放血，斩成大块，加上花椒，生姜，茱萸放在锅里一起煮，肉块在锅底随着汤汁的沸腾而翻滚。味道沁入鼻子中，下达脚底涌泉，上达天灵盖，浑身通透。
<br/>　　杜茂坐在当中，碗到酒干，脚下滚着几个酒坛，已经不知喝了多少。
<br/>　　“大哥海量，喝这么多，不妨事吧。”一个小个子道。
<br/>　　“哈，这你就是有所不知了，大哥喝一碗酒，便有一分力气，喝两碗便有两分力气，等到喝醉了，才有十分力气，当年打虎，如果没有好酒助兴，也不会那样痛快。”
<br/>　　一片嘲讽声，似乎都在奚落小个子的无知。小个子脸皮发烫，低头抿了一口酒。
<br/>　　“卖酒喽，好酒……”
<br/>　　正在这时候，院外传来两声叫卖。杜茂皱起眉头：“是谁在这里聒噪，少我们兄弟的兴，疤子你出去看看。”
<br/>　　酒已喝得半酣，疤子挺身站起来，脸上的刀疤像浸血一样红。
<br/>　　“我去看看，是谁这么不开眼！”
<br/>　　他推开门，见离门口百步外一个拐角，围着一堆人，喧哗声就是从那边传来。
<br/>　　人群中间有一辆推车，上面放着一只硕大的木桶。卖酒的是一个年青人和一个老者，两人看上去像是父子。
<br/>　　疤子走过去的时候，那老者正和一个驼子推搡。
<br/>　　“一碗酒一百文，哪里有这么贵，你这个奸商想坑我么！”驼子道。
<br/>　　“你这人怎么不讲道理，事先已经说好，一碗酒一百文，能连喝十碗者分文不取。你这人做不到，明明是想赖账。”老者苦着脸道。
<br/>　　“什么赖账，老子进城下馆子都不要钱，喝你一碗酒怎么了。”
<br/>　　“诸位，大家给评评理，我们是小本生意，一家老小都靠我卖酒养活呢，这人分明是不讲道理么。”
<br/>　　驼子满脸涨红，也不知道是喝醉了还是怎样，他举起拳头：“再敢乱说话，小心老子一拳打得你见阎王。”
<br/>　　身后忽然一只手抓住了他的手腕，驼子回头，挣了两下，感觉手腕像是被山熊捏住一般。
<br/>　　疤子走过来时已大概听明白是怎么会事，车上搁着几个酒碗，周围还有几个汉子醉醺醺的模样。
<br/>　　而这驼子很显然是没能喝到十碗，想要赖账。
<br/>　　疤子的目光看着这驼子，只看了一眼就不愿再看下去，独立翻江倒海一阵恶心。这驼子穿着粗布短打衣服，后背好大一个驼峰，下巴乱糟糟黄胡子，太阳穴贴着膏药，脸上乱糟糟的麻子。
<br/>　　人能长成这样，也真是不容易。
<br/>　　疤子一把将驼子推开，差点将那驼子推倒在地。
<br/>　　“是好汉子做了就要认，真有本事便把十碗酒喝下去，喝不下去就给钱。”
<br/>　　驼子被推了个趔趄，揉着发红的手腕，道：“你是那个，敢多管闲事，有能耐你连喝十碗。”
<br/>　　这驼子也不知哪里来的，自己从来没见过，的确，这么丑的如果以前见过，肯定过目不忘。长得这么丑，也不知道他爹妈怎么生他的，和他相比，自己简直就是英俊魁梧了。
<br/>　　“来，给我盛碗酒！”疤子。
<br/>　　老者从酒桶中舀出一碗酒递到疤子手中，感激涕零道：“多谢这位好汉，这碗酒算我送的。”
<br/>　　“什么话，我给不起你钱嘛，今天我就要连喝十碗，还要给十碗的钱。”
<br/>　　疤子端起酒碗，咕咚咚一饮而尽。入口他就知道坏了，这酒不知道怎么做的，喝得不像是酒，倒像是喝火一样。喉咙炙热滚烫，四肢百骸就像着了火一般。
<br/>　　一碗酒喝完，他就觉得眼睛迷迷糊糊，身体晃晃悠悠，手中的酒碗差点掉在地上。
<br/>　　“哈哈，怂了，大家看到没有，他也怂了。”那驼子放声大叫。
<br/>　　疤子怒从心头起，恶向胆边生，如果不是顾忌这么多人看着，他早就一刀给这驼子一个透心凉，替人间清理这腌臜货。
<br/>　　“这位好汉，如果你不能喝就算了，我这酒劲儿是大些。”
<br/>　　“什么叫不行，你才不行呢，老子行得很。”
<br/>　　话是这么说，但疤子知道自己已经够呛了。可恨那驼子在一旁嘻嘻哈哈，不停的嘲笑自己。
<br/>　　“老头，我带你去一个地方，哪里有人能喝你十大碗酒，不知道你敢不敢去。”说着，疤子指了指大槐树后面的院子。
<br/>　　大家都知道哪里住的是谁，人群中已经有胆小的散开，不敢去招惹那里住着的人。
<br/>　　老者也是害怕，他道：“好汉，我们是小本生意……”
<br/>　　“怕什么，一文钱都不会少你。”
<br/>　　“那能不能让这些人一起去，不然老汉是不敢去的。”
<br/>　　“好！也让你们见识见识什么叫真正的好汉。”
<br/>　　刀疤说了一句，见那驼子想逃，一把把他揪了过来：“就你娘的不能走。”
<br/>　　
<br/>(本章完)
