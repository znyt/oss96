　　龙脉或地灵塔出问题了？
<br/>　　巫天御脸色阴沉了下来，这怎么可能？
<br/>　　此塔存在了无数年月，可是从来没有出过问题，一旦龙脉灵气耗尽，族中祭司就会抓取新的龙脉，绝对不可能让修炼室断了灵气。
<br/>　　“天御大哥，你无需着急，地灵塔出问题了，族中长老肯定会第一时间修复的！”巫长远走到了巫天御身边，沉声说道。
<br/>　　“嗯！”
<br/>　　巫天御闻言微微颔首，随即打量了巫长远一眼，赞道：“长远，很不错，达到武帝之境了！”
<br/>　　“呵呵！与天御大哥相比，我这点实力不足挂齿！”巫长远谦虚的说道。
<br/>　　他虽然自信自己的实力，但也不敢在巫天御面前自夸。
<br/>　　……
<br/>　　修炼室中，苏莫缓缓睁开了眼眸。
<br/>　　“巫天御！”眸中杀机一闪，苏莫的眼眸微眯了下来，他一直在留意着外界的情况，对方一出现，他便立刻发现了对方。
<br/>　　不过，苏莫并没有出去，现在显然不能动手。
<br/>　　他虽然心中杀意沸腾，但不得不暂时压制下来。
<br/>　　只要在巫族，他有的是机会击杀对方。
<br/>　　“武帝境二重巅峰！”苏莫微微有些意外，因为他发现巫天御的修为，居然达到了武帝境二重巅峰。
<br/>　　对方的修为进展，居然如此之快，比他的本体都不慢多少了！
<br/>　　不过，即便对方修为再高，他也丝毫无惧。
<br/>　　在离开巫族之前，他必定斩杀此人。
<br/>　　听着外面众人的议论，苏莫并没有停止修炼，他现在正在进行功法转换。
<br/>　　此地如此浓郁的灵气，又不需要耗费太多的灵晶，他怎么可能错过！
<br/>　　至于其他人没有灵气可用，那管他什么事？
<br/>　　少倾，苏莫不再理会，再次闭上了双眼，继续修炼。
<br/>　　滚滚魔气在他身上涌动，威势森然霸道，魔威滚滚，但修炼室能完全隔绝气息，让外界的人无法查探得到。
<br/>　　此时，苏莫体内的神丹，已经变成了墨色，充满了澎湃的魔气。
<br/>　　强烈的魔道意志，不断的冲击着他的身心，他牢牢的固守心神，极力运转混天魔功，将所有灵气和玄力，都转化成了魔力。
<br/>　　这功法的确强大异常，他的玄力远远没有彻底转化，但玄力的威力已经暴涨了近两倍。
<br/>　　当然，这功法远不止玄力强大，还有很多强大之处。
<br/>　　随着修炼，苏莫吞噬战魂催动的更加大力，滚滚灵气从下方阵法中吸扯而出，如同冲销的狼烟。
<br/>　　……
<br/>　　修炼室之外，地灵塔第一层空间之中，一众巫族年轻人个个面色无奈，没有灵气修炼室就是无用之地。
<br/>　　有些人甚至已经离开了此地！
<br/>　　“玉儿，修炼室不能用了，我们也走吧！”一名高大俊朗的青年，向身边娇美的女子说道。
<br/>　　此人正是上次击杀巫尘的巫烈，其身边的娇美女子，便是巫玉儿。
<br/>　　两人刚来没多久，还没来得及进修炼室，就发生了这种无奈的状况。
<br/>　　“嗯！”巫玉儿点了点头，就欲转身离开。
<br/>　　修炼室不能用了，留在此地也是浪费时间。
<br/>　　“那个修炼室内是谁啊？没有灵气居然还不出来？”
<br/>　　就在此刻，一声惊讶的声音响起，只见一名巫族青年抬手指向苏莫所在的修炼室，满脸的疑惑之色。
<br/>　　众人闻言，纷纷循声望了过去，巫烈和巫玉儿也转头看了过去。
<br/>　　只见那座修炼室大门紧闭，不少人都是心有疑惑。
<br/>　　不过，众人也没有太过在意，没有灵气不代表就一定要出来，也许其内的人，正在用灵晶修炼。
<br/>　　不过，仔细一看之下，却是有人发现了不对劲。
<br/>　　“不对，这修炼室有古怪！”一名修为高达武帝境五重的青年，眼眸紧紧的盯着修炼室，面现惊奇之色。
<br/>　　“怎么了？”
<br/>　　“哪里不对？”
<br/>　　“有什么古怪？”
<br/>　　众人闻言，个个面带疑惑之色，纷纷开口询问。
<br/>　　“你们仔细感受一下就能知晓！”那名青年说道。
<br/>　　众人立刻静下心来，仔细感受那座修炼室，顿时心中一震。
<br/>　　他们虽然感知不到修炼室内的情况，却是能清晰的感知到，修炼室的大地之下，有浓郁至极的灵气不断的钻入修炼室之中。
<br/>　　灵气源源不断，全部向那一座修炼室中涌去。
<br/>　　所以人都是有些闷逼，一脸怔怔的模样，完全搞不清楚为什么会出现这种状况。
<br/>　　他们的修炼室内，灵气稀薄不已，可以说几乎是断了灵气，那座修炼室中为何灵气如此磅礴？
<br/>　　少倾之后，一声怒喝顿时响起，震动了整片空间。
<br/>　　“卧槽，不会是所有灵气，都被这座修炼室吸过去了吧？”
<br/>　　这一声怒喝，顿时惊醒了所有人，立刻让得众人的脸色纷纷难看了起来。
<br/>　　“这修炼室里是谁啊？”
<br/>　　“玛德，我说怎么会断了灵气，灵气分明都是跑到这座修炼室里了！”
<br/>　　“他怎么做到的？”
<br/>　　“难不成里面的人重新布置了厉害的聚灵阵？”
<br/>　　“太可恶了！”
<br/>　　嘈杂的声音此起彼伏，现场几乎炸开了锅，所有巫族之人都怒了！
<br/>　　太无耻了！
<br/>　　太自私了！
<br/>　　族中公用的修炼室，对方居然将灵气都掠夺到了自己的修炼室中，让得别人无法修炼，这简直是可恶到了极点。
<br/>　　哗啦啦！！
<br/>　　一众巫族之人，全部都围了上去，将那座修炼室围的水泄不通。
<br/>　　“里面是谁？赶紧给我滚出来！”巫天御站在最前方，面对着修炼室的大门，面色铁青，直接怒喝一声。
<br/>　　他是真的怒了，修炼室里的人，破坏了他的修炼，让得他本该突破的境界中断了。
<br/>　　这简直是让他心中怒火中烧。
<br/>　　若是地灵塔的问题，或者是大地龙脉的问题，那也就算了，没想到却是人为的原因。
<br/>　　“滚出来！”
<br/>　　“哪个杂碎，赶紧出来！”
<br/>　　“自私自利的家伙，真是可恶！”
<br/>　　不只是巫天御怒了，所有人都怒了，这种事情以前从未发生过，简直是太让人气愤了。
<br/>　　众人人多势众，也不怕修炼室内是什么高手！
<br/>　　毕竟，那些高阶武帝或者是武帝境四、五重以上的族人，不可能在地灵塔第一层的修炼室修炼，那些高手都在更下面几层的修炼室之中。
