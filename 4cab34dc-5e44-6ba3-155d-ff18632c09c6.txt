        &lt;h3&gt;第三卷第二十六章  试探&lt;/h3&gt;
<br/>
<br/>        陈子昂只能在家里吃一顿饭就得走，江俞轩还在洛城等他们。
<br/>
<br/>        母亲做的饭菜就是香，她已经吃得撑了，还舍不得放下碗筷，而爸爸和妈妈还在不停的给她夹菜。
<br/>
<br/>        陈子昂很珍惜和父母亲在一起的时间，她放下手机只想静静的陪他们吃顿饭，说说话，可是手机不屈不挠的响了好几次。
<br/>
<br/>        一个小孩跑过来：“姑姑，姑姑，那边有响声。”
<br/>
<br/>        小孩看见陈子昂的包放在旁边了。
<br/>
<br/>        陈子昂起身拿电话：谷强
<br/>
<br/>        “你好，谷总。”陈子昂和谷强分开后，这个是谷强第一次打电话给她。
<br/>
<br/>        “陈总，别来无恙？”
<br/>
<br/>        “一切都好。谷总怎么记得给我打电话了呢？”
<br/>
<br/>        “像陈总这么美丽的姑娘，谷某怎么能忘记呢，只是最近特别忙。”谷强嘴里的话语，
<br/>
<br/>        “我是多么想天天给你打电话啊，我是多么想天天见到你啊，我亲爱的小妹。”谷强心里的声音。
<br/>
<br/>        “谷总要劳逸结合，注意身体。”
<br/>
<br/>        “谢谢关心，陈总打算什么时候过来？”“不过来最好。”谷强心里很少矛盾，既希望妹妹过来，又希望妹妹不要过来。
<br/>
<br/>        陈子昂看了一眼年迈的父母，心里一动。
<br/>
<br/>        “我现在在老家看父母，如果能去甸城的话还需要半个月。”
<br/>
<br/>        对面停顿了一会。
<br/>
<br/>        “这样啊，那你什么时候来了提前告诉我，我好迎接你。”
<br/>
<br/>        “好的。我今天才回到老家，乡亲们很热情，在家门口摆了几桌呢，谷总要不要看看我们老家的饮食？”
<br/>
<br/>        “哈哈，肯定很好吃，如果有机会了我一定让陈总带我去品尝品尝。陈总，祝你愉快，我这里还有事情，先挂了。”对方滴水不漏，连问候她父母亲都没有。
<br/>
<br/>        陈子昂看着挂掉的电话微微有些发愣。
<br/>
<br/>        “子昂。”
<br/>
<br/>        “哥。”陈子昂回过神。
<br/>
<br/>        “快点吃饭吧，吃完我们就走。”
<br/>
<br/>        “怎么回来就要走？”
<br/>
<br/>        “洛城还有人等着呢。”
<br/>
<br/>        爸爸和妈妈相互看了一眼，便也低下头开始吃饭，他们吃得很慢。
<br/>
<br/>        “你吃快点，子昂等会要走，别磨磨唧唧的。”爸爸催妈妈。
<br/>
<br/>        “你不是也正在吃着呢嘛。”妈妈不满爸爸。
<br/>
<br/>        “爸，妈，不急，你们慢慢吃。”
<br/>
<br/>        “子昂让我慢慢吃呢，就你着急。”妈妈狠狠的瞪了爸爸一眼，但是明显的吃饭的速度变快了。
<br/>
<br/>        树欲静而风不止，子欲养而亲不待，陈子昂从上高中后，陪父母亲的日子很少，看着爸爸妈妈的头发在变白，她心里在祈祷，祈祷父母亲的身体健康，长命百岁，这是给了她生命，教育她做人的爹娘啊。
<br/>
<br/>        等西城买的那一套房子好了，将父母亲接过去住，二哥和二嫂在西城，如今七哥也在西城，让爸爸妈妈也住在西城，这样都有个照顾。她回西城了也能和爸爸妈妈住一起。
<br/>
<br/>        陈子昂虽然奋斗了这么几年，她赚的钱都投资了，做生意的主要靠资金的流通，其实看着这几年的事业蒸蒸日上，但是手头上却没有多少可以动用的钱，尤其是还投资了五姐的超市，想到五姐当时那种辛苦的日子，陈子昂就想支持五姐。
<br/>
<br/>        投资五姐的超市，她立马就感觉到了资金的不流畅，后期还要研发，西城的项目资金也没有回笼，陈子昂的压力山大，买房也只能交个首付了。
<br/>
<br/>        原本她还想着把爸爸妈妈接到北京，但是北京有俩个小萌宝在，小宝一直都是个秘密的存在。目前她并不想让家人有太多的想法，虽然纸是包不住火的。
<br/>
<br/>        九月的山里已经有些冷了，人们都穿上了外套，子昂还穿着裙子，妈妈给子昂找了一件她上学时候的外套，一件已经洗得发白了的，后面开叉的蓝色西服。
<br/>
<br/>        这个是六哥原来穿的衣服，六哥走了七哥穿，七哥又留给子昂穿，子昂经常穿着这件西服去上学，惹得同学们经常笑话她。
<br/>
<br/>        陈子昂看见这件衣服，眼前又晃过谷强的身影。
<br/>
<br/>        “妈，这件衣服还在啊？”
<br/>
<br/>        “这个衣服是你六哥的呢，你六哥走了你七哥穿，后来你又穿，我一直舍不得扔掉。”妈妈叹了口气，他们老六一直杳无音信，也不知道还活着没。
<br/>
<br/>        陈子昂拿过衣服，轻轻的披在身上，她一定要去甸城看一看。
<br/>
<br/>        陈子昂和陈子霖告别了父母又踏上征程，前路漫漫，有家人的祝福和陪伴也很不错。陈子昂看了一眼开车的陈子霖，他们父母亲生下了八个孩子，前面四个长得像爸爸，后面四个长得像妈妈，六哥和七哥都是人中俊杰，站在人群中都是闪闪发光的人，唯一的一点缺憾就是没有读多少书，但是俩个人脑子都很聪明。
<br/>
<br/>        爸爸妈妈恩爱一辈子，也给孩子们做了一个榜样，几个大的夫妻感情都不错，七嫂虽然是定的娃娃亲，经过磨合，七哥和七嫂的感情也不错了。
<br/>
<br/>        现在兄弟姐妹们几个就剩她孤家寡人一个了。
<br/>
<br/>        其实，她现在的状况也不错，俩个孩子很乖巧，只是缺父爱。陈子昂心里还是有些难受，那两个小家伙以前经常问她要爸爸，后来压根就不提了。
<br/>
<br/>        团团已经是个小男子汉了，将妹妹嘟嘟照顾得很好，春姨和财叔也一直照顾尽心尽力的照顾着俩个孩子，只是，落雪结婚之后就再也不和她联系了。
<br/>
<br/>        她心里充满了愧疚，是她成了落雪的软肋。
<br/>
<br/>        窗外的风景很美，青山玉水，蓝天白云，群山环绕、公路蜿蜒曲折、穿过山腹有一处雾气蒙蒙，感觉置身在云山雾海，陈子昂似乎从来没有发现家乡竟然这么美。
<br/>
<br/>        陈子昂收回思绪，人生路上，谁也不知道后面的路是什么样子的，纵使计划得再好，也难免有意外出现，只是活在当下，努力为未来奋斗，便不辜负这一生吧。
<br/>
<br/>        “子昂，你不休息会？”
<br/>
<br/>        “我不累，哥，你累不累？累了我来开车？”
<br/>
<br/>        “我不累，这才多少路程，前几年我跑车的时候，一天上千里的路程。为了赶时间，可以一天不吃饭。”
<br/>
<br/>        “真辛苦。”
<br/>
<br/>        “年轻人啊，不辛苦怎么成？不过现在好了。”
<br/>
<br/>        “是啊，我们现在都好了。”
<br/>
<br/>        “其实，子昂啊，现在日子过好了，全是政策好。要不然就我们这一帮农村的人，哪有出路啊。”
<br/>
<br/>        “绿水青山秋意浓，暖风吹过万花踪，国富带动民心齐，小康路上秀种种！”陈子昂随口溜了一首诗。
<br/>
<br/>        “子昂，为啥是秀种种？”
<br/>
<br/>        “各行各业啊，你看看现在各行各业是不是都在齐头并进？百花齐放？”
<br/>
<br/>        “确实，还是你有才。”陈子霖没有念多少书，现在想想还是很遗憾的，妹妹说的有时候他都不懂。
<br/>
<br/>        陈子昂并没有接话，她的思绪已经飞回北京了，她也不知道为什么她经常会跳跃性的断电，这一会她在想俩个小宝宝，想----张函！
