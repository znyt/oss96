　　之前因为没钱，再加上自己身体虚，一般都是喝粥，今天总算是尝到了翠云的手艺，叶少阳吃得很开心。
<br/>　　翠云还烫了一壶酒，亲自给他倒酒，自己坐在一边，看他又吃又喝，在叶少阳强烈要求下，她才跟着一起吃。
<br/>　　“我给你买了几套衣服和鞋子，装在包袱里了，路上远，肯定用的上的。还有一些药水，我从镇上买的，说是西洋药，能管跌打损伤的，你一路上不免磕磕碰碰的，没准用的上。”
<br/>　　“还有一些干粮，肉干什么的，应该是够你一路吃的了，记得住店的时候吃点热饭，别老吃干粮……”
<br/>　　翠云絮絮叨叨地说着，叶少阳仔细听着，不住点头，心里头暖暖的。
<br/>　　虽然只相处了短短几天，不过他能感觉到，翠云是个善良淳朴的姑娘，而且对自己很好，自己能有幸被她救下，并且相处了这几天，其实也是一种缘分。
<br/>　　叶少阳瞅着她说话的空隙，望着她，说道：“你有什么打算？”
<br/>　　翠云耸了一下肩膀，道：“我一个乡下寡妇能有什么打算，就是这样过到老了。”
<br/>　　叶少阳道：“不考虑嫁人什么的？”
<br/>　　翠云摇摇头，“镇上人迷信，说我克死了丈夫，没人敢来说亲，我也没那心思啦，好在我也没有公婆，也没孩子，自己一个人过挺好的。”
<br/>　　叶少阳道：“是啊，你没有公婆和孩子，又这么年轻，为什么搬走，再找个人嫁了？”
<br/>　　翠云道：“傻子啊，我虽然是寡妇，但也是嫁了的，娘家肯定是回不去的，去外地更不用说了，我在这有夫家的几亩地种，能养活自己，去了外地靠啥子生活啊。再说我在这里住久了，镇上人都熟悉，有啥子也能互相帮帮忙，自己也习惯在这。”
<br/>　　叶少阳点点头，道：“反正开心就好。”
<br/>　　翠云道：“你呢，你家里是不是给你订过亲？”
<br/>　　一句话勾起了叶少阳的心事，说道：“家里没定亲，不过我有女朋友。”
<br/>　　“女朋友？”翠云对这个词不太理解。
<br/>　　“未婚妻吧。其实基本上都快办婚事了，然后出了变故，她……她去了很远的地方。”
<br/>　　翠云歪头看着他，说道：“那你为啥不去找她？”
<br/>　　“找啊，我会去找的。”
<br/>　　翠云恍然想到什么，说道：“那你这次去江西，是不是去找她啊。”
<br/>　　“是找她，不过她不在江西，怎么说呢……我是去寻找她的线索。”
<br/>　　说这句话的时候，他脑海中浮起的是小九和芮冷玉两个人的面孔，不过话中的“她”并不是针对谁，不管是救小九，还是救冷玉，自己都要先找到徐福，回到自己的世界再说。
<br/>　　翠云道：“你未婚长的一定很漂亮吧。”
<br/>　　“很漂亮，个子很高……”突然想到什么，把钱包拿出来，拿出自己和芮冷玉的合影给她看。
<br/>　　翠云并没有看照片上的人，而是碰着照片发起呆来。“这是……画上去的？”
<br/>　　“这是照片，照出来的。”叶少阳想到这个时代可能还没有照相机，正不知道该怎么跟她解释。翠云自己先吃惊地叫起来：“照片啊，这就是照片！”
<br/>　　“呃，你知道？”
<br/>　　“我听我们镇上从南方回来的人说过，说上海那边有专门给人照相的地方，叫……”
<br/>　　“照相馆。”
<br/>　　“对对，照相馆，说是有一个机器，对着你拍一下，就可以把你样子自己画出来了，跟真人一模一样呢！”
<br/>　　叶少阳笑笑，想解释那是照相技术，不是画，不过没有解释。
<br/>　　翠云捧着照片，翻来覆去地看，“确实好漂亮啊，不过你们穿的好奇怪啊。这都什么时代了，你们怎么还穿着这么古代的衣服啊。”
<br/>　　照片上是汉服，芮冷玉从网上订的，完全还原的盛装汉服。
<br/>　　民国当然早就不穿这种样式的衣服了，在自己那个时代，穿这种复古的衣服是一种新潮，在民国人的眼中，却变成真正的古典了。叶少阳也是无语的。
<br/>　　“这照片是在上海拍的吗？”不等他回答，翠云笑着看了他一眼，接着说道，“我就知道的，你不是一般的我们这种人，你果然是去过上海的。”
<br/>　　然后开始跟他打听上海的样子，叶少阳哪里知道这时候的上海是什么样子，只要把在电视和书上看到的那些跟她随口说了一些，留声机夜总会黄包车什么的，翠云也没去过上海，听的一愣一愣的，充满了向往。
<br/>　　吃完饭，叶少阳要帮翠云收拾碗筷，被翠云严厉拒绝，在当地，家里的男人是绝对不会做这种活的，客人那就更不行了。叶少阳只好作罢，回房间去继续吐纳。
<br/>　　从前在人间，他因为用的是天下独一份的大周天吐纳心法，一天只需要吐纳两个周天，但是丹田破碎之后，叶少阳开始疯狂吐纳，希望能够恢复得快一点。虽然收效并不太大，这几天下来，也只是把碎裂的丹田修复起来，至于恢复法力，他自己判断了一下，以自己目前的恢复速度，想要恢复全盛时期的法力，至少也要几个月的时间。
<br/>　　想到这个，他就忍不住有点心烦，也只有安慰自己慢慢来了。
<br/>　　翠云洗刷完毕，又去院子里喂了猪和鸡，回到自己房间躺下了。
<br/>　　这个时代本来就没什么娱乐，女人的娱乐方式更少，天黑之后，点灯还费油，一般都是早早上床躺着了。
<br/>　　这个男人明天就要走了……
<br/>　　翠云躺在床上，想到这件事，心中竟然有着强烈的不舍和难受。她自己也觉得很是意外，毕竟才相处短短几天。
<br/>　　翠云对自己这种表现，感到有点不好意思。当初只是想着救人一命，并没有想到别的，甚至还有些犹豫会不会因为收留他而坏了自己的名声。
<br/>　　她守寡几年，一个人生活，虽然有时候不免有些孤独，但也算是心如止水，并没有想过太多，但是多了叶少阳的这几天，她内心……的确是开心的。
<br/>　　她仔细想了下，自己对这个突然闯入自己生活的陌生人，喜欢肯定是有的，不过更多的……好像是一种类似于倾慕的心理。
