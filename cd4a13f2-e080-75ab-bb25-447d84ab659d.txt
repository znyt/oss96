　　至于蛇和乌龟、鲛人这些，灵气更甚，是可以修成正神的，比这五种要厉害的多，不在此列。
<br/>　　这五种水族生灵，一旦修成人形，很容易成为一方祸害，因此阴司立下了五通神的神位，只要这五种精怪修为达到一定程度，都可以由人间法师上表，册封为五通神，位列仙班，镇守一方。
<br/>　　虽然这五通神地位不高，但是好歹也是神，人间法师是没资格把他们怎么样的，历史上也有很多五通神犯戒，被罢免或斩杀的，不过那都要先上表于天——其实就是阴司，由阴司的正神拟化成人来执行，人间法师，只有商标参他们的资格，没资格跟他们打。
<br/>　　“你看这个五通神，是五通中的哪一通？”叶少阳问。
<br/>　　毛小方想了一会，摇了摇头，道：“这个不重要，我告诉你一件事，我之前说是方便，其实是偷看那所房子——就是陈老汉住的隔壁，我们走的时候不是听见里头有声音吗，之后我特意去看了一下，侧面有个窗户，你猜我透过窗缝看到什么了？”
<br/>　　“别卖关子。”叶少阳盯着他。
<br/>　　“一个女子。”
<br/>　　叶少阳白了他一眼，“你可真八卦，那可能是陈老汉的老伴，有什么好奇的。”
<br/>　　毛小方道：“陈老汉说他没娶妻。”
<br/>　　“那就是相好呗，这也正常。”
<br/>　　毛小方摇摇头，说道：“那姑娘看着才二十多岁，长的挺漂亮，穿着红衣服，坐在床上对着一面铜镜梳头，面容看上去很是愁苦。”
<br/>　　叶少阳一听就愣住了，眼前浮现出陈老汉那张满是皱纹的老脸，而且他一个庙祝，也不可能多有钱，无论如何，好像都不应该有女人看上他，尤其是年轻漂亮的女人。
<br/>　　想到这里，叶少阳抓住毛小方的手说道：“你实话说，那姑娘真的是年轻漂亮，你眼神……没什么问题吧？”
<br/>　　“我眼神能有啥问题，我看姑娘那眼神可是准的很。”
<br/>　　“是是，我知道你是老手。”叶少阳坏坏地笑起来。
<br/>　　“什么！”毛小方脸一红，“没吃过猪肉，还没看过猪走啊。那确实是一个年轻漂亮的姑娘嘛。”
<br/>　　叶少阳道：“那么问题就来了，你说那妹子愁容不展？难不成……这陈老汉金屋藏娇，强行绑了一个姑娘在房间里？”
<br/>　　毛小方急忙摇头，“那不能，姑娘自己在梳头，行动自由，嘴巴也没被堵上，那山上人来人往的，她如果是被迫的，完全可以大喊大叫，找人救出自己，而且陈老汉也不可能有这个胆子，把人关在庙里。”
<br/>　　叶少阳道：“那你说这姑娘是自愿的？”
<br/>　　毛小方白了他一眼，“你为啥总把这姑娘跟陈老汉扯在一起？”
<br/>　　“不然呢，两人住在隔壁，那庙里又只有陈老汉一个人，跟他没关系跟谁有关系？”
<br/>　　毛小方也说不出话来，沉吟道：“那你说怎么办？”
<br/>　　“我有办法，晚上的时候，我去偷看一下，就什么都知道了。实在不行就直接问问那姑娘。”
<br/>　　毛小方觉得这也是个办法。
<br/>　　“不过，那五通神应该是在庙里，我感觉得到。你直接过去的话，可能会打草惊蛇。”
<br/>　　“我也感觉到了。”叶少阳道，“不过我有办法。对了，你刚才站在山顶，看对面的蛤蟆谷，想到什么没有？”
<br/>　　“蛤蟆谷是个龙脉，山谷为龙颈，那些溪流就是龙须，这座五通神的山神庙座落的位置，正好就是压住了龙头，这暗合了堪舆术中的某种法术，嗯……风水方面我是最差，叫不上这法术的名字，你懂我的意思吧？”
<br/>　　毛小方点头说道：“这叫藏龙术，是地师用的法术，其中的机关要理我也不懂，但是原理我明白的。只是……凭一个五通神，能镇压得住蛤蟆谷里的邪物？我对五通神不太了解，实力如何？”
<br/>　　“我听我家军师说过，大明朝靖难之前，有五通神闹事，因为是神祇，法术界没法把他怎么样，最后还是请到了下凡的文曲星方孝孺，斩杀过一尊五通神，具体实力不知道，不过既然有资格被封神的，想来也绝不是一般的邪物所能相比的。”
<br/>　　“是了。但愿不要得罪这尊山神。”毛小方耸了耸肩说道。
<br/>　　两人在山下找了个地方呆着，吃了干粮，在叶少阳撺掇下，又去蛤蟆谷里走了一趟，因为是大白天，除了阴气弥漫之外，没有什么情况发生。
<br/>　　在山谷的中间，叶少阳注意到，山谷中有一处山体滑落，露出了一个黑乎乎的洞穴，被一丛乱草掩盖，昨天来的时候并没有发现。
<br/>　　洞口阴风阵阵，两人想到昨晚上遇到那只僵尸怪，怀疑就是从这里钻出来的，不敢随便进去，干脆先离开，回到山神庙所在的山下，讨论这个洞***小方的猜测是，那洞穴里可能就是阴巢尸穴的所在，因为有邪物在里面修炼，逐渐成了祸害，至于这山神庙，估计是有道门天师供迎五通神在这里镇压，多少年来相安无事。
<br/>　　叶少阳听完他说的话，也不表态，从兜里摸出来一个东西，是一块青砖。
<br/>　　“咦，你怎么随身挟带这东西，多重啊。”毛小方感觉很好奇。
<br/>　　“这是我在那山洞附近捡的，我怕有邪物盯着，当时不方便跟你说。你看看这是什么东西。”
<br/>　　毛小方把青砖接过来，看了一下说道：“这就是盖房子的青砖吧，有什么不对？”猛然想到什么，“不对！这荒郊野外的，怎么会有青砖，那这是……”
<br/>　　两个人的目光对上了，叶少阳点点头，吐出两个字：“墓砖！”
<br/>　　毛小方愣了一会说道：“啊，那山洞附近山体剥落，山洞因此才显露出来，这么说，那山里有墓？对对对，我之前还纳闷那僵尸哪里来的，敢情是古墓里爬出来的！”
<br/>　　叶少阳道：“山葬？”
<br/>　　“山葬！那里是龙脉，想埋在那里，必然要掘山而葬，不过工程浩大，那墓主人绝对不是一般人。”
