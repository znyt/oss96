<br/>				陈默不知道谁在打自己的主意，但是有了警醒的心思。这次有了德林叔的提醒，那么也就有了心里准备，无论是谁来，他都会让其牙口崩碎！<br/>　　<br/>　　转身也去菜地和工地转了一圈，有看看工棚怎么样了。由于前天才将地坪打好，所以里面的水泥还要养护几天，就是水泥上盖个稻草什么的，然后再洒水，不让水泥外层和内层干湿不一样，从而造成干裂。<br/>　　<br/>　　转了一圈后，正要回去，没有想到的是几辆牲口运输车开了过来。<br/>　　<br/>　　等开到葫芦谷附近，大海和村里的姚爷就下了车。<br/>　　<br/>　　“大海，姚爷！你们怎么跟车来了？”陈默有些疑问道。<br/>　　<br/>　　“这不是考虑路途长，万一牲口有问题怎么办？”姚爷说道。<br/>　　<br/>　　“啊!姚爷，这样不是太累了么？”陈默有些不好意思的说道。<br/>　　<br/>　　“没有什么，我还没有七老八十的，还能动！”姚爷虽然是老把式了，但是年岁也就不到六十。<br/>　　<br/>　　“大海，辛苦了！你先带姚爷去休息一下，这里我来招呼着！”陈默笑着说道。<br/>　　<br/>　　“不用，先前在路上都吃过了，没有什么，还是将牲口先招呼着卸下里，车等着走呢！”大海说道。<br/>　　<br/>　　“那行，你看着先招呼，牲口棚我都让人清理过的。”陈默看见大海要先干活，也就没有多劝，都是一起长大的小伙伴，没有什么好矫情的。<br/>　　<br/>　　陈默转身，将陈金贵叫过来。<br/>　　<br/>　　“金贵叔，你去安排厨师弄些菊~花茶过来，让所有的人都喝点，另外让丁兰花将我上次放在办公室公共的香烟拿出来几条，给每个司机一包，然后工人和其他的你看着给，另外就是姚爷，你也给一条。”<br/>　　<br/>　　有时候，这个找人干活，就要招呼人吃饭喝水、抽烟！这在农村是必不可少的。当然，你要是不拿这些出来，谁也说不上什么，可是保不准有人在身后议论你。<br/>　　<br/>　　所以为了让人说点好话，另外就是让别人感觉你还不错，那么就要做好这些。<br/>　　<br/>　　其实这些真的花不了多少钱，香烟什么的都是比较便宜的，但是也没有人会计较。简单的说，就是看心意。大家赚钱都是赚个顺心钱，要是心意人情不到位，可能你雇人干活就要花大价钱。<br/>　　<br/>　　村里的人，要是你人情到位，就算是工钱低一点，也是可以认可的。谁没有个高峰低谷的，有钱就多给，没钱就少给！只要会做人，人情到位就成。<br/>　　<br/>　　所以陈默就买了几箱香烟放在了丁兰花哪里，要是有需要就从她哪里拿就成。还有瓶装水等等，都买了好多。<br/>　　<br/>　　他安排好以后，也就没有回去，看着众人卸车。<br/>　　<br/>　　这里总共有四辆车，都拉的是牛犊和猪苗。由于长途运输的原因，牛犊和猪苗都有些蔫了。陈默就悄悄走到了牲口棚，在水槽中释放了一些溪水。<br/>　　<br/>　　动物可能对灵气非常的敏感，等陈默放入溪水后，牛犊和小猪仔都相继跑到各自的水槽中，痛快的喝起水来。<br/>　　<br/>　　后面一些刚下出，却闻到溪水喂到的牛犊和猪仔，顿时有些骚乱，赶紧超前挤着要去喝水。<br/>　　<br/>　　于是众人都有些傻，牛犊们非常自觉的下车，然后~进到牛棚中开始喝水，猪仔也是一样，卸车的人都不用插手，所有的牲口不一会就装卸完毕了。<br/>　　<br/>　　这次卸车也是最休闲和快速的一次卸车了，往常都是很多牲口不走路，或者害怕出去，所以每一次卸车都是累死累活的，赶牲口也是非常累人的。<br/>　　<br/>　　这个时候，陈金贵也提了个大茶壶，里面是泡的菊~花茶，给每个人都倒了一碗，让大家解渴！<br/>　　<br/>　　陈默正好上前找到姚爷，这是村里陈默他们这一辈年轻人的叫法，也就是这个人姓姚，但是叫着叫着，都不知道他的本名了。<br/>　　<br/>　　“姚爷！这一趟辛苦了！”陈默先感谢了一下。<br/>　　<br/>　　“没有的事情，辛不辛苦也就那样！年轻的时候，比这累的活也不是没有做过！”姚爷端起大碗，一口气将一碗的菊~花茶喝完，陈金贵趁势又给倒满。<br/>　　<br/>　　姚爷看着陈金贵和旁边的陈默，心中暗自点头，对已二娃这个小家伙会做人，很是高兴。只要在生活中被人尊重，他的内心就会非常的舒服和高兴。<br/>　　<br/>　　“姚爷，我看了看这些牲口，牙口都非常的好，看来你还真是宝刀未老啊！”陈默小小的拍了个马屁。<br/>　　<br/>　　“哈哈！那是，这附近十几个村子，按个有我看牲口的把式好？绝对没有，你找我算是找着了！”姚爷在这个上面，却没有什么谦虚劲，还尽都是自夸！<br/>　　<br/>　　陈默也就笑笑，但是不可置否。他虽然相信姚爷挑牲口等的能力比较好，但是也保不准有那个人有比姚爷更好的能力。<br/>　　<br/>　　“姚爷，你看我这牲口棚怎么样？”陈默为了后面的打算，也就指点起自己的牲口棚。<br/>　　<br/>　　“嗯！还不错，用心了！”姚爷笑着点头说道。<br/>　　<br/>　　“那姚爷，你看你现在也是闲着，每天也没有什么事情，是不是每天到这里转转？”陈默说道。<br/>　　<br/>　　“哦？你想雇我喂养牲口？”<br/>　　<br/>　　“是！”陈默点头说道：“主要是没有个熟悉牲口养殖的人，所以想让你老过来帮帮忙！”<br/>　　<br/>　　姚爷也没有多说，就那么蹲在地上抽着烟，然后看着一众卸车的人和陈默雇佣的工人。<br/>　　<br/>　　半天后，姚爷问道：“你准备给我多少钱工资？”<br/>　　<br/>　　“嗯～！工资基本每月３０００元，你看怎么样？”这是陈默早就想好的。<br/>　　<br/>　　“行！”姚爷虽然有儿有女的，但是孩子们都在外地打工，所以自己如果能赚一些也是好的。另外就是陈默开的工资也不少，所以也就同意了。<br/>　　<br/>　　“那成，姚爷，我这里的这些牛犊和猪仔就都靠你了，那个马有为几个人都在这里负责照顾牲口，有什么事情你就安排他们。大海是畜牧主管，你要是有什么事情就和他商量！”陈默交代了一番。<br/>　　<br/>　　“行，没有问题，大海这个小子不错！”姚爷也是几十岁的人了，陈默说什么话，他心里也有底。<br/>　　<br/>　　陈默有转身，将大海拉到了一边。<br/>　　<br/>　　“大海，我将姚爷招呼过来，让他负责照看牛犊和猪仔。”<br/>　　<br/>　　“哈！没有想到你还挺快的，我还想找你说说这个事情呢！这次去挑牛犊和猪仔，要不是姚爷，我绝对会被那帮卖牲口的人给骗了！”大海说道。<br/>　　<br/>　　“哦？怎么回事？”<br/>　　<br/>　　“还能怎么？以次充好呗！牛犊中夹杂了一些杂交的小牛犊，还有猪仔里也混了一些黑猪冒充藏香猪，要不是姚爷一眼看出来，都他么的装车运回来了！”<br/>　　<br/>　　大海想起这个事情，就有些生气，要不是自己实在是需要货源，都有放弃的打算。<br/>　　<br/>　　“那卖家怎么说？”<br/>　　<br/>　　“还能怎么说，赔礼道歉，将所有货源的换回来了，然后送了两只小牛犊算是赔礼道歉！”<br/>　　<br/>　　“哈！那还不错！不过这事你要记着。这样，你等会去丁兰花哪里，领出来四千元，你和姚爷每人两千，算是辛苦费。”<br/>　　<br/>　　“我就算了，就给姚爷就成，我差点把事情办砸了！”<br/>　　<br/>　　“没事，只要结果好就成，你记得去领啊，我等下就去说一声。”<br/>　　<br/>　　“那行！”既然自己的兄弟给，那么拿着就是了。以后给二娃好好干活就成，因此也没有什么矫情，推了一次没有推掉，也就接受。<br/>　　<br/>　　陈默去了一趟办公室，现在靠着菜地这一侧的小二层楼，已经盖好，所有的人都在这里办公，食堂也在一层，还有储藏室等等。另外就是陈金贵，在这里住了一间房子，算是值班室。<br/>　　<br/>　　这里大门口位置挂了一个牌子，叫做绿源有机农场，这是他申请办理的公司牌照，主要是方便业务往来。<br/>　　<br/>　　嗯！公司的名称就是这个绿源有机农场，名字非常普通，也就表达出来陈默的起名能力。<br/>　　<br/>　　见到丁兰花以后，交代了一些事情，陈默才走回去，中午饭的时间到了。<br/>　　<br/>　　：。：<br/><br/><br/>Ps:书友们，我是小学嗣业，推荐一款免费小说App，支持小说下载、听书、零广告、多种阅读模式。书友们快关注起来吧！<br/><br/><br/>
