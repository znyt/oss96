　　岐山十绝阵前，阴风悲嚎，杀机四伏，黑云遮天，血光涌现。
<br/>　　燃灯见十天君入阵，也只好无奈说道：“善哉，贫道既然来了，便要入阵一探究竟。”
<br/>　　说着，他取出一盏琉璃宝灯，悬于头顶，泥丸宫中青莲七瓣，舒展开来，七色之光闪耀，将他周身笼罩，他便踏步入了阵中。
<br/>　　燃灯是阐教三大准圣之一，更是有幸听过一次道祖讲道之人，而十绝天君，却都是金仙之辈，就算依靠周天十绝大阵，他也不曾害怕。
<br/>　　阵内昏暗一片，黄雾蒙蒙，分不清东南西北方向，没有光色，无边无际，就算有慧眼，也看不清对方十天君身处何处。燃灯正看着，忽然听到一道声响，阵内景色忽变，平地之上血海滔滔，波涛如山，直涌上来，又有阴风吹来，肆掠狂放，冰花如箭。
<br/>　　一时间晴空万里，骄阳似火，一时间有乌云密布，雷霆炸响。一时间刀山火海，冰川遍地，一时间风雷四起，黄沙滚滚。悲风飒飒，冷雾飘飘，天空中满是鬼哭狼嚎之声，令人心惊胆颤。
<br/>　　燃灯头顶一片青莲花瓣落下，坠入血海之中，瞬间便化为乌有。燃灯修炼至今，早已堪破色空，超出三界，不入五行，入火不焚，入水不溺。他手中柄杖一指，大声喝道：
<br/>　　“三千大千，尽皆虚幻，世间万法，无非心造！”
<br/>　　左手拇指与食指一捻，一点火光于指尖燃起，火光飞出，落入琉璃灯上。此灯为无尽意，无尽光，无量光，自虚空中来，与燃灯共生，能照破幽冥，堪破邪魔！
<br/>　　此灯一亮，燃灯周边十丈之内景色皆散，一时不能迫害燃灯。
<br/>　　燃灯头顶琉璃宝灯，光耀四方，随着阵中一走，瞬间便将大阵看得一个精光，之后不急不缓，走出大阵。
<br/>　　“今日我入阵一游，以观虚实，如今已然知晓其中阵势，明日再会，必破此阵！”燃灯遥遥说道。
<br/>　　“哈哈，休要大言不惭，我等且看你明日如何破阵？”十天君哈哈哈大笑。
<br/>　　回到周营，燃灯将十绝阵的情况与众仙说明，十绝阵的凶残让众仙心中担忧不已。
<br/>　　“这该如何是好？”
<br/>　　燃灯道：“诸位不必担忧，此十阵连为一体，相互辅助，缺一不可，我等明日进阵，全力攻之，只需破了一阵，气机外漏，十阵不可连贯，各自为政，到时分而破之即可。”
<br/>　　“如此甚好！”
<br/>　　翌日，众仙再次临阵，杨戬等三代弟子并未进阵，只有燃灯、赤精子，金翅大鹏、黄金道人四人前往。其余众仙都手持兵器，护在大阵之外，只待大阵一破，便群起攻之。
<br/>　　众人进得阵内，只见阵内别有乾坤，黄沙漫天，四下相望，并不见十天君身影，众人暗暗提防。虽然众人看不见十天君，但十天君却能看见众仙，他们各自进阵，在云板上一拍，手中主幡摇动，将阵法发动。
<br/>　　顿时，哭喊之声四面响起，带着种种悲怆、苍凉、绝望、愤恨、痛楚、酸、甜、苦、辣等主办感情，从空中覆盖下来。燃灯等人都是得道真仙，早已炼就心如磐石之境，这些干扰，自然不可动摇其心。但若是换一个三代弟子来，便会被这些情感乱了道心，直至身死道消。
<br/>　　落魄阵不行，主幡一摇，阵势一变，平地之上寒气直冒，雾气凝结成冰，从四面席卷而来将众人包围，在与众仙护体真气相触之时，又化为熊熊真火，随即又化为滔天血海，上下翻滚，一条条血蟒自虚空之中钻出，口吐红沙黑雾，向着众仙杀来。
<br/>　　众仙周边的灯火俱皆熄灭，幸亏有燃灯的无尽琉璃灯高悬上方，火焰相护，自然没有出现意外。
<br/>　　突然虚空无端出现一条裂缝，静静扩大，赤精子一个不慎，落了下去，那裂缝瞬间无声闭上。下一瞬，赤精子又出现在众人面前。
<br/>　　“道兄，你没事吧！”
<br/>　　“无事，幸好贫道有至宝护身，有精通遁地之术，在察觉到危机那一刻，便遁地而走，否则便凶多吉少了，真是大意了。”赤精子叹道。
<br/>　　“大家小心！”燃灯说了一声，之后不知从何取出一七彩宝珠，神光扩散，四人瞬间便隐形起来，众人看不见对方。
<br/>　　金光圣母见此，心道：真狡猾！如今谁也看不清谁，倒是公平！
<br/>　　主峰摇动，十天君催动大阵，只见虚空中亿万刀刃齐齐涌出，如狼牙，白森森地插满整片虚空，冰、火、雷、电、风、沙十绝齐出，阵中雷霆大作，岐山之外，黑云翻滚，如龙卷风一般，自高空垂下，探入阵中，这便是十绝阵，可借助天地之间的煞气战斗。
<br/>　　阵外杨戬等人见此，彼此都心忧不已，但却没有办法。
<br/>　　阵中四人依然没有显现出身影，十天君见此，也渐渐收手，若不收手，这反而会白白浪费法力。风吼阵主董全站起来，张口轻吐，一阵无影无形的风刮过，此封无声无息，杀人于无形，姚宾见此，放出几个鬼面骷髅，吸引阵内四人注意。
<br/>　　风吹过，金翅大鹏张口一吸，便将之全部吸入腹中，那董全借助风来感应众人，却是只能无功而返。董全急躁不安，这时金翅大鹏再张口一吐，一阵无相罡风刮过，混合着毁灭神光，顺着刚才那风迅速过了回去。
<br/>　　风吼阵中，董全正在吹风，心头一跳，忽然危险，却不知危机从何而来，身体急转，并无所得，正要坐下，突然全身一震，一道毁灭神光扫来，他全身化为飞灰。
<br/>　　金光圣母、秦完等九天君俱皆身形一颤，口吐鲜血，皆大声悲呼道：“董兄，你死的好惨！”
<br/>　　杨戬等人在阵外，突然听到一阵悲叫声，紧接着，便看到岐山上空，黑云散开，太阳之光照射下来，众人大喜，自然知道十绝阵已破，连忙出手，誓要破灭其他大阵。
<br/>　　董全一死，十绝阵已然不复十绝之名，不过依然十分厉害。艳阳高照，十绝阵的真容显现在众仙面前。
<br/>　　岐山之上，分别有十门，立于十方，每一方都是一道大阵，风吼阵已破，众人便不再去管，直接杀入其余九阵。
<br/>　　金光阵中，金灵圣母面色严肃，高坐法台之上，手持金镜，镜光一指，电光雷火四面冲来，刚刚进入阵中的萧秦还来不及闪过，便魂飞魄散了。
<br/>　　紧接着，黄金道人走进大阵之中，看见金光朝他射来，黄金道人也不闪躲，抬手也是取出一面金色镜子，那金光一来，射到镜上，由金光圣母射出的湮灭金光竟然反射了回去，金光圣母见此，大感不妙，想要闪躲，可是这时黄金道人再次出手，手中一道金色圣光打出，如同沐浴在圣光之中，一条金色大道出现，照亮了整座大阵，金光所过之处，一切都将毁灭。
<br/>　　金灵圣母还来不及闪躲开，便被金色圣光照射到，瞬间化为一座金色的雕像，静止不动，只留一点真灵飞入封神榜中。
<br/>　　金光阵，破！
<br/>　　金光圣母乃是十阵之主，如今圣母一死，其他人因此心神大动，心神一乱，便导致大阵出现失误，不过半天时间，十绝阵便全部被阐教众仙破灭，只留下五大天君还在苦苦作战。
<br/>　　至此，十绝天君的周天十绝阵被破！
