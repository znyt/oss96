　　真圣界内。
<br/>　　苏莫已经过关，只能静静的等待了起来，一边等待他一边观看其它平台上的战斗。
<br/>　　他主要是先了解一番巫族之人的实力，比如巫人王和巫殇，还有其他几名巫族天才。
<br/>　　巫人王的实力，的确不弱，但也没有出乎苏莫的预料，此人正在所在的平台上乱战，战斗的难分难解。
<br/>　　圣雨降临，他们就立刻停止战斗，抵挡或者躲避剑雨，而后继续混战。
<br/>　　只不过，圣雨的威力极为强大，每一轮圣雨落下，都能淘汰不少人。
<br/>　　至于巫殇，此人的确是恐怖，而且让苏莫有些看不错深浅，此人用掌，每一掌击出都能击败一名天才，几乎是没有一合之敌。
<br/>　　而对方所在的平台上，已经只剩下了三十余人。
<br/>　　其它平台上，还有几名巫族的天才，实力虽然不如巫殇，但貌似比巫人王还要略强三分，也是不可小觑。
<br/>　　至于其他几名真圣榜排名前列的天才，比如巨摩、飞长云、黑锋几人，战力亦是极度强大，手下无一合之敌。
<br/>　　不过，这些人也因为太强，而遭到所在平台上所有人的联合攻击，战斗的颇为吃力，几乎险象环生。
<br/>　　这些人可没有吞噬战魂，面对所有人的围攻，要承受强大的压力。
<br/>　　好在还有圣雨的攻击，随着一轮轮圣雨快速的降临，每一个平台上都有大量的人被不断的淘汰。
<br/>　　有些身受重伤的人，则是担心陨落，立刻自行退出平台。
<br/>　　随着人数的急速减少，巨摩、黑锋等人压力大减，攻击越发狂猛，很快就要将所有人击出平台。
<br/>　　至于姜神夜，此人也是极强，但或许对方没有发挥出多少实力，看起来并不比巨摩黑锋等人出众。
<br/>　　时间流逝，巨摩过关了、姜神夜过关了，巫殇过关了，很多强大的天才都一一过关了。
<br/>　　过关的人只能继续等待，因为还有众多的平台，依旧在战斗，在抵挡圣雨的攻击。
<br/>　　很多平台上的人实力比较均衡，因为没有特别强大的人，则是需要耗费大量的时间。
<br/>　　比如尤天韩所在的平台，比如巫人王所在的平台，两人的实力虽然较强，但还没有达到远超别人的地步。
<br/>　　不过，两人也没有以一己之力，抗衡所有人，而是抵挡圣雨的同时，尽量将同平台的人一个个击败。
<br/>　　八百个平台，耗费了半个时辰的时间，终于全部结束了，每个平台只有一人过关，合计八百人。
<br/>　　总体来说，每个平台上都是经过了激烈的争锋，才角逐出胜利者，只有苏莫所在的平台，是最为简单和轻松的。
<br/>　　除了真圣榜排名靠前的人之外，很多过关的人都受伤了，甚至有人身受重伤。
<br/>　　比如巫人王，全身浴血，颇为狼狈，可谓是险而又险的才能撑到最后。
<br/>　　“苏莫，你的死期要到了！”巫人王遥望苏莫，眸中杀机森然，他若不是被苏莫掠夺了天玄武体八成的血脉力量，怎么可能会战斗的这么吃力。
<br/>　　若是感觉到自己的实力减弱，他对于苏莫的恨就越深，接下来就是第三关了，每个人都有三次挑战的机会，被挑战人无法拒绝，只能应战。
<br/>　　他第一个就是要挑战苏莫，一举擒拿。
<br/>　　尤天韩也过关了，他虽然也受了点伤，但无伤大雅。
<br/>　　对于尤天韩来说，苏莫自有巫族之人对付，并不是他的目标，他的目标是真圣碑留名，得真圣之气。
<br/>　　苏莫目光扫视着所有的平台，心中暗叹，任云都被巨摩轰出了平台，整个神武学府的弟子之中，也就他和尤天韩两人过关。
<br/>　　随着第二关的结束，巨大的真圣碑再起变化，朦胧的青光散发而出。
<br/>　　剩下的八百人，个个眸光坚定，最后一关了，只要能过关，便能在真圣碑留名，得真圣之气。
<br/>　　朦胧的青光急速扩散，很快就将八百人全部笼罩。
<br/>　　“嗯？”
<br/>　　当青光笼罩在自己的身上之时，苏莫顿时双眉一挑，他有一种感觉，感觉到青光在沟通他的意思，询问他的名讳。
<br/>　　当然，这只是一种感觉，在苏莫心底生出的玄妙感觉。
<br/>　　而后，苏莫没有拒绝，意念沟通青光，说出了自己的名讳。
<br/>　　接下来，惊奇的一幕出现了，只见真圣碑上，突然出现了‘苏莫’二字。
<br/>　　“这是……这就留名了？”苏莫见此顿时心中疑惑，但很快他就明白了，并不是如此。
<br/>　　因为他的名字并没有出现在真圣碑上，只是浮现在真圣碑表面的青光之中，而且，并不是只有他一个名字。
<br/>　　只见真圣碑表面的青光之中，一个个名字不断的浮现，正是通过第二关的八百人的名字。
<br/>　　八百个名字浮现在青光之中，杂乱无章，没有任何的规律。
<br/>　　苏莫心中恍然，这应该是第三关的挑战形式。
<br/>　　随着众人的名字出现，笼罩在众人身上的青光迅速收敛，而后在真圣碑的前方，再次形成了一个青色的平台。
<br/>　　这个青色的平台，比之第二关的平台要巨大无数倍，方圆足有上千里，并且非常的凝视。
<br/>　　“第三关要开始了！”
<br/>　　众人个个深吸一口气，能否得到真圣之气，留名真圣碑，就看这一关了。
<br/>　　对于真圣界的考验，众人都比较了解，这种考验方法并没有什么意外。
<br/>　　苏莫亦是深深的吸了口气，三次挑战他心中已经有了对手，那就是巫人王、巫殇、还有另外一名强大的巫族天才。
<br/>　　他要一举擒拿三人，有这三人的性命在手上，他出了真圣界之后，方有一丝活命的可能。
<br/>　　只是，他倒是忘了问任云都，挑战三次皆胜算是过关，那别人挑战自己呢？是不是也算在三次之中？
<br/>　　还是说，别人的挑战不算在三次之中，只有自己挑战三次，才算是三次挑战？
<br/>　　嗡嗡~~
<br/>　　就在苏莫思索之时，真圣碑外的八百个名字之中，突然有一个名字青光大放，无比的耀眼。
<br/>　　这个名字是，飞长云！
<br/>　　飞长云见此，顿时身形一动，飞到了巨大的青色平台上。
<br/>　　众人见此，顿时心中凝重了起来，第一个挑战的人居然是飞长云这位真圣榜排名第三的高手。
<br/>　　很多人都暗暗祈祷了起来，祈祷自己不会被对方挑战，不然败多胜少。
<br/>
