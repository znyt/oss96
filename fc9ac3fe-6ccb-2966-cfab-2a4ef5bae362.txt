　　清晨，朝阳初升，苏莫和吕罡便离开了客栈，向六壬商会而去。
<br/>　　六壬商会举行的拍卖会，并不在商会的所在地，而是在另外一条街道。
<br/>　　这是一座巨大的石质宫殿，占地方圆千顷，乃是专门建造的拍卖会场。
<br/>　　当苏莫和吕罡来到拍卖会场门前之时，发现会场门前已经排成了长长的队伍，正在缴纳灵晶入场。
<br/>　　“每人缴纳一万块下品灵晶或者十块中品灵晶，方可进入拍卖会场！”
<br/>　　会场大门之前，伫立着十数名守卫，其中为首的一名中年大汉朗声大喝。
<br/>　　这名中年大汉身上气息极强，浩瀚无比，分明就是一名高阶武尊。
<br/>　　苏莫和吕罡两人，有序的排在了队伍之后，等待缴纳灵晶入场。
<br/>　　此时，外面的街道之上，依旧不断的有人陆续到来，虽然前方的人不断的在进入会场，队伍依旧是排的越来越长。
<br/>　　“扬天成！”
<br/>　　苏莫感知比较敏锐，突然感受到了一丝较为熟悉的气息，回头一看之下，发现扬天成居然来了！
<br/>　　此刻，扬天成和一男一女，从远处的街道上，大步向拍卖会场走来。
<br/>　　“武尊境三重、武尊境一重！”苏莫眸中精光一闪，立刻转过了脑袋。
<br/>　　扬天成和那名绿衣女子，均是武帝境一重的修为，而另一名青年，却是修为高达武帝三重的巅峰。
<br/>　　苏莫倒是不怕扬天成三人，以他现在的实力，根本无需有任何惧意，但若是被三人认出来，估计难免又会有一番冲突。
<br/>　　而且，他和吕罡在一起，很容易被认出来。
<br/>　　略一沉吟，苏莫取出了银缤面具，戴在了脸上，身上立刻气息全无。
<br/>　　不过，虽然带上了面具，对方不可能认出自己，但苏莫还是非常无奈。
<br/>　　因为等到拍卖会开始，金色大弓和三支庚金破虚箭一拍卖，到时候扬天成肯定知道他来了。
<br/>　　但是也无所谓，到时候拍卖会结束，灵晶到手，凭扬天成三人，还留不住他。
<br/>　　“这是……!”吕罡见苏莫突然戴上面具，神情一怔，不过他马上便明白了原因。
<br/>　　很快，扬天成三人便来到了会场之外，排在了队伍的后方。
<br/>　　“吕罡！”扬天成发现了吕罡，眸中露出一丝冷笑，自从他和苏莫结怨之后，对于任何道宗之人，均是不自觉的带上了一丝敌意。
<br/>　　不过，他也没有找吕罡的麻烦，毕竟两人无冤无仇。
<br/>　　至于吕罡身前的银面人，扬天成更是没有在意，毕竟出门在外，什么样的人都有。
<br/>　　苏莫和吕罡有序排队进场，除了扬天成之外，苏莫还发现了不少熟人。
<br/>　　比如黒炎族的炎澜天、御剑谷的卓剑豪，他们均不是单独一人，都是三五成群。
<br/>　　让苏莫意外的是，两三个月不见，这些人居然都达到了武尊之境。
<br/>　　“苏兄，我们好像不需要缴纳灵晶！”吕罡示意前方，小声的对苏莫说道。
<br/>　　苏莫抬眼看去，便见到有人不需要排队进场，只需出示请帖，便能直接进入拍卖会场。
<br/>　　“我们走！”苏莫点了点头，直接走出队伍，来到了会场门前。
<br/>　　而后，他出示请帖，便轻易的进入了拍卖会场之中。
<br/>　　“这人是谁？”
<br/>　　扬天成、炎澜天等人，都注意到了吕罡和一名银面人出示请帖，直接进入了会场，不由的微微好奇。
<br/>　　吕罡他们都熟悉，所以和吕罡在一起的银面人，自然也引起了他们的好奇。
<br/>　　能有请帖的人，要么是厉害的强者，要么是身份非凡之人，要么是卖主。
<br/>　　此人带着面具，不敢以真面目示人，想必不会是前两者，那就是第三者卖主。
<br/>　　不过，一般的小卖主，没有资格得到六壬商会的请帖，只有大的卖主才有资格。
<br/>　　苏莫和吕罡走进会场，穿过一条长廊，眼前豁然开朗。
<br/>　　这是一个中型大小的拍卖会场，与其它的拍卖会场没有太大的区别，中央是高达三丈有余的拍卖台，四周是一排又一排环形的坐席。
<br/>　　这种规模的拍卖会场，只能容纳十万人左右，但是完全够用了。
<br/>　　毕竟，这是高规格的拍卖会，来参加拍卖会的人，基本上都是武尊境的武者，人数不可能太多。
<br/>　　拍卖会场的上方，有环形的跃层，其上是一间间的贵宾包厢。
<br/>　　苏莫和吕罡两人，出示情帖，通过跃层的入口，来到了跃层之上。
<br/>　　请贴上标有包厢的编号，二十九，两人很快便找到了二十九号包厢。
<br/>　　包厢不大，只有方圆五、六丈左右，但是布置的相当奢华，其内地面、桌椅之上，全部铺上了一层精美的兽皮。
<br/>　　位于拍卖台的一方的墙壁，是一层透明的水幕，从包厢内可以一览整个拍卖场，而外面却无法窥探包厢内的情景。
<br/>　　“苏莫，有不少天虚宗的弟子也来到了拍卖场！”刚进入包厢，吕罡便迫不及待的说道。
<br/>　　出了扬天成和其同行的一男一女之外，吕罡还发现了不下十名天虚宗的弟子。
<br/>　　“那位和扬天成同行的青年是谁？”苏莫沉声问道，其它人他不在意，那名青年武尊境三重巅峰的修为，气息深邃无比，实力很强。
<br/>　　“那是扬天威，在天虚宗真传弟子中，实力仅次于厉恨天，排在第二名。”吕罡说道，他自幼加入道宗，对于七大势力中有名的人物，都是相当了解。
<br/>　　“嗯，不足为惧！”苏莫默然的点了点头。
<br/>　　吕罡闻言暗暗心惊，苏莫的自信之强烈，让他感觉惊讶不已。
<br/>　　不过，他并没有认为苏莫自大，因为无论是在会武中，还是在魔渊，苏莫都已经证明了他逆天的战斗力。
<br/>　　“七大势力中都有不少真传弟子来此！”
<br/>　　吕罡走到水幕之前，目光扫视下方，随即，他突然指着一名女子，惊讶的道：“苏莫你看，那是乙薇师姐！”
<br/>　　苏莫顺着吕罡手指的方向看去，便看到了一名身姿高挑的红衣女子，在一群青年的簇拥下，走进了会场之中。
<br/>　　这女子身材颇为惹火，呈完美的S形，其容颜清丽，虽不算绝美，但也算是花容月貌。
<br/>　　最主要的是，此女的修为，居然高达武尊级四重境界。
<br/>　　“不简单！”苏莫轻声自语，在道宗之内，武尊境四重的修为，已经是真传弟子中巅峰的存在了。
<br/>　　真传弟子之中，并没有更高修为的人。
<br/>　　因为真传弟子，有年龄的限制，就算修为达到了武尊境五重，但年龄超过了五十岁，也不可能成为真传弟子，而会成为宗内执事。
<br/>　　苏莫一眼便能看出，此女不足五十岁，而能达到武尊境四重的修为，天赋可想而知。
<br/>　　“当然不简单，宗内真传弟子之中，无人是乙薇师姐的对手。”吕罡点头道。
<br/>　　他的眼眸中神采奕奕，显然对于乙薇颇为仰慕。
