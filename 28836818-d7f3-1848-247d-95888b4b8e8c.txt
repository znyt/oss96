　　修炼室之中，听到巫勋长老的声音，苏莫睁开了眼眸，无奈的叹了口气。
<br/>　　族中长老来了，他不出去也不行了，已经无法再修炼下去了。
<br/>　　不过，他虽然修炼的时间不长，但有海量的灵气支撑，功法的转换，进展也是颇为快速。
<br/>　　虽然距离完全转换成功，相距甚远，但一切都还算顺利。
<br/>　　随即，他停止了修炼，将剩下阵法中还未耗尽的灵晶收取，停止了阵法的运转。
<br/>　　随即，他收敛了身上的气息，长身而起，缓缓的打开了修炼室的大门。
<br/>　　咯吱！
<br/>　　随着大门缓缓的打开，苏莫看到了一双又一双眼眸，均在死死的盯着他。
<br/>　　几万双眼眸，很多均是眸中森寒，充满着怒火，盯的他心中有些发毛。
<br/>　　有少量认识苏莫这具肉身的人，见到他的面容，不禁满脸愕然之色。
<br/>　　这坑爹的人，居然是巫尘！
<br/>　　巫长远嘴巴微张，心中愕然不已，他怎么也没想到，修炼室内的人居然会是‘巫尘’。
<br/>　　而一直没有离开的巫烈和巫玉儿，看到苏莫的面容，顿时心中一震，面现不敢置信之色。
<br/>　　巫尘？
<br/>　　居然是巫尘？
<br/>　　这怎么可能？
<br/>　　巫尘不是被他们杀了吗？怎么可能会在这里修炼？
<br/>　　两人对视一眼，均是看到了彼此心中的疑惑，随即两人的脸色立刻阴沉了下来，眸中杀机一闪而逝。
<br/>　　巫尘居然骗了他们，根本就没有死！
<br/>　　嘭！
<br/>　　修炼室的大门彻底打开了，苏莫环视一圈，目光扫过众人，最后落在了巫勋长老的身上。
<br/>　　“拜见长老！”苏莫走出修炼室，向巫勋长老抱拳一拜。
<br/>　　“巫尘？居然是你！”巫勋长老眉头一皱，他对于黑耀城中武帝境以上的年轻人，大多都认识，自然也认识苏莫这具肉身。
<br/>　　“是我，勋长老！”苏莫点了点头。
<br/>　　“到底怎么回事？为何所有的灵气全部都涌向了你的修炼室中？”巫勋沉声问道。
<br/>　　“这……晚辈也不清楚为何我这间修炼室的灵气如此浓郁！”苏莫摇了摇头，开始装傻充楞。
<br/>　　“你也不清楚？”
<br/>　　巫勋长老心中疑惑，随即走进修炼室中查看了一番，发现修炼室中的阵法并无任何异常。
<br/>　　这让巫勋长老更疑惑了，难道真的是刚才大地之下的龙脉出问题了？
<br/>　　他倒不认为是苏莫搞的鬼，因为以他武尊境的修为，都没有本事将其它修炼室的灵气，全部掠夺过来。
<br/>　　那就更遑论眼前这名只有武帝境三重的巫尘了！
<br/>　　或许真的是龙脉出问题了！
<br/>　　巫勋长老暗想。
<br/>　　“那你为何一直不出来？”冷漠的声音响起，巫天御凌厉的目光冷视着苏莫。
<br/>　　若不是有巫勋长老在此，他早已出手教训眼前的这个混蛋了！
<br/>　　“修炼室中灵气这么浓郁？我为何要出来？”苏莫冷笑一声道。
<br/>　　“你……！”巫天御顿时语塞，无法反驳。
<br/>　　的确是这样，修炼室中聚集了本该流向其它所有修炼室的灵气，灵气必定浓郁到极致，谁会愿意出来！
<br/>　　“巫尘，老子本来在冲击境界瓶颈，都让你这个滚蛋给破坏了！”
<br/>　　“你害我修炼不成，必须给予补偿！”
<br/>　　“不错，你破坏了我们的修炼，必须拿出补偿！”
<br/>　　这时，有不少巫族年轻人叫嚣了起来，个个面带怒色。
<br/>　　苏莫见此，心中冷笑，这些巫族之人真是天真的可以。
<br/>　　“我破坏了你们修炼？简直是笑话！”
<br/>　　苏莫不屑的摇了摇头，嗤笑道：“这间修炼室就在这里，你们也可以进去修炼，你们也来破坏我修炼吧！”
<br/>　　“你……真是嚣张！”
<br/>　　“破坏了我们修炼，居然还如此张狂！”
<br/>　　“简直找死！”
<br/>　　众人闻言大怒，他们本来心中就有火气，现在再听到苏莫如此言语，顿时怒火冲天。
<br/>　　不少人手中玄力涌动，都忍不住想要出手了。
<br/>　　“肃静！”
<br/>　　这时，一声厉喝震动八方，巫勋长老阴沉着脸从修炼室中走了出来。
<br/>　　众人顿时安静了下来，现场落针可闻，没有人敢违逆长老之言。
<br/>　　“都散了吧！地灵塔第一层所有修炼室暂时关闭，待老夫全部检查清楚，再行开放！”巫勋环视众人，朗声吩咐道。
<br/>　　“是！”众人应声，随即有不少人纷纷离开。
<br/>　　但有一部分人却是并没有离开，均是目光冷视着苏莫，眸中有冷笑之意。
<br/>　　这个巫尘如此的嚣张，他们怎么可能放过，无论如何今日也要找机会教训一番，方可解心头之恨。
<br/>　　反正这个巫尘在族中也没有什么背景，实力也不怎么样，教训了也就教训了。
<br/>　　“巫尘，你很狂，但没有实力还敢狂妄，就是白痴了！”巫天御冷冷的说道。
<br/>　　“你可以试试！”苏莫淡淡的说道，他目光直视对方，没有丝毫惧意，若是能和巫天御生死一战，光明正大的击杀对方，他非常乐意。
<br/>　　巫天御闻言，顿时双眸一眯，一个在族中没有任何地位的人，居然敢挑衅他，真是吃了熊心豹子胆了！
<br/>　　一瞬间，巫天御心中杀机大起。
<br/>　　只不过，巫勋长老还在旁边检查其它修炼室，他暂时不好动手。
<br/>　　巫烈和巫玉儿站在后方，两人目光对视，均是看到了对方心中的想法。
<br/>　　这个巫尘绝对不能留，不然的话，将他们两人上次之事捅出去，他们就有大祸临头了。
<br/>　　此时，苏莫得罪了所有人，正是他们借刀杀人之时。
<br/>　　“巫尘，你这个畜生，居然还敢回来？”
<br/>　　巫玉儿一声娇喝，随即分开众人，和巫烈一起大步走到了苏莫的身前。
<br/>　　这一声娇喝，让得所有人均是神情一怔，不解的看着巫玉儿。
<br/>　　很多人都认识巫玉儿，知道对方是巫尘的未婚妻，但两人如此的关系，巫玉儿为何会辱骂巫尘？
<br/>　　苏莫亦是神情一怔，随即扫了巫玉儿一眼，忍不住眉头一皱。
<br/>　　这个女人心思狠毒，联合巫烈杀死了巫尘，现在见到他回来，担心东窗事发，率先发难了吗？
<br/>　　“玉儿姑娘，这个巫尘做了什么恶事？居然让你如此愤怒？”巫天御好奇的问道。
<br/>　　巫天御的心中，已经对‘巫尘’动了杀心，但碍于族规，不能肆意出手。
<br/>　　若是巫尘真的做了什么人神共愤的事，他就有理由清理门户，当场镇杀对方。
