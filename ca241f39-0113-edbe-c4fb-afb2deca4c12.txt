夏家有三兄弟，夏老爷子是老大，当年他跟着林水瑶去了海城，帝都城这位是老二，夏邦就是老二的儿子，现在也就是夏家的大当家。
<br/>
<br/>        听夏夕绾要低调点，夏邦抬手摸了摸自己的头，憨厚又听话的呵呵笑了，“好的绾绾，都听你的，你让我向东，我绝对不向西！”
<br/>
<br/>        这一幕可把夏家那些下人看呆了，要知道夏邦可是江湖老大哥，没人不怕的，就连孙进都发憷的，但是他竟然对夏夕绾言听计从。
<br/>
<br/>        夏邦是一个妻管严，除了怕家里那位母老虎之外，夏夕绾可是第二人。
<br/>
<br/>        那些下人心里对夏夕绾越发的敬重，本就是仙子一样的人儿，夏夕绾刚来夏家时他们都惊为天人。
<br/>
<br/>        当然这也跟夏家的整体颜值有关，夏家的男人们个个高大彪悍，女人也是女汉子一个，在夏夕绾没来之前，这里没有“颜值”这两个字。
<br/>
<br/>        “绾绾！”这时一道风风火火的身影跑到了房间里，说母老虎，母老虎到，这位是夏邦的妻子蓝烟。
<br/>
<br/>        蓝烟跑到夏夕绾的面前，哗啦一声将一堆照片倒在了书桌上，拉住夏夕绾的小手就将她拉了过来，一脸兴奋道，“绾绾，你快看看，这些人中有没有你喜欢的？”
<br/>
<br/>        夏夕绾看了一眼，这照片上都是各式各样的男人，她眨着羽捷看着蓝烟，“蓝妈，这些都是谁啊？”
<br/>
<br/>        “绾绾，这些都是帝都城还没有婚配的豪门公子们，基本都在这里了，你看看你喜欢哪一个，孙进那个龟孙子我们早点将他踹了，这里的人随便你选，绾绾你看上哪一个，我就去将他掳来，我们直接拜堂成亲，让他做我们夏家的姑爷！”
<br/>
<br/>        掳…
<br/>
<br/>        又是掳…
<br/>
<br/>        夏夕绾抬头抚了一下额，这对夫妻的行事作风真的是如出一辙。
<br/>
<br/>        这时夏邦走了过来，开心的夸赞道，“这个主意好，孙进那个龟孙子还敢嫌弃我家绾绾，绾绾你就找个比他好上千倍万倍的，气死他，绾绾你看，你这是周家的公子，青年才俊，很有名气，但是瘦里吧唧的，没有男人味，换一个…”
<br/>
<br/>        “这个吧，”蓝烟拿起一张照片，跟夏邦激烈的讨论开了，“这个是钱家的公子，将门之子，现在是救援队长，你看这一身的肌肉，肯定能保护我家绾绾，就是长得不太好看，再换一个…”
<br/>
<br/>        夏夕绾看着这对夫妻，额头三根黑线，她就想问一问被cue了N遍的龟孙子孙进是怎么想的。
<br/>
<br/>        “蓝妈，”夏夕绾抱住了蓝烟的手臂，“把这些照片都收起来吧，这些人我都不喜欢。”
<br/>
<br/>        “一个都没看上？”蓝烟想了想，“帝都公子基本都在这里了，除了帝都四大豪门厉陆徐夜家的太子爷们，这些太子的确都没有婚配，除了陆家大少陆寒霆，这陆寒霆和厉家的千金厉嫣然是一对，估计很快就会订婚的，现在帝都城估计没有不知道陆厉联姻的。”
<br/>
<br/>        “绾绾，除了陆寒霆，你看上谁了，我们夏家跟这四大家一直都是井水不犯河水的，他们看不上我们，我们还嫌弃他们呢，不过要是你喜欢，我们也可以跟他们结亲，我们看上的姑爷，抢也要抢来。”
