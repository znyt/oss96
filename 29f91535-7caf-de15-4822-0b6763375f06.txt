　　看着空中圣品灵石化为的长河，无数武者心中都有些蠢蠢欲动，恨不得要直接出手抢夺。
<br/>　　这一千万圣品灵石，谁如果拿走了，简直是足以开宗立派，在九域做出一番成就了。
<br/>　　但是动心的人虽然很多，可却没有人敢真正出手。
<br/>　　归墟城的规矩森严，就连那身为战魔宗少宗主的纣绝阴，都不敢出手。
<br/>　　更何况是这些人，就算是今日将圣品灵石夺走了，只怕也根本没有可能从归墟城里离开。
<br/>　　有命拿没命花。
<br/>　　只是一道道的目光，还是充满着贪婪的盯着楚风眠。
<br/>　　楚风眠能够拿出这一千万圣品灵石，那想必楚风眠的身上的财富，比起这还要更多。
<br/>　　在归墟城里不能出手，但是一旦是离开了归墟城，可就没有这个规矩了。
<br/>　　时间一分一秒过去。
<br/>　　那老者也是足足过了五分钟的时间，才将所有的灵石，清点完。
<br/>　　“这剑柄归你了。”
<br/>　　老者手一挥，将那剑柄对着楚风眠扔了过来。
<br/>　　随即便是转过头去，开始收拾摊位，准备离开。
<br/>　　“这老家伙要走。”
<br/>　　看到老者的举动，一名名武者的眼神之中都露出了几分异样。
<br/>　　这老者现在的手中，可是足足有着千万圣品灵石，只要是谁可以杀了他，这千万圣品灵石也就是属于谁的。
<br/>　　悄然之间，一道道身影，便是偷偷跟着老者离去了。
<br/>　　“你们几个，跟着这老家伙，等到他出了城，就杀了他，把他的灵石夺过来。”
<br/>　　纣绝阴脸色阴冷，小声对着他身旁的几名六道魔门弟子命令道。
<br/>　　“放心吧少主。”
<br/>　　几名六道魔门弟子听到命令，纷纷跟随这老者离开。
<br/>　　而且不止是这老者，楚风眠也已经是被人盯上了。
<br/>　　楚风眠感觉到，至少是有着数十道目光，在一直盯着楚风眠。
<br/>　　楚风眠走到哪里去，这些人就跟到哪里去，归墟城只是一个小地方，少有武者会在归墟城内长久的停留。
<br/>　　这些人也就是等着楚风眠离开归墟城的一刻，便是发起雷霆手段，轰杀了，夺取了楚风眠身上的财富。
<br/>　　楚风眠也知道这一群人的想法，他也懒得理会，以楚风眠的手段，瞬息之间就足以将这些跟随的人，全部轰杀。
<br/>　　不过在归墟城内，楚风眠还不打算展现实力，以免打草惊蛇，他在归墟城内闲逛着，突然见到了一座高楼。
<br/>　　通天楼。
<br/>　　通天阁旗下的客栈，也算是整个九域算是最好的客栈了。
<br/>　　楚风眠刚刚走过去，几名通天阁的弟子便是围了上来，招揽道。
<br/>　　“前辈需不需要洞府休息？我们通天楼的洞府，是归墟城最好的，而且每一座洞府，都连接这灵泉，灵气浓郁，足以让前辈修炼。”
<br/>　　这几名通天阁弟子，从人群之中一眼看见楚风眠。
<br/>　　楚风眠虽然隐藏了实力，气息，但是身上的那一股气质却是无法改变的。
<br/>　　这种气质，只有身居高位，甚至是一些绝代强者，才有的气质。
<br/>　　这几名通天阁弟子也知道楚风眠肯定是有些身份的人物，才急忙围了上来。
<br/>　　“通天楼？正好。”
<br/>　　楚风眠本来是想要解决了跟随他的那些麻烦，在找个地方，去参悟这古籍，玉符，剑柄，三件宝物的秘密的。
<br/>　　这三件宝物，虽然是放在了空戒之中，可是每时每刻都在吸引着，让楚风眠恨不得早一点的参悟了这三件宝物的秘密。
<br/>　　通天楼倒是一个好地方，楚风眠也住过通天楼，知道其中洞府的隐秘性，正好是一个好地方，让楚风眠参悟这三件宝物。
<br/>　　而且只要是在归墟城内，那些跟着楚风眠的人，也不敢贸然出手对付楚风眠。
<br/>　　“给我来一间天级房间。”
<br/>　　楚风眠对着一名通天阁弟子说道。
<br/>　　“什么？前辈要住天级房间？”
<br/>　　那几名通天阁弟子，听到楚风眠的话都是一惊。
<br/>　　他们虽然是猜想到了楚风眠肯定是有些身份的，在这归墟城内通天楼的天级房间，就连不朽之境的无上老祖都未必住得起。
<br/>　　只有准帝，甚至是真正的帝尊，才有这个财力。
<br/>　　楚风眠看起来如此年轻，绝不可能是一名帝尊，居然就要住天级房间？
<br/>　　“怎么？没有吗？”
<br/>　　楚风眠看着那些通天阁弟子的惊讶，眉头一皱道。
<br/>　　“有有，当然有，前辈，这通天楼的天级房间，一天需要一百枚皇品灵石，或者一枚圣品灵石。”
<br/>　　一名通天阁弟子，小声的对着楚风眠说道。
<br/>　　他也是怕楚风眠不知道通天楼的价格，所以提醒了一下。
<br/>　　“一百枚一天？倒是贵了不少。”
<br/>　　楚风眠记得上一次参加九域天才战的时候，这通天楼中的天级房间，只是五枚皇品灵石一天。
<br/>　　现在已经是翻了二十倍了。
<br/>　　不过现在九域之中的皇品灵石，随着大量远古遗迹的出现，已经是没有之前那么稀少了。
<br/>　　所以这通天楼的价格，也是水涨船高。
<br/>　　不过对楚风眠来说，涨再多也一样，这点灵石，楚风眠随手就足以凝聚。
<br/>　　“带路。”
<br/>　　楚风眠手一挥，一个袋子便是落入到了一名通天阁弟子的手中。
<br/>　　他打开袋子的一刻，精粹的灵力便是喷涌而出，一枚枚晶莹纯净的灵石，摆放在里面。
<br/>　　“圣，圣品灵石？”
<br/>　　那名通天阁弟子都是吓了一大跳，他还是第一次碰到直接以圣品灵石交易的。
<br/>　　圣品灵石兑换皇品灵石，虽然是一比一百。
<br/>　　但是实际上，圣品灵石可是要比起皇品灵石少得多，没有人愿意去这样兑换的。
<br/>　　大部分的时候，都是一比一百五，甚至是一比二百的比例。
<br/>　　直接拿圣品灵石付账的，就连帝尊，都没有能这么奢侈。
<br/>　　“我绝对是碰到大人物了。”
<br/>　　这名通天阁弟子心中一惊，随即反应过来，脸上堆积起慢慢的笑容道。
<br/>　　“前辈请随我来。”
<br/>　　说着，这名通天阁弟子，便是带着楚风眠步入到了通天楼之内。
<br/>　　住通天楼本是需要一些复杂的手续的，打算在这通天阁弟子的安排之下，手续即可办完，便是带着楚风眠向着通天楼内走去。
