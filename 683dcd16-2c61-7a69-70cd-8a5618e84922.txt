　　“破！”
<br/>　　楚风眠口中淡然吐出一个字，裂天一剑，彻底破碎。
<br/>　　“咔嚓！”
<br/>　　一声巨响，这裂天一剑，就这样的全部破碎了。
<br/>　　反噬的力量，全部落在了司空景胜的身上，将他的身躯直接打飞出去。
<br/>　　“噗。”
<br/>　　一口鲜血，直接喷了出来，司空景胜的身躯都是在颤抖这，心中惊骇万分。
<br/>　　“这怎么可能？”
<br/>　　司空景胜，可是司空家的老祖，生死境巅峰的人物，只差一步便是足以超越生死，踏上生死台阶的人物。
<br/>　　在整个北域都足以称得上是一位排的上号的大人物。
<br/>　　可如今，却是被楚风眠这样一个小辈，一个甚至只有神力境小辈，直接击败了。
<br/>　　在剑术上，他居然都不是楚风眠的对手。
<br/>　　“这……”
<br/>　　冰漓长老也都面露呆滞，似乎是无法相信眼前发生的一幕。
<br/>　　司空景胜，居然败了，在剑术上，居然败了。
<br/>　　剑术，可是司空家的根本，司空家的强大，都是因为他们血脉中遗留下来的剑术，才足以号称武道家族。
<br/>　　刚刚那一剑，司空景胜可所谓是没有任何一点的手下留情，甚至是将家族中的剑术，都给全部释放了出来，如今居然都被楚风眠这样轻易的给化解了。
<br/>　　这一幕，除非是亲眼所见，不然的话就算是传闻出去，也不会有着任何人的相信。
<br/>　　以神力境的境界，击败生死境巅峰，就算是先天神体，都做不到这一点。
<br/>　　“这到底是得到了多么大的奇遇，才能做到，我们北邙学院，居然是出现了这样一号人物。”
<br/>　　冰漓长老有些呆滞的自言自语道。
<br/>　　本来见到寒月漓，冰漓长老便是认为，寒月漓几乎是整个北邙学院，资质最高的弟子了。
<br/>　　可眼前，楚风眠展现出来的实力，甚至是还要比起寒月漓要强大几分。
<br/>　　更令人骇然的是，楚风眠才神力境极限的境界。
<br/>　　甚至是都还没有度过生死大劫。
<br/>　　一尊武者，一旦是度过生死大劫，身上的实力将会数倍，数十倍的暴涨。
<br/>　　现在的楚风眠，实力便是就足以击败生死境巅峰的司空景胜，那要是楚风眠度过了生死大劫，实力该会暴涨到什么地步。
<br/>　　超越生死境？甚至是一跃踏上生死台阶，超脱生死？
<br/>　　冰漓长老如今的心中似乎都有些不敢想象。
<br/>　　“司空景胜？司空家的老祖？楚某给过你们一次机会了，既然当初让你们滚的时候不滚，现在也别想离开了！”
<br/>　　楚风眠高高在上，看着如今吐血连连的司空景胜，冷冷的开口道。
<br/>　　“你就跪下，来陪你的孙子吧！”
<br/>　　轰！
<br/>　　楚风眠的大手，猛然一挥落下，一股巨大的压力，压在司空景胜的身上，将司空景胜的身躯都给死死的镇压，就算是司空景胜如何反抗，都逃不出楚风眠的手掌心。
<br/>　　“不可能！一个小小的神力境武者，怎么可能镇压我，怎么可能击败我！不可能，是我是无敌老祖，我绝不会败！”
<br/>　　司空景胜疯狂的咆哮起来，奋力挣扎，似乎是想要打破楚风眠的镇压，从中脱困而出。
<br/>　　只是面对这司空景胜的挣扎，楚风眠只是冷笑一声。
<br/>　　他的心神一动，身躯之中的战龙血脉，被全然催动起来，这镇压的力量，便是又变得更加可怕的许多。
<br/>　　楚风眠身上的气息，都是化为了四尊巨大无比的远古战龙，突然之间便是从天而降，直接便是冲着司空景胜镇压下去。
<br/>　　这四尊远古战龙的力量，镇压而下，简直是要将司空景胜的身躯都给直接撕裂。
<br/>　　只见司空景胜的身躯，不断的颤抖这，拼劲一切力量，想要来抵挡楚风眠的镇压，但是却没有丝毫的意义。
<br/>　　“不！”
<br/>　　司空景胜大声的怒吼这，一股股的灵力，凝聚在极致，在做这最后的反抗。
<br/>　　“还想要反抗？给我跪下！”
<br/>　　楚风眠冰冷的开口道，他的大手一抓，这司空景胜身上凝聚出来的最后灵力，也是被直接打破。
<br/>　　顿时司空景胜身上的防御，被全部打破，楚风眠的威压，直接压迫到了司空景胜的身上。
<br/>　　眨眼之间，这司空景胜的身躯，便是似乎是在这瞬息之间，都失去了力气，双膝一软。
<br/>　　就跟刚刚那司空弘一模一样，直接跪倒在了地上，就跪在了司空弘的身边。
<br/>　　这司空家的一对爷孙，如今全部都被楚风眠击败，一起跪在了楚风眠的面前。
<br/>　　“司空家，可是武道世家，你们的财富，也都拿过来吧！”
<br/>　　楚风眠看着跪倒在地的司空景胜，司空弘二人，他的大手一动，直接对着这二人抓了过去。
<br/>　　这二人，似乎还想要反抗，但是如今他们跪在地上，被楚风眠的力量死死的镇压这，想要反抗都没有任何机会。
<br/>　　楚风眠右手一抓，便是两枚空戒，直接落入到楚风眠的手中。
<br/>　　楚风眠打开一看，在其中有着无数天地奇遇，无数财富。
<br/>　　司空家，毕竟可是武道世家，这司空景胜作为司空家的老祖宗，财富简直多的惊人，至少得有着数百亿灵石的财富。

<br/>　　楚风眠冷哼一声，直接不理会这二人，走了回去。

<br/>　　楚风眠回去，看到还在惊讶中的冰漓长老，笑着开口道。
<br/>　　“这两个人，上门挑衅来，也该给一个教训。”
<br/>　　“楚风眠，你，你真的将他们二人，就这样的给镇压了？”
<br/>　　冰漓长老似乎还有些不可置信。
<br/>　　司空弘被镇压，跪下，已经是出乎冰漓长老的意料了，不过却不至于让冰漓长老震惊，真正令她震惊的，还是下一幕，这司空景胜都被楚风眠击败镇压。
<br/>　　司空景胜可不是弱者，可是生死境巅峰的大人物，生死境高阶的强者，放在外面去都是称霸一方的大人物。
<br/>　　这司空家的老祖，都被司空景胜这样镇压了，冰漓长老还是有些心惊。
