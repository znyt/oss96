        呼，推荐结束了六月初过去了呢
<br/>
<br/>        没当新开一卷的时候，我总是很方，原因你们都懂，因为大纲在心里
<br/>
<br/>        所以最近几天状态是真的不好，可能是因为天天都在写，好久没休息了，灵感匮乏
<br/>
<br/>        而且主要也还是因为上个月太肝了，总感觉都把自己肝透支了
<br/>
<br/>        其实我也很纠结，好不容易的获得的成绩，我也不想懈怠，毕竟经历过以前惨淡的我真的明白今天一切的来之不易
<br/>
<br/>        但是又着实的累，最近气温一直很高，干什么都是又热又累，疲倦的要死，写东西感觉和挤一样
<br/>
<br/>        虽然我很想多写一点，但是又怕赶工质量下降，变的没意思
<br/>
<br/>        能挣到很多钱的确是很大的诱惑，
<br/>
<br/>        但为了度失去了质量就麻烦了，
<br/>
<br/>        我都连续更新4o多天了吧，好累啊，主要心态上的累，一直写，感觉灵感都缺乏了，需要积蓄一下
<br/>
<br/>        成绩上去了虽然是件好事，但也稍稍有点压力，我一直害怕万一写的没意思了吸引不了这么多读者怎么办
<br/>
<br/>        哈哈哈，苦笑，深切的感受到我果然还是个俗人，虽然嘴上说着靠兴趣写书，也不再求票了，但既然挣到了那么多钱，总感觉自己还是担上了一份作者的责任，不能瞎几把乱来，三天打鱼、两天晒网了
<br/>
<br/>        没法像以前二月份一样肆无忌惮了，唉，现在请个假都要犹犹豫豫的，我堕落了啊
<br/>
<br/>        总之，今天久违的去采次风吧，当然你们看见的时候，我可能就已经在摸着鱼了

<br/>        话说我昨晚在上一卷末尾的小彩蛋你们看见了么
