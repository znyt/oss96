　　　　阳鼎天、殷棕等五人，全部冲出了通道，离开了阵法的笼罩。
<br/>　　五人乍一离开，那处通道瞬间关闭了，彻底将天海城封死了。
<br/>　　“他们要干什么？”海明江见此，眉头紧皱了起来，心中有些疑惑。
<br/>　　按理来说，他们被困在阵法中，对方五人联手，绝对能打败他们两人。
<br/>　　但现在，对方五人居然离开了阵法，是想要做什么？
<br/>　　“海阁主，阳鼎天之前说这是什么阵法？”苏莫心中涌现不好的预感。
<br/>　　“好像叫火炼八荒阵！”海明江道，他对于阵法涉猎不多，对于这种阵法也是不太了解。
<br/>　　“火炼八荒……难道他们想用大阵将我们炼化？”苏莫皱眉道，因为这阵法听着就是一个拥有炼化能力的大阵。
<br/>　　“什么？”
<br/>　　海明江闻言一惊，面带不可置信的道：“这……这不可能吧？天海城可是有上亿人口呢！”
<br/>　　海明江感觉苏莫的猜测有些疯狂，炼化整个天海城，阳鼎天他们都疯了吗？
<br/>　　苏莫亦是心中震动，他也感觉这个猜测很是疯狂，但这一切确实有很大的可能！
<br/>　　很快，苏莫两人便知道了答案。
<br/>　　只见阵法之外，天海城的上空，五大强者以及所有阴阳神宫的弟子并未离开，全部在阵法气罩之外，伫立在虚空之中。
<br/>　　阴阳神宫的弟子着实不少，足有二、三十万人，其中还有不少黄泉魔宗、北堂世家和殷家的强者。
<br/>　　“苏莫、海明江，你们将会被炼化成虚无，魂飞魄散，永世不能超生！”
<br/>　　阳鼎天目光如电，穿透略显透明的阵法，朗声大喝：“不过你们放心，整个天海城，还有无数人陪伴你们，你们不会孤单！”
<br/>　　阳鼎天的声音，浩浩荡荡，如同天地雷音，响彻整个天海城。
<br/>　　“什么？”
<br/>　　“他们居然要将整个天海城炼化？”
<br/>　　“疯子！你们都是疯子！”
<br/>　　“快放我们出去！”
<br/>　　天海城中的无数武者，闻听阳鼎天之言，顿时沸腾了起来，放声疾呼。
<br/>　　天海城被炼化，那他们都要跟着陪葬，没有人能逃脱。
<br/>　　先前虽然天海城被阵法笼罩，众人只要远远躲避，也能挡住战斗余波，但现在，所有人都怕了。
<br/>　　“真是一群畜生！”苏莫脸色铁青，对方为了对付他们两人，正是无所不用其极啊！
<br/>　　连炼化整个天海城这种事情，居然都能做的出来，真是灭绝了人性。
<br/>　　“哈哈哈！！”
<br/>　　阳顶天放声大笑了起来，大量的阴阳神宫之人，亦是大笑了起来。
<br/>　　疯狂？
<br/>　　只要能达到目的，整个天海城陪葬又如何？没什么大不了的！
<br/>　　炼化整个天海城，是阳鼎天和二魔老等人商议出的策略，这个办法非常的稳妥。
<br/>　　不然的话，在大阵之内战斗，以他们五人的实力，虽然也能斩杀苏莫和海明江，但肯定会非常的麻烦。
<br/>　　毕竟，海明江是准帝，而苏莫也有准帝的实力。
<br/>　　作为准帝级别的强者，谁没有强大的底牌，到时候拼起命来，他们五人中难免会有人受创。
<br/>　　想杀死一名准帝可不容易，上次二魔老和殷棕、北堂平华三人，合力都没有杀死帝释，便可见准帝强者的难杀。
<br/>　　这倒不是帝释强大，只不过到了他们这个级别，没有绝对的碾压之力，很难杀死对方。
<br/>　　所以，阳鼎天等人，才会想出这么一个稳妥之法，以炼化整个天海城，不费吹灰之力的诛杀苏莫和海明江。
<br/>　　咚！咚！咚！
<br/>　　这时，便见到一名黑衣青年从阴阳神宫弟子中走了出来，虚空踏步，走到了阵法边缘。
<br/>　　青年身上的杀机极为浓郁，每走一步，便让得虚空崩裂。
<br/>　　青年正是上官昊。
<br/>　　上官昊来到了阵法边缘，和苏莫隔着阵法气罩，相距不足十丈。
<br/>　　他的目光，透过阵法气罩，和苏莫对视在了一起，仿若针尖对锋芒，让得阵法气罩都微微波动了起来。
<br/>　　“苏莫，好久不见！”上官昊枯瘦的脸庞上，露出了一丝森然的笑意。
<br/>　　“上官昊，你还配做人吗？”
<br/>　　苏莫脸色阴沉无比，同样透过气罩望着上官昊，冷声道：“你背叛自己的师尊，背叛自己的师门，连畜生都不如！”
<br/>　　“背叛？”
<br/>　　上官昊闻言，森然一笑，道：“你说错了，是帝释背叛了我！”
<br/>　　“背叛了你？”苏莫闻言无语。
<br/>　　“不错！”
<br/>　　上官昊点头，咬牙切齿的低吼道：“因为你的到来，帝释将属于我的一切，都给了你，这不是背叛是什么？他不仁就不能怪我不义！”
<br/>　　苏莫暗暗摇了摇头，对方的心灵已经扭曲了，已经完全陷入了一个极端。
<br/>　　帝释对于上官昊一直很看重，一直是将对方当成传人培养，哪里有什么不仁义！
<br/>　　至于夕儿，他和夕儿两情相悦，水到渠成，这完全是在情理之中。
<br/>　　而这在上官昊看来，居然是帝释对他的背叛，正是让苏莫无语至极。
<br/>　　“上官昊，你会自食恶果！我不会放过你！”苏莫冷冷的说了一句，便不再看对方，心中暗暗思索脱身之策。
<br/>　　“哈哈哈！！”
<br/>　　上官昊闻听苏莫之言，却是疯狂大笑了起来，满脸阴森的道：“你不会放过我？你还有机会吗？”
<br/>　　“今日，就是你的死期，待你死后，帝释也要死！”
<br/>　　“至于夕儿，我不会杀她，我要让她后悔终身，我要让她受尽折辱，我要让她成为了一个万人骑的婊.子!”
<br/>　　“哈哈哈！！！”
<br/>　　上官昊疯狂大笑，整个人显得异常的兴奋，兴奋的脸色都扭曲了起来。
<br/>　　“简直找死！”苏莫彻底被激怒了，猛然出剑，长剑洞穿空间，直接向上官昊刺去。
<br/>　　嘭！
<br/>　　一声闷响，帝皇剑刺在阵法气罩上，却是根本穿透不了阵法的阻隔。
<br/>　　“哈哈哈！苏莫，好好享受死亡的感觉吧！”
<br/>　　再次大笑一声，上官昊飞身后退，退到了大魔老的身后。
<br/>　　“阳鼎天，快快催动阵法吧！”
<br/>　　这时，殷棕已经等不及了，开始催促阳鼎天，因为这阵法主要还是靠阳鼎天催动。
<br/>　　要说此地最想杀苏莫的人，除了上官昊之外，那就是殷棕了。
