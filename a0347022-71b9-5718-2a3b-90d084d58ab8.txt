　　迷宫之中，苏莫大步而行，这迷宫非常浩大，即便二十万人进入其中，也如同恒河之沙，轻易不会遇见。
<br/>　　不过，由于人数众多，每时每刻，都有人被淘汰出去，甚至犹如陨落其中。
<br/>　　若是被重创，就会被立刻传送出去，免于一死，但是，若是被人强力轰杀，那就彻底陨落了。
<br/>　　龙族高层，可不会心疼这些奴仆的性命，他们要的是天赋强大的奴仆，能为龙族所用，而不是白白享用龙族资源的废物。
<br/>　　“五百多件宝物，在这么大的迷宫之中，无异于大海捞针！”一边快速行走，苏莫一边心中暗想。
<br/>　　虽然说，以他的速度，一个多时辰的时间，足以将整个迷宫转一遍。
<br/>　　但是，等到他将迷宫走遍，宝物早已被别人取完了。
<br/>　　苏莫不由的叹了口气，他也不求能得到多少宝物，能得到更好，无法得到他也不会强求。
<br/>　　嗖嗖嗖！！
<br/>　　就在此刻，一阵破空声，传到了苏莫的耳中。
<br/>　　“嗯？”
<br/>　　苏莫闻听此声，顿时眼眸微眯了下来，因为，这破空之声显然速度极快，而且不止一人。
<br/>　　“难道有宝物？”苏莫心中暗想，若是没有宝物，这些人不会如此速度。
<br/>　　这个情况，倒像是有人在追逐。
<br/>　　随即，苏莫立刻身形一闪，向着破空声的源头追去。
<br/>　　追了四条通道，怪了三个弯，苏莫终于发现了破空声的始作俑者。
<br/>　　这是两名男子在追一名女子，很可能那位女子得到了宝物。
<br/>　　两名男子均是人族，而女子并不是人族，而是妖族。
<br/>　　女子身穿兽皮长裙，脸庞上长满了细密的银色鳞甲，不知道是何种种族。
<br/>　　不过，显然女子不是那两名男子的对手，正在快速的逃窜。
<br/>　　咻！
<br/>　　苏莫身形如同利箭洞穿长空，快速追了上去，这三人之中，显然那名女子很可能得到了宝物。
<br/>　　苏莫的速度极快，可以说，在同样是一品低级修为的情况之下，他的速度无人可比。
<br/>　　两个呼吸之间，苏莫便追了上去，拦住了那名女子的去路。
<br/>　　“将宝物交出来吧！”苏莫面色冷漠，目光凝视着那名女子。
<br/>　　女子停下了身形，望着眼前的速度，又瞥了一眼后面停下的两人，面色凝重无比。
<br/>　　“小子，你居然要和我们抢夺宝物？”两名男子面色阴冷，他们追击女子有一段时间了，现在居然有人插手。
<br/>　　“你们三人，将身上的储物戒、空间宝器等储物的宝器，全部交出来吧！”苏莫不容质疑的说道。
<br/>　　“狂妄！”
<br/>　　闻听苏莫之言，女子与两名男子，不约而同的开口了，眼前之人实在是自大狂妄。
<br/>　　轰隆隆！
<br/>　　苏莫并未多言，大手一探，直接幻化出一只源力大手，向三人抓了过去。
<br/>　　这一次，他动用的依旧是火属性之力，不过其中掺杂了大量的木属性源力。
<br/>　　木生火，木属性源力会让火属性源力的威力，大涨一截。
<br/>　　对于五行属性，苏莫早已玩的炉火纯青，如臂指使。
<br/>　　硕大的火焰巨手，如同一道天幕，充满了整个通道，向三人急速盖压而去，根本无处躲避。
<br/>　　“这么强？”
<br/>　　三人见此，顿时大惊之色，因为这火焰大手的威力，足以擒拿二品武者，让他们心中巨颤。
<br/>　　若是修为没有被压制，他们三人自然无惧这火焰大手，但是，现在他们修为都只是一品低级，根本不是对手。
<br/>　　“一起出手！”
<br/>　　那名女子娇喝一声，手掌结印，击出一道印法。
<br/>　　而另外两名男子，一人出剑，一人出枪，纷纷施展最强绝招，攻击火焰大手。
<br/>　　在这通道之中，空间狭隘，他们唯有联手对敌，方有逃脱的可能。
<br/>　　但是，他们三人均是失望了，三道攻击击中火焰大手，只是让火焰大手稍稍一顿。
<br/>　　而后，三人的攻击纷纷破灭，火焰大手碾碎一切，直接将三人，全部擒在了手中。
<br/>　　啊啊啊！！
<br/>　　惨叫声此起彼伏，火焰大手的力量巨大无比，以及其上裹挟的火属性之力，立刻让三人惨叫连连。
<br/>　　苏莫擒住了三人，然后将三人身上的储物戒指，全部抢夺了过来。
<br/>　　“你们被淘汰了！”苏莫将三人丢了出去，并且火焰大手将三人重创。
<br/>　　嗡嗡~~
<br/>　　空间一阵波动，三人被传送了出去，离开了迷宫。
<br/>　　随即苏莫开始查看，三人的储物空间，那两名男子的储物空间，空无一物。
<br/>　　而那名女子的储物空间，却是有一枚神果。
<br/>　　此果，只有樱桃般大小，透体泛紫色，神果的气息极为玄奥。
<br/>　　“什么神果？”苏莫心中疑惑，随即，他立刻将这枚果子吞服了下去。
<br/>　　神果入腹，药力立刻化开，流入了苏莫的四肢百骸，以及全身经脉。
<br/>　　霎时之间，苏莫感觉到，体内的源力流速暴涨了一截，混沌的源力在全身经脉之中飞速的奔腾。
<br/>　　不仅如此，他的经脉的药力的作用之下，在缓缓的改变，变得韧性越来越强，能承受更强大的源力。
<br/>　　“这是……？”苏莫感知到体内的情况，顿时心中一震，这果然是至宝。
<br/>　　这神果的药力，实在是太适合修炼了，若是修为达到瓶颈的情况下，他吞服这种神果，突破修为将能容易数倍。
<br/>　　“果然是至宝！”苏莫眼眸闪亮，随便得到一件宝物，居然就有如此的奇效，这龙族的宝物果然不一般。
<br/>　　而且，貌似这神果的药力，会在体内留存很长的时间。
<br/>　　以目前药力在体内消散的速度，起码能在体内留存一个月。
<br/>　　而这一个月的时间，修炼的速度，会提升七八倍以上。
<br/>　　一瞬间，苏莫心中火热了起来，本来，他来到此地，对于龙族的宝物，并没有多么热衷。
<br/>　　毕竟，他天赋绝世，战力逆天，对他有大用的宝物已经不多了。
<br/>　　但是，没想到随便一件龙族的宝物，就能有如此的奇效。
<br/>　　“一定要多得到一些！”苏莫心念电转，这种神果若是能得到几百枚，他达到神王的时间，能缩短一小半。
<br/>　　虽然这五百多件宝物，不可能都是这种神果，但其他的宝物，想必也不会差多少。
<br/>　　“老方法！”
<br/>　　苏莫心中又了想法，随即身形一闪，急速离去，他要去找迷宫的大门，用出他抢夺宝物的老办法。
