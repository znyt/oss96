　　听完陈老汉的介绍，叶少阳二人领了香烛，前往庙堂去祭拜。
<br/>　　从陈老汉的卧房出来的时候，叶少阳看了一眼边上还有一间房，房门关着，叶少阳一开始也没在意，以为是仓库之类的，从旁边走过的时候，听见里面有人的脚步声，不由问陈老汉：“这房间里还有人？”
<br/>　　“啊，你们别管了，进香去吧。”陈老汉催促他们快去。
<br/>　　叶少阳注意到，这陈老汉的眼神之中，闪过一丝不自然，心中疑惑，但也没有多问，走进了庙堂。
<br/>　　庙堂里只有一尊神像，是一个凡人的模样，身穿绿袍，手拿一把钢叉，身材矮胖，口阔眼大，两个眼珠子像是要瞪出来一样。
<br/>　　在神像前面的牌位上，写着“灵感大王”四个字，毛小方和叶少阳一看就皱起了眉头，让他们感到好奇的并不光是这尊神的名字，而是刻着神祇名字的灵牌，这灵牌已经很旧了，看上去已经快朽了，但是在灵牌的上段，有几条金色的纹路，看上去像是装饰，但是叶少阳和毛小方一看就知道，这是道门符文的一种，叫册封符。
<br/>　　这种灵符，在一些小庙里都可以看到，追述起源头的话，跟神仙的体系有关：
<br/>　　不管是道教还是佛教的神祇，神都是分为几等的，佛教有佛、菩萨、尊者、罗汉……诸天神佛，道门的体系一样很庞大，有三清四御五老六司七元八极九曜十都之说，如此种种，都是所谓的正神，人类虽然无法确切知道这些神的存在，但是在宗教典籍记载的里面，这些都是有确切姓名的。
<br/>　　这是第一等的神，还有一些神祇，是人死后受封的这种，其中最有名的就是关二爷，在道佛两家都封了神，还有孔圣人、门神二将……这些也都是正神，而且大部分都是真实存在的，除了去轮回的，有一部分都在阴司呆着。
<br/>　　还有一种神，就是邪神。
<br/>　　所谓邪神，在古时候，大部分都是一些将军或者土匪之类的，总之杀人要多，最后自己也死于非命，这样的人戾气深重，有很多盘踞在人间闹事，一般有法师对付不了的，就会让当地百姓建庙供奉，慢慢化解他的戾气，逐渐地就成了镇守一方的守护神，也就是山神、土地、河伯、海神这些神的来源。
<br/>　　这样的鬼魂如果想要封神，必须经由法术界有足够威望的宗师上表册封，阴司同意之后，会挂玄名，立牌位，成为官方认可的神祇。
<br/>　　不管是法术界还是阴司，一般遇到不是特别凶残、杀人不多，想要成神并且有一定实力的，都会默许封神，说白了就是招安，这样做也是没奈何，毕竟很多厉鬼大妖无法讨伐，就算法术界一起上勉强能灭掉，也会损失惨重，不如封神，让其护佑一方，反而成为阴司的一份子。
<br/>　　所以封神这种传统，从上古一直流传到现在。
<br/>　　据说当年封神一战，两教共签封神榜，连纣王这样的都封神了，就是为了给自己减少麻烦。不过那时候人族兴亡，阴司的实力有限，后来阴司不断吸收历朝历代的法师、人杰，实力总算是一步步加强，后来这种事干的就比较少了，不过也不是没有。
<br/>　　这种神，虽然不是正神，但也是阴司官方认可的，在人间有一定的权力，一般来说，在封神之后，会保佑一水土，也不会再干邪修害人的事。
<br/>　　这种地方性的小神，因为是法师上表册封，所以在灵牌上会有道家或者佛家的符号，代表自己的身份地位，有区别那些在人间游荡的恶鬼厉妖。
<br/>　　叶少阳跟毛小方躬身上香，但是不跪，陈老汉看在眼里，有些不爽，说道：“见到灵感大王，你们怎么不跪拜。”
<br/>　　“我膝盖有伤，跪不下去。”叶少阳找了个借口，心中冷笑，这什么破神，就算是阴司册封的，自己一代天师，灵仙牌位，这什么神有资格让自己跪？
<br/>　　“我去方便一下，劳驾打听一下……”毛小方拱手。
<br/>　　“出门右转，院墙下面有茅房。”陈老汉有些不耐地挥了挥手。
<br/>　　叶少阳站在神像前，仔细打量这尊神像，神像是石膏塑的，当然是个死东西，但是叶少阳能感觉到神像的两只眼睛在自己身体上下扫视。他相信，这不是自己的错觉。
<br/>　　虽然自己跟毛小方已经将身上的法器用灵符封住，不会有灵力外泄，打扮也是普通人，不过，这一切瞒不过神的目光。
<br/>　　等毛小方回来之后，叶少阳二人告别了陈老汉，在山顶转了转，在这山神庙的后方，俯瞰下去，正好就是蛤蟆谷。
<br/>　　从这个角度看过去，山谷前方的几条小溪，被这座山所挡住，绕道而行。
<br/>　　“明白了吧？”叶少阳看了毛小方一眼，问道。
<br/>　　毛小方俯瞰山谷，点了点头。“这是……”
<br/>　　叶少阳做了个噤声的手势，然后招呼他下山。
<br/>　　到了山脚下，两人并没有立刻离开，而是找了一块被山体挡着的岩石，在这里吹风乘凉，一边讨论起这个“灵感大王”。
<br/>　　叶少阳最初一眼看到这四个字的时候，就想到了《西游记》，《西游记》续集里第一集的妖怪，那个要吃小孩的金鱼精，就自称灵感大王。
<br/>　　不过在法术界，灵感大王这四个字，指的不单单是鱼精，而是一切水生的妖怪。
<br/>　　“既然这个山神取名‘灵感大王’，那也泄露了他的真身，一定是来自水中的某种精怪！”毛小方判断道。
<br/>　　叶少阳冷哼了一声，道：“什么灵感大王，就是个五通神。”
<br/>　　“五通神！”毛小方一惊，猛地一拍脑门，道，“对啊，我怎么没想到，五通神……对，多半是五通神。”
<br/>　　所谓五通神，是水中的五种生物成精：黑鱼、鲤鱼、团鱼（王八）、蛤蟆、蚌。
<br/>　　水族之中，能够自己修炼成精的，就是这五种生灵，像小鱼小虾之类的，如果没有奇遇点化，永远都成不了精。
