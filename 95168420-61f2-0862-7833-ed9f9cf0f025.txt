　　萧逸的脸上，蓦地闪过一丝凝重。
<br/>　　心头亦是一凛。
<br/>　　恰在此时，顾怜星已将洞府大门打开。
<br/>　　“洞府已彻底开启，诸位随我进来。”
<br/>　　顾怜星说了一声，率先进入。
<br/>　　身后一众武者跟随而入。
<br/>　　萧逸暂时停下思索，也跟随其中，他最主要的目的，还是想看看这天星禁地。
<br/>　　至于别的好处，例如里头有什么重宝之类的，萧逸则没有想过。
<br/>　　想也知道，天星禁地，每数年便由天星府开启一次。
<br/>　　若真有重宝，也早被拿光了。
<br/>　　或许唯一的好处，便是里头有对星光一道修炼的增幅效果了。
<br/>　　进入洞府，扑面而来的，是一股厚重气息。
<br/>　　而映入眼帘的，则是一片空荡荡的巨大范围。
<br/>　　不错，进入洞府后，一眼看去，里头空旷一片，大概在千米范围左右。
<br/>　　“咦，空荡荡一片，怎么什么都没有？”
<br/>　　“我还以为洞府之内是星光荟萃之地呢。”
<br/>　　一些武者，面露疑惑之色。
<br/>　　一些年龄较大的武者笑笑，刚要回答。
<br/>　　前方，顾怜星轻笑一声，道，“诸位中，有不少人是第一次进入天星洞府。”
<br/>　　“怜星便在此为诸位解释一遍。”
<br/>　　顾怜星顿了顿，深呼吸一口气，缓缓道，“这里，便是天星禁地的洞府之内。”
<br/>　　“但如诸位刚才在洞府之外所见，洞府异常庞大。”
<br/>　　“故这里仅仅是洞府的最外围。”
<br/>　　“至于洞府里的其它地方，则皆在洞府禁制之内，包括我们天星府在内，亦开不了这些禁制。”
<br/>　　顾怜星边说着，再次顿了顿。
<br/>　　她的身体，确实虚弱，说话一阵便要停一停。
<br/>　　“诸位将会在这洞府之内，修炼3天。”
<br/>　　“现在看似周遭空荡荡一片，但只要诸位凝聚元力，开始修炼，星光便会聚拢而来。”
<br/>　　“待得一天左右，聚拢而出的星光达到一定程度，主府便会牵引而出。”
<br/>　　“届时，主府一出，洞府之内的星光之力会陡然暴走。”
<br/>　　“诸位修习星光一道，将有极大的增幅。”
<br/>　　“……”
<br/>　　顾怜星认真地解释了一遍。
<br/>　　这其中，有不少武者不是第一次进入天星禁地，早便知晓情况。
<br/>　　但还是礼貌地拱拱手，“有劳怜星小姐解释。”
<br/>　　不多时，一众武者，开始在这片空荡荡的如广场一般的范围内盘膝而坐，就地修炼。
<br/>　　萧逸同样盘膝坐下，开始修炼。
<br/>　　果然，随着一众武者的修炼，道道星光，被牵引而出，凭空出现在这空旷广场之内。
<br/>　　如果说，这洞府是一整体，里头充斥了无数星光力量的话。
<br/>　　那么在场的一个个武者，便是分散的个体，利用他们星光一道上的武道力量，各自吸引着这些星光力量出现。
<br/>　　萧逸亦在修炼着，同时，也在思索着。
<br/>　　“当星光力量达到一定程度时，主府便会牵引而出吗？”
<br/>　　萧逸心头暗暗想着。
<br/>　　想起来，之前洞府之外时，同样存在禁制阵法。
<br/>　　任何一个洞府之外，都必有禁制阵法存在，以守护洞府，这是再正常不过的事。
<br/>　　而刚才的禁制阵法，乃是被顾怜星手中令牌压制，众人方能进入。
<br/>　　而那块令牌，又明显借助了天空星光之力。
<br/>　　难怪天星禁地的开启，需要入夜时分，因为这时才是天地星光之力最为浓郁之时。
<br/>　　换句话说，亦是她那块令牌力量最强之时。
<br/>　　如果没有猜错的话，这整个洞府之内的禁制，应该都与星光幽光。
<br/>　　需要借助星光的手段与力量，方能解开。
<br/>　　虽然具体怎么回事，萧逸不知晓。
<br/>　　但他之前的感知中，洞府外的禁制阵法，虽是星光萦绕，却无论气息、气势，还是禁制阵法的构成，都与他在金光禁地那洞府之外感知的差不多。
<br/>　　不，简直如出一辙。
<br/>　　萧逸眉头微皱，思索着。
<br/>　　忽然，一声惊呼，打断了他的思索。
<br/>　　“嘶。”一个武者忽然倒吸一口凉气。
<br/>　　“你们看，那个家伙…好…好可怕的牵引速度。”
<br/>　　一众武者，盘膝修炼，一道道星光，被牵引而出。
<br/>　　可这在其中，又有一人，牵引而出的，不是星光，而是整整一条巨大的星光长河。
<br/>　　“那个家伙，星光一道上的武道层次，达到了何等惊人的地步？”西门孤夜双眼一眯。
<br/>　　几乎所有人的目光，都聚集在了萧逸身上。
<br/>　　不错，自萧逸盘膝坐下修炼开始，星光之力便以极快的速度被牵引而出。
<br/>　　隐藏于洞府之内的星光，在他的武道牵引下，几乎化作了一条长河。
<br/>　　不过十数分钟，以萧逸为中心，周遭百米，俨然已成一片星光海洋。
<br/>　　而别的地方，其他武者所坐之处，仅仅是缠绕着稀薄的星光力量。
<br/>　　萧逸一人之力，牵引的星光之力，竟是被在场近千武者加起来的还要多，还要快。
<br/>　　想也知道，萧逸如今的星幻剑道层次，乃是掌控八万五千道星光，武道层次接近圣皇境九重。
<br/>　　周遭这群不过圣王境修为的武者，论星光一道的层次，自是远远比不上他。
<br/>　　仅仅一个时辰后…
<br/>　　轰…
<br/>　　整个宽阔广场，忽然剧烈抖动。
<br/>　　广场的尽头，千米之外，某个建筑，缓缓升起。
<br/>　　“主府要出来了。”顾怜星惊呼一声。
<br/>　　下一秒，目露惊骇之色地看向萧逸，“不愧是萧逸公子，好可怕的实力。”
<br/>　　“正常来说，要牵引出主府，集一众武者之力，也起码需要一天有余的时间。”
<br/>　　“现在，却是短短一个时辰便牵引出来了。”
<br/>　　不过片刻，主府轰然伫立。
<br/>　　广场上的抖动，也顷刻消失。
<br/>　　同一时间，轰…广场之内，原本被众人牵引而出的星光力量，霎时暴走。
<br/>　　在场武者，纷纷大喜。
<br/>　　他们已经能感受到周遭暴走的星光，感受到修炼速度的飞速飙升。
<br/>　　而最重要的是，原本按照以往惯例，他们需要用一天时间牵引出主府，而在洞府之内的时间，只有三天。
<br/>　　也就是说，只有两天时间是用于真正修炼。
<br/>　　而现在，他们将有3天时间，享受这修炼速度百倍增的效果。
<br/>　　天星禁地，数年才开启一次，一众武者，连忙闭目盘膝起来。
<br/>　　这时，一道身影缓缓从盘膝中站起。
<br/>　　正是顾怜星。
<br/>　　顾怜星，缓缓朝主府走去。
<br/>　　主府之内，肉眼看之，同样是空荡荡一片，范围也只有百米大小左右。
<br/>　　可里头，却有一石坛。
<br/>　　石坛之上，一块块星光耀眼之物整齐摆放。
<br/>　　数了数，刚好是18块。
<br/>　　“上品圣器？”在场武者，感受到那18块星光之物，脸色一惊。
<br/>　　“咦不对，17块。”萧逸认真看了眼，皱了皱眉。
<br/>　　那一块块星光之物，他凝视了半分，蓦地觉得有些眼熟。
