　　“什么，陛下他……”可能是消息过于让老孔难以接受，他习惯性的对李二用出以前的称呼。
<br/>　　不过这些已经不重要了，李承乾根本不在乎他说的是什么，只是抬起手，制止老孔继续说下去：“这件事情眼下知道的人并不多，所以最好不要宣扬出去，孔师能做到么？”
<br/>　　老孔机械的点着头，哆嗦的更加厉害了。
<br/>　　太上皇可能在海上被人袭击了，这么劲爆的消息老孔真的不知道自己应该如何消化。
<br/>　　“孔师，现在你知道朕为什么会这样做了吧？”这个时候的李承乾显的很固执，用的虽然是问句，但显然他并不想听老孔的任何劝说。
<br/>　　孔老头儿听懂了李承乾话里的意思，叹了口气说道：“老臣明白了，老臣这就回家养病！”
<br/>　　虽然老孔是孔雯的爷爷，算来李承乾私底下也应该跟他叫爷爷，可是当事情关系到李二的生死时，就算是爷爷也没有办法阻止孙女婿的行动。
<br/>　　……
<br/>　　而且不要说老孔，现在就算是朝堂上的那个些文武大臣，也没有谁能阻止李承乾，二十万大军几乎是在一夜之间便已经整装待发。
<br/>　　长安到西域的客运列车全部停运，取而代之的是一列列满载着士兵和武器弹药军列。
<br/>　　半现代化的大唐军队已经完全没有了以前的样子，土黄色的沙漠迷彩作战服，土黄色的战靴，土黄色的钢盔，多功能战术军刀，多功能单兵武装带，再加上新式步枪，将每一个战士几乎都武装到了牙齿。
<br/>　　而除了他们随身携带的单兵武器之外，火箭弹、贞观炮、推土机等等重型装备也随着队伍装上了列车。
<br/>　　大食、拜占庭两国的使者几乎都吓傻了，曾经还在使用冷兵器的大唐就已经让他觉得很难对付，而现在全部装备了最新式热武器的大唐十六卫对他们来说简直就是要了亲命的存在。
<br/>　　可是他们自己干过什么只有他们自己知道，大食的使者很清楚南海那边发生了什么，拜占庭人同样知道南海那边发生了什么，让他们没想到的是李承乾的反应竟然如此激烈。
<br/>　　而与那些异族不同的是，大唐百姓在看到那一队队与以往大不相同的军队时，一个两个几乎都要乐疯了。
<br/>　　虽然大唐雄师这几年已经蛰伏，但大唐尚武的精神却没有丢，曾经的大唐铁骑是打遍天下无敌手的存在，只是这段时间因为皇帝陛下想要休养生息，所以蛰伏起来。但是现在，大唐雄师的‘獠牙’再一次刺出，不知道哪个倒霉鬼会成为大唐口中的美食。
<br/>　　北海舰队这个时候已经出发，除了必要的物资以外，船上所有的位置都塞满了武器弹药，就差没把水手的枕头换成炮弹了。
<br/>　　李震、李佑两人正坐着火车赶往泉州，他们将在泉州与北海舰队汇合，再带上新近下水的五舰钢铁战舰一同南下寻找李世民的踪迹。
<br/>　　从审问隐王一脉所得出的线索来看，李世民在南海的消息就是他们泄露出去的，再联系太上皇接连一个多月没有任何消息回传的情况，几乎所有人在心中都有了一份猜测。
<br/>　　敢打大唐主意，并且敢于实施的国家现在还有多少？大食算是一个，拜占庭这个国家不好说，虽然他们如果有机会的话也会这样做，但他们距离南海实在太远了，根本没有足够的力量到达那里。
<br/>　　李佑神情严肃的站在船舷之上，他的身边便是李震，猛烈的海风吹在他们两个的身上，将两人的衣襟吹的猎猎作响。
<br/>　　良久之后，李佑突然开口：“李兄，我听说那些大食人在天竺边境的时候就跟尉迟交过手，只是他们当时没有实力，被尉迟给打了回去，这事儿你听说了吧？”
<br/>　　李震在船舷的栏杆上拍了一下，有些恼火的说道：“大食人对土地的渴望从来就没有减退过，这一次一定又是他们在背后搞鬼。”
<br/>　　“我也觉得是这样！”李佑看着天边的夕阳，有些出神的说道：“只是眼下我们的任务是去南海搜寻父皇的踪迹，否则我真的很想跨海过去，在他们的后方来上那么一下。”
<br/>　　李震挑了挑眉毛，嘴角微微一挑：“放心吧，会有机会的，陛下这次不是派了程老将军去天竺道与大食的边境了么，估计一个不好就能打起来，我们只要抓紧时间快点找到太上皇，到时候说不定还能赶上这一波。”
<br/>　　李佑叹了口气说道：“到时候再说吧，茫茫大海，找一支舰队不是那么容易的。”
<br/>　　自从混入军方，在倭国打了几场大的战役之后，李佑明然成熟了许多，不在像以前那样肆无忌惮，遇事也知道冷静考虑问题，与历史上记载的那个他简直就是天地之差，只是他自己并不知道而已。
<br/>　　而事实上历史上的李佑之所以会那样嚣张跋扈，为的也不过就是想要引起别人的关注而已，李承乾正是发现了这一点，所以才会给他机会，让他有了表现自己和得到别人认可的机会。
<br/>　　也正是因为这样，李佑的精力得到了发泄，所以才会与从前的那个他有了强烈的反差。
<br/>　　而且这次老头子这么久没有消息，估计真的是被人给围了，如果他李佑能在关键时刻赶到，正好可以好好的在老头子面前露上一把脸。
<br/>　　虽然说皇位已经定下来了，但李佑在乎的不是皇位，他真正在乎的还是老李的认可。
<br/>　　想着这些，李佑的不禁握紧了双拳，一双眼睛死死盯着远处渐渐暗下来的海面，心中决定这次必须要好好表现一下，争取改变自己和母亲在老李心中的印象。
<br/>　　“呜……”一阵气笛声突然打断了李佑的思绪，回过神来之后他发现战舰不知什么进候竟然已经停了下来，在战舰的左右朦胧中似乎有一个黑色的影子。
<br/>　　那是一座海岛，在倭岛来回走过几趟的李佑在看到那个黑影的瞬间便反应了过来，只是让他疑惑的是，为什么那么大一座岛，竟然没有一丝的灯火。
