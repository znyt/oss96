        只是，傅夜沉跟她在一起的时候，从未主动提及过他所认识的女性的任何事情，包括他的这位干妹妹。

<br/>        或许，这跟他傅夜沉的性子有关吧！
<br/>
<br/>        他不会没事找事地在自己爱人面前去提其他无关紧要的女人。
<br/>
<br/>        “现在，应该是妻子了吧？”夏凌焕嘴角微扬。
<br/>
<br/>        童筱颖怔愣地看向夏凌焕，他似乎知道她和傅夜沉的私事。
<br/>
<br/>        “她叫童筱颖，你两应该一样大吧！”夏凌焕这才正经八经地向夏玄灵介绍起童筱颖来。

<br/>
<br/>        倒是在饭后，夏凌焕一边收拾着餐盘，一边看着正拿牙签叉着水果沙拉吃的夏玄灵，问道：“今天见到金澈了吗？他答应帮你了吗？”
<br/>
<br/>        夏玄灵拿着牙签的手滞了滞，刚刚还在为这嘴里的水果尝到甜头，结果被哥哥夏凌焕这一泼冷水下来，嘴里如同嚼蜡。
<br/>
<br/>        在一旁帮着夏凌焕收拾碗筷的童筱颖，只是静默地听着这兄妹两的对话，不插嘴也不吭声。
<br/>
<br/>        “我没机会跟他谈，他拒我于千里之外。”夏玄灵眸光黯然。
<br/>
<br/>        她真的没想到，三年不见，裴金澈像是脱胎换骨了一样，变得更加完美优秀了。
<br/>
<br/>        听说，他是夜沉哥花重金，从国外“挖”回来的，想必，他现在一定有他的过人之处。
<br/>
<br/>        傅氏集团，如此庞大的一个“帝国”，傅夜沉竟然轻易地交付到裴金澈手中，还真是令所有人感到意外。
<br/>
<br/>        与此同时，童筱颖也深刻地感应到，傅夜沉已经变得不如从前了。
<br/>
<br/>        至少，在三年前，她能感觉到，他还很信任傅昼景。
<br/>
<br/>        现在……
<br/>
<br/>        一种直觉告诉她，傅夜沉在堤防着傅昼景，不然，他的集团，为什么不交付给傅昼景打理？
<br/>
<br/>        明明，傅昼景也有这个本事，可以管理好一个庞大的集团。
<br/>
<br/>        “你十八岁高中毕业，放弃念大学的机会，跟夜沉哥借钱开了咖啡店，三年的时间里，你确实在好几个大城市里开了连锁店，但你那连锁咖啡店，不是资金出了问题，而是管理层出了问题，才会导致经营不下去。我觉得，为了你的咖啡店着想，你还是厚着脸皮再去找一下金澈。就连你夜沉哥的意思，也是要你去求金澈帮你解决这个大难题！”夏凌焕耐着性子劝说。
<br/>
<br/>        “我知道，但是他把我当成了陌生人，我该拿他怎么办？”夏玄灵瘪嘴道。
<br/>
<br/>        童筱颖按捺不住地问：“你和那个裴总之间是不是有什么误会？”
<br/>
<br/>        听他们之间这对话，她可以肯定，他们都是旧相识。
<br/>
<br/>        提起误会，夏玄灵不禁皱了下眉头。
<br/>
<br/>        夏凌焕看向童筱颖，深沉地回答道：“跟你老公有关。”
<br/>
<br/>        大哥？！
<br/>
<br/>        “跟他有什么关系？”童筱颖感到难以置信。
<br/>
<br/>        夏玄灵随之瞪了夏凌焕一眼，示意他不要在外人面前多嘴。
<br/>
<br/>        夏凌焕会意后，苦笑了笑，向童筱颖无奈地耸了耸肩。

<br/>        不过，他们不说也没关系，她大不了去问大哥。
<br/>
<br/>        夏玄灵放下手中的牙签，看着童筱颖微笑着转开了话题：“筱颖，你和夜沉哥是怎么认识的？”

<br/>        很明显，夏玄灵似乎真的不希望她童筱颖知道她的私事。
