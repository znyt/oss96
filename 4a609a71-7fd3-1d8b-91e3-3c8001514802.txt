　　这一切发生得太快，几乎就是瞬息之间，他们这几百号人就成了案板上的鱼肉。身边有不少人围过来，找叶少阳询问这是什么水，从他们身上流过，为什么把他们体内的修为也带走了，不光如此，那些邪物发现他们泡在水里的身体，也在一点点分解，血液和皮肉，也被水一点点带走。
<br/>　　这水就像是有腐蚀性，但是不疼也不痒。
<br/>　　一时间军心大乱，大伙鬼哭狼嚎起来。
<br/>　　小马和吴嘉伟等人游到边缘去，试图冲开电弧的封锁，但试了半天没用，回来告诉叶少阳，这电弧的组成，是非常纯粹的灵力，以此可以断定，背后施法的人一定是正派法师。
<br/>　　“圣灵会，一定是他们，妈个把子，一眨眼我们这些人就被困住了，妈蛋，你们冷静一点，还没死呢！”小马抓住身边几个情绪最激动的邪物，朝水中丢了出去，大声骂着。
<br/>　　“你也别激动，赶紧保护我吐纳，把灵虫赶出来再说！”叶少阳飞快地说道，自己现在右手不能作法，那就不能握剑，最强的手段也就无法施展出来。
<br/>　　法术公会从来不做没用的事。
<br/>　　既然用了那么隐蔽的手段，让自己被灵虫纠缠，目的必然就是现在这样，至少废了自己一只手，抛开七星龙泉剑不说，一些厉害的法术也是要双手结印，单手结印的威力要差很多。
<br/>　　不管怎样，自己一定要先恢复最佳状态，接着才能想办法解决眼下的危局。
<br/>　　“大家静心凝神，调整气息，能减缓道行的流失！”叶少阳发声提醒大伙，便在小马和吴嘉伟的保护之下，入定调息起来。
<br/>　　清波乖乖地站在他身后，紧张地看着四周。以他的实力和地位，别说跟叶少阳吴嘉伟他们相比，就是身边这些倒霉卷入这个世界的邪物，也个个都比他厉害得多，但他身为弟子，一直默不作声地跟在叶少阳身边。他还是第一次经历如此可怕的事情，反应却比身边那些嗷嗷乱叫的生灵要冷静地多，他低头看着碧绿的水面，能一直看到自己的双脚，虽然冰凉刺骨，但是……有一种很奇怪的感觉，他把手臂从手中拿出来，一下子呆住了，喃喃道：“这水为什么没打湿手臂，连衣服都是干的！”
<br/>　　小马听见了，也把双手抬起来，果然手从手上滑下去之后，没有一滴流下来，整条手臂都是干的。
<br/>　　吴嘉伟听说之后，把自己随身的法药掏出来，连法药都是干的。
<br/>　　“怪啊，这特么什么水！”小马喃喃道。
<br/>　　“无色海！这是无色海！”身后一个人叫起来，小马回头看去，是一个和尚，一把抓住他的肩膀，问道：“什么无色海，你怎么知道？”
<br/>　　“这……是这样的，佛经中有说，三界为无色界、色界、欲界，无色界中，有无色海，水碧绿长青，细看无色，人于海中，滴水不沾……却不就是现在这样么，阿弥陀佛，贫僧一生修行，没想到竟然到了这无色海中……真是天大的机缘……”
<br/>　　这和尚喜不自胜，双手合十，不断祷告着。
<br/>　　“你吃错药啦！”小马在他肩膀上打了一巴掌，“我们被困在这儿了，你反倒高兴？”
<br/>　　这和尚不为所动，对小马说道：“你有所不知，这无色海不是一般生灵能到的，一切生灵，要往生极乐，都要过这无色海，在水中洗濯身上一切欲望，空无一物，方可去往婆娑净土，那才是真正的天堂！”
<br/>　　“你去上天堂吧！”
<br/>　　小马把他随手丢到水里去，瞪眼看着吴嘉伟，道：“这是什么情况啊！什么无色海，就算真有，不也是在什么……那什么地方来着，怎么会在这里？”
<br/>　　“我不知道！”
<br/>　　就在这时，靠近“锅盖”上游的四周的人突然炸锅一般叫起来，吵闹着不知道发生了什么事。
<br/>　　小马和吴嘉伟就近冲到一边，透过电弧组成的栅栏朝外看去，原来是有碧绿色的莲叶从水下面长了出来，在水面上展开，中间抽出了花茎，眨眼的工夫，上面就长出了粉红的花苞，一点点展开，中间居然坐着一个人，随着花苞的展开，小人也越来越大，总算能够看清，是一个光头的和尚，盘着双腿，端坐在荷花的花蕊中间，闭着眼睛，口中念着咒语。
<br/>　　不远处那一只莲叶上，同样坐着一个和尚，都穿着丝瓜筋一样的袈裟。
<br/>　　小马愣了一下，绕着“锅盖”转了一圈，总算是看到了全貌：一共八朵莲叶，上面各自坐着一个人，四个是和尚，另外四个，因为电弧的阻挡，看不清楚脸，但能看到有两个是道士，另外两个穿着古装常服。
<br/>　　这八个人的顺序也有讲究，和尚是四面一面一个，都在念着一样的咒语，另外四个人都在不断结印，手法也不一样，不知道在做什么。
<br/>　　“叶少阳，哈哈……总算让老夫等到这一天了！”
<br/>　　一阵喋喋的怪笑，从端坐在莲花上的一个道士口中发出，“叶少阳，你看不到我吗，老夫是玉玑子啊，哈哈……”
<br/>　　玉玑子？
<br/>　　叶少阳勉强分出来一丝注意力，听见这三个字，心头微微一怔，猛然想了起来，玉玑子，昆仑山掌教？
<br/>　　这还真是冤家聚头了。自己跟他，算是真正额仇人了，从最初自己杀了凌宇轩、后来道风挑了昆仑山的山门，夺了玉清符，玉玑子就对他们师兄弟恨之入骨，那一次悬空观之战，玉玑子煽风点火，不惜玩起了绑架，后来被橙子的鲛人家族驱赶，失败之后，依然不思悔改，后来一直找机会报复，甚至不惜加入了法术公会。
<br/>　　这老东西出现在这里，倒是一点也不奇怪。
<br/>　　叶少阳一边调息恢复，一边释放出了一缕魂魄，观察附近的情况——只有灵仙以上才能做到，不过这一缕魂魄不能跟人战斗，也不能离开本尊太远，更不能说话分心，也就算是代替了本尊的视角。
