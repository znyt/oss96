        &lt;h3&gt;第三卷第二百三十四章  呼延寰宇到北京&lt;/h3&gt;
<br/>
<br/>        西城分公司出事，大家心里都有些难过，在一起工作了好多年，突然来了那么一下，都感觉被背叛了，但是陈子昂觉得是人之常情，在这个世界没有什么是无坚不摧的。
<br/>
<br/>        呼延寰宇要和陈子昂一起去甸城，所以陈子昂先期将呼延寰宇调离西城，进入北京工作，让他和赵绾绾先熟悉一下俩个人之间的工作方式，后期减少工作上的摩擦。
<br/>
<br/>        陈子昂到甸城不完全是做项目，她最主要的是想弄清楚谷强的身份。
<br/>
<br/>        在这一两年的时间里，陈子昂通过渠道了解了甸城各个方面的关系，甸城的状况深深的烙在她的脑海中，谷强的身份也越发让她起疑。
<br/>
<br/>        陈子澜的出事更是引起她的警觉，她本来想找陈子澜了解一下甸城具体情况，但是想着陈子澜的工作性质，只好作罢。
<br/>
<br/>        如果谷强真的是他的哥哥，她有权利拉他走出污浊，如果谷强不是她的哥哥，她会送他一程，只是因为他和陆家相勾结。
<br/>
<br/>        陈子昂从新年起就已经改变了她一贯忍让的态度，她不再认为她的忍让会让对方得到愧疚，反而是会让对方变本加厉。
<br/>
<br/>        她的脑袋里也不是人们看到的那样，她是一个天才，既能搞破坏也能安分守己，她能分析并且纵观全局，落雪在美洲遇袭更是给她提了一个醒，强大如斯的落家人都会有人动手，何况她陈子昂只是一个蝼蚁，与其等着对方出手被动挨打，不如主动出击。
<br/>
<br/>        她和张家、陆家必定有一战！她的书房里有她画出的一张草图，这张草图的人物关系和形象在她的笔下栩栩如生，而这些关系被她理顺之后一切都明白无误的摆在她的面前。
<br/>
<br/>        她的人生改变从夏颜开始，而夏颜这几年频繁的出现在她的生活中，从夏颜到冯睿延伸到张函再到陆家再到她身边的所有人，那根从来都不被她在意的线被她画出来了，陈子昂开始关注这些人的一切。
<br/>
<br/>        她想知道的是没有人能挡得住的。
<br/>
<br/>        赵绾绾是她开始做信息化的时候就跟她在一起，她对赵绾绾充满了信任，呼延寰宇重义气，讲诚信，就算是呼延临庆的亲戚也不曾对公司做过任何不利的事情，相反还拒绝了呼延临庆的高诱惑。
<br/>
<br/>        赵绾绾和呼延寰宇还是高手中的高手，能写能破。
<br/>
<br/>        .......
<br/>
<br/>        陈子昂带着赵绾绾亲自到机场接机。
<br/>
<br/>        “你怎么亲自来了，我这打车也就到公司了。”
<br/>
<br/>        “为了表示我对呼延人才的重用啊。”陈子昂笑着说。
<br/>
<br/>        “来，介绍一下，你们俩个都是只闻其声不见其人的，赵绾绾、呼延寰宇。”
<br/>
<br/>        赵绾绾伸出她那敲键盘的瘦弱而纤细的小手握住呼延寰宇手：“久闻盛名，以后多多关照。”
<br/>
<br/>        呼延寰宇有些不好意思：“听陈总说你是拼命三娘，还以为你和她一样人高马大，原来如此娇小玲珑。”
<br/>
<br/>        “嗨嗨嗨，呼延，好歹我也是你貌美如花的老板，你能不能给我留一点面子？什么叫人高马大？”陈子昂表示不满意。
<br/>
<br/>        “我觉得，女人还是娇小玲珑惹人爱。在你面前，说实话，我都有点颤巍巍。”
<br/>
<br/>        “哈哈哈，我这是压倒一切恶，势力，行的端站得直！”陈子昂挺了挺身子。
<br/>
<br/>        “你别嘚瑟了，不就是比我高吗？也就是高那么一点点。”呼延寰宇气馁了。
<br/>
<br/>        赵绾绾被呼延寰宇和陈子昂之间的互动逗笑了。
<br/>
<br/>        原来他们俩个人真的是朋友相处，她和陈子昂心意相通，但是那只限于工作上，平时她总是觉得陈子昂是她的上司，多多少少还是对陈子昂有一丝丝下属对老板的那种心理。
<br/>
<br/>        现在看着如此可爱的陈子昂，她觉得她似乎错过了什么。
<br/>
<br/>        “子昂要不是人高马大怎么能抗得起担子，放我早被压垮了。”赵绾绾笑着接话。
<br/>
<br/>        “还是你会说话，我以为像我一样的程序猿都是大老粗，没想到绾绾温温柔柔一江南小女子。”
<br/>
<br/>        “什么温温柔柔江南小女子，我是深圳的。”
<br/>
<br/>        “啊，哈哈哈，那就是南方小女人。”
<br/>
<br/>        赵绾绾和呼延寰宇就这样认识了。
<br/>
<br/>        陈子昂见俩个人说话都比较随和，绾绾也没有了之前的那种疏离感，心中大喜。
<br/>
<br/>        “呼延，你的住宿都已经安排好了，和绾绾他们一个大楼，你先放下行李，收拾收拾，晚上我们一起吃饭，有些事情我要和你们商量一下。”
<br/>
<br/>        “好嘞。”
<br/>
<br/>        “绾绾，我还有事情，你带着呼延先熟悉一下公司的情况，我们晚上见。”接到呼延寰宇，陈子昂将呼延寰宇交给赵绾绾就离开了。
<br/>
<br/>        “你忙你的去吧。”
<br/>
<br/>        这么几年，呼延寰宇是很少来北京的，每次都是开会急匆匆的来，急匆匆的走，在信息问题上他和赵绾绾也都是电话进行沟通，开年会的时候，赵绾绾经常都是钻在工作室里鼓捣她的研发，很少和大家见面。
<br/>
<br/>        这一次能见到赵绾绾并且和赵绾绾搭档，呼延寰宇是怀着期待的心情的，英雄惜英雄就是说的他们这种人。
<br/>
<br/>        “听说公司所有的研发都是你带人搞的，我真的是很佩服你，以为你是男生。”呼延寰宇有些感慨。
<br/>
<br/>        “佩服我？我们公司最厉害的人你知道是谁不？”
<br/>
<br/>        “难道不是你吗？”
<br/>
<br/>        “就是你说的那个人高马大的人，她的脑袋就是一台活的计算器，也是一台活的编程器。她的思路和想法往往超乎常人，我们所做的一切都是按照她的要求来做的，我从来都不敢在她的面前开玩笑，你知道为什么吗？”
<br/>
<br/>        “为什么？”
<br/>
<br/>        “因为敬畏！”是啊，她的计算恰到好处，她的思路超出想象，因为跟在她的身边久了，才知道她有多强大。
<br/>
<br/>        呼延寰宇有些质疑的看了一眼赵绾绾，他只知道陈子昂只会带着大家做生意，只会闷着头跑业务。
<br/>
<br/>        “别怀疑，你马上就会体验到，我想陈子昂将你这么早调到北京不会让你这么简单的就跟着她一起去甸城的。”
<br/>
<br/>        晚上算是给呼延寰宇接风，但是只限于他们三个人，吃完饭陈子昂说：“呼延，将你调到北京，是有事情让你来做，我这里有一套软件需要你们破解，看看你们能在多长时间内破解完成。”
<br/>
<br/>        赵绾绾对着呼延寰宇微微一笑。
<br/>
<br/>        三天后，呼延寰宇和赵绾绾走出工作室来到陈子昂的办公室，他们已经破解了三天三夜！俩个人眼眶发红，但是很兴奋。
<br/>
<br/>        “完成了？”
<br/>
<br/>        “完成了！”
<br/>
<br/>        “很开心？”
<br/>
<br/>        “当然！”
<br/>
<br/>        “我用三个小时写的程序，你们用了三天，难为你们了！”陈子昂微笑着看着他们。
<br/>
<br/>        呼延寰宇终于知道赵绾绾说的话是真的！
