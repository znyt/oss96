　　　　此刻，那怕猿猴王现在再恨涂小安这条蛇精也没工夫追杀他了，立刻着急忙慌的发号施令，让它的猴子猴孙赶紧远离这片山谷，有多远躲多远。
<br/>　　渡劫的天雷天火可不开玩笑的，不小心落到一座山峰上，山峰顷刻间就化为了废墟。
<br/>　　暗红色的天窒息般的仿佛就在头顶一般，给人一种强大的压迫感，修炼低的人，都难以呼吸，浑身颤栗。
<br/>　　普通猴，在这种环境下，很多甚至会心肌梗塞而死。
<br/>　　随后，暗红色天空开始旋转了起来，仿佛一个巨大的海中漩涡不停地旋转了起来，一道道紫色电蛇犹如无中生有，就从暗红色天空中出现，而后被吸入了一个巨大的漩涡之中。
<br/>　　漩涡愈加湍急，方圆数百里范围内天地灵气完全混乱了，一道道巨大的电蛇在漩涡中蜿蜒曲折，渐渐的，漩涡停止了旋转，变成了一朵巨大的紫色劫云。
<br/>　　暗红色的天空，紫色的巨大劫云，劫云中闪烁的电蛇，一切是那么的不可思议，那么的玄幻。
<br/>　　牛妖王跟赤阳巨猿原本打的风生水起，热火朝天，它们都是三十二级的实力，就算打上个三天三夜，也难分胜负。
<br/>　　但此时天空的异象让它们无法在打下去了，数以万计的猴族仿佛树倒猢狲散，大难临头各自飞。
<br/>　　牛妖王这时身形闪动，出现在了涂小安的身边：“主人怎么回事，上空怎么显化渡劫劫云？”
<br/>　　涂小安眨巴眨巴眼道：“你主人我马上要渡劫了”
<br/>　　“什么？”牛妖王大吃一惊，震撼无比：“之前怎么没听你说过”
<br/>　　“我也有点措手不及”
<br/>　　“主人，你渡劫渡的那么随意啊”牛妖王看着上空越来越可怕的异象，脸色难看道：“一般来说，修炼到了巅峰要证道渡劫，是有一段时间的缓和期，让你做准备”
<br/>　　“可主人你的渡劫，说来就来，老牛我一点心里准备都没有啊”
<br/>　　涂小安一个白眼丢过去：“你要屁的准备啊，我还没准备呢”
<br/>　　上空的天越发愈演愈烈，此时，不远处一颗茂密的大树已一种诡异的爬行速度奔跑而来。
<br/>　　“卧靠，吓死查理了，哪个乌龟王八蛋要渡劫，主人，我们快离开这猴儿山”
<br/>　　涂小安咬牙切齿道：“那个乌龟王八蛋就是你主人我，信不信我在渡劫先，先将你当柴火给烧了”
<br/>　　树精呆滞了下，意识到说嘟噜嘴了，干笑道：“主人，你这是要一飞冲天啊，恭喜主人要超凡脱俗，成为三阶的存在”
<br/>　　“别拍马屁了，一边呆着去”
<br/>　　牛妖王脸色却不好看的道：“主人，你上空显化的是紫色劫云，威力非同小可，这是正版的天雷啊，就是不知道会落下几道来”
<br/>　　“你渡劫劈下来几道天雷”涂小安问。
<br/>　　“六道”牛妖王继续道：“这个是天地法则根据渡劫之人的实力修为而定，修为越逆天，天雷劈的越多，越狠”
<br/>　　“已主人你一条蛇精就收服老牛我这等三阶妖灵来看，不给主人你落下十几道天雷，都没天理”
<br/>　　“呵呵...”涂小安冷冷一笑，眉目一挑恶趣味道：“牛啊，你要不要帮主人我抗几道天雷啊，让你也酸爽下”
<br/>　　那条上古巴蛇曾经就要求涂小安帮它抗天雷，也就是说，这个渡劫是可以组团的，只要牛妖王悬浮在涂小安的上空，帮着抗雷就行。
<br/>　　牛妖王闻言，脸色就煞白煞白起来：“主人你别闹，你的天劫威力跟老牛当时可不是一个级别待遇，况且别人抗雷，天雷的威力更是会倍增的”
<br/>　　“倍增？”涂小安诧异，这点上古巴蛇可没有说明。
<br/>　　该死的邪蛇，居然敢坑自己，涂小安当时就疑惑，天劫是那么好糊弄的吗，组团都成？
<br/>　　只能怪他当时太傻太天真，居然还信了。
<br/>　　原来别人帮忙，天劫的威力就会变得更大。
<br/>　　天地法则，天威浩荡，眼中容不得沙子。
<br/>　　其实涂小安并不怕惧怕天雷天火，因为他有先天的优势，便是掌握雷电属性跟火焰属性。
<br/>　　这两种属于在手，简直就是为了对抗天劫的。
<br/>　　“来吧，就让天劫来的更加猛烈一些吧”涂小安悬浮与空，忽然霸气凌然的一声长啸，这一刻，他就是战神，天底下仿佛已经没有什么能够阻挡他要成为三阶的存在。
<br/>　　三阶之下，皆蝼蚁，唯有三阶，才是超凡脱俗的存在，才能凌驾天地万物之上。
<br/>　　轰隆隆...
<br/>　　忽然，上空暗红色的天显化的一朵巨大紫色劫云仿佛是不满涂小安的装逼，一朵化两朵，两朵化四朵。
<br/>　　宛如道生一，一生二，二生三，三生万物。
<br/>　　不过半晌，暗红色的天直接显化出了三十六朵紫色劫云，如漫山遍野的鲜花，百花齐放。
<br/>　　“我的乖乖...”
<br/>　　牛妖王望着天完全惊呆了，三十六朵？
<br/>　　一朵之中蕴含一道天威浩荡的天雷，整整三十六道天雷洗礼啊。
<br/>　　遥想当年，牛妖王被六道天雷都劈的半死不活了，最后苟延残喘的才活下来。
<br/>　　“主人莫装逼，装逼被雷劈，祝上帝保佑你”牛妖王做了一个耶稣信徒的手势。
<br/>　　涂小安脸色一下子难看，阴郁了下来，我去，我特么就随便装个逼都不行嘛，要不要这样给我面子。
<br/>　　暗红色的天三十六朵紫色劫云显化完毕，与此同时，另一片对立的天空变得炽焰如火，赤红色的天映红一切，如火如荼，仿佛要将一切都给焚烧了一样。
<br/>　　一个巨大的风暴漩涡汇聚而成，整个猴儿山震动起来，狂风呼啸，天地变色。
<br/>　　巨大的风暴漩涡之中，一块天外陨石滚滚如火的显化，大的遮天蔽日，落下来，可以说整个猴儿山都会不复存在。
<br/>　　无数生物诚惶诚恐，笼罩在巨大的恐惧之中，瑟瑟发抖，匍匐与地。
<br/>　　“不好主人，你的天雷异象居然也迸发了，正常来说，天雷之后，会有时间缓和期，才会显化天火的异象”
<br/>　　牛妖王惊骇的难以复加：“要是同时显化，说明天雷天火会形成连环攻击模式，主人果然不愧是主人，待遇就是不一样”
<br/>　　要是别人出现这样事情，牛妖王会直接丢出一句：“等死吧你”
<br/>　　这还玩个屁。
