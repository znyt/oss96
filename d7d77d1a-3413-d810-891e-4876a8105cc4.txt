        楼上，傅夜沉还在开跨国会议，但是，等到傅夜沉的跨国会议一开完，落以柔就立马去了总裁办告知：“大哥，夫人她有来过，说她在民政局里等你。”
<br/>
<br/>        话语刚落，落以柔便看到傅夜沉在不经意间地蹙了下眉头，她随即喜上眉梢，高兴地阳奉阴违道：“大哥是不是要发喜糖了？你和夫人这是要去领结婚证了吗？太好了！”
<br/>
<br/>        “我接下来的行程是什么？”傅夜沉却答非所问。
<br/>
<br/>        落以柔立马恢复到工作状态，拿起平板电脑，一丝不苟地向傅夜沉汇报起今日的行程来。
<br/>
<br/>        这一天下来，傅夜沉整整忙了一天，但是他并未忘记筱颖还在民政局里等她的这件事情。
<br/>
<br/>        他以为，她是个傻丫头，没想到，她不傻。
<br/>
<br/>        知道要去民政局里跟他领离婚证，才算真正的结束两个之间的婚姻。
<br/>
<br/>        可是为什么，真到了这一刻，他又是那般不舍？
<br/>
<br/>        他在逃避，一直逃避到下午离下班的时间还有半个小时。
<br/>
<br/>        而童筱颖真的就在民政局门口，等了他整整一天。
<br/>
<br/>        她的听力好像越来越差，还是说，是这个世界越来越安静了。
<br/>
<br/>        童筱颖呆呆地坐在了民政局门外的台阶上，直到一双锃亮的黑色皮鞋出现在她视野里，她的目光才顺着这双皮鞋，缓缓地往上移动。
<br/>
<br/>        当她的目光撞上傅夜沉那双凌冽的黑眸时，她冷不丁地打了个寒颤。
<br/>
<br/>        “大哥……”童筱颖讷讷地开口。
<br/>
<br/>        傅夜沉面无表情地说道：“还有五分钟，办理离婚证，还是来得及。”
<br/>
<br/>        他淡漠地说完，转身准备往民政局里走时，手腕上的袖口突然被一只小手给扯住。
<br/>
<br/>        童筱颖聋拉着脑袋，低沉地说道：“大哥……你答应过我的……不会离开我……会永远守护着我……还有愿得一心人，白头不相离。还有……”
<br/>
<br/>        “出尔反尔的人，是你童筱颖，不是吗？你的一颗心，用在了两个男人身上，你不觉得累，我都替你累了！”傅夜沉微微侧身，冷冷地打断了童筱颖的话。
<br/>
<br/>        童筱颖声音哽咽了起来：“大哥，你爱我的同时，为什么不把信任也一并给我？”
<br/>
<br/>        “信任？是你在一次又一次地挑战我的底线！是你一次又一次投入阿景的怀中，在他怀里可以睡得那么香甜，在我怀里，你却是皱着眉头。我强迫你跟我结婚不假，但是我们俩婚后的这段日子，你被我几次捉‘奸’在床，你可记得清楚？而每一次，你贪恋完阿景的温暖后，又可以做到继续若无其事地跟我在一起。你心里，到底是怎么想的？童筱颖，我可以忍受你不爱我，但是我忍受不了，你一边假装来爱我，一边又跟阿景藕断丝连！”傅夜沉一把拽住她的手腕，将她拉到自己跟前，黑眸里布满血丝，铿锵有力地质问。
<br/>
<br/>        原来，她这段日子对他的喜欢，在他眼里，不，在他心里，只是她假装出来的爱。
<br/>
<br/>        童筱颖的心猛地揪了一下，突然间，她彷佛开窍了一般，强颜欢笑起来，眼里含着眼泪，却故作爽朗地回答道：“我们之间的确应该结束这段荒唐的婚姻。”
<br/>
<br/>        他们两是今天最后一对办理离婚证的夫妻。
<br/>
<br/>        傅夜沉将办理离婚证所需的资料都带齐了，唯独缺了她童筱颖的户口本，但是他傅夜沉的权利摆在这里，就算没有她的户口本，这婚也离成功了。
<br/>
<br/>        最后，她连那本离婚证也没要，便先行离开了民政局。
<br/>
<br/>        当她没走多远，感觉到身后有人追上来，心里腾地升起一丝欣喜之际，她的手腕再次被那道熟悉的力度抓住，随之一张冰冷的银行卡，落入了她的掌心。
<br/>
<br/>        “这是你应得的。”傅夜沉将银行卡放入了童筱颖的手中后，松开了她的手，然后对她毫不留恋地转身径直上了一旁早已停在路上等候他多时的越野车。
<br/>
<br/>        童筱颖微微垂眸，面无表情地看着手里的这张卡。
<br/>
<br/>        还真是讽刺……
<br/>
<br/>        原来，她对他付出的真心，只值一个亿啊！
<br/>
<br/>        有钱人的世界，她还真不懂。
<br/>
<br/>        当天，她将这张银行卡附上写有密码的纸，以傅夜沉的名义捐赠，直接打包快递寄给了当地红十字会。
<br/>
<br/>        回到家后，这天晚上，童筱颖躺在床上捂着耳朵翻来覆去，难受地蜷缩着身子，只觉耳朵里有无数只蚂蚁在撕咬着她一般，痛了一整晚。
<br/>
<br/>        第二天天亮后，这个世界，于她而言，好像从此安静了。
<br/>
<br/>        童筱颖去了医院才知道，自己现在就医已经晚了。
