
<br/>　　<br/>　　    王惊蛰闻言，顿时眯了眯眼睛，说道：“我肯算了，您未必会算了吧？放任墨家巨子这么个仇家在外面，我怕是睡觉都要睡不着了，与其防狼千日不如将狼一棒子打死，省心了……”<br/>　　<br/>　　    墨家巨子笑了笑，手一直搭在凉绣的脑袋上，他很风轻云淡的说道：“若是你家长辈过来和我说这话，我可能还有几分忌惮，至于你么？”<br/>　　<br/>　　    这老人说完，又看向了孔德荣说道：“你们孔府中人对你我是不知道的，所以我也不必把你放在心上，若是换成衍字辈的老家伙们来我可能还会当回事，而且还得是两三个才行，你这一辈的人还是算了吧”<br/>　　<br/>　　    孔德荣当即皱眉，抿了抿嘴唇却没还嘴，要是换个人说这话他可能当场就给顶回去了，但一代墨家巨子说的，他还真就得听着了。<br/>　　<br/>　　    老人随后又朝向茅小草，看了半天后，叹了口气，说道：“绣儿唯一的败笔就是在你身上了，千防万防，却疏忽了她最不在乎的一个环节，也是由此万劫不复了！”<br/>　　<br/>　　    “把女子和小人相提并论这话可不是平白无故说出来的”小草指了指自己，说道：“我都清纯了多少年了，偶尔就精明了这么一回，没想到却让你们家的凉绣姑娘赶上了，只能说啊，她命中该有此一劫罢了”<br/>　　<br/>　　    这老人点了点头“嗯”了一声，似乎挺赞同她的话，随即说道：“姑娘，说来她对你是存了坏心，但总归没有要你命的意思，掠走之后也并未为难你，我之前说这一页能不能掀过去，真是诚心诚意问你们的，现在我在问一嘴，放过这孩子，给他解了蛊，从今以后我带她归隐山林直到老死为止，你们若是不信我可以墨家先祖发誓，如何？”<br/>　　<br/>　　    “誓言都是狗屁，这年头多少人都发了誓也没见被天打五雷轰啊，老人家我看还是算了吧”茅小草拧着眉头说道：“斩草还是得除根才行，你今天说放了我们，万一哪天回想起今日的种种，你再怒上心头可怎么办啊？”<br/>　　<br/>　　    墨家巨子脸色当即一垮，声音微怒的说道：“你们真是不打算放过我这老人家和一个都快疯疯癫癫了的姑娘？”<br/>　　<br/>　　    凉绣此时真是有点魔怔了，两眼呆滞，喃喃自语不停，明显是之前的刺激，让她的大脑处于混沌的状态了，真有点失心疯的意思了。<br/>　　<br/>　　    茅小草咬牙说道：“我都说了，唯小人和女子得防着，您是会算了万一哪一天她好了后在心生报复，我们还得防着贼呢……”<br/>　　<br/>　　    小草一说完，突然间猛的就向后退了过去，王惊蛰的余光瞥见她动了后，就知道这是该动手了，忽然就伸手朝着墨家巨子一掌拍了过去。<br/>　　<br/>　　    孔德荣当即一愣，哪里想到这对男女说动手就动手，一点点的防备都没有，太突然了。<br/>　　<br/>　　    小草轻声说道：“孔先生，正面碰上墨家巨子可是百年能难得一遇的巧事，错过这个一网打尽的机会，以后恐怕再难遇到了，你还等什么呢？”<br/>　　<br/>　　    “啪”<br/>　　<br/>　　    此时王惊蛰一掌过去后，正好跟墨家巨子对上了，这老人看着仿佛风烛残年一样，可抬手之间就将王惊蛰那一巴掌给挡了回去，力道之大愣是让他退了两三步。<br/>　　<br/>　　    孔德荣听闻茅小草的话，心里稍微盘算了几下，然后就下定决心，果断出手了。<br/>　　<br/>　　    孔德荣和王惊蛰同时出手夹击之下，墨家巨子就稍稍往后退了一步，然后再次抬手，抬手间微微一晃，就看见他唐装宽大的袖口中闪过两道寒芒，随即两把寸许的小刀直接奔着王惊蛰和孔德荣的面门就急射而去。<br/>　　<br/>　　    王惊蛰反应稍快，见这老头抬手却没动就知道有猫腻，看都没看对方是什么动作，就猛地偏过了脑袋，那把小刀几乎是擦着他的鼻梁飞过去的，然后“噗”的一声插在了他后面的墙壁上，他反应虽快，但孔德荣离得有点近，就有点措手不及了，只是往下低了低脑袋，就感觉脑袋上一凉，脑瓜皮顿时给削掉了一块，头发都飞下来一撮。<br/>　　<br/>　　    墨家巨子伸手一拉凉绣就将她从椅子上给扯了起来，然后连续往后退了几步抬腿朝后踹在了房门上，拉着她就退了出去。<br/>　　<br/>　　    这老头你别看一副老态龙钟的模样，但动作真特么麻利，拉着一个人瞬间就出去几米外了。<br/>　　<br/>　　    “穷寇莫追啊……”孔德荣皱眉说道。<br/>　　<br/>　　    茅小草语速极快的说道：“不追？开什么玩笑不追，凉绣犯险要八字借命还不算，最想要的是一页生死薄，为的就是多延长几年寿命，她年轻力壮的还不至于吧？所以，凉绣为的肯定是墨家巨子，这老头应该是强弩之末了，你们只要能困住他，耗都能把他给耗死了……”<br/>　　<br/>　　    小草说的还真没错，凉绣找上韩观山，为的不只是八字借命那几年，她想要的更多，十年八年都不止，为的就是她爷爷，这一代的墨家巨子续命，所以这才有了墓崖村之后，凉绣依旧为了八字借命而算计王惊蛰的后续。<br/>　　<br/>　　    墨家巨子，最多至少还有不到十年的阳寿了，因为差不多十多年前的一场交锋，让他受伤之后有了暗伤到现在都没有治愈，精气神一直都在超常耗费，本来至少能活个八九十岁的墨家巨子，差不多七十来岁左右就得寿终正寝了，不然凉绣何至于抓着八字借命不放？<br/>　　<br/>　　    听闻小草这么一说，王惊蛰和孔德荣当即就明白了，他俩知道在这种情况下，不至于和对方拼命，只要浪费他的精力，没准就能给人拖的撑不住了。
