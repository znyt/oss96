　　“晓晓，这蛮荒三妖之中的大哥，你见过吗？”
<br/>　　苏莫周身气劲卷动，裹挟着翼晓晓，飞驰在天穹之上。
<br/>　　他对于三只蛮荒大妖中，素未蒙面的另外一位，非常的好奇。
<br/>　　光头大汉是老三，黑衣中年人是老二，那老大是谁？是什么妖兽？
<br/>　　“没有，我在圣山快三年了，都没有见过！”翼晓晓闻言，立刻摇了摇头。
<br/>　　“那你知道对方本体是什么妖兽吗？”苏莫再次问道。
<br/>　　“不清楚！”
<br/>　　翼晓晓再次摇头，又道：“不过，我知道肯定是禽类妖兽！”
<br/>　　“哦？禽类妖兽？”苏莫闻言诧异，居然是禽类的妖兽。
<br/>　　“嗯！我曾经偷偷到他所属的宫殿中看过，里面有很多奇怪的羽毛！”
<br/>　　翼晓晓道，她也不确定是不是禽类，一切只是猜测，他师父也没有告诉她。
<br/>　　苏莫和翼晓晓一路闲聊，数日之后，便回到了苍穹神宫。
<br/>　　从闲聊之中，苏莫了解到，翼晓晓作为八翼飞龙蛇的后裔，她的修炼主要是让血脉返祖，和提升等级。
<br/>　　至于人族的功法，她也能修炼，不过效果如何，就尚未可知了。
<br/>　　不过，龙腾同样是妖族都没有问题，想必翼晓晓也不会差。
<br/>　　翼晓晓被苏莫安排在了雷霆宫，由雷一安排其接受传承。
<br/>　　这个时候，九宫传人终于齐聚了，只待他们九人彻底成长起来了。
<br/>　　待龙腾、青雅、夕儿、司空炎、战乱五人的修为再提升一段时间，苏莫便可让五人辅助他修炼，强大他的五行属性之力，加速提升他的功法境界。
<br/>　　回到苍穹神宫之后，苏莫便再次清闲了起来，大多数时间都是在修炼之中渡过。
<br/>　　他只是静心等待，等待苍穹世界各大势力的宗主级人物到来，以及等待海明江的好消息传来。
<br/>　　为了加大给予各大势力的压力，苏莫将再次突破修为的消息，让门下之人故意传了出去。
<br/>　　这个消息再次如同重磅炸弹，在苍穹世界炸开，引起一片哗然。
<br/>　　武皇境四重的苏莫，已经苍穹世界无敌，提升到武皇境五重境界之后，那还得了？
<br/>　　本来有几大超级势力，已经悄悄组成联盟，打算联合起来反抗苏莫。
<br/>　　但听到这一消息之后，所谓的联盟立刻土崩瓦解，烟消云散。
<br/>　　在强大的实力压迫之下，很多人都不愿自寻死路。
<br/>　　两个月的时间转瞬即过，苍穹世界几乎是所有武皇级以上的势力，无论是妖族还是人族，九成九的宗主级人物，都来到了苍穹神宫，浩浩荡荡数万人，如同朝拜一般。
<br/>　　苏莫接见了这些人，当场宣布让所有势力臣服，并每年向苍穹神宫缴纳数量不等的灵石资源。
<br/>　　这个命令引起一众强者哗然，当场就有几十个大势力的宗主极力反对。
<br/>　　臣服就算了，居然每年还要缴纳大量灵石？这分明就是压榨和剥削！
<br/>　　这几十个大势力，愤怒不已，即便实力不敌苏莫，还是站出来反对了。
<br/>　　当然，一般的大势力，哪敢做出头鸟，站出来反对苏莫？
<br/>　　这些反对的势力，乃是由人族九大势力之一，灭世剑宗宗主牵头，一致反对苏莫。
<br/>　　没有任何的意外，灭世剑宗的宗主，堂堂的准帝强者，一代枭雄，被苏莫当初斩杀，血染苍穹神宫。
<br/>　　这一下，当真是震慑诸雄，霎时间，再也没有了任何反对之声。
<br/>　　所有人都一致认为，苏莫的这种做法非常好，更加的有条理，更加的有助于统御天下。
<br/>　　说白了，这个世界就是实力为尊的世界，只要拳头硬，便可随意的颠倒黑白，指鹿为马。
<br/>　　现在的苏莫，拥有绝对的实力，他说是黑的便是黑的，没有人会说是白的。
<br/>　　至此，苍穹神宫真正的做到了统御天下，成为了苍穹世界的霸主，苍穹世界的绝对统治者。
<br/>　　对于这些大势力需要缴纳的灵石，也是按照各大势力的级别和实力，分成了不同的档次。
<br/>　　像东洲青龙圣宗、天灵宗等势力，每年只需缴纳五千万极品灵石。
<br/>　　而人族九大势力这种级别的势力，以及同等的妖族势力，所缴纳的灵石最多，每年高达百亿极品灵石。
<br/>　　当然，有两个势力是例外，那便是北海冰宫和东海蛟龙一族。
<br/>　　前者已经穷的叮当响，苏莫自然不会再为难他们，至于后者，苏莫更不可能要其缴纳灵石。
<br/>　　另外，苏莫需要天下归心，力量凝聚，自然不会一味的剥削，也会给众势力很多甜头。
<br/>　　比如，一些对苍穹神宫比较忠心的势力，他会赐予帝级功法和武技，以及其它的宝物。
<br/>　　这些大势力中，有特别出色的弟子，可以进入苍穹神宫的宝地，提升修为实力。
<br/>　　苍穹神宫作为远古霸主级实力，九大宫之中，有很多快速提升实力的宝地，比如厚土塔，只是其中之一而已。
<br/>　　苏莫的方法，就是打一棒再给颗甜枣，让众势力臣服苍穹神宫的同时，不会生出反叛之心。
<br/>　　如此的话，久而久之，便能万众归心。
<br/>　　毕竟，他给的好处，完全不比众势力缴纳的灵石差。
<br/>　　当然，想得到帝级的功法和武技，以及来苍穹神宫修炼的机会，就要看各个势力的表现了。
<br/>　　苏莫有信心，有此种方法，可以轻易在几年之内，将各大势力和苍穹神宫牢牢的绑在一起，形成不错的凝聚力。
<br/>　　时间过得很快，转眼间又过去了一个月。
<br/>　　咚！咚！咚！
<br/>　　这一日，苏莫正在宫殿中静静打坐，殿外响起了沉闷的敲门声。
<br/>　　时隔数月，海明江终于回来了。
<br/>　　在苏莫的示意之下，海明江走进了宫殿中，他的手中还抓着一名老者。
<br/>　　这名老者身穿黑袍，身形消瘦，披头散发，两只手臂已经被斩断，身上伤痕累累，气息虚浮不已，显然遭受了重创。
<br/>　　“是你！”苏莫见到这名老者，顿时眼眸一凝，因为他认识此人。
<br/>　　此人，便是当初和阴霆一起追杀他的戚老，好像是阴罗殿的护法。
<br/>　　“宫主，你认识此人？”海明江见苏莫神情，疑惑的问道。
<br/>　　“嗯！”
<br/>　　苏莫点了点头，问道：“他会搜魂术？”
<br/>　　“不错！”
<br/>　　海明江向苏莫抱拳，道：“这几个月，我们抓捕了六千多名黄泉魔宗的余孽，经过严刑拷问，只有他一人会搜魂术！”
