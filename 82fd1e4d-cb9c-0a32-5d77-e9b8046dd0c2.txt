　　严家是阎罗星域的四大家族之一，蕴含的实力也是无比恐怖，拥有两名九劫古帝。
<br/>　　在如今赤红大陆的无数势力之中，都算得上是中游以上的存在了。
<br/>　　山峰之上。
<br/>　　在楚风眠看到这严家众人的一刻，那为首严君的目光，也看向了楚风眠的方向。
<br/>　　“此人居然也来了！”
<br/>　　严君目光看向楚风眠的一刻，眼神之中杀机疯狂的涌现而出。
<br/>　　上一次在跟楚风眠交手的时候，他们严家损失不少，就连那窥神镜，都被楚风眠所夺走。
<br/>　　吃了这样一个大亏，严君如今对楚风眠可谓是恨之入骨，只是失去了窥神镜，他在想要寻找到楚风眠，几乎是没有任何的可能。
<br/>　　今日当楚风眠刚刚落到了山峰之上的一刻，这严君便是就注意到了楚风眠的存在。
<br/>　　“君少，要不要我们出手去杀了他！”
<br/>　　几名严家弟子，看到严君眼神之中的杀意，急忙在旁边开口道。
<br/>　　“不要轻举妄动，此人可不好对付。”
<br/>　　严君挥了挥手，旋即对着身后的二位九劫古帝开口道。
<br/>　　“二位太上长老长老，能否是那二人的对手？”
<br/>　　严君开口之际，伸手指向了燕皇跟青梦。
<br/>　　以他的实力，自然也看的出来，燕皇跟青梦二人，可都是九劫古帝。
<br/>　　两位九劫古帝，在加上楚风眠，实力未必是比起他们严家一行人要弱。
<br/>　　正因如此，严君也不敢贸然出手去对付楚风眠。
<br/>　　“这二人，不好对付，那男子名为燕皇，算是大罗天星域的一名强者，不过刚刚突破，老夫一人对付他足以。”
<br/>　　其中的一尊老者，扫视了一眼随即开口道。
<br/>　　“但是那女子可不好对付，此人是圣龙宗弟子的实力，不亚于我们阎罗星域之中阎罗宗的弟子，我们能镇压住拖延住他们二人，但是无暇分身来帮助少主。”
<br/>　　各大星域虽然距离相隔甚远，但是严家毕竟也是阎罗星域的大家族之一。
<br/>　　对大罗天星域的强者，还是有所了解的，楚风眠只是一个刚崛起的新人，所以他们对楚风眠的了解不算多。
<br/>　　但是燕皇跟青梦，可都是成名已久的强者了，这两位严家太上长老，自然是认识了。
<br/>　　“这么说，那小子是圣龙宗弟子了？”
<br/>　　严君听到这话，目光闪烁，似乎是在考虑着什么。
<br/>　　他观察着楚风眠身上的气息，也看得出来楚风眠已经是突破了，正式踏入了帝尊境。
<br/>　　一身实力比起上一次跟他交手的时候只怕是要强横了无数倍了。
<br/>　　如果这严家二位九劫古帝无法出手去对付楚风眠的话，单单是凭借剩下的人，压制住楚风眠还有可能。
<br/>　　但是想要杀楚风眠，还需要另外的帮手才行。
<br/>　　“君少，慕家的人来了。”
<br/>　　就在这时，旁边的一名严家弟子突然看着远方飞来的身影，突然出声道。
<br/>　　只见十几道遁光，向着这严家众人所在的山峰之上飞了过来，其中为首的一名年轻男子，年纪跟严君相差不大。
<br/>　　在他的背后，也跟随一名老者，身上散发着滂湃的天地之力，也是一尊九劫古帝。
<br/>　　这慕家众人遁光飞了过来，便是都全部落在了严家众人所在的山峰之上，有着几分联合起来的意思。
<br/>　　“严家跟慕家的人，居然联合起来了。”
<br/>　　“为首的那是慕家的天才，慕九。”
<br/>　　“严家跟慕家，同属于阎罗星域，还都是阎罗星域的四大家族之一，联合起来也很正常。”
<br/>　　“三尊九劫古帝啊，看来这严家慕家联合起来，是想要冲着这佛寺之中的宝藏而来啊。”
<br/>　　一道道目光也看到这一幕，许多武者都议论纷纷起来。
<br/>　　严家慕家一旦联合，也是一股极强的力量了，三尊九劫古帝，接近十名八劫古帝。
<br/>　　在如今赤红大陆之中，已经是，除了大罗天星域的五行天，齐林山，沧海都三方势力之外，最强的势力之一了。
<br/>　　“慕兄快座。”
<br/>　　这严君看到慕家的众人到来，一群严家弟子都是让开了一个位置，同时一名严家弟子拿出了一把椅子，放在了严君的身旁。
<br/>　　那慕家为首之人，也不客气，直接坐在了上面。
<br/>　　“严兄，看来这一次想要得到了佛寺之中的宝藏，可不容易啊。”
<br/>　　慕家为首的年轻男子慕九，看了一眼严君，随即开口道。
<br/>　　他这一路到来，也知道在这佛寺之中，到底是聚集了多少的强者。
<br/>　　上百位九劫古帝啊。
<br/>　　他们两家联合，才不过只有三名九劫古帝，一旦是这佛寺的开启，争抢起来，绝对是无比凶险。
<br/>　　“强者虽多，但都不属于同一方势力，到时候一旦是佛寺开启，自然是各自为战，不足为据。”
<br/>　　严君扫视了一眼，突然间脸色一变的说道。
<br/>　　“不过我这想要对付一个人，不知慕兄有没有想法？”
<br/>　　“严兄想要对付谁？”
<br/>　　听到这话，慕九好奇的问道。
<br/>　　在这个时候，最为重要的一定是佛寺的宝藏才对。
<br/>　　可听严君的意思，居然是想要对付一个人？这也让他无比好奇。
<br/>　　“就是对付此子。”
<br/>　　严君伸出手，对着楚风眠指了过去，慕九目光顺着严君所指的方向看了过去，看了一眼脸色微微变化。
<br/>　　“两名九劫古帝？”
<br/>　　“那人是圣龙宗的楚风眠？严兄为何打算对付此人？”
<br/>　　这慕九见到楚风眠的一刻，也认出了楚风眠的来历，虽然楚风眠的名声还没有传到阎罗星域，但是在这赤红大陆之中，也有关于楚风眠实力的流传。
<br/>　　这个消息是三大势力一方故意放出去的，正是想要让其他人，也来对付楚风眠。
<br/>　　楚风眠拥有足以抗衡九劫古帝的实力，一身奇遇的流传，慕九也听过。
<br/>　　那其中的种种奇遇，慕九都想要得到，在看向楚风眠的时候，他的双目之中贪婪涌现。
<br/>　　“此人可不好对付。”
<br/>　　不过慕九可没有一口答应下来，楚风眠简直是一个移动宝藏，在赤红大陆之中的强者都知道这一点。
<br/>　　可楚风眠依然是足以站在这里，也证明了，想要对付楚风眠，可没有那么容易。
