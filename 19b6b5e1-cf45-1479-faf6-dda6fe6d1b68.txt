　　第50章
<br/>　　柳家小姐今年芳龄二八，生得比花花解语，比玉玉生香，从前是养在深闺人未识，如今是旧时王谢堂前燕，飞入寻常百姓家。
<br/>　　可即便是柳芷沦落到这烟花之地，从贵族落为四等贱民，却也不是谁想见就能见的。
<br/>　　首先必有一贯铜钱做敲门砖，如此才有机会踏入柳芷闺房，可距离见到柳芷，中间还隔着一道帘子。至于如何才能让柳芷走出帘子，为你倒上一杯酒，说上几句话，却要看个人本事。诗词歌赋，治世安邦随你怎么侃，只要你侃得柳小姐开心，柳小姐愉悦，她才会懒洋洋的说上一句。
<br/>　　“今日奴家倦了，还请公子明天再来。”
<br/>　　然后你明天屁颠颠过来，想要见面还需一贯铜钱，如此三番四次，最后能不能见着，还得听天由命。最后想要同赴鸳床，还得拿出一份滔天的银子。
<br/>　　一贯铜钱是什么概念？一贯等同于一两银子，四五贯就是普通农家一年的花销。
<br/>　　可即便如此，想要来花钱的人还是络绎不绝，这里面有某些心理学的因素在，类似于得不到的永远在躁动，被偏爱的有恃无恐。
<br/>　　而其实得到了就那么回事，过三五次便没甚意思，黄花成败柳，不管这位胭脂红如今被抬得多高，到最后也得和这大厅内的姑娘一样，陪酒卖笑。
<br/>　　所谓才子佳人的故事，听听而已，没那么多风流与人说。
<br/>　　不过至少如今，胭脂红还是被捧得很高。
<br/>　　像许茂一干人等，都是奔着胭脂红来的，谁如果能摘得美人芳心，可是了不起的荣耀。但到现在为止，还没人能做到。
<br/>　　许茂将龟公喊来，在托盘上放上去一块绢布，又仔仔细细放上一封信札。用虔诚的语气问：“问问胭脂红姑娘，今日是否有空。”
<br/>　　沦落到风尘之地，一般不用自己的本名，有辱祖上，所以大家纵然知道胭脂红的名字，也没人用柳芷称呼。
<br/>　　绢布作价大约是一贯，是可以当钱使的，五陵少年争缠头说的就是这个意思。信札里有一首小诗，这个时代对书生文士还是格外推崇的，至于这诗是许茂自己作的，还是花钱买的，就说不好了。
<br/>　　龟奴在大厅内走一圈，托盘上大概就是此类物事，互相之间便有比较的意思。不争不抢，胭脂红没有这么大魅力。
<br/>　　程大雷是想见胭脂红一面的，他需要扩充一下山寨人口，想要买些奴隶，而爱贤慕美之心人皆有之，他也青春年少，也想人前显贵。
<br/>　　“取笔墨来！”程大雷呼了一声，挽起袖子。
<br/>　　许茂将视线落在程大雷身上：“癞蛤蟆真想吃天鹅肉呀。”
<br/>　　徐神机见过程大雷写字，那叫一个丑啊，此刻他以怀疑的语气道：“公子，你行嘛？”
<br/>　　程大雷饱蘸浓墨，笔走龙蛇，瞬间纸上便出现一道道鬼画符一样的文字。说是文字，当真是辱没了这两个字。
<br/>　　“蛤蟆本是地中生”
<br/>　　有人将笔下纸缓缓念了出来，然后是一片讥笑，这第一句只是泛泛，黄口幼儿也能做得出。
<br/>　　“独卧地上似虎行”
<br/>　　第二句又是令众人大失所望，或者说，根本没人对程大雷报有希望，许茂发出一声嗤笑，将头扭到一边，表达自己的轻蔑。
<br/>　　“……”
<br/>　　“……”
<br/>　　程大雷将纸封了，搭在托盘上，搁下一贯铜钱，然后施施然坐下，静听风吟。
<br/>　　龟奴将托盘送进房间，众人翘首期盼着，大厅内的歌乐之声，也显得漫不经心起来。
<br/>　　“也不知道今天胭脂红姑娘会选择谁……”
<br/>　　“当是许公子吧，许公子来了有十几天，银子都抛出去不知多少。”
<br/>　　“我听说许公子找高人写了一首诗，偏偏胭脂红姑娘也是个爱诗的。”
<br/>　　大家交口结舌讨论着这些事情，热情且激动，美女与酒总是大家爱聊的话题。过不知多久，有丫头从房间出来，许多人都追随着她的身影，心情竟隐隐有些紧张。
<br/>　　丫头从楼上下来，绕过众人，偏偏走到程大雷面前，低声道：“欧公子，小姐请您到房中一叙。”
<br/>　　“这……”
<br/>　　许茂猛地睁大眼睛，满脸难以置信。程大雷施施然站起来，走到他面前，往其后背上拍了拍，然后迈着大步走向胭脂红房间。
<br/>　　“凭什么，凭什么！”许茂忍不住就说出声来：“我连续已经来了十几天，昨日明明与胭脂红姑娘说好的。”
<br/>　　“大概是他写的诗吧……”
<br/>　　“诗，什么诗？”刚才许茂只顾得表达嘲讽，倒是没有仔细看程大雷写的诗，自然，程大雷的字迹对任何一个人来说，阅读都是一种折磨。
<br/>　　许茂似乎在寻找什么，其实是想让别人把程大雷写的诗告诉他。
<br/>　　“许公子，别动！”有人高呼。
<br/>　　许茂身体保持僵硬动作，问：“怎么了？”
<br/>　　“你背后贴着什么！”
<br/>　　这人过来，将许茂背后贴着的一块小纸片揭下来，大家都凑过头来，尤其是许茂，眼睛睁得好大。
<br/>　　只见在纸片上，用毛笔简陋画着一只动物，这只动物是——蛤蟆。
<br/>　　一片寂静，众人目光对视，安静得能听到每个人的心跳。
<br/>　　“这是……蛤蟆大王……”有人忐忑不安的问。
<br/>　　“难道他就在这里？”有人左右看看，一身冷汗。
<br/>　　“不，这是千里眼顺风耳！”有人发出一声高呼：“我说什么来着，我说什么来着！”
<br/>　　许茂感觉脑袋发晕，眼睛发黑，身体不由自主的摇晃。
<br/>　　“还真有千里眼顺风耳啊！”
<br/>　　程大雷踏入了房间，木屋木几，一道纱帘拦着床帏，背后隐隐能看到纱帘后有个人影。古色古香，朴素中见精致，木桌上摆着一只花瓶，瓶内有一朵红花，芳香扑面而来。
<br/>　　“蛤蟆本是地中生”
<br/>　　“独卧地上似虎行”
<br/>　　“春来我不先开口”
<br/>　　“哪个虫儿敢吱声”
<br/>　　大厅中，已经有人凭着记忆将这首咏蛙诗缓缓念出，最后一个字落地……
<br/>　　满堂寂静。
<br/>
