　　虚空之中，突然平息。
<br/>　　楚风眠身上力量，也逐渐平息，看起来似乎是并无任何变化。
<br/>　　但是现在任由是谁，看向楚风眠身上的气息，都会感觉到了有些不同，在楚风眠的身上，仿佛是有着翻天覆地的变化。
<br/>　　度过了万雷帝劫，毁灭了雷帝，楚风眠便是有着一种蔑视一切的气质。
<br/>　　就算是在强大的强者，如今在楚风眠的面前，楚风眠都不会丝毫惧怕。
<br/>　　楚风眠就连天意的审判都不怕，其他人，更是又能如何？
<br/>　　度过万雷帝劫，楚风眠力量的转变，虽然极大，但是远远抵不上他的心态上的转变。
<br/>　　一尊强者，必须有着泰山崩于前而色不变的心态，不然的话，遇到强者，心生惧意，哪里还会有着任何可能对抗的机会。
<br/>　　终于是踏入了神力境，更是度过了万雷帝劫。
<br/>　　楚风眠的心中，露出了几分喜色，现在的他，终于已经是脱离了蝼蚁的范畴。
<br/>　　“走。”
<br/>　　度过万雷帝劫，凝聚了本命剑元，现在楚风眠的实录，又进一步，这一次前往虚空海，就算是遇到真的颜古龙帝的洞府，楚风眠也有自信，得到其中的一切机缘。
<br/>　　他的身形一动，一下子从虚空深处离开。
<br/>　　但是这万雷帝劫的影响，还远远不止如此。
<br/>　　万雷帝劫度过的一刻，简直是偌大的九域中，都有着感觉，其中隐藏的无数老古董，都是纷纷睁开了双眼，一个个的眼神中，满是惊骇。
<br/>　　“万雷帝劫，有人度过了！”
<br/>　　“是谁？什么人？”
<br/>　　“这气息，来自北域？”
<br/>　　“荒凉的北域，居然是出现了如此人物。”
<br/>　　“已经千年，没有人度过万雷帝劫了吧。”
<br/>　　“这九域中，又是一尊恐怖强者，要出现了。”
<br/>　　一些老古董，觉察到了这一股气息，都是抬起头，不由的感叹一声。
<br/>　　九域震动，毫不夸大，无数消息，在一日之间，不胫而走，传遍了整个九域。
<br/>　　尤其是北域中，北邙学院，他们感觉到的气息，更加明显。
<br/>　　“我们北域，什么时候出现了这样的人物，度过万雷帝劫，别说是度过，就算是万雷帝劫的出现，都是千年之前了。”
<br/>　　北邙学院的深处，无数老古董，都被一下子惊醒了起来，纷纷议论道。
<br/>　　一声古老道极点的声音，突然响起，语气中也露出了几分惊喜。
<br/>　　“这万雷帝劫的位置，就是我们北邙学院附近。”
<br/>　　又一尊古老声音，突然出现，语气大喜。
<br/>　　“莫非是我们北邙学院的弟子？我们北邙学院若是出现了这样的弟子，绝对是好好培养才行！度过万雷帝劫的天才，培养的好，足以助我们北邙学院跟七大宗门争锋了！”
<br/>　　很快，又是一道古老声音响起，语气中微微有些失落。
<br/>　　“不是，根据消息，内门弟子，种子弟子，核心弟子中，这几日都没有人度过生死大劫。”
<br/>　　“那是谁？距离我们北邙学院很近，莫非是地魔门的人？”
<br/>　　这声音的主人，极为惊异。
<br/>　　在北域之中，无数势力之中，最有可能出现如此天才的，只有北邙学院跟地魔门，地魔门在北域的势力，虽然不如北邙学院，但是他座第二，也无人敢去争。
<br/>　　“若是地魔门的弟子中，出现了这样的天才，必须要扼杀才行，地魔门一直被我们北邙学院压一头，但是他们的势力，也已经壮大了许多。”
<br/>　　“一旦是让这样的天才成长起来，地魔门再添一员强者的话，已经是足以跟我们北邙学院真正抗衡了。”
<br/>　　一道杀气腾腾的声音，猛然响起。
<br/>　　“这样的天才，必须要扼杀才行。”
<br/>　　一声声争论声，不断响起。
<br/>　　“安静。”
<br/>　　一道最为威严的声音响起，将这无数争论，全部都给压了下去。
<br/>　　“根据地魔门那边的消息，他们如今也在寻找着到底是谁，看来应该不是地魔门的弟子。”
<br/>　　“不是我们北邙学院的弟子，也不是地魔门的弟子，已经是得到了一些奇遇的天才。”
<br/>　　“这种没有势力的天才，更应该拉拢，将他拉拢到我们北邙学院来，我们就有着勉强跟七大宗门去抗衡的机会了。”
<br/>　　“速速派人，前去寻找此人，不能让地魔门的人抢先一步，寻找到他！若是拉拢不成，就灭杀了此人，不然迟早成为祸害！”
<br/>　　这声音，斩钉截铁，便是无可置疑的命令，他开口，无人敢反驳。
<br/>　　“是！”
<br/>　　无数声音，同时响起，便是纷纷离开，消散。
<br/>　　同时另一边，再一次地底洞窟之中，几尊强者，也在互相交谈。
<br/>　　“万雷帝劫出现了，有人度过了万雷帝劫！”
<br/>　　“此人，不是北邙学院的弟子，应该是一个得到了大奇遇的天才，找到啊，将他带到我们地魔门中来！”
<br/>　　“能够找到此人，培养起来，可以成为我们对付北邙学院的利剑，颠覆北邙学院，指日可待！”
<br/>　　这样的声音，还很多很多，几乎北域的各方势力，都在议论着到底是谁，度过了万雷帝劫。
<br/>　　拉拢，扼杀，无数不同的意见。
<br/>　　短短不到一日世间，整个北域，都突然变得热闹纷繁，偌大的北域，已经是千年来没有如此热闹了。
<br/>　　真正的九域震动，便是今日。
<br/>　　而身为这一切的核心，楚风眠，还全然不知，他度过万雷帝劫，居然是引来了如此之大的震动。
<br/>　　他只是突破空间，回到了北邙学院之中。
<br/>　　楚风眠一回到北邙学院，便是发现空戒中玉符的响动，楚风眠打开一看，便是白萱的呼唤。
<br/>　　“楚风眠，我的洞府在五等山脉云雾山脉中，你来我的洞府吧。”
<br/>　　白萱的声音，从中响起，一句说完，便是没有了声音。
<br/>　　楚风眠身形一动，立刻便是来到了那五等山脉，云雾山脉。
<br/>　　几乎无声无息之间，便是进入到了其中。
<br/>　　这一座座修炼山脉中，本来也都是布置了无数禁制，阵法的，若是以前，楚风眠也只能等待白萱将他接进去。
<br/>　　但是现在，楚风眠踏入神力境，更是度过了万雷帝劫，实力不知道是提升了多少倍。
<br/>　　这禁制阵法，在楚风眠的面前，简直如同小孩子的玩具一般，根本拦不住楚风眠。
<br/>　　楚风眠轻而易举穿越这无数禁制阵法，来到了云雾山脉之中，寻找白萱的气息，直接来到了白萱的洞府前。
