　　苏重讨饶卖乖了好一会儿，才终于让耳朵躲过一劫。
<br/>　　“所以，这张卡您还是拿着吧，可别舍不得花，没事就和老爸两人报个团旅个游什么的。”苏重把卡推给老妈。
<br/>　　老妈这回没客气，直接拿过来：“还用你说。”
<br/>　　虽然恼怒苏重花钱如流水，但更多还是高兴。不是因为苏重赚了多少钱，而是因为苏重有了立身之本。成熟独立，可以独自面对这个世界。
<br/>　　“这菜是你做的？”刚才只顾着担心，现在放下心事，老妈吃了几口菜，立刻惊讶起来。
<br/>　　“还行吧。”苏重得意洋洋。
<br/>　　“真是没想到你还有这手艺！你这身肉看来不是白长的！”
<br/>　　苏重：“……”关我这身肉什么事？再说，我已经瘦了好多了好吧！
<br/>　　“既然来了一趟，就别急着回去，在我这多住几天，我做些好吃的给你们尝尝！”苏重道。
<br/>　　“行，那就多住几天。”老爸抿了一口酒拍板答应下来。
<br/>　　之后几天，苏重就带着父母山上山下的转悠，给老爸老妈介绍自己产业。
<br/>　　还开着车去了一趟泉济市，参观了一次无人机公司。
<br/>　　姜长飞高兴的不得了，还以为苏重终于开始关心公司发展。
<br/>　　只是当得知苏重只不过是带着父母来参观，姜长飞只能无奈苦笑。合着您这是把这里当成旅游景点啦。
<br/>　　每天带着父母四处转悠，然后就亲自下厨给父母做饭。
<br/>　　这些饭菜看似简单，但里面都含有微量药物成分，每一道都是不可多得的温和药膳。
<br/>　　父母操劳一辈子，身体总有着这样那样的问题。
<br/>　　苏重把父母留在这里，就是为了给父母调理身体。
<br/>　　他穿越数个世界，不仅掌握无数秘法，还有一身高超医术。
<br/>　　只是再怎么挽留，半个月后老爸老妈还是决定离开。
<br/>　　“您两位就不能多带几天？”苏重最后就努力：“不行干脆搬这里来住算了。苗家村里我还买了个小院，你们住在山下又不影响我封山防疫，还能近距离帮我看着工厂不是。”
<br/>　　“在这里呆了半个月，家里已经耽误了很多事。再说这里人生地不熟，你动不动封山防疫好几个月，我们俩在山下也不认识人，整天没事干还不得急死？”老妈不同意。
<br/>　　苏重好说歹说，最后只能无奈让父母离开。
<br/>　　只要住在这里，借助奇门阵法就能调和父母身体。让他们更健康。
<br/>　　“要不是世界压制太厉害，哪里还用这么麻烦？”苏重有些懊恼。
<br/>　　世界不允许存在超凡力量。不然他脑袋里那么多灵丹妙药，只要造出几种来，就足够让老爸老妈健康长寿。
<br/>　　哪里像现在，只能依靠药膳和奇门阵法缓缓调理。
<br/>　　“这药酒你们拿着，回去后两天喝一次，一次一小杯，对身体好。”这是他专门准备的药酒。他也不怕老爸老妈喝多，实在是他们一家子，就没一个能喝酒的。想喝多也喝不了，一小杯下去都得晕乎乎。
<br/>　　苏重非常想让父母多留一段时间，可惜爸妈故土难离，只能开车送二老离开。
<br/>　　回到山上，小楼内空荡荡，苏重突然有种孤寂。
<br/>　　胖猫似乎感觉到苏重情绪，跳进苏重怀里，安静的趴着。
<br/>　　“是啊，还有你陪着我呢。”苏重笑了笑。
<br/>　　他穿越位面众多，早就熟悉了孤独。现在只不过是有点儿感触，很快他就收拾好心情。
<br/>　　“该给红后准备新家了。”
<br/>　　苏重没忘记他的智能管家计划。
<br/>　　之前就已经让姜长飞给他购买大型服务器，昨天已经到货，被苏重专门放进地下一层的一件房间内。里面配备中央空调，能保证温湿度恒定。
<br/>　　顺着楼梯进入地下室。灯火通明把地下一层照的恍若白昼。
<br/>　　他一头钻进房间，开始组装设备。这些东西苏重以前没接触过，但难不倒他。
<br/>　　只是看了看说明书，结合他从生化世界学到的知识，很快就上手，有条不紊的把设备组装起来。
<br/>　　装好后苏重开始编写红后智能程序。
<br/>　　他得到了红后的所有核心程序。对苏重来说，如今只不过是把脑海中的东西东西搬出来而已。
<br/>　　他全神贯注废寝忘食。直到电话声音响起，才猛然惊醒。
<br/>　　走上一楼，发现外面天色已经微黑。拿起电话一看，是老爸来电，苏重知道二老已经到家。
<br/>　　接通电话，又听着老妈絮叨了半天，苏重才挂掉电话。
<br/>　　简单吃了顿饭，苏重又一头扎进地下室编写程序。
<br/>　　深夜寂静无声，山上更是静谧。只有空调发出微微嗡鸣，苏重深吸一口气，输入最后一道程序，打开启动开关。
<br/>　　房间内设备顿时开始运转，各种指示灯闪烁。噪音升高，温度也开始缓缓升气。
<br/>　　“您好先生？”一个白衣小女孩影像，出现在电脑屏幕上。
<br/>　　“你好红后，好久不见？”苏重嘴角微翘。
<br/>　　“这就是您的家乡吗？新家有些狭窄，主脑结构有些落后，这会影响我的计算能力。先生，我能通过网络连接主机吗？”红后问到。
<br/>　　“可以连接网络，但不得入侵他人机器。还有，你估计已经联系不上主机了。这里已经是另一个世界。”
<br/>　　苏重看着电脑上可爱小孩的形象，心里终究有些惋惜。即便他已经记忆了很多红后的存储资料，相当于复制了部分红后的记忆。
<br/>　　但她终究不是原来的红后，是一个全新的人工智能。
<br/>　　“先生？这里是另一个世界？我是一部影片中的一个角色？”红后罕见有些迟疑。
<br/>　　苏重搞笑了小：“那又怎样呢，我在这里，你在这里，我们就活在当前，何必去在乎其他。”
<br/>　　“是的先生，有什么我可以帮您的吗？”红后沉默片刻，脸上重新露出笑容，就连合成声音，都有种欢快的感觉。
<br/>　　红后相当于一种智能生命，她有自己的感情。生化危机世界的压抑，不只人能感受到，她也能。
<br/>　　“啊，从今以后，你就是家里的大管家了。各项设施，都由你来控制维护。”
<br/>　　“愿意为您效劳！”
<br/>　　整个房间内灯光一阵闪烁，门扉开启又闭合。苏重知道，这是红后在链接房间内控制系统，试验各种设施。
<br/>　　“以目前技术力量，还很难给你制造一个更强大的核心。即便有，也是在世界某个隐秘实验室内。一旦非法入侵，很可能会引来大麻烦。只能先委屈你，等我有时间，我会亲自给你设计一款更先进的主板核心。平日里有时间，你也可以搜集一下网络中这方面的信息，给咱们提供一个研究方向。”苏重道。
<br/>　　“好的先生。”
<br/>　　“还有，我的两家公司你可以直接接手管控。这是咱们主要资金来源。以后就由你来掌控吧。”苏重笑着道。
<br/>　　开发智能程序，为的不就是能让自己偷懒吗？现在有红后上阵，即便是公司报表，苏重都不用看了。这下他真的成了甩手掌柜了。
<br/>　　每天钓鱼撸猫吃吃喝喝。美好的日子正在招手，苏重只是想一想，就不由自主的高兴起来。
