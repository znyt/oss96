　　“行了，内院需要注意的事情就这些，天色也不早了，你们收拾收拾屋子就能休息，在内院中，你们随意，相信不会让你们失望的。”
<br/>　　将一些事情交代完毕后，夏管事就转身离去。
<br/>　　内院中并没有修炼课程，更没有导师来教导你修炼，想要变强，唯独只有依靠自己。
<br/>　　只要拥有足够的火能，便能够进入天焚炼气塔中修炼，在那里修炼，比任何课程都管用，甚至外院的导师都向往那里。
<br/>　　另外，内院中还有‘斗技馆’，‘功法阁’等等场所，学员想要获得斗技或者功法，就可以去那里。
<br/>　　当然，这也是必须在，拥有足够火能换取的前提条件下，与外院相比，内院更加自由，完全是放任式管理。
<br/>　　其中，赚取火能的方法也有很多种，奖励的火能越多，就需要相对应的实力。
<br/>　　...
<br/>　　...
<br/>　　只是稍微的打扫了一下，卧室就能住了，其中本就很干净，估计以前有人定期来这里打扫。
<br/>　　打开窗户，一股清新空气便涌了进来，整个房间中就充满了生气。
<br/>　　虽说是两室一厅，但另一间卧室，慕清霄压根就没有提起。
<br/>　　两人住在一起，早已成为了习惯，根本没必要去分开，况且，他也舍不得分开。
<br/>　　半个时辰后，吃过晚餐，夜幕降临，皎洁的月光倾洒而下，微光笼罩着整个内院。
<br/>　　小阁楼中，卧室中充满温馨的气息。
<br/>　　轻轻伸手搂住她的纤腰，慕清霄坏笑一声，道：“仙儿，我们休息吧，明日一早就在内院中转转。”
<br/>　　见他不怀好意的笑容，小医仙柔美的脸颊上绯红如霞，微微点头，任由他抱着走上床榻。
<br/>　　紧接着，卧室中响起一阵令人面红耳赤的声音。
<br/>　　夜以至深，皎洁的月光穿过窗檐，倾洒在卧室中，床榻上面，慕清霄抱着沉沉睡去的佳人，思绪万千。
<br/>　　在进入内院后，他就一直能感觉到，在西边方向，有一股熟悉的气息，异火的气息。
<br/>　　“明天先去看一下吧，尽快下去一趟最好。”
<br/>　　抛开杂念，慕清霄正准备闭上双眼，却感觉到，胳膊处的七彩吞天蟒微微扭动身躯，爬了出来。
<br/>　　望着可爱的小家伙，慕清霄伸手就去抚摸它的小脑袋。
<br/>　　然而，小家伙并不像平时，摇着漂亮的小尾巴凑过来亲昵，反而在空中顿住。
<br/>　　当慕清霄手掌触碰到它小脑袋的瞬间，它的淡紫色瞳孔中闪过一丝复杂，随后一阵强烈的紫光，自其身体中透射而出。
<br/>　　慕清霄微微一愣，光芒闪烁间，七彩吞天蟒的体积瞬间变大起来，同时，似乎变得更加柔软。
<br/>　　就在他愣神之际，左手空荡荡的感觉消失，取而代之的是一具柔软至极的娇躯，就这样压在他的身上。
<br/>　　回过神来，映入眼中的是一副完美无瑕，妩媚天成的面容，美丽的淡紫色瞳孔中，尽是复杂之色。
<br/>　　“你...恢复了吗？”
<br/>　　慕清霄并不感到意外，他早就知道，美杜莎醒来只是时间问题，左手非常自然的搂上她的纤腰，轻轻的问道。
<br/>　　此时，两人以一种极为旖旎的姿势，躺在床榻上，慕清霄在下，美杜莎在上，胸前硕大的峰峦，挤压在他的胸膛上。
<br/>　　虽然有些危险，但慕清霄却觉得极其具有快感，眼中没有一丝惧意。
<br/>　　“早就醒来，只是灵魂未完全恢复罢了，先让本王起来。”
<br/>　　瞥了眼左侧沉睡的小医仙，美眸盯着面前神骏而熟悉的面容，美杜莎心中有些无奈。
<br/>　　自己确实进化成功了，但身体却多出了另外一个灵魂，还非要跟在面前的男子身边，且言听计从。
<br/>　　其实，一周前她就已经醒来，亲眼目睹了，慕清霄在黑角域枫城，将四星斗宗轰杀成渣的场面。
<br/>　　虽早就见识过他的强大，但斗宗与斗皇根本就是另外一个层次，否则她也不至于冒着生命危险，想要蜕变。
<br/>　　但慕清霄八星斗皇，虐杀四星斗宗的画面，让她思考了许久，或许，这就是血脉天赋的差距。
<br/>　　与生俱来，根本没有可比性，哪怕在如何努力。
<br/>　　而且，由于七彩吞天蟒的缘故，跟在他的身边久了，她心中也渐渐升起一丝异样的情感，这种感觉，她自己都说不上来。
<br/>　　感受到挤压在胸口处的柔软，望着近在眼前的倾城佳人，一股邪火，自慕清霄小腹处升腾而起。
<br/>　　两人面对着面，相隔半个拳头的距离，彼此的呼吸，清晰可闻。
<br/>　　幽香钻进鼻孔中，可能是因为小医仙缘故，压抑的太久，慕清霄竟情不自禁的，朝那诱人的红唇吻了上去。
<br/>　　美杜莎微微一愣，显然没想到，他会如此乱来，一时间竟没去躲闪，回过神来，一条舌头已经带着炙热的气息，钻进她的口中。
<br/>　　“唔...!”
<br/>　　淡紫色美眸瞪如铜铃，其中尽是不敢置信，却不知为何，又不想拒绝。
<br/>　　卧室中渐渐升温，两条舌头痴缠在一起，彼此交缠......
<br/>　　片刻后，慕清霄坐起身来，左手搂着她的纤腰，右手放在她的发髻中，径直向着屋外走去。
<br/>　　推开房门，径直来到另外一间房间，压抑的邪火早就忍耐不住。
<br/>　　来到床榻上，两人忘情拥吻，慕清霄抽出左手，熟练的解开紫色锦袍上的纽扣，雪白的事业线，使得邪火一瞬间，涌上一截。
<br/>　　纽扣陆续解开，雪白的峰峦已经隐隐可见，左手伸进锦袍中！
<br/>　　“咝咝...”
<br/>　　可就在下一秒，舌尖就传来一股刺痛，唇齿分离，慕清霄舔了舔嘴角的血迹，望着身下的佳人，脸色有些尴尬。
<br/>　　“怎么了？”
<br/>　　“我...我要休息了。”
<br/>　　此时，美杜莎已经完全清醒过来，但绝美的脸颊上，布满羞涩的红晕，说罢，娇躯上就闪烁起紫光......
<br/>　　下一秒，望着身下漂亮的七彩色小蛇，慕清霄脸上肌肉狠狠的抽了抽，坑死人不偿命啊！
<br/>　　见主人盯着自己，七彩吞天蟒淡紫色瞳孔中充满了无辜，吐了吐蛇芯，蹭了蹭他的手心，慢慢爬上他的手腕。
<br/>　　叹息一声，慕清霄无奈极了，身边两个倾城美人，一个不能碰，一个不让碰，简直要人老命啊！
