　　Derrick转身朝着办公室门走去，在握住门把手的时候，Derrick对站在办公桌背后的霍斯曼德说道：“其实你也跟我一样，迷恋着queen。桌子上两瓶东西就送给你了，你可以试着自己吃下一粒，然后再跟queen发生点什么，这样queen或许就会死心塌地的爱上你。”
<br/>　　霍斯曼德看着自己办公桌上那两瓶“水晶珠”，有那么一刻，他居然对Derrick的提议有些心动，但下一刻他就惊醒过来，看向桌子上那两瓶“水晶珠”就像是看到了两只面目狰狞的恶魔一样，伸手抓起桌子上的两个瓶子扔进了办公桌旁边的垃圾桶中，唯恐避之不及。
<br/>　　霍斯曼德跌坐在椅子中，喘着粗气，好不容易才把自己起伏的情绪平定下来，冷静下来之后，霍斯曼德又朝着办公桌旁边的垃圾桶看去，看着垃圾桶中那两瓶淡蓝色和粉红色的水晶珠，最后还是弯腰把两个瓶子捡了起来，放进了自己的抽屉中。
<br/>　　虽然霍斯曼德不会承认，也不敢承认，但是他确实是对Derrick刚才的提议有了那么一些心动。
<br/>　　这个世界上有那么一种女人，会让人想要去征服，即便是粉身碎骨也在所不惜，玛丽科娃就是这样的一种女人。
<br/>　　收拾好情绪，霍斯曼德打开笔记本电脑，跟玛丽科娃建立了视频通讯。
<br/>　　“事情怎么样了？”玛丽科娃抚摸着自己大腿上的白色波斯猫，对霍斯曼德问道。
<br/>　　“他拒绝了。我私自给他增加了一千万美金，以六千万美金的价格，想要买断他手中‘水晶珠’配方，可他还是拒绝了。”霍斯曼德摇头说道：“他提出了两个条件，他说满足任何一个条件，他都可以交出‘水晶珠’的全部配方。”
<br/>　　“什么条件？”
<br/>　　玛丽科娃垂下眼眸，逗着波斯猫，眼睛中寒星闪烁，杀机涌动。
<br/>　　“第一个条件，他开价一亿两千万……欧元。”
<br/>　　“胃口很大。”玛丽科娃显得波澜不惊，好似Derrick这个要价并没有被她放在心上，继续逗弄着自己大腿上的波斯猫，问道：“第二个条件呢？”
<br/>　　“第二个条件……”
<br/>　　霍斯曼德犹豫着要怎么开口，他不知道玛丽科娃在知道Derrick这第二个荒唐的条件时，会是有什么样的反应。
<br/>　　“说！”
<br/>　　“第二个条件，他愿意以六千万美金的价格出售‘水晶珠’配方，但是他还有附加要求。”霍斯曼德犹豫了一下，说道：“他要你跟他一起走。”
<br/>　　玛丽科娃抚摸波斯猫的动作停了下来，抬起眼眸，脸上竟然露出笑容，只是玛丽科娃此时脸上的笑容，却是让霍斯曼德感觉到了无尽的寒意。
<br/>　　“没想到我只值这么一点钱。”
<br/>　　玛丽科娃放在波斯猫脖子上的手，猛地用力，趴在她大腿享受着温柔待遇的波斯猫，凄厉的惨叫一声，四只脚奋力的挣扎起来，想要从玛丽科娃手里逃脱。
<br/>　　感觉到猫爪子抓在自己身上的痛楚，玛丽科娃松开了自己掐在猫脖子上的手，重新获得自由的波斯猫跳下玛丽科娃的大腿，逃到了一个角落里，不敢再接近玛丽科娃。
<br/>　　“后面应该怎么做，你应该很清楚，不需要我再教你。”玛丽科娃看着电脑屏幕中的霍斯曼德，说道：“在没有足够的实力之前，就要学会妥协。学不会妥协，就意味着会被摧毁。”
<br/>　　玛丽科娃结束了跟霍斯曼德的视频通讯，对着藏在一旁角落中的波斯猫招了招手。波斯猫显然是对刚才的事情还感到害怕，一时半会儿没有动，直到玛丽科娃再次招呼它的时候，跟一团雪球一样的波斯猫才跑到玛丽科娃身边重新跳上她的大腿。
<br/>　　玛丽科娃抱着波斯猫，靠在椅子上，闭上眼睛，她眼前再次出现了当年她世界小姐夺冠后所参加的酒会，就是这个酒会改变了她的人生轨迹。
<br/>　　她在这个酒会上认识了以影视投资人身份参加酒会的小丑，她被小丑过人的见识和丰富的阅历所吸引。在酒精的推动下，他们发生了关系，第二天早上，当玛丽科娃醒来，看着床单上那一滩殷红时，懵了，彻底懵了！
<br/>　　小丑在离开酒店房间时，留下了一个电话，并且告诉玛丽科娃，女人的美丽是上帝的恩赐，这是上帝送给天使的礼物，但也是一把罪恶之刃。如果没有能力守护好自己的美丽，这种美丽就会摧毁掉拥有它的人。
<br/>　　玛丽科娃一开始根本没有打算要拨打小丑留下的那个电话，对于她跟小丑之间所发生的事情，也只是当成一次意外，当成了以后进入娱乐圈会发生的事情，提前上演。
<br/>　　然而事情却并没有玛丽科娃想的那么简单，两个月后，玛丽科娃发现自己怀孕了，孩子是谁的不言而喻，她纠结着是不是要拨通自己钱包中纸条上的那个电话，但最后玛丽科娃还是放弃了，她打算自己把孩子生下来，然后抚养长大。
<br/>　　玛丽科娃的父亲是个赌鬼，欠下了一大笔他根本无法偿还的赌债，遭受到赌城老板的威胁，玛丽科娃的父亲选择把自己的女儿拿来抵债。
<br/>　　有了两个多月身孕的玛丽科娃选择反抗，途中跳车逃跑，这直接导致了玛丽科娃的流产，被追得走投无路的玛丽科娃最终还是拨通了小丑留下的电话。
<br/>　　很快一架直升机出现在玛丽科娃头顶上，一个接一个全副武装的佣兵跳下直升机，在自动步枪的扫射下，那些追捕玛丽科娃的保镖全部遭到射杀，玛丽科娃被小丑带走。
<br/>　　玛丽科娃流产导致了大出血，最后她的命虽然保住了，但是她被确诊为终身无法再怀有身孕。
<br/>　　获悉自己终身不孕的那一刻起，世界小姐玛丽科娃消失了，取而代之的，是后来的国际通缉要犯“毒皇后”！
<br/>　　玛丽科娃变得冷血和残忍，她从医院离开后，杀掉的第一个人就是她的赌鬼父亲！她彻底认同了小丑跟自己说过的那句话，女人的美丽是上帝的恩赐，它是上帝送给天使最好的礼物，但同时它也是一把罪恶之刃！
