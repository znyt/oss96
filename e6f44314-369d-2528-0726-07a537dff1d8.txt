        &lt;h3&gt;第三卷第五十五章  寒伯安的愿景&lt;/h3&gt;
<br/>
<br/>        寒伯安酷爱收藏和喜欢射击，经常去射击点练习一下他的射击技术，今天约了三五个好友去射击馆，他基本都是九环或者是十环，看来这一段时间的忙碌并没有让自己的技术倒退。
<br/>
<br/>        和朋友们分别之后，他开着车回到公司，将车停到车库，他并没有及时下车，而是很惬意的靠在座椅上，点起一根雪茄，吸了一口，吐出一团圆圆的烟圈，白色的烟雾飘飘袅袅在空中飘曳着。
<br/>
<br/>        他想到了去西城和兰州的事情，西城的那个师傅很是健谈，将西城的出租车行业细细的给他介绍了一番，也带着他走访了一下小型的物流部，西城有一个专门的市场，哪里就是一个集散中心，来来去去的货物都会从哪个集散中心分流出去，只是管理混乱，每个部分都是各自为阵，他决定将他们统一收购，进行集中管理，但是每个地区的实施方案也不一样。
<br/>
<br/>        他心里还有一个模糊的想法，改变出租车行业状态，提升服务质量，让出行的人有更好的出行体验。、
<br/>
<br/>        随着社会的发展，随着经济水平的提高，人们的消费观念也会提高，所要求的服务质量和服务体验也会提高，如何让更多的人能够感觉到出行更满意和更舒适呢？
<br/>
<br/>        寒伯安决定在深入的调查一下。
<br/>
<br/>        回到他那豪华的总裁办公室，他叫来了助手。
<br/>
<br/>        “我需要你帮我去做一个民意测验。”
<br/>
<br/>        “什么民意测验。”
<br/>
<br/>        “在全国各地进行出现满意度调查，比喻说打车？”
<br/>
<br/>        “你是要全国各地进行都进行调查？”
<br/>
<br/>        “是的！”
<br/>
<br/>        助手并不知道他们睿智的总裁为什么要进行人们的出行满意度调查，他们每次出门都是专车接送，飞来飞去，很少坐出租车的。
<br/>
<br/>        所以老板让他调查出租车行业是个什么道理？虽然有疑问，但是他还是很快去执行了，他先是从网上进行了调查，当拿给老板看时，寒伯安皱着眉头：“这就是你给我的各地的调查？”
<br/>
<br/>        “我是从网上调查的。”
<br/>
<br/>        “不到一百个人的想法，这也叫民意？”
<br/>
<br/>        “我需要确切的数据，不是这样模糊的概念和想法。”
<br/>
<br/>        “那？”
<br/>
<br/>        “亲自去考察、体验或者是查访。”
<br/>
<br/>        “是！”老板要的东西似乎和他想的不一样。
<br/>
<br/>        经过几个月之后，助手拿到了厚厚的一沓调查问卷，放在了寒伯安的面前。
<br/>
<br/>        寒伯安仔细的看了一下这些调查，很多的都是吐槽出租行业的服务糟点，服务态度差，拒载或者是拼坐。
<br/>
<br/>        人们普遍希望能有一次愉快的出行，希望能够安全、快捷、舒心。
<br/>
<br/>        寒伯安的那个想法越发清晰。
<br/>
<br/>        计划也提前到来，每一个时代都会有一小部分人走在时代前沿，他们都会进行不断的尝试和变革，寻找合适的机遇，等待最好的时机，他们从来都不会坐以待毙，从来都不会躺在安逸窝里寻求自我安慰和自我保护，他们总是会不停的接受挑战，不停的实现自己的价值。
<br/>
<br/>        寒伯安决定和陈子昂见面聊一聊。
<br/>
<br/>        他将调研报告拿给郝景文看。
<br/>
<br/>        “大哥，你想另辟捷径？”他们的业务竞争愈来愈激烈，虽然他们还是站在前沿，但是他们已经感觉到了危机，每个行业都有一定的周期性，他们必须寻找新的突破口。
<br/>
<br/>        “不，我只是想拓展业务。”行业虽然竞争激烈，但是他们还是保持着大佬的位置，优胜劣汰的规则适用于任何一个行业，现在能留下来的就是几家了，谁能抗到最后谁就是赢家。
<br/>
<br/>        “这个行业目前服务满意度不好，可以开辟一片新的天地，你让财务看看我们公司的资产情况，能拿出多少资金。还有联系一下陈子昂，问一下那个app软件可不可以在进行修改。”
<br/>
<br/>        郝景文毕竟年龄小了那么几岁，社会经验还没有寒伯安丰富，但是经过寒伯安微微的一提点，他突然就恍然大悟，他们现在做的是面对大众，如果改善出租业的服务质量那么面临的人群又会是不一样的，虽然都是和运输相关，但是确又是另外的一个产业链。
<br/>
<br/>        只是想要改变另外的一个产业链，怕是会碰头的。
<br/>
<br/>        “这个事情恐怕比我们现在做的事情要困难很多，毕竟取代的人群也是不一样的。物流只是互联网衍生下来的新生事物，但是，这个行业却关乎到了很多人的利益。”
<br/>
<br/>        “适者生存，社会的发展需要大家都做出改变，他们还一直在原来的那种状态下生活，不进行整顿和改变，那么就不要怪别人来抢他们的饭碗。
<br/>
<br/>        随着社会的发展，人们的生活质量提高了，相对应的服务体验也会要提高。你想一想啊，当你旅途劳累了，想叫一辆出租车，可是上车之后，车子卫生不好，而且服务态度又不好，你的心情是不是越来越差？
<br/>
<br/>        你再想一想，当你累了，有人穿着西装打着领带，给你开车门，你上车之后还有轻柔的音乐，你的疲劳感是不是就会消失？取而代之的是满满的幸福感？”
<br/>
<br/>        “嗯，你说的很对，现在人们的生活水平都提高了，人们的需求确实是在改变着。我们这个事情......”
<br/>
<br/>        “有得做！”俩个人同时说出来后面的几个字。
<br/>
<br/>        “那我下午联系俞轩和陈子昂。陈子昂已经研发出来了新产品，真在准备投入使用，前期在找人试用，后期大量投入的时候，份额不会很低！”
<br/>
<br/>        “他们研发的我们就不参与了，毕竟当初开始的时候并没有参与。目前做好我们手上的事情。”
<br/>
<br/>        “我们现在的资金量是可以托起我们的这个项目的，不过软件部分需要陈子昂他们来做，我建议我们还是按照原来的方式走。”
<br/>
<br/>        “可以按照原来的方式走。我相信陈子昂也会同意的。”
<br/>
<br/>        “陈子昂怕是没有时间来操作这个事情，估计她得交给轩轩了，她要去甸城开展业务了。”
<br/>
<br/>        “交给他们两个谁都一样，反正我们已经是合作关系了。”
<br/>
<br/>        什么叫志同道合？在生意场上，能合作的都是彼此很优秀的人，他们往往有相同的志向，而且彼此信任，他们能预知未来，能在时代洪流中站稳脚步，能在改变中发现商机，他们做的不仅仅是生意，而是一种情怀。
<br/>
<br/>        他们年轻，他们热血澎湃，他们是社会中的一种现象，他们代表着年轻一代奋勇向前的精神，他们大胆尝试，在他们的字典里没有失败，他们心里都藏着一个没有说明的梦想和情怀。
