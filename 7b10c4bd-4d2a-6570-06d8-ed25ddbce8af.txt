<br/>				陈默和李瑞通完电话后，又接着给凌经理打了电话。既然要将鱼肉上市销售，那么两个供应商也就不能一个通知了一个不通知。<br/>　　<br/>　　“陈先生，你好啊！”凌经理见电话显示上是陈默，立刻接起电话并问好。对于陈默，已经被港区列入了重点名单。<br/>　　<br/>　　陈默将自己鱼塘中的鱼类和重量，还有鱼类的价格，给凌经理说了一下。<br/>　　<br/>　　“价格不是问题，你有多少数量，我们港区这里全部都承包了。”<br/>　　<br/>　　凌经理霸气的说道。对于陈默他那里出产的食材，可以说是非常的认可。这种认可，不仅是在监测上，还有客户的口碑上，只要是看到上个月港区餐厅的销售额度，就能知道他们港区赚了多少钱。<br/>　　<br/>　　港区的高层，不仅在开会上将陈默这个供货商，列入重点关系户，并且是重点关注，只要陈默哪里出产什么，第一个就要弄回来。<br/>　　<br/>　　“哈哈！凌经理，谢谢你这么认可我的东西。不过全部给你们港区，还是不成的，主要是另外还有一个销售商要供货。”陈默笑着说道。<br/>　　<br/>　　凌经理是知道这个销售商的，都已差不多一天签订的供销合同，并且还知道这个人是陈默的同学，也就只能认了。<br/>　　<br/>　　“那也只能这样了！不过陈先生，你可一定要给我保证啊，就两家销售商了，不能再增加了。”<br/>　　<br/>　　陈默听到以后只能呵呵了，如果真的要是这么做了，那么不就是棒槌了么。万一有人过来再给价格高些，只要不违反合同就成。<br/>　　<br/>　　要是一直保证只是两家经销商，其中一家好在是自己的同学，要不然，陈默绝对会再找几家销售的。<br/>　　<br/>　　因为一但货源被两家联合起来控制，那么就轮到陈默成为打工的人了。这不是没有发生过，做生意的，就没有什么节操可言。<br/>　　<br/>　　就算是有合同又能怎么样，有的时候就是会违反合同，只要控制了进货渠道，那么价格还不是谁控制谁说了算么？到时候陈默除了种菜以外，还真的没有其他动作可言！<br/>　　<br/>　　所以对于凌经理这样的话语，陈默也只能打哈哈绕过去。凌经理也是老江湖了，咋么不可能听出陈默的否认，但是也没有说什么。谁都不是傻~瓜，真要是那样的话，恐怕凌经理还真的不敢和陈默合作。<br/>　　<br/>　　与凌经理说好条件，得到确认后也就这样了。说的再多也没有用，还不如用实物说话，只要鱼做成菜以后，经过品尝才能知道到底好不好。<br/>　　<br/>　　陈默挂断电话后，又给大黄和大灰准备了些吃的。现在大黄和大灰的饭量可以说是与日俱增，不过好在陈默并没有喂什么狗粮，而是弄了只在乾坤珠内养殖的鱼，和昨天剩的饭菜，混合了一点溪水以后，直接喂食。<br/>　　<br/>　　对于狐狸，也就更加的简单，直接用乾坤珠内的鱼就成。两只小狐狸对于体长有些和自己差不多的鱼，似乎有些无从下口，只能围在鱼的周围，不是上去咬上一口，但是却更不没有任何的作用。<br/>　　<br/>　　顿时急了，小狐狸直接使用最大~法宝，转身对母狐狸哼唧的叫着，这让陈默有些想笑，这两只小狐狸还真是聪明。<br/>　　<br/>　　母狐狸优雅的走了过来，是的，优雅的走了过来！虽然左前爪有伤，但是因为它不是人，所以陈默能够想用什么药就用什么药，所以母狐狸的伤口恢复的相当快。<br/>　　<br/>　　就这么几天的功夫，身体上所有的伤口都已经愈合，而且就算是毛发，也是长出了一层短毛，再过一阵，就完全看不出这只母狐狸受过伤。<br/>　　<br/>　　母狐狸用一只前爪，将鱼按住，然后一口口的鱼撕裂开来，小狐狸立刻凑上前来，开始吃起这些鲜嫩可口的鱼肉。<br/>　　<br/>　　陈默给大黄和大灰的鱼，都是换着来的，反正乾坤珠内的鱼生长快速，吃什么都成。<br/>　　<br/>　　不过给狐狸的就只是鳗鱼，因为其刺少，而且软软的，可以直接吃下去。这是他考虑到两只小狐狸，要是吃其他鱼，被鱼刺卡住了就会很麻烦。<br/>　　<br/>　　给它们的鱼，都是被陈默刚刚杀死的，所以鱼肉非常的新鲜可口。又因为是从乾坤珠内拿出来的，所以家里这些动物们都非常的爱吃。<br/>　　<br/>　　摸了摸母狐狸身上开始变得光滑的毛发，想着是不是给母狐狸起个名字？要不然都不知道怎么叫它。<br/>　　<br/>　　参考大黄和大灰的名字，陈默思索了半天，这次一定要给狐狸起个酷拽霸狂炸吊天的名字，普通名字怎么可能和这么聪明的狐狸想匹配呢？<br/>　　<br/>　　轻轻抚摸着地头喂孩子的母狐狸，然后坚定的说道：“以后你就叫小赤了！你的两个孩子就叫小一和小二！”自我感觉了一下，感觉莫名的有些良好。<br/>　　<br/>　　不过母狐狸本来好好的喂着孩子，但是却在陈默说出名字后，有些停顿，然后抬头看着陈默。<br/>　　<br/>　　母狐狸的目光，似乎、大概、可能有些揶揄，或者是鄙视？嗯？这是什么目光？<br/>　　<br/>　　陈默顿时恼怒了，说道：“你吃我的喝我的，完事还要让我给你养孩子，作为狐狸的你，要是还要对我起的名字挑三拣四，小心我……”说道这里，却说不下去了。<br/>　　<br/>　　母狐狸一家三口，虽然是求收养过来的，但是看着两只小小的萌萌的小狐狸，心中就是一阵的喜欢，如果现在推出去，陈默也是舍不得的。<br/>　　<br/>　　“小心我不给你们吃好吃的！”陈默只能这样说了，至于说母狐狸能不能理解，已经不再他的考虑范围之内了。<br/>　　<br/>　　母狐狸看着陈默，那清澈的目光中还是有些揶揄，但是更多的是疑问，因为刚才陈默说了那么多，它也没有听懂。虽然这只母狐狸很聪明，但是要听明白这么委婉的话语，还真是有些困难了。<br/>　　<br/>　　陈默继续摸着母狐狸，说道：“看看，是不是没有办法说了，那么我给你起的名字，如果你不说话，我就算是你答应了啊！你叫小赤，这两个家伙叫小一和小二！”<br/>　　<br/>　　看着狐狸的目光还是那么的揶揄，似乎说道名字，它就明白一样，真的是搞不懂！不过算了，就这么叫吧。<br/>　　<br/>　　如果有外人见到陈默对着一只狐狸这么说话，绝对会让人怀疑是不是他的智力有问题，或者是精神病患者？<br/>　　<br/>　　看着两只小狐狸在抢着吃鱼肉，心里非常的喜欢，忍不住也摸了摸两只小家伙，到是让两只小家伙有些不满的叫了几声，打扰它们吃饭的罪过有些大！<br/>　　<br/>　　：。：<br/><br/><br/>Ps:书友们，我是小学嗣业，推荐一款免费小说App，支持小说下载、听书、零广告、多种阅读模式。书友们快关注起来吧！<br/><br/><br/>
