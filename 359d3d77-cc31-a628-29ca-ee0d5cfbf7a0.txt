　　“嘭！嘭！嘭！”
<br/>　　不时的有一名名弟子，钻进熔岩池之中，溅起岩浆迸射。
<br/>　　这些弟子，大多都是虚神境的外宫弟子，也偶尔有真神境的内宫弟子。
<br/>　　苏莫没有急着进入熔岩池之中，而是仔细观察了片刻。
<br/>　　这熔岩窟，据说有十八层，越是向下越是危险。
<br/>　　而每一层之中，均是有很多火元精。
<br/>　　所谓的火元精，便是强大的火属性力量形成的灵物。
<br/>　　每一层的火元精都不同，只要抓到了某一种火元精，便算是达到了某一层，可以得到相应的奖励。
<br/>　　“所有准备进入熔岩窟的弟子，前来登记！”
<br/>　　这时，一声朗喝响起。
<br/>　　苏莫转头一看，在山洞的边缘，有着一张石台，一名中年人端坐在石台之后。
<br/>　　苏莫走了过去，登记了‘段惊天’这个姓名。
<br/>　　而后，苏莫没有过多停留，再次来到熔岩池边，身形一闪，便飞了下去。
<br/>　　苏莫离开之后，山洞入口处，光芒一闪，褚彦来了。
<br/>　　褚彦打量了一圈，见苏莫不再，便心中了然，苏莫已经下去了。
<br/>　　他特意等了片刻，就是要等苏莫下去之后，才来到此地。
<br/>　　“褚彦哥！”
<br/>　　“褚彦大哥！”
<br/>　　“褚彦大哥怎么来了？”
<br/>　　众多弟子，见到到来的褚彦，立刻纷纷抱拳。
<br/>　　褚彦，内宫天才弟子，年纪才一百岁出头，修为便以达到一品高阶真神。
<br/>　　不仅如此，褚彦还是褚家之人。
<br/>　　人王宫之中，因为经过悠久岁月的发展，一些强者的子孙众多，早已形成了家族。
<br/>　　当然，所谓的家族，并没有明确的划分，但是却真实存在。
<br/>　　而褚彦的褚家，在人王宫是响当当的存在，褚家最强者，乃是一位神王境的长老。
<br/>　　所以，众人见到褚彦，多时客气有佳。
<br/>　　“刚才那个青衣青年进了熔岩窟？”
<br/>　　褚彦向众人点了点头，随即来到了熔岩池边缘，淡淡的问道。
<br/>　　“是有一位青衣青年下去了，修为好像是虚神境四变！”
<br/>　　“刚刚才下去。”
<br/>　　“褚彦哥，他是你朋友吗？”
<br/>　　众人闻言，纷纷好奇的问道。
<br/>　　“是我朋友，所以，哪位兄弟下去照顾照顾？”
<br/>　　褚彦嘴角勾起一丝冷笑，环视众人问道。
<br/>　　他丝毫不介意不远处的执事，一个执事，不会插手他的事。
<br/>　　众人见此，均是眉头一挑，看到褚彦面上的邪异冷笑，他们瞬间就明白了。
<br/>　　褚彦要教训此人啊！
<br/>　　“褚彦哥，此人是何身份？”有人问道，他们都不傻，可不会随意等罪人。
<br/>　　“一个蝼蚁而已，今日刚加入人王宫！”褚彦不在意的说道。
<br/>　　“原来是刚入宫啊！”
<br/>　　“既然是褚彦哥的朋友，我们当然要照顾照顾了！”
<br/>　　“褚彦哥，我去吧，绝对照顾好他！”
<br/>　　众人分分开口，不少人主动请缨，要下去照顾苏莫。
<br/>　　“魏仑，你下去吧！”
<br/>　　褚彦看向一名身材瘦弱的黑衣青年，吩咐道。
<br/>　　这位黑衣青年，同样是虚神境四变的修为。
<br/>　　当然，褚彦认识对方，知晓对方战力强大，一般的虚神境四变武者，根本不是魏仑的对手。
<br/>　　“好！”黑衣青年魏仑点了点头。
<br/>　　“记住，不要让他得到任何火元精，还有，不要弄出人命！”褚彦吩咐道，虽然对于苏莫有些怒意，但他还没想过杀人。
<br/>　　“明白！”魏仑点头，随即身形一闪，便冲进了熔岩池中。
<br/>　　不远处，负责管理熔岩窟的中年执事，见此无奈的摇了摇头。
<br/>　　不过，只要不会弄出人命，他自然不会多管。
<br/>　　……
<br/>　　熔岩池之下，到处都是沸腾的紫色岩浆，炙热无比。
<br/>　　“这不是岩浆！”
<br/>　　苏莫身处在滚滚岩浆包裹之下，体外五行玄力涌动，仔细观察了片刻，心中暗道。
<br/>　　这是浓郁的火属性力量形成的灵液，类似岩浆，但根本就不是岩浆。
<br/>　　苏莫想尝试能不能吞噬，但很快便放弃了这个想法。
<br/>　　他不能为了这点蝇头小利，就动用吞噬元神。
<br/>　　不然，一旦被人发现，后果难料。
<br/>　　“这里应该是第一层！”
<br/>　　苏莫心中暗道，他继续下沉，向深处潜去。
<br/>　　“嗯？”就在此刻，苏莫发现，他的左右两侧，岩浆涌动，有东西在飞速的靠近。
<br/>　　吼！吼！
<br/>　　而后，两只凶猛的妖兽，一左一右，向他猛扑而来。
<br/>　　这两只妖兽，均是蜥蜴，全身紫红，张开血盆大口，就想将苏莫吞下。
<br/>　　“这就是火元精所化的妖兽吗？”
<br/>　　苏莫身形一闪，便躲过了两只妖兽，而后继续下潜。
<br/>　　他没有出手，这第一层的火元精，根本没有出手的必要。
<br/>　　下潜了千丈左右，下方的岩浆改变了，变成了赤色，温度更加的惊人。
<br/>　　苏莫体外包裹的混沌五行玄力，都滋滋作响，被炙热的温度不断的挥发。
<br/>　　“第二层了！”
<br/>　　苏莫心中暗道，不过，第二层远远不是他的极限。
<br/>　　况且，到达第二层，抓取一只火元精，只是得十枚源石，实在太少了。
<br/>　　每向下一层，得到的源石能翻一倍，他会尽力下潜。
<br/>　　就在此刻，上方岩浆涌动，仿似又有火元精所化的妖兽，向他攻击而来。
<br/>　　苏莫没有理会，加速下潜，不过，这妖兽速度极快，飞速而来。
<br/>　　很快，这‘妖兽’拦住了苏莫的路，而他也看清了，这并不是妖兽，而是一位黑衣青年。
<br/>　　“小子，你不用下去了！”魏仑望着眼前的苏莫，冷笑着说道。
<br/>　　“什么意思？”苏莫眉头一皱，他也看出来了，此人来者不善。
<br/>　　他不禁有些郁闷，怎么走到哪里，都有不开眼的人。
<br/>　　“意思很明显，我手痒，拿你出出气！”魏仑面带调笑之色，眼前的人虽然和自己修为相当，但是，他丝毫不在意。
<br/>　　对于自己的实力，他极有自信。
<br/>　　在整个外宫之中，实力能超过他的人，也就那寥寥十几人而已。
<br/>　　“我和你有仇吗？”苏莫冷着脸问道，心中却是猜到了，可能是褚彦搞的鬼。
<br/>　　毕竟，在人王宫，他除了和褚彦有过节之外，没有和任何人有任何交集。
<br/>　　沁儿姑娘不可能找他的麻烦，而那位骆妤姑娘，也不至于命人找他的麻烦。
<br/>　　“你和我没仇！”
<br/>　　魏仑冷笑一声，有恃无恐的道：“不过，我就是看你不顺眼，怎么看怎么不舒服！”
<br/>　　“你给我滚吧！”苏莫懒得和对方啰嗦，立刻一拳轰出。
<br/>　　轰隆隆！
<br/>　　一拳出，耀眼的五色拳茫，立刻贯穿了岩浆，直击魏仑。
<br/>　　由于距离即将，强大的五色拳茫，几乎是瞬息即至，瞬间便临近了魏仑的身前。
<br/>　　“什么？”魏仑见此，顿时大惊失色，因为这一拳的力量，让他头皮发麻。
<br/>　　现在，他恍然大悟，这个褚彦要教训的人，根本不是普通的虚神境四变武者，而是超级天才。
<br/>　　“开！”
<br/>　　电光火石之间，怒喝一声，魏仑手中出现了一柄长刀，他身形一边暴退，一边狠狠的斩向拳茫。
<br/>　　轰隆！
<br/>　　一声爆响，长刀和拳茫相击，魏仑全身剧震，瞬间倒飞而去。
<br/>　　哗啦啦！
<br/>　　周围岩浆疯狂涌动，而魏仑的身形，早已不见了踪迹，不知道被轰到了何处。
<br/>　　“应该不会死了吧？”
<br/>　　苏莫见此，自喃一声，他已经极力的控制自己的力量了。
<br/>　　毕竟，虚神境四变的武者，对于如今的他来说，弱小的如同一只苍蝇。
<br/>　　不过，苏莫也没有太担心，他用的力量很小，对方不至于陨落。
<br/>　　最多肉身崩溃而已，元神是不会死的。
<br/>　　随即，苏莫便不再多想，继续下潜，他要看看，自己能下潜到多少层。
