　　洪荒之中，大战依旧还在继续，战到如今死伤何止千万，那简直是千千万，千亿万不止。
<br/>　　洪荒大地之上，早已血流成河，尸骨堆积如山，天空之上，各种残值断臂、残鳞败甲漫天飞舞，血雨倾盆而下，伴随着雷闪电鸣，狂风呼啸，战场格外惨烈。
<br/>　　战场之中，巫族的十二柄戮妖幡在虚空之中闪动，无尽煞气冲击而来，形成一道道戮妖之力，向着人群之中的妖族斩杀而去，十二枚戮妖幡是根据人族的魂魄，在意每个祖巫自身的本命法则，最后吸收无尽地脉煞气而形成的，专门克制妖族的妖力。
<br/>　　戮妖幡遇风而涨，在虚空之中随风飘荡，一道道神力于戮妖幡之中斩出，划破虚空，如同收割韭菜一般，在肆意收割着无尽妖族的性命。不过瞬息之间，便死去无数妖族。
<br/>　　帝俊等妖族强者看到这一幕，目眦尽裂，怒发冲冠，大喝一声：
<br/>　　“布置周天星辰大阵！”
<br/>　　虚空之中一阵波动，顿时出现三百六十五位大罗金仙境界的妖王，帝俊手持河图与洛书两件先天灵宝，玄光绽放，道法演绎，古朴的妖字，沧桑的画卷在虚空之中平面铺开。三百六十五位大罗金仙按照某种特殊的规律，相互运转起来。
<br/>　　顿时，以河图洛书为媒介，接引出九天星空深处的三百六十五颗主星辰，帝俊与东皇太一为两颗至尊星辰的阵眼，无尽星辰光辉形成三百六十五道巨大的光柱，降落下来，战场顿时变得虚幻起来，周遭星辰散落，虚空扭曲，好似脱离了洪荒世界。
<br/>　　三百六十五位大罗金仙化身三百六十五颗星辰，星光璀璨绚烂，绽放出耀眼无比的光芒，玄光带着无尽星辰之力，将周围变成一个浩大的星辰世界。
<br/>　　各星辰随着某种特定的轨迹运转起来，仿佛世界在循环，时空变幻，岁月苍茫，帝俊手持屠巫剑，剑身漆黑，杀机内敛，但却又透露着无尽锋锐之气，这是专门为克制巫族祖巫肉身所锻造出来的凶剑。
<br/>　　东皇太一手持混元剑，凶气弥漫缭绕其身，无尽杀意割破苍穹。
<br/>　　无尽星光汇聚而来，汇聚成一个身披星辰法袍的帝皇影像巨人。巨人身高数十万丈，模样极像了帝俊，他手握星辰之剑，全身星辰璀璨，耀眼无比。
<br/>　　三族之中，无数修士被包围在大阵之中，巫族的实力被压制下来，无数妖族却能来去自由，毫无影响，屠杀就此开始。
<br/>　　“可恶，布置小都天神煞大阵！”
<br/>　　祖巫帝江见此，大感不妙，连忙大声向着虚空喊道。
<br/>　　“嗡！”
<br/>　　虚空一阵翁响，十二柄戮妖幡汇聚而来，以都天神煞大阵的形式汇聚运转，无尽凶煞之气从地表汇聚而来，帝江等祖巫归位，各自持有一柄与自己相配的戮妖幡。
<br/>　　当然，因为死去了四大祖巫，所以剩下的，只能以大巫去填充，但大巫的身体强度又有些不够，所以只能用戮妖幡去替代一部分。
<br/>　　“轰隆隆！”
<br/>　　随着一阵声响，虚空之中，出现一个模模糊糊的人影，正是凝聚中的盘古真身。大阵阵眼之处，帝俊操控着整片大阵，光芒笼罩起来，一颗颗浩大的星辰在他与东皇太一的操控下，朝着那盘古真身撞击过去。
<br/>　　那盘古真身虽然模糊，但已经具有一定的实力了，右手挥拳打去，那虚幻的星辰瞬间破灭，只是这星辰一颗接着一颗，虽然撞击在盘古真身身上也只是让他微微一顿，但是同时也让盘古真身无法真正凝聚起来。
<br/>　　这也是小都天神煞大阵的局限性，若是十二祖巫全在，布置出真正的都天神煞大阵，就连真正的圣人都敢战，更不用说这帝俊与太一不知的周天星辰大阵了。但，真正的都天神煞大阵如今已经因为祖巫的缺失变得不全了，以后更是可能不会在洪荒中出现了。
<br/>　　帝江等祖巫也不是傻子，自然看出了帝俊的心思，于是索性便不管那些星辰，只管凝聚盘古真身。
<br/>　　帝俊在控制星辰撞击盘古真身的同时，也操控星辰真身来战玄天。
<br/>　　“哈，妖族周天星辰大阵，久仰大名，今日正好见识一番。”
<br/>　　玄天看见这高达数十万丈的星辰巨人毫不畏惧，他一声朗笑，面色略带凝重之色，大喝一声，道：
<br/>　　“天地为法，武祖为相，武祖法相，出！”
<br/>　　“轰！”
<br/>　　虚空一阵波动，金光闪烁，一尊足有数十万丈的浩大身影出现在形星辰世界之中，他全身散发着不朽之光，苍茫、亘古的气息在他全身流转，无尽威压从虚空中降临。空间为之一滞，星辰仿佛也变得暗淡。
<br/>　　他，此刻就仿佛是天地之中的唯一主宰，执掌着众生的生死。
<br/>　　巨大的金身武祖法相缓缓睁开眼睛，双眼之中暴射出无比璀璨的金色光芒，光芒穿过虚空，洞穿两颗向他袭来的两颗星辰。
<br/>　　金身武祖脚踏星河，他一手横贯苍穹，穿过无尽虚空，捏爆一颗星辰，他再次一指点出，绽放出无尽光辉，与那浩大的星辰真身撞击在一起。
<br/>　　“轰隆！”
<br/>　　星辰破灭，星辰之光幻灭，便得黯淡起来，星辰之剑也降落下来，巨大的星辰真身与武祖法相同时倒退几步。
<br/>　　“好，再来！”
<br/>　　没有一击灭杀星辰真身，玄天眼中非但没有沮丧，反而更加兴奋，他大喝一声，挥手之间，一道乳白色的转轮从他手中飞出，划破天际，斩向星辰法相。
<br/>　　“可恶！该死！”
<br/>　　帝俊大叫一声，眼中闪过冷厉之色，操控星辰真身，挥动长剑，星辰连成一串，对着转轮撞击而去，同时浩瀚的星辰之剑斩向玄天的法相。
<br/>　　“帝剑斩天！”
<br/>　　“哼！就凭你，你敢妄称斩天，玄天剑！”
<br/>　　武祖法相一声冷哼，嘴中满是不屑，他手中武道法则一阵幻化，出现一柄闪烁着九彩之光的神剑，剑光璀璨，对着对面斩来的星辰之剑同样毫不畏惧的斩去。
<br/>　　“轰！”
<br/>　　剑光对轰，宇宙震动，无数星辰破灭，虚空破碎，唯留两道锋利的剑光在对峙着。
<br/>　　“啊——”
<br/>　　只听一声惨叫，原来是东皇太一的叫声。
<br/>　　玄天一声冷笑，他的玄天剑斩的可不只是肉身，它还能斩灭元神。玄天剑是他玄天九技中的一种。九天神技不仅是武法技，更是神魂技。也就是说，它不仅能够进行物质能量攻击，还能进行神魂攻击。刚刚东皇太一一声惨叫，便是中了玄天剑的神魂攻击。
<br/>　　玄天剑，幽天锁，钧天引，罗天网，遮天手，诛天矛，浩天盾，逸天影，赤天箭。
<br/>　　九天神技，是玄天一身武技的总结，不仅能够修炼神魂，而且还能修炼功力，不仅有攻击手段，还有防御手段，遁法手段，可为一举多得。
<br/>　　“轰！”
<br/>　　此时，巫族的小盘古真身也凝聚成功了，一道闪烁着苍茫气息的浩大身影于无尽黑色煞气之中走了出来。
<br/>　　三道巨人法相真身，武祖法相玄天，妖皇星辰真身，巫族盘古真身，形成三角对峙之势。
<br/>　　一场大战，就此展开。
