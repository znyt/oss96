　　黑耀城非常的大，占地方圆八九千里，浩瀚广阔。
<br/>　　若不是后面还有个天冥圣城，以及周围的另外八座卫城，黑耀城绝对是苏莫有生以来见过的最大的城池。
<br/>　　黑耀城的城门之前，有数十名巫族守卫，个个都是武帝境强者，甚至领头之人还是武尊强者。
<br/>　　不过，苏莫现在丝毫不惧，大摇大摆的走进了黑耀城中。
<br/>　　他现在是巫尘的肉身，身上的气息也全部是巫尘的气息，只要不催动战魂，他相信没有人能识破他。
<br/>　　再说了，巫尘在巫族地位不高，熟悉他的人也不多。
<br/>　　没有人阻拦，苏莫径直的走进了黑耀城中。
<br/>　　黑耀城中，和一般的人族城池差距不大，商铺林立，商贩众多。
<br/>　　城中也有人族，只不过非常的少，而且全部是奴仆、随处之流。
<br/>　　巫族只是一个统称，巫族之中还有一个个大小家族，但无论是大家族和是小家族，都要效忠支脉的族长。
<br/>　　巫尘之所以在巫族地位不高，是因为其父母早亡，没有什么背景！
<br/>　　而唯一的爷爷，也在半个月前，陨落在了其它星辰之上。
<br/>　　所以，巫烈和巫玉儿才敢肆无忌惮的动手杀他。
<br/>　　正因为巫尘是个孤家寡人，没有人在意，苏莫才会最终选择对方的肉身。
<br/>　　苏莫在黑耀城中大步而行，没过多久便来到了一处古朴的宅院之前，随后走了进去。
<br/>　　这处宅院便是巫尘的家，只有他一人居住。
<br/>　　进入宅院以后，苏莫坐在一间偏殿之中，思索着接下来的计划。
<br/>　　巫尘认识巫天御，当然，也只是知晓对方，两人并不熟悉。
<br/>　　巫天御在族中有些背景，父母均是族中的长老。
<br/>　　当然这种地位在巫族中也是很普通，因为巫族长老数以万计，巫天御的身份与巫娴不能比。
<br/>　　从巫尘的记忆中，苏莫知道巫天御的家族在何处，但巫天御作为巫族的天才子弟，一般很少待在自己的家族中。
<br/>　　巫天御的行踪，还要仔细查探才行。
<br/>　　并且，找到了对方也不能直接杀，要确定宏青璇的下落，还在不在对方的手中？
<br/>　　而且，肯定不能光明正大的杀，必须要暗中进行，不然会为自己带来灾难。
<br/>　　“苏莫，汝真是大胆，居然敢混进巫族支脉的大本营中！”
<br/>　　这时，阎魔鬼圣的声音，在苏莫的脑海中响起。
<br/>　　“鬼圣前辈，你这段时间怎么了？”苏莫轻声问道，这段时间对方一直沉寂，若不是他能感受到对方的存在，真以为对方死了。
<br/>　　“没什么！吾只是在思考汝之命道而已！”阎魔鬼圣叹道。
<br/>　　“命道？什么意思？”苏莫完全不理解。
<br/>　　“这个以后再言，汝此行好自为之吧！”阎魔鬼圣悠悠说完，言语中充满了莫名的感慨，随即便不再多言。
<br/>　　苏莫眉头微皱，对方神神叨叨的，真不知道在想些什么！
<br/>　　摇了摇头，他也没有再理会对方，起身大步的离开了院落。
<br/>　　他要去查探巫天御的行踪。
<br/>　　在黑耀城中逛了片刻，苏莫远远的看到了一座较为恢弘的府邸，这座府邸便是巫天御的家族。
<br/>　　苏莫在远处徘徊了片刻，便直接走了过去。
<br/>　　畏手畏脚的查探，很容易引起别人的注意，还不如直接去询问。
<br/>　　很快，苏莫便来到了府邸的大门之前。
<br/>　　大门前，有四名巫族护卫，修为倒是不高，三人是武皇境修为，只有一人是武帝境。
<br/>　　“站住，来者何人？”见苏莫大步而来，那名武帝境的护卫立刻厉喝一声。
<br/>　　“请问天御兄是否在家中？”苏莫向对方抱了抱拳，微笑着问道。
<br/>　　“你找少爷？”护卫沉声问道。
<br/>　　“正是！”
<br/>　　“少爷在地灵塔！”护卫淡淡的说道。
<br/>　　“地灵塔？”苏莫闻言，脑中立刻查探搜魂得来的记忆，顿时知道了地灵塔是何地。
<br/>　　“多谢！”苏莫微微一笑，向对方道谢，随即转身大步离去。
<br/>　　地灵塔，是黑耀城的修炼宝地，此塔存在了百万年之久，下方连接着天冥星的大地龙脉，是所有黑耀城年轻巫族的修炼圣地。
<br/>　　“看来巫天御果然达到了武帝之境！”
<br/>　　苏莫心中冷笑，因为地灵塔并不是什么人都能进去修炼的，只有五十岁以下，并且修为达到了武帝境界的巫族之人，才能进入地灵塔。
<br/>　　这就说明，巫天御现在的修为，已经达到武帝之境。
<br/>　　苏莫没有丝毫犹豫，立刻加快速度，向地灵塔而去。
<br/>　　片刻之后，他来到了黑耀城的中央，一片广场之上，看到了地灵塔。
<br/>　　地灵塔只有三丈高，占地方圆万丈，犹如一个巨大的蒙古包一般。
<br/>　　这是因为，地灵塔是修建在地下，是一座倒插在大地之上的塔楼。
<br/>　　此塔共有十层空间，每向下一层，修炼效果都能好上几倍。
<br/>　　地灵塔的大门，犹如是一片朦胧的水幕，让人看不清其内的情景。
<br/>　　这水幕是一个阵法，任何不到武帝境修为的人，或者是年岁超过五十的人，都无法进入。
<br/>　　此时，地灵塔的门前人来人往，大量的巫族年轻人进进出出，显得热闹非凡。
<br/>　　苏莫脚步不停，直接便走进了水幕大门之中。
<br/>　　唰！
<br/>　　下一刻，他眼前场景变换，来到了一片略显昏暗的空间之中。
<br/>　　这片空间不大，方圆不足万丈，四周布满了大量的修炼室，所有修炼室连接起来，围成了一个巨大的圆形。
<br/>　　而在这片空间的中央，有一个硕大的光门，通向下一层。
<br/>　　不少巫族年轻人，都通过光门进去了下一层，但也有很多人在中央空地上盘坐。
<br/>　　因为修炼室有限，他们需等待别人出来之后，才能修炼。
<br/>　　来到了这地灵塔之后，苏莫立刻在第一层、第二层和第三层中搜寻了一番，却并没有找到巫天御。
<br/>　　他没有再向下搜索，因为这地灵塔的修炼室，是需要支付昂贵的灵晶才能使用。
<br/>　　再向下的话，价格非常的昂贵，以巫天御的修为应该不会在下面几层，只会在上三层之中。
<br/>　　这就让苏莫再次犯难了，想必对方肯定在修炼室之中，但每一层都有上万个修炼室，而修炼室又有阵法隔绝，根本无法窥探丝毫。
<br/>　　这要如何寻找对方？
<br/>　　最后，没有办法的他，只能选择等待，在第一层的空间中盘膝而坐，等待巫天御出来。
<br/>(本章完)
