　　第644章
<br/>　　徐州，琅琊城。
<br/>　　大战过后，一片狼藉。这原本的诗书礼乐之乡，已成了人间魔窟。青石板铺成的街道上，东倒西歪躺着一具具尸体，两边房屋传出阵阵惨叫声。
<br/>　　十日前，刘顶天的大军已经攻破琅琊城。琅琊城是个好地方，人人晓礼，路不拾遗，夜不闭户，但相应的，守军的战斗力便稍弱些。况且，刘顶天并不是从城门正面突破，而是绕道海上，率大军从港口登陆，一举将琅琊城攻破。
<br/>　　十日前破城，但烧杀抢掠的工作十日后尚未完成。刘顶天手下的大将，疯狂掠夺城中的金银珠宝与女人。如果不是为了这两项，大家为何要将脑袋拴在裤腰带上做活。
<br/>　　而刘顶天本人，其实来历也并无太特殊之处。本是徐州某世家的护院家丁，因与主妇勾搭成奸，事情败露之后逃到海上，成了一无法无天的山贼。
<br/>　　如他这样的人，帝国没有一千也有八百，如果没有这乱世，他只能是海上漂泊的一小小贼寇。但这乱世给了他机会，本是海上亡命徒的他，摇身一变成了正义教林少羽的传人。
<br/>　　靠此竟然拉起一支人马，也是天时地利人和，并没有费多大力气，便打下一座琅琊城。
<br/>　　如今刘顶天占了城主府，原先琅琊城城主的尸体挂在城墙上，暴晒十日，受那风吹雨打之辱。而城主府的一干女眷，原先城主的姬妾，便成了刘顶天的囊中之物。
<br/>　　他手下大将都在搜刮金银财宝，更何况是刘顶天，如今城主府堆积的珠宝如山，金银成塔。刘顶天春风得意，帝国之大，如今自己也算得上是一方诸侯。他每日醉卧金银山中，玩弄女人为乐，日子过的十分逍遥。
<br/>　　城中十室九空，侧目静听，只能听到一阵连着一阵的哭声。
<br/>　　如此状况，人人避祸，能逃得就逃了，能躲得便躲了，既不能躲也不能逃，便也就只靠着天灵盖来顶。照道理来说，城中街道上本不该有什么行人，但偏偏有三人急行在偏僻的胡同中。
<br/>　　三人都穿着宽大斗篷，用兜帽遮脸，穿过大街小巷，三人拐进一座荒废的院子。
<br/>　　此时，为首之人才轻吁口气，摘掉脸上兜帽，正是来自落凤城的李行哉。
<br/>　　而他身边随行两人，自然就是楚云生与戚继光。
<br/>　　“王上，琅琊如今已是一座危城，本不需要您亲自出马的。”戚继光道。
<br/>　　李行哉摇摇头，表情有些难看：“看来要打一打琅琊城了。”
<br/>　　知道李行哉亲身涉险，便已蠢蠢欲动，如今他做出这个决定，戚继光并不意外。
<br/>　　楚云生皱眉思索，他不是个很爱说话的人，辅佐李行哉后更是惜字如金，因为需要他考虑的事情太多了。
<br/>　　他沉思良久，道：“王上，我们现在的情况并不适合打琅琊城。”
<br/>　　李行哉看着他：“说来听听？”
<br/>　　楚云生以树枝为笔，以黄土为纸，便说便画：“落凤城与琅琊城虽然都在徐州境内，但两地相距颇远，这中间隔着几座城池，都不是我们的势力，如此陆路不通，只能走水路。可我们如今虽有些根基，但还是不适宜大动兵戈，而刘顶天本身就是海盗出身，能打下一座琅琊城，必然擅长水战，我们与他们动手，并无多少胜算。”
<br/>　　李行哉摇了摇头。
<br/>　　“回城，准备兵马，尽快动兵。”
<br/>　　“王上……”楚云生觉得自己一席话说得句句在理，但李行哉似乎半点也没听进去。
<br/>　　“我是帝国的王爷，如今受苦正是我帝国百姓。”李行哉直起身子，重新将兜帽戴到头上：“这件事算不清楚，我不管让谁去管。”
<br/>　　……
<br/>　　程大雷跨着黑牛，片刻不敢停，冒雨回了蛤蟆城。他生怕自己在路上耽搁一时半刻，华正则便被刘悲忽悠死了。
<br/>　　“华大夫在何处，华大夫在何处……”
<br/>　　程大雷回了蛤蟆城后，没有回城主府，直接去了蛤蟆城的医馆。
<br/>　　只见屋子里，华正则正一把眼泪一把鼻涕，哭得撕心裂肺。
<br/>　　刘悲坐在他面前，细声安慰：“何必呢，不至于，自古天下谁无死，包括你我，早晚都是黄土中的白骨，不值得，当真不值得。”
<br/>　　“不要拦我，让我去死……”
<br/>　　华正则悲痛欲绝，一头向屋中柱子撞去。
<br/>　　“哎呦呦，华大夫使不得……”
<br/>　　程大雷此刻刚好进屋，连忙使出轻身功夫，拉住他的胳膊，算是将他在鬼门关前拉了回来。
<br/>　　“华大夫，你这是何苦来哉。”
<br/>　　说着，程大雷用眼神示意刘悲闭嘴，双目盯着华正则，饱含殷殷关切。
<br/>　　华正则一张丑脸哭得涕泗纵横，如今更加丑陋。不过他已死过一次，求死的心已经缓了许多。
<br/>　　程大雷细语安慰：“华大夫节哀，只是一些传言过来，东海的具体情况如何你我都不知道。也许华大夫怕的事情，并未发生。”
<br/>　　华正则一听这话，隐隐升起一些希望。他抹干眼泪，立刻道：“我要回东海，去寻我的族人。”
<br/>　　程大雷摆摆手：“此地和东海相距太远，华大夫单枪匹马上路，又无自保之力，即便能活着走到东海，凭华大夫一人之力，又怎么寻找失散的族人？”
<br/>　　华正则在医术上有极高的天赋，但做人做事的确没多少主意，他此刻有些糊涂，抬起头道：“程当家说我该怎么办？”
<br/>　　“华大夫一诺千金，程某人也并非忘恩负义之徒。华大夫放心，我现在立刻就派人去东海，华大夫的族人无事最好，若是不幸遭难，我便将华大夫的族人聚起来，引到蛤蟆城也能求个安稳。”
<br/>　　华正则本来心神沮丧，此刻才渐渐镇定下来。他双手一抱拳：“多谢程当家。”
<br/>　　程大雷点点头：“不过，还需华大夫身上一件信物，以为凭证，不然我担心，我手下人说的话没人信。”
<br/>　　华正则想了想，道：“我身上也没有什么信物，不过，华家有种暗中联络的密语，乃是以药材之名联络。我写一剂药方，将它示给我的主人，他们便会明白其中的意思。”
<br/>(本章完)
