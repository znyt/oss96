<br/>        “现在能就职吗？”人事部部长又问。
<br/>
<br/>        诺筱颖微笑着点了点头。
<br/>
<br/>        对于人事部招了一个每个月才四千块钱的大专在校实习生的消息，在地利置业内部不胫而走。

<br/>        当设计部部长领着诺筱颖走进办公室的时候，格子间里的同事，纷纷像看猴一样，打量着诺筱颖。

<br/>        他们的傅氏集团旗下地利置业竟然招收实习生？！

<br/>        最爱八卦的姚明娜于是也来设计部凑了个热闹。
<br/>
<br/>        当透过设计部办公室的落地窗，姚明娜看到诺筱颖正笑盈盈的和设计部里的同事做自我介绍时，顿时傻眼了。

<br/>        她不由地嘟起了红唇，双拳握紧，愤愤不满地跺了跺脚。
<br/>
<br/>        看到诺筱颖，姚明娜心里便明白了，这一定是傅昼景的意思。
<br/>
<br/>        既然不能跟傅昼景对着干，那么她便好好整整诺筱颖这个女人，幸好她和设计部副部长孙莉莉的关系还不错。

<br/>        设计部，管外的基本上是部长的事，主内的便是副部长了。

<br/>        姚明娜私底下跟孙莉莉打了声招呼，诺筱颖便成了在办公室里打杂的了。
<br/>
<br/>        刚上班一下午，诺筱颖被同事们呼来喝去，不是端茶倒水，就是打印设计图送资料什么的，走的路太多，脚上的负担一重，又痛了起来。
<br/>
<br/>        眼看着就快到下班时间了，副部长孙莉莉突然走过来，敲了敲诺筱颖的格子间。
<br/>
<br/>        “小梁，你会不会做效果图？”孙莉莉问道。
<br/>
<br/>        诺筱颖微笑着点了点头。
<br/>
<br/>        当孙莉莉将手中的一张客厅装饰图递给诺筱颖，并让诺筱颖做好效果图，明天一早就交给她的时候，诺筱颖顿时便为自己刚刚的点头而后悔了。
<br/>
<br/>        副部长这意思不是逼着她诺筱颖晚上加班么？
<br/>
<br/>        “我明天早上开会要用，你可别耽误了我的时间！”孙莉莉一脸严肃地说道。
<br/>
<br/>        诺筱颖刚想解释她一个人没法在这么短的时间内完成时，孙莉莉便径直转身离去了。
<br/>
<br/>        她做不到，就是做不到。
<br/>
<br/>        诺筱颖忍着脚腕上传来的疼痛，加快步伐，吃力地追上孙莉莉：“孙副部长，请留步。”
<br/>
<br/>        听到背后有人呼唤自己，孙莉莉驻足转过身来，看到诺筱颖一瘸一拐地朝自己走来，她还是很好心地问了一句：“你脚怎么了？”
<br/>
<br/>        “崴伤了，还没痊愈。”诺筱颖微微一笑。
<br/>
<br/>        但孙莉莉并未因为她带伤上班，而对她的看法有所改观。
<br/>
<br/>        “有事吗？”
<br/>
<br/>        “噢！是这样的，您明天早上就需要这张效果图的话，我想我可能没办法在这么短的时间内完成……”我怕耽误您，我请您再安排一个同事和我一起做。

<br/>        后面的话，诺筱颖还未说完，孙莉莉便疾言厉色地呵斥道：“我们这里在座的人，都能完成，为何就你一个人偏偏不能完成？”

<br/>        诺筱颖扫了一眼大家，又一脸抱歉地看着孙莉莉，微微颔首：“对不起，孙副部长，我真的……”

<br/>        诺筱颖顿时错愕了，她之所以能进傅氏集团旗下的地利置业，是被他们人事部的人给招进来的啊！

<br/>        孙副部长为什么要反过来问她这个问题？

<br/>        跟嫂嫂唐倩兮打了个电话报告自己不会回家吃饭后，诺筱颖独自一个人坐上了去往市中心的公交车。
