　　周宋捷把钱鑫和陆岑的照片，投影到投影布上，继续说道：“钱鑫，某小区保安，于半个月前上班过程中失踪，至今下落不明；陆岑，‘碧海潮生’夜总会小姐，数天前，住在她隔壁公寓的小姐妹，发现多日不见陆岑，感觉到意外，前来报案。目前，我们初步怀疑，钱鑫和陆岑之间的失踪有一定联系。”
<br/>　　底下，慕容飞看到钱鑫照片的时候，眼睛轻轻眯了眯，他没想到居然会是佟千雪所住小区的保安！
<br/>　　“关于李霄力还有什么其他情况？”罗昊对周宋捷问道。
<br/>　　“李霄力这个人很狡猾，我们跟了他很长时间都没有发现什么特别的地方，但是我们怀疑，李霄力可能藏有枪械，而且还不止一把！”
<br/>　　周宋捷的语气变得有些严肃起来，撤掉了钱鑫和陆岑的照片，重新放上一张照片，说道：“这个人叫刘三，身边人一般称呼他‘三儿’或者‘三爷’，李霄力的左右手，目前‘碧海潮生’的‘crystal-ball’拍卖就是刘三在全权负责。刘三以前是杭山市市散打队的，身手不错，前杭山市散打冠军，因为受伤的关系没能入选省队，加之年纪渐长，选择退役。”
<br/>　　“退役之后一直没有固定的工作，一次甚至因为琐事跟同事吵架并且发生肢体冲突，一拳把对方打进了医院，被拘留了十五天。拘留所出来后，工作丢了，也没有人再愿意用他，刘三就在社会上瞎混，给人看场子当打手。后来被李霄力发现，李霄力看中了刘三出色的身手，把他留在自己身边。不过跟着李霄力做事之后，刘三也一直被压着，直到最近这段时间才冒头。这是一个值得关注的对象。”
<br/>　　周宋捷看了眼底下坐着的龙隐部队战术中队队员，突然觉得自己最后一句话有些多余，刘三是前散打冠军那又怎么样？在这些人面前，别说是一个市级的前散打冠军，就是世界级的散打冠军，那也是白搭。
<br/>　　罗昊也的确没有太过重视刘三这个人，如果是在拳台上打比赛，罗昊不敢拍着胸脯说，自己一定可以打赢刘三，但战场不是拳台，出手就是要人命的杀招，所以如果是以杀人为目的的话，罗昊杀死刘三最多只要三招！
<br/>　　见过面了解过情况后，罗昊就带着战术中队队员离开了行动简报室。特种部队和特警队是完全不同的地方，罗昊有自己的行动部署，而且这次行动也是由战术中队为主导，杭山市警方负责外围的配合。
<br/>　　“好牛逼啊！”
<br/>　　等到战术中队的队员离开后，一个年轻的特警队员眼睛中直冒星星，一脸羡慕的说道：“原来我以为我们装备已经都牛的了，现在跟别人家的一笔，简直土得掉渣了。他们身上那些武器，我只在游戏和杂志上看到过，看看人家，ELITE公司生产过万元的消音器，就这么装在武器上，再看看我们，就那种普通的消音器，使用还要打申请要领导批准，真是一把辛酸泪。”
<br/>　　特警，是警察队伍中的利刃，只有最出色的警察才能加入这样的队伍，每一个加入特警队的警察，心中都是带着这样一份自豪；可是现在跟别人家一比，那种自豪瞬间荡然无存！
<br/>　　简报室中，那些特警队员情绪有些低沉，刘东来自然也是看出了自己那些队员低沉的情绪，这让他忍不住皱了皱眉。
<br/>　　刘东来也不会把责任推到罗昊他们身上，刘东来很清楚，他们这些人就是这样“高调”，当年他自己还在部队服役的时候，第一次看到特种部队那牛逼哄哄的样子，先进的装备，再对比对比自己，也有股同样的失落感。
<br/>　　当年他的老连长就冲着他们喊，“羡慕吗？肯定羡慕，你们羡慕，我也羡慕！那是从十几万大军区中挑选出来的尖子，他们不仅装备比你们好，训练条件比你们好，吃得还比你们好，用得还比你们好，但是羡慕有什么用？不努力，你们也就只有看着别人流口水的份！”
<br/>　　从那天起，他们连队里不管是服役了几年的老兵，还是刚入伍的新兵，都是卯着劲拼着命的练，第二年的特种部队选拔中他们连队中有五个人得到了参加特种部队选拔的资格，虽然最后进入特种部队的只有两个人，但是就比例来说，这已经算是很高的入选比例了。
<br/>　　“看看你们这要死不活的样子！”
<br/>　　刘东来瞪着下面那些特警队员，说道：“他们每个人都是国家投入了上千万培养出来的战争机器，他们随时都会为了这个国家去死，甚至练眉头都不会皱一下，你们以为你们现在看到的安定社会是哪里来的？是他们拿命换来的！没有他们，你们以为自己周末的时候能跟女朋友出去约会？你们觉得你们下班回家，能有热菜热饭等着你们？做梦！就你们这幅样子，老子做你们队长都觉得丢脸！现在散会，两个小时后集合，自己去想清楚，如果再让我看到你们这个样子，晚上的行动都别参加了，免得出去丢人现眼！”
<br/>　　等到行动简报室里的特警队员都离开后，周宋捷对刘东来说道：“老刘啊，他们中间还有些只是孩子，你这样说他们是不是有点过了？我们公安系统每年都有那么多人因公牺牲……”
<br/>　　“老周，你说得我何尝不明白？”刘东来叹了口气，说道：“可是你也看到他们刚才的样子了，如果不用重点的语气把他们骂醒，更误事！你没上过战场，不知道真正的战场是什么样子，但是我见过，在战场上，面对的敌人不会用土制猎枪，随时会炸膛的‘化隆造’来跟你战斗，就是边境线那些毒贩，用得武器都是外国货，M16、M4、反坦克火箭筒，大口径狙击步枪，全都是你想都不敢想的东西。我亲眼看到子弹打中我身边人脑袋，把脑壳都打飞的画面，他还只是一个孩子，二十岁都不到，就死在距离我不到二十米的地方……”
