　　　　猿猴王没想到制服住对方，依然无法阻止对方的屠杀行为，吓得直接不敢用力了。
<br/>　　他的猴子猴孙，死的太冤枉了，这条蛇精到底是什么鬼，手段那么的诡谲。
<br/>　　“还不放开吗”涂小安明明被人掐着，反而露出了胜利者的高傲姿态。
<br/>　　猿猴王从来没如此憋屈过，咬牙切齿道：“将你的冰元素取消，我的猴子猴孙不过一二阶，祸不及它们”
<br/>　　“你别忘记了，我也才二阶而已，放开我，一切好说”
<br/>　　“你在威胁本王吗”猿猴王睥睨霸道：“本王从来不受威胁”
<br/>　　涂小安平淡道：“真的吗”
<br/>　　他的眼眸浑然一眯...
<br/>　　嘭嘭嘭...
<br/>　　远处再次数百具冰雕顷刻间粉碎，漫天碎冰挥洒，冰中掺杂着猴类的断肢残臂。
<br/>　　“给本王去死”
<br/>　　猿猴王的手猛然的用力，涂小安的脖子部位金光迸发凝固，金元素能力重点保护。
<br/>　　心中暗暗的吃惊，这猴王还真是脾气暴躁，明明心疼自己的猴子猴孙，但却不甘就范，性如烈火，狂傲不驯。
<br/>　　“好啊，你都不心疼，我就更不心疼了”
<br/>　　涂小安的脖子被勒的越来越近，他不怕窒息，顶多这身体溃散而已。
<br/>　　“给我爆...”
<br/>　　嘭嘭嘭...嘭嘭嘭...嘭嘭嘭...
<br/>　　大面积的冰雕集体爆炸起来，一只只觉醒猴精怪猴灰飞烟灭。
<br/>　　猿猴王顿时杀意暴涌而来，若狂风飚射：“本王还不信掐不死你这条蛇精”
<br/>　　啾...
<br/>　　涂小安蛇王左眼凝固，一道眸光化为实质的骤射出来，猿猴王大惊失色，两者距离太近，根本不可能躲，他下意思的转头，顿时连连倒退，松开了涂小安。
<br/>　　眸光攻击，射在了猿猴王的右脸上。
<br/>　　他右脸上的皮肤消失了，露出了毛茸茸的猴毛，毛发中，还有一丝殷红的鲜血溢出。
<br/>　　徒然，猿猴王一半人脸，一半猴脸，不伦不类起来，但却显得尖嘴猴腮起来。
<br/>　　涂小安恢复自由，却反而因为系统的提示音，陷入了一场深思中...
<br/>　　“叮...恭喜宿主成功完成十万进化能源，获得了证道的资格，即将准备渡劫...”
<br/>　　这就渡劫了吗?
<br/>　　涂小安貌似一点心里准备都没有啊，他没想到一来猴儿山，一下子就将进化能源给全部完成了。
<br/>　　还真是计划赶不上变化，本来涂小安是想来猴儿山吞噬妖灵，快速的提升进化能源。
<br/>　　可一来，妖灵的实力出乎他的预料，而附近又密密麻麻的汇聚了那么多的觉醒生物。
<br/>　　这随便搞一搞，十万点就达到了。
<br/>　　在涂小安愣身的时候，一股睥睨而恐怖的王者气息铺天盖地的笼罩在他的身上。
<br/>　　回神，只见半人半妖的猿猴王暴跳如雷，一双眼睛金光四射，火眼金睛似的，体内爆发的气息连空气都沸腾起来，他堂堂猴王，既然被一条蛇精给伤了，而且还被他屠杀了那么多的猴子猴孙，简直就是天大的耻辱。
<br/>　　不可饶恕，不可饶恕。
<br/>　　眼看这只猴王要彻底发飙，涂小安忙道：“等等”
<br/>　　猿猴王岂能听，一掌霸道的排出，爆响声连连，涂小安连忙腾空而起，意念转动，爆喝一声：“给化解”
<br/>　　嗯？
<br/>　　猿猴王倏然看到地面的冰一下子消失了，那些被冰冻的猴子也恢复自由，好像什么事情都没发生过。
<br/>　　但是一只只被冻的不轻，叽叽喳喳的叫嚷起来。
<br/>　　“你以为现在解除了冰元素，本王就能饶了你吗”
<br/>　　猿猴王杀气沸腾，之前那些猴子猴孙可不能白死，这条该死的蛇精必须将他挫骨扬灰了不可。
<br/>　　它什么时候吃过这种亏。
<br/>　　“你若不想让您的猴子猴孙死的更多，就快让它们散去，不要汇聚在这里了”涂小安要证道渡劫了，之所以解除了这些猴子的冰冻，也是不想让它们在自己渡劫之中惨死。
<br/>　　这绝对是一片好意啊。
<br/>　　“有本王在，你休想在伤我一只猴子猴孙”猿猴王骤射上去，误解了涂小安的意思。
<br/>　　它无边戾气徒然生出，方圆数百米卷起狂暴气流，想将对方碾碎了一般。
<br/>　　涂小安厉芒闪烁，不知好歹，见对方杀来，他只能先忙着闪躲。
<br/>　　如今，涂小安没了跟对方打架的心思，突然让渡劫，心中有点忐忑起来，隐隐更是激动。
<br/>　　他要是渡劫证道了，就可以真正的化为人形，拥有血肉之躯，不在需要靠什么化形药剂来维持。
<br/>　　三阶之后，涂小安的实力就更加强大了，可以做到真正的有恃无恐。
<br/>　　“给本王去死吧”
<br/>　　猿猴王追着涂小安打，一副不死不休的样子，这三十三级的超等妖灵说实话真的很凶残，涂小安用尽手段也不一定拿的下。
<br/>　　更别说现在马上要渡劫，没那闲工夫。
<br/>　　陡然
<br/>　　猴儿山狂风骤起，席卷天地，风卷残云，山谷上方数百里范围内的天空浮云不知道被卷飞到了哪里去，同时山谷上空的天空颜色渐渐变了颜色，变成了暗红色。
<br/>　　一股压抑的窒息感侵袭了整个山谷，天空似乎要垮塌下来了，山谷中所有猴撑着惊骇的眼神望着天，诚惶诚恐，惊恐不安起来。
<br/>　　“快让你的猴子猴孙散去，本蛇王要证道渡劫了，想让它们尝尝天雷天火的味道吗”
<br/>　　涂小安心中冥冥之中已有预感，抬头望天，天的颜色都变了。
<br/>　　“什么？证道渡劫？”猿猴王骤然停止追杀，看着暗红色的天，心中惊涛骇浪了起来。
<br/>　　没错，这是渡劫的先兆啊。
<br/>　　“该死的蛇精，你居然跑我猴儿山渡劫”
<br/>　　猿猴王差点没一口血喷出来，一般要渡劫了，都会选择一个上佳而无人打扰的地方，也是为了不殃及池鱼。
<br/>　　因为渡劫的异象太可怕了，绝大部分生灵都会在渡劫上灰飞烟灭。
<br/>　　这位到好，来他猴儿山渡劫，这是什么仇什么恨啊。
<br/>　　其实猿猴王完全是误会涂小安了，他也不知道自己怎么就渡劫了。
<br/>　　杀了别人那么多猴子猴孙，还要在别人的地盘渡劫，引起一场浩劫，涂小安心中还真的有点过意不去。
<br/>　　所以，为了减少杀戮，涂小安才解除冰冻的猴子，让它们快点逃命去。
