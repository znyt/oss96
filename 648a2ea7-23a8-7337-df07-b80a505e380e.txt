　　值得一提的是，这一次的参观，长安百姓竟然有幸目睹了御书房的真容。
<br/>　　这是他们第一次感觉自己距离皇帝如此之近，看着房间中摆放整齐的一件件物品，他们仿佛可以看到皇帝陛下坐在那里与大臣们讨论问题时的样子，又仿佛可以看到皇帝陛下批阅奏折时的认真。
<br/>　　随着进一步的参观，随着百姓的脚步一点点的走过皇宫的各个地方，大唐皇帝的形象在他们的脑海中一点点变的真实起来，不再像以前那样只是一个模糊的概念。
<br/>　　最后不知是谁可能是参观的过程让其太过激动吧，竟然振臂高呼：大唐，万胜！吾皇，万胜！万胜！万胜！
<br/>　　四周众人有那么一丝错愕，但很快便全部都加入这个行列之中。须臾间，金碧辉煌的大唐皇宫之中响起山呼海啸般的‘万胜’之声！
<br/>　　而随着这声音，一颗礼花升上夜空，一声巨响过后，在夜空中绽放出璀璨夺目的光辉，除夕盛典在这一刻正式开始。
<br/>　　……
<br/>　　李承乾的位置在太极殿的正前方，他的身侧是他的几个老婆，而在他的前方，则是为大唐立下赫赫战功的大唐将军，和那些文武百官。
<br/>　　征战沙场近十年时间，终于一朝得以归国，使得这些为国征战的将军几疑身在梦中。
<br/>　　回看身周的一切，看着那如同星河般璀璨的皇宫，看着那些进入皇宫参观百姓兴奋的脸庞，一种叫做自豪的感觉油然而生。
<br/>　　就像刚刚皇帝陛下在大殿中说的一样，如果没有他们在外征战，大唐不会有今天这样的盛世；如果没有他们在外征战，百姓也不会有今天这样幸福的笑容；如果没有他们在外征战，同样也不会有今日的万国来朝。
<br/>　　哦不，今后不会有万邦了，因为万邦只存在于过去，现在他们都是大唐人，没有什么异族和本族之分，他们全都生活在大唐皇帝陛下的羽翼之下。
<br/>　　而与下面年轻的将军们不同的是，那个老兵痞们一个个此时正挤眉弄眼的用目光交流着。
<br/>　　这帮老人渣们资格老，年龄高，李承乾也没有什么好办法约束他们，反正只要这些老头儿不当着外面那些正在参观的百姓面打起来就好。
<br/>　　没办法，今年的人太多了，除夕夜宴在太极殿根本摆不开，所以李承乾十分干脆的将宴会搬到了外面，弄成了西式自助餐的形势。
<br/>　　整个皇宫里面不管是文武百官还是普通百姓，全都可以参与到其中，只要饿了便可以过来自由取食。
<br/>　　当然，食物也并不是什么特别精美的东西，山珍海味之类的东西一概没有，有的只是很普通的几样糕点和一些美酒、饮料，毕竟在这样的日子里，很少会有人跑到皇宫里面来一饱口腹之欲。
<br/>　　另外还有一点就是太过奢华的生活其实影响并不好，会给那些进来参观的百姓留下十分不好的印象，这与李承乾最初的想法是互相违背的，不可取。
<br/>　　……
<br/>　　但尽管如此，也已经让百姓们大开眼界了。
<br/>　　各种各样的宫中糕点对那些孩子们有着致命的吸引力，而一些大人往往要矜持许多，面对孩子们递上来的果品，只是象征性的尝上一口，接着便回塞进孩子们的口中。
<br/>　　东西其实并不怎么好吃，不过跟皇家沾边了，自然也就成了美食，而事实上那些糕点在李承乾吃来，甚至还不如宫外寻常小馆子里的素包子好吃。
<br/>　　不过，这些已经都不重要了，重要的是开心，看着四周百姓辛福洋溢的笑脸，李承乾就有一种说不出的满足。
<br/>　　……
<br/>　　这里值得一提的是，这次夜宴李承乾还邀请的整个长安城中所有年过七旬以上的老者，无论男女，此时的他已经走下了高高的大殿，挤在一群老人中间，挨着个的问候。
<br/>　　遇到腿脚不灵便的，问问病情，遇到精神矍铄的竖起拇指赞叹一句，总之没有冷落任何一人。
<br/>　　在古代，能活过七十的老人其实并不多，整个长安城算一算，去掉那些真的来不了的，能进入太极宫的人数一共不超过五十人。
<br/>　　这不得不让李承乾对大唐医学院表示出严重的不满，看着脸色讪讪的贺彬乾小老头儿，如果不是因为时间不对，非得把他拖出去打板子不可。
<br/>　　苏玫跟在李承乾的身边，自然看到了他的脸色有些不大好看，神情显的有些忐忑，借着身边无人的时候赔着小心说道：“陛下，都是妾身不好，在医疗方面的工作做的不全面。”
<br/>　　“小枚啊，你是宫里的贵人，你说的话对下面的人来说那就是懿旨，但如果你想做好一件事情，最好还是要下去看看，不要单单只听下面人给你汇报上来的一些数据。毕竟眼见为实，耳听为虚，你说对么？”李承乾小声的对苏枚说道。
<br/>　　“是，妾身知道了！”苏枚脸色微红。
<br/>　　“这次记住，等上元节过了之后，先把这些老家伙都拉到医学院做一下身体检查，而这个要形成一个福利，凡年满七十的，每年都必须体检两次，不要怕麻烦，明白么？”
<br/>　　“明白了，陛下放心，妾身回去就办这件事。”苏玫重重一点头。
<br/>　　……
<br/>　　从一群老头子中间挤出来，李承乾又不顾众人反对的挤进了一群百姓中间。
<br/>　　遇到有带着小孩子进来的，就过去逗弄一下，然后从一边的果品堆中拿一些小的零食塞到孩子们手中，在小娃娃甜的腻死人的谢谢声中哈哈大笑。
<br/>　　大唐的百姓还是朴实的，面对大唐的最高统治着，大人们显的很拘束，一个个严肃的就像是要上刑场一般。
<br/>　　李承乾虽然并不喜欢这样子，不过心中却对杜构和马周等人很满意，因为至少他们没有找一些禁军装成百姓来忽悠自己。
<br/>　　而为了表示自己的满意，李承乾特地嘱咐了孔雯，马上准备一些纸笔，作为送给那些孩子们的礼物，等到这些百姓出宫的时候，凡是带着孩子的，都可以领到一份。
