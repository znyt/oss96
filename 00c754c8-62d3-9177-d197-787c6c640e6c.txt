　　当何峰将名单全部念完，声音静止了良久之后，那些新兵们都还没回过神，死去的狼牙老兵之中不仅有名声响亮的名将，更有曾经带领他们训练了两个月的教官。
<br/>　　狼牙战团的全体老兵都聚集在第一战区，当七百万丧尸群冲入的时候，他们收到的是第一波也是最强的冲击。
<br/>　　死了很多人，才换来能让迷宫区的新兵们有一个缓冲时间，将他们的死伤率降到最低。
<br/>　　而哪怕如此，新兵还是有一万人死亡……
<br/>　　在经历了这场最大战役之后，再如何什么都不懂的新兵，也或多或少清楚这一数字对比之下代表着什么。
<br/>　　甚至很多新兵都自叹一句何德何能？
<br/>　　新兵们在很久之后才渐渐回神，老兵们则是强行抑制着情绪，忍着悲痛站的笔直。
<br/>　　何峰望着下方空地上的这些人，看着数量空前壮大的狼牙战团，长期未能得到休息的他精神同样疲惫，但他还是沉默了很久，与陈少爷等一干站在人群中的中将们无声对视。
<br/>　　对他们来说这场大战空前浩大，战绩也恐怖无比，新兵们更是在这一战之后蜕变般的一跃成为老兵，实战永远比任何训练都更能让人进步。
<br/>　　客观上来说，他们不仅赢了，还赢的很逆天！
<br/>　　华夏最强战团，恐怕在今天过后，就会成为世界最强战团！
<br/>　　但狼牙战团向来凝聚力惊人，此时他们心中所想的没有半点这场大战后的荣耀，而是那些不再能活蹦乱跳的可爱面孔。
<br/>　　“将名单送回狼牙。”何峰终于开口，朝着副官下令：“立碑。”
<br/>　　立碑，是每一场大战过后的重要程序，就是将死去战士的名字印刻在巨大墓碑之上，并给每一个牺牲的人建造独立墓碑。
<br/>　　狼牙墓地，是最为神圣的地方。
<br/>　　通常每一次的立碑都不是一两个人，而是每场战争过后的统一行动，也有狼牙战团成员的集体悼念。
<br/>　　但这一次不一样，这次立碑对于死人来说，异常孤独！
<br/>　　“战争还没有结束！”何峰再次开口，声音骤然一变，严厉而充满气势：“接下来等待我们的是接连不断的尸潮，或大或小，或多或少，直到南方的那块地方丧尸不再攻来，直到丧尸王被我们斩首示众！”
<br/>　　边境尸潮的第一战一旦开打，就是一场为期不知道多长的持久战，战区的建设时刻进行着，军队的扩张和训练更不能松懈。
<br/>　　当下的末世纪元三年，没人任何军队可以进入南方的那片埋骨之地还活着出来，唯有被动的等待着敌军进攻，然后将其分批解决。
<br/>　　南方边境在上一世的末世十年都还在持续战斗，这里的一整条长线地带都属于战区，不断建立起来的防御堡垒，更是时刻准备着应对突发而来的尸潮。
<br/>　　因为战争永无止境，楚涵才会将狼牙那些部门及储备资源，全部都一次性的搬迁而来。
<br/>　　他一开始就准备好了，在这里驻扎，与这个世界最邪恶的力量对抗！
<br/>　　听着何峰的话，心里早就有底的狼牙战团全体成员都面色认真，捏紧了拳头共同对抗。
<br/>　　“现在集体休整，为下一场战役做准备！”何峰的结束词简短无比，深得楚涵真传。
<br/>　　话落，肃静的场面气势一泄而空，尤其是三战队的那几百名成员，在不休不眠的大战了七天七夜后，早已疲惫到头昏眼花毫无力气，站在这里做大战后的最后总结，完全是一颗受到刺激的心在吊着。
<br/>　　当何峰的结束语落下后，三战队内从队长到队员，无论是高达八阶甚至九阶的新人类，还是最低等阶刚入队不久的新成员。
<br/>　　都全体就地而睡！
<br/>　　三战队的人站在最前方，也站的最笔直，更站的最有个性，但当这些人全部一声不吭的躺在地上，跟集体昏厥一般似得呼呼大睡时，还是将身后的那大几万新兵给震惊的不行。
<br/>　　他们还是第一次知道，人可以累到这个地步！
<br/>　　与不休不眠战斗不止的三战队成员不同，新兵们在区域大的迷宫战区，后期面对的也大多数受到音波影响的丧尸群，在战斗之余有合理的休息时间分配，虽然时间很短也同样疲惫不堪，但再坚持几个小时也没问题。
<br/>　　在这种对比之下，新兵们对三战队的敬佩，也随之到达了高峰。
<br/>　　紧接着三战队倒下就地而躺的，是陈少爷所率领的射击队，虽然只有空战的三十人死亡，也没有和丧尸有过正面交手，但射击队与三战队一样，在这七天七夜里不眠不休，时刻注意自己射击范围内的一切异动，准备随时开枪。
<br/>　　高度集中的精神让他们早就坚持不住，要不是同样分配数量众多的银净水吊着，恐怕战斗还没结束就已经有人昏死过去。
<br/>　　当这站在不同方位的两批人前后倒下后，战斗在第一战区的侦查队，精英队及警戒队成员，也哗啦啦的接二连三倒下。
<br/>　　第一战区的战斗只有后四天时间的不眠不休，与前两批队伍相比好上很多，但相对的分配到他们手中的银净水也数量有限，所以论说疲惫程度，所有狼牙老兵都程度一致。
<br/>　　那就是累到了极限，甚至想一睡不起！
<br/>　　看着狼牙战团的老兵一个个躺下，不管地上有多脏都瞬间入睡的样子，新兵们在呆滞之中也渐渐感受到了困意袭来，这场大战的结束，让他们终于可以放下警惕，不再一有动静便惊跳而起。
<br/>　　震惊过后，新兵们全体放弃挣扎，各自找了个地方就躺了下来。
<br/>　　指挥台的参谋部成员同样没怎么休息，更别说完全没睡过觉的何峰，在三战队倒下的时候，这些人也全体撒手不管，随便找了个地方就打起了呼噜。
<br/>　　甚至包括了何峰本人……
<br/>　　对他们来说最重要的名单已经派人送往狼牙，最高长官不在场的情况下，没什么事比睡觉更重要。
<br/>　　知会段氏基地？去他·妈·的吧老子要睡觉！
<br/>　　传讯给南部联盟军？去他·妈·的吧老子要睡觉！
<br/>　　再如何重要的事都等睡够了再说！
<br/>　　于是在这场大战后，没有欢呼没有庆祝，更没有所谓的战绩统计环节，参战者以一种闻所未闻的状态，迎接了这次史上最大战役的结束。
