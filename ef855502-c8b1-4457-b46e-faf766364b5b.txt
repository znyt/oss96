　　“怎么出动这么多军人啊。”
<br/>　　“不知道，好像这个店有问题。”
<br/>　　“新开的店？这店名字还真是够任性的。”
<br/>　　“我听说啊，好像是他们没交保护费，这才把事情闹这么大。刚才还有几个宪兵被丢出来呢，这个店老板是个暴躁老哥。不好惹。”

<br/>　　王队长道。

<br/>　　这个军官挥了挥手，立即一群人就来到了店面里面。
<br/>　　王队长第一个冲了进去。店里林峰和吴三在忙着收拾东西。
<br/>　　看到林峰没走他冷笑道：“小子，这一次你完蛋了，敢动手打我？”他指着林峰恶狠狠道：
<br/>　　“牛队长就是这个家伙！抗拒执法不说！还对我动手！”
<br/>　　敢在众目睽睽之下对他动手，让他丢这么大的脸面，这下，他要十倍讨回来！！
<br/>　　“还有这种暴徒？既然如此，交给我了。”
<br/>　　牛队长大手一挥一群人扛着步枪就是冲入店内把林峰包围起来。
<br/>　　牛队长杀气腾腾的来到了店内。
<br/>　　“这群垃圾，我们在外面抛头颅洒热血，上次更是差点回不来了，这群暴徒在后面吃着喝着玩着还特么一天到晚没事找事，一个个都给老子抓去枪毙！”
<br/>　　牛队长骂骂咧咧道。
<br/>　　“可不，必须枪毙！绝不宽恕！”王队长也连忙应付道。
<br/>　　“暴徒就在眼前，就是那个小子！“
<br/>　　王队长指着林峰得意洋洋道，仿佛他已经看到了林峰被吓的跪地求饶的样子。旁边的李德也是满脸冷笑，看你还能不能牛逼的起来。
<br/>　　然而牛队长却忽然不说话了，只是擦了擦自己眼睛旋即无比狂喜的喊道。
<br/>　　“林峰战神！？居然是你？”他的一句话让一群人都愣住了。
<br/>　　“都给我把枪放下！！”
<br/>　　牛队长立即挥手让自己人把武器放下。
<br/>　　“王队长，你说这个家伙是暴徒？”
<br/>　　王队长再笨也察觉到事情不对劲了。
<br/>　　“难道他不是暴徒？”
<br/>　　“呵呵，王队长，这个人可是战神林峰！”
<br/>　　牛队长摇了摇头道。
<br/>　　“他救过陆将军和兵王的命！”
<br/>　　“什么？”
<br/>　　王队长和李德彻底的呆住了，目瞪口呆的看着林峰。
<br/>　　“你是？”
<br/>　　林峰看着牛队长糊涂道。
<br/>　　“战神，我在先遣部队里见到过战神您一面。”牛队长十分客气的说道。
<br/>　　然后给林峰来了个标准的敬礼动作。
<br/>　　他带来的一些军人也都给林峰齐刷刷的敬礼。看着林峰的眼神也充满了好奇和狂热。他们都是先遣部队的人。林峰不知道他们，但他们知道林峰。
<br/>　　“林峰战神，虽然敬礼不能表示什么，但这能够代表我们对你的感激之情，若不是战神出手帮忙，先遣部队肯定遭受毁灭性的打击！以至于威胁到曙光基地的安全！”
<br/>　　牛队长认真的说道。
<br/>　　王队长和李德虽然不知道具体发生了什么，但是这寥寥数语就已经让他们听得心惊胆战。

<br/>　　“是吗？王队长还有这种事情？”
<br/>　　牛队长立即皱着眉头看着王队长。
<br/>　　王队长连忙摇头：“没有啊牛队长，这，这都是他的一面之词，我可没有这么说。”
<br/>　　“呵呵，林峰战神会冤枉你一个小队长？王队长，没想到你居然是这种蛀虫！给我带走！”
