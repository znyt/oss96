　　“不。”
<br/>　　顾怜星连连摇头。
<br/>　　萧逸眉头一皱，虽然早有预料顾怜星不会愿意离去，但他还是更倾向于自己独自去查探。
<br/>　　顾怜星似是看出萧逸心头所想，沉声道，“萧逸公子不必担心我。”
<br/>　　“怜星早便说过了，反正也只剩那数月性命，便也不在乎那些许危险了。”
<br/>　　“我也去。”公孙火舞连忙说道。
<br/>　　萧逸眉头皱了皱，他也早猜到公孙火舞一定会跟着顾怜星。
<br/>　　“算了，随你们吧。”萧逸摇了摇头。
<br/>　　看了四周，此刻魂力的暴走早已停下。
<br/>　　天空中，除了幻光连连，斑斓之色氤氲环绕之外，再无别的异象。
<br/>　　想来，暂时也不会有什么危险。
<br/>　　嗖…
<br/>　　萧逸一把捉过二人的手臂，瞬间御空折返。
<br/>　　方向，正是险地深处，远方那座幻光洞府。
<br/>　　若是幻光险地内的魂兽，真的爆发兽潮的话，萧逸自问也没那个实力去挡。
<br/>　　险地之内，最弱的妖兽，都是圣王境魂兽。
<br/>　　强的，大多在圣皇境，甚至有着达到圣皇境巅峰，乃至比肩人类武者中的武道大能。
<br/>　　最重要的是，抵御兽潮，非一时三刻之事；若没有绝对的碾压之力，便会成为持久战。
<br/>　　自己一个圣王境武者，自是远远抵挡不了。
<br/>　　即便他全力爆发下的战力，不弱于武道大能，可那也是极其耗费元力的战斗。
<br/>　　简单来说，他根本无法在这等拉锯战中鏖战许久，这也是若魂兽兽潮爆发，他也没辙的原因。
<br/>　　若他此刻有圣皇境以上的修为，倒是两说。
<br/>　　不过，虽然拿魂兽兽潮没辙；但他一心要远遁，区区魂兽兽潮也奈何不了他。
<br/>　　若真有危险，带着顾怜星二人逃离还是有把握的。
<br/>　　十数分钟后。
<br/>　　萧逸带着二人来到险地深处的正中央。
<br/>　　从高空落下，便是幻光洞府之前。
<br/>　　洞府，在一旁氤氲幻光包裹之下，璀璨而美丽至极。
<br/>　　周遭，空荡荡。
<br/>　　没有半只妖兽，半只魂兽，更无半个人类武者。
<br/>　　“进去。”公孙火舞说了一声。
<br/>　　那里，或许便有让顾怜星虚弱的武魂修补的方法。
<br/>　　她自是急切。
<br/>　　“慢。”萧逸凝重地说了一声，细细地感知了起来。
<br/>　　几乎是他落到洞府之时的瞬间，他已眉头紧皱，心头闪过一丝诡异之感。
<br/>　　整个洞府，以及洞府周遭，给他的感觉，便是‘静’。
<br/>　　对，静，静到了极点。
<br/>　　方圆百里，无风无声，连半道低弱的妖兽呼吸声都没有。
<br/>　　“稍后跟紧我。”萧逸认真地说了一声。
<br/>　　这是这些天来，萧逸头一次如此凝重地吩咐二人。
<br/>　　说罢，萧逸率先一步，走向洞府。
<br/>　　顾怜星二人，紧紧跟随身后。
<br/>　　来到洞府前，萧逸忽然愣了愣。
<br/>　　洞府之外，并无半分守护之物。
<br/>　　没有上品圣器，亦无禁制。
<br/>　　萧逸思索了一下，脸色凝重，随后直接进入洞府。
<br/>　　一入洞府，首先映入眼帘的，便是一个巨大宽阔，且空荡荡的广场。
<br/>　　倒是有些像天星洞府之内的景象。
<br/>　　萧逸待了一阵，感知四周，发现并无危险。
<br/>　　这才传音给顾怜星二人，让二人进来。
<br/>　　二人闪身而进。
<br/>　　公孙火舞打量了眼四周，惊讶道，“咦，这就是幻光禁地吗？没想到倒是和天星洞府那里一模一样。”
<br/>　　顾怜星笑笑，道，“幻光险地，被誉为十八府险地中最危险的险地。”
<br/>　　“这里的传闻，又骇人至极；每次月圆之前，哪里会有武者敢留在险地之内。”
<br/>　　“想来，幻光洞府多年来，也只有我们三人进过来窥其究竟。”
<br/>　　此时，萧逸的目光，正直视广场的尽头。
<br/>　　尽头处，空无一物。
<br/>　　但在他很早之前的猜测中，十八府险地，必有联系。
<br/>　　如各险地，都存在异象；各险地，都存在洞府，且洞府需要特定的条件方能出世。
<br/>　　又如那些上品圣器，等等。
<br/>　　现在，这幻光洞府之内，连布局都与天星洞府一模一样。
<br/>　　这便更加证实了萧逸之前的猜测。
<br/>　　那么，也就是说，这空荡荡的广场尽头，必然也如天星洞府一样，存在主府。
<br/>　　萧逸身影一闪，来到广场的尽头。
<br/>　　“果然。”萧逸感知了一番后，面露喜意。
<br/>　　在他的感知中，面前这尽头处，空荡荡，但却有有着一隐秘的屏蔽阵法。
<br/>　　“散。”萧逸低喝一声。
<br/>　　阵法，并非攻击型或者防御型的强大阵法。
<br/>　　只是一个普通的屏障阵法。
<br/>　　萧逸轻松便将其破去。
<br/>　　哗…
<br/>　　一阵阵法之力，在空气中溢散。
<br/>　　不过片刻，阵法之力消散一空。
<br/>　　面前，主府轰然而现。
<br/>　　“果真如此。”萧逸面露喜色。
<br/>　　天星洞府那里，拥有主府；这里，同样如此。
<br/>　　如果没有猜错的话，其他各险地的洞府，也应该差不多。
<br/>　　不过，天星洞府那边，存在着人为因素。
<br/>　　多年前的一位天星府府主，拼了性命，取走了一盏天星盏。
<br/>　　故天星洞府内的主府，禁制不全。
<br/>　　每次武者进入，需要先吸摄出大量的星光力量；让得整个洞府广场的星光力量达到足够浓郁的程度，方能让主府出现。
<br/>　　而幻光洞府这里，很显然已经无数年无人进入了，也不存在有武者干预的情况。
<br/>　　自然的，主府仅仅是在一道屏蔽阵法之内。
<br/>　　阵法一散，主府便自动现形了。
<br/>　　萧逸快步进入，映入眼帘的，是一色彩斑斓的巨大‘圆球’。
<br/>　　圆球，约莫数米大小。
<br/>　　通体浑圆，看起来，只觉是一团能量团。
<br/>　　整个能量团，氤氲萦绕，充斥着各色光彩，耀眼至极。
<br/>　　“这是？”萧逸皱了皱眉。
<br/>　　“魂力？”
<br/>　　不错，在他的感知中，这个‘圆球’，是一团魂力。
<br/>　　而最让人惊讶的是，这团魂力所充斥着的浑厚以及精纯，远远超出了他能理解的范畴。
<br/>　　或许，这已经不能单纯称之为魂力了。
<br/>　　萧逸也不知该怎么形容。
<br/>　　总之，这团魂力，精纯到极点。
<br/>　　正常的魂力与之相比，就如充斥天地间的普通灵气与之珍贵无比的上品灵石之相比。
<br/>　　其差距，起码在千百倍以上。
<br/>　　“这就是幻光洞府内的秘密，或者说奥秘？”萧逸疑惑自语一声。
