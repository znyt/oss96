　　接下来的整整一夜，都没发生什么事情，直至第二天的清晨，刘哔感觉到周围的空气一阵波动。
<br/>　　“即将开始进行位面传送，请宿主做好准备。”
<br/>　　系统提示音从脑海中响起，让刘哔停止了修炼，舒展了一下有些不太舒服的身体。
<br/>　　昨夜没有找到住宿的地方，他只能等到雨停之后，搭了个建议帐篷，但因为太过于潮湿，让他身体微微受了一些寒气。
<br/>　　不过对于一个丧尸来说，这点寒气也造不成什么影响。
<br/>　　“10……8……3……1……”
<br/>　　随着系统的倒计时进入尾声，一个带着金色的光圈从刘哔身体四周亮起，随后一阵强烈的眩晕感传来，等清醒过来的时候，四周的景物却是已经变了。
<br/>　　这是他一开始进行穿梭的小巷，看清四周的环境之后，刘哔立即反应了过来。
<br/>　　和他离开的时候似乎没有太大变化，只是街道上的丧尸似乎要少了一些，但粗略的看上去，依旧是一副密密麻麻的样子。
<br/>　　手边没有表，他也没办法辨别现在的时间，从太阳的高度上来开，现在大致应该是中午一两点钟吧。
<br/>　　稍稍犹豫了一下，刘哔径直从小巷中走出，对于他的突然出现，四周的丧尸没有丝毫反应。
<br/>　　果然，丧尸是不会对这幅身体感兴趣的吧。
<br/>　　放下了心，刘哔来到已经看不出原本样子的马路上，随手找了一辆越野车，向记忆中基地的方向驶去。
<br/>　　车辆这种东西，在末世的大街上随处可见，尤其是市区，在重重丧尸的守卫下，也有许多汽车还残留着汽油。
<br/>　　街道上的丧尸被汽车声音吸引，密密麻麻的从四周的建筑物上涌了出来，刘哔视若无睹的向前开着，没过多久就被尸群堵死，方才无奈的将汽车收入空间戒指中，一直到城外方才重新开了起来。
<br/>　　在不使用翅膀的情况下，汽车要比他步行的速度快上许多，马上就要回基地了，他不想暴露自己身体上的异常。
<br/>　　只是，他始终有些担心自己的弟弟，不知道离开了这么久的时间，那边的情况是否还好。
<br/>　　虽然有基地的保护，但没什么劳动能力的人，基地恐怕也不会收留，那种情况，也是在让人放心不下。
<br/>　　大概行使了一个多小时的时间，刘哔已经看到了基地熟悉的轮廓，艳阳高照，将眼前的一切照的白花花一片。
<br/>　　稍稍整理了身上的衣物，他向远处的城门驶去。
<br/>　　虽然已经刻意减缓了车速，但是这样短短的一段路，也只花费了几分钟的时间就赶到了。
<br/>　　基地门外如同往常一样，排起了一串熙熙攘攘的队伍，入口处站着几个守卫，穿着特殊材料打造出的盔甲，有说有笑的交谈着什么。
<br/>　　他的到来没有引起任何人的注意，虽然已经在这个地方生活了六七年，但像刘哔这样底层的幸存者，基本没人会记住他。
<br/>　　在这样残酷的世道，弱者是没有朋友的，大家都忙着生存，既然彼此都带来不了什么帮助，那还不如各走各路。
<br/>　　毕竟是附近唯一的大型幸存者基地，每天都有无数幸存者进出，门外排队等待进入的人数，自然也不会太少。
<br/>　　看着面前排成长龙的队伍，刘哔眉头深深皱起，这要是排队，至少要花费两个多小时，甚至更长的时间。
<br/>　　记挂着自家兄弟的安危，他的心情有些烦躁。
<br/>　　将手搭在方向盘上，刘哔没有多加犹豫，径直开车冲入了长龙前方，硬生生插入了由幸存者组成的队伍。
<br/>　　这样的举动，自然引起了很多人的愤怒，他周围三五成群的幸存者，皆是骚乱了起来。
<br/>　　前方负责检查入城的守卫，淡淡的往这边瞥了一眼，随即收回了目光，像是什么事情都没看到一般。
<br/>　　这样的事情，几乎每天都在发生，并不稀奇，反正这些人都还没有进入基地，守卫也懒得去搭理他们。
<br/>　　毕竟，多一事不如少一事，万一哪天碰到一个脾气古怪的高阶异能者，出手击杀了他们，他们也没有地方说理去。
<br/>　　弱肉强食，这项法则适用于末世的每个地方，他们可不相信基地会为了庇护他们，而去得罪一个高阶异能者。
<br/>　　“喂，有你这么插队的吗，我们排了半天，你什么不说就插在我们前面？”一片嘈杂的指责声中，三四个壮硕男子推开人群冲了过来，将刘哔所在的车辆包围，伸手重重的拍打了几下车窗。
<br/>　　玻璃缓缓落下，刘哔平静的看着几人，道：“怎么，想说说？”
<br/>　　像是被猛兽盯上一般，几人心中涌上一股惊悚的感觉，经过无数战斗所养成的直觉告诉他们，眼前的这个男人，并不好惹，他身上的那股气息，绝对不是一般的异能者。
<br/>　　“不是，主要是我朋友差点被你的车撞伤，这……”众目睽睽之下，几人都有些骑虎难下，最前方的一个光头男子，硬着头皮开口，语气却是软了下去。
<br/>　　平息了体内调动起来的灵气，刘哔转过头去，打断了光头男还未说完的话语：“滚！”
<br/>　　周围议论纷纷的声音逐渐小了下来，在场的人都不是傻子，自然猜到了大概是什么情况，估计这个越野车上的男子，也不是什么好相与之辈。
<br/>　　光头男子脸色一阵转换，却也没有继续纠缠，挥了挥手，带着众人后退了一段位置，排在刘哔车后。
<br/>　　毕竟是在基地门口，刘哔了解规矩，也没有做的太过分，故此，刚才的冲突中并没有动手。
<br/>　　这样插队的现象，也不是个例，他之前排队的时候，也经历过类似的事情，只不过当时他所扮演的角色，是弱势的一方罢了。
<br/>　　而他也没觉得自己的做法有什么不妥，之所以这般努力的修炼，不就是为了成为强者，比那些弱者要多拥有一些资源吗。
<br/>　　总有人说，变强是为了守护，而不是争夺，刘哔一直觉得，那都是一些屁话。
<br/>　　难道所谓守护，就不算是争夺吗，归根结底，守护也是强者行使权力的一种体现罢了。
<br/>　　归根结底，也没有什么不同。
