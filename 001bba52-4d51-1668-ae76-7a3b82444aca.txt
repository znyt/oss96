　　“握草！”
<br/>　　叶尘忍不住爆了一声粗口。
<br/>　　眼看着，空间通道就要崩塌，一旦被卷入空间乱流之中，以如今的修为，几乎是九死一生！
<br/>　　“琉璃金身！”
<br/>　　“法相天地！”
<br/>　　叶尘也来不及细想，当即收起赤羽飞舟，将防御开到最强，然后召出紫琼剑，狠狠一剑斩落，
<br/>　　“虚空斩！”
<br/>　　轰隆隆！
<br/>　　叶尘全力一剑之下，在空间通道没有崩塌之前，率先强行撕开了一条空间裂缝，然后迅速钻了过去。
<br/>　　就在叶尘钻入裂缝中的那一刹那，空间通道也彻底崩塌！
<br/>　　刺啦啦！
<br/>　　叶尘顿时感觉到一股强大的吞噬力量，想要将他整个人撕裂一般。
<br/>　　“给我，破！”
<br/>　　叶尘大喝一声，强行动用自己掌握的那一丝空间之力，硬生生闯出一道出口。
<br/>　　总算在通道彻底崩塌之前，侥幸逃脱了虚空乱流的吞噬，回到了外部世界。
<br/>　　呼呼呼！
<br/>　　叶尘瘫坐在地上，足足喘息了好一会，方才回过劲来，低头看了一眼身上的情况，顿时苦笑不已。
<br/>　　只见他全身的衣服，都已经被彻底绞碎，浑身上下都布满了一道道伤口，几乎成了一个血人！
<br/>　　原本体内的伤势，都还没有完全愈合，如今旧伤之上，又添新伤，五脏六腑几乎全部报废！
<br/>　　若非他体内有万古长青珠护着心脉，这一次就算不死，恐怕也得脱层皮。
<br/>　　饶是如此，没有一两个月的时间，恐怕也难以恢复。
<br/>　　“这到底是什么情况？难道这是逍遥仙珠有什么问题？回头一定要好好问问九老！”
<br/>　　叶尘郁闷不已，
<br/>　　他眼下这种状态，赶路肯定是不行了，只能先暂时休养一阵。
<br/>　　就在叶尘准备就地调息的时候，忽然远处传来一道威严的声音，而语气中透着惊疑，
<br/>　　“那边怎么突然这么大动静？随我去看看！”
<br/>　　原来，叶尘强行破开空间通道，引发了不小的动静，竟然惊动了周围的人。
<br/>　　唰唰唰！
<br/>　　随着声音落下，一道道人影在山林中闪过，一阵躁动过后，一群人出现在叶尘的面前，
<br/>　　为首的是一名中年男子，骑着一头浑身长满黑色鳞甲的庞然怪物，气势十足！
<br/>　　而中年男子的身后，还跟着一队人马，应该是他的下属。
<br/>　　众人看到一身血迹的叶尘，顿时一个个紧张起来，自动围成一个圈，将叶尘包围了起来。
<br/>　　“你是何人？为何会出现在我黑岩城的后山之中？”
<br/>　　为首的那名中年男子，死死盯着叶尘，暴喝了一声道。
<br/>　　叶尘抬头扫视了一圈众人，心中暗暗叹息，
<br/>　　虽然这些人的实力并不强，他若是强行调动真元，也足以将这些人轻松干掉。
<br/>　　可是，他体内伤势太过严重，若是强行动武，伤势将会更加严重，到时候恐怕就要休养更长的时间。
<br/>　　更何况，他虽然是迫不得已，闯入别人的宗门之中，可毕竟是他理亏，如果再动手杀人，就更说不过去了。
<br/>　　一念及此，叶尘缓缓站起来，淡淡道：
<br/>　　“抱歉！不小心闯入贵地，我这就离开！”
<br/>　　说完，叶尘便要转身离开。
<br/>　　不料，那中年男子顿时冷冷一笑，
<br/>　　“朋友！你擅闯我黑岩城禁地，这样就想离开，未免也太不把我黑岩城放在眼里了吧？”
<br/>　　叶尘顿时眉头微微一皱，
<br/>　　“那你想怎样？”
<br/>　　中年男子冷哼一声，
<br/>　　“你身上血气如此之重，又突然出现在我们黑岩城之中，肯定有所图谋！”
<br/>　　“给我老实交代！你是不是邪光阁的人，来我们黑岩城，到底想要干什么？”
<br/>　　叶尘听到这话，眉头顿时皱的更紧了，
<br/>　　他本不想伤人，可是眼前这家伙，却咄咄逼人，他可不想在这里干耗着。
<br/>　　就在叶尘暗暗调动体内真元，准备突围而出的时候，忽然又响起一道清脆悦耳的声音，
<br/>　　“庞叔叔，发生什么事了？”
<br/>　　随着这道声音落下，一头纯白色妖兽从人群后方，缓缓走到了叶尘跟前。
<br/>　　而在那白色妖兽的背上，坐着一名身穿淡紫色裙衫的少女。
<br/>　　这少女容貌可爱精致，看上去也就十四、五岁，宛如一个瓷娃娃一般，还扎着一对冲天辫，显得越发可人。
<br/>　　众人看到这少女出现，顿时纷纷躬身行礼，
<br/>　　“二小姐，您怎么来了？”
<br/>　　可爱少女一出现之后，一双乌溜溜的大眼睛，就一直盯着叶尘在看，
<br/>　　“庞叔叔，这人是谁啊？怎么身上这么多血？”
<br/>　　可爱少女一边说着，便要催动胯下的妖兽，朝叶尘靠近。
<br/>　　那中年男子当即身体一闪，闪电般挡在了可爱少女的跟前，一脸紧张道：
<br/>　　“二小姐小心！这人突然出现在后山之中，行迹十分可疑，属下严重怀疑，他极有可能是邪光阁派来的奸细！”
<br/>　　那可爱少女听到这话，顿时眨了眨明亮的大眼睛，又在叶尘身上打量了一番，噗嗤一笑，道：
<br/>　　“庞叔叔，你一定是搞错了吧！你看上他身上的伤这么严重，邪光阁怎么会派一个身受重伤之人，来我们黑岩城呢？”
<br/>　　“而且，我看这位大哥哥，也不像是坏人呢！”
<br/>　　中年男子顿时大急，
<br/>　　“二小姐！这人不可貌相啊！就算他不是邪光阁的奸细，鬼鬼祟祟出现在这里，也绝对十分可疑……”
<br/>　　没等中年男子说完，可爱少女摆了摆手道：
<br/>　　“庞叔叔，你看他受了这么重的伤，再耽搁下去说不定会死人的，不如就让我先把他带回去医治，等先把身上的伤势只好再说吧！”
<br/>　　“二小姐，这……”
<br/>　　中年男子还想再劝，可爱少女直接俏脸一板，
<br/>　　“怎么？难道我连这点小事都做不了主吗？”
<br/>　　“我……”
<br/>　　中年男子无奈之下，只好乖乖站到一旁。
<br/>　　可爱少女这才重新露出一抹可爱的笑容，冲着叶尘笑道：
<br/>　　“大哥哥，我叫灵溪，是一名药师呢，你身上伤很重，必须要尽快医治！你跟我走，我帮你疗伤好不好？”
<br/>　　叶尘双目和可爱少女对视了片刻，见她眼神清澈见底，丝毫不像是在伪装，心头也颇生好感，
<br/>　　如此心善、干净的女孩，在尔虞我诈的修真界，还真是不多见！
<br/>　　“好呀！那就多谢你啦！”
<br/>　　叶尘也报以微笑，点头答应了下来。
