<br/>　　“十王者中，大魔王这边就有两个，估计对面想赢都难。”
<br/>　　“话也不能这么说啊，大魔王这边有两个十王者选手，对面的肯定也不会少于这个数字的，就算是少了，那么在其他位置上肯定也会补回来的。”
<br/>　　“没错，双方肯定不会出现明显的实力差距的，系统会自动将双方拉成55战力的样子。”
<br/>　　看到叶帝这边的几个选手，直播间里的众多观众纷纷开口说道。
<br/>　　“这一局god应该在对面了，刚刚我看好友列表里面显示god也在排队。”叶帝笑着说道，因为还没有正式开始游戏，所以他现在倒是还显得轻松。

<br/>　　最终叶帝他们这边选择了：上单波比、打野男枪、中路沙皇、下路则是女警和扇子妈这一对强势组合。
<br/>　　而对面则是选择了：上路巨魔、打野盲僧、中路机械先驱、下路则是ez和娜美这对组合。
<br/>　　“这一局大魔王应该带净化啊，对面已经选择盲僧了。”
<br/>　　“带你妹的净化，这种局和那种十王者战不一样好么，而且时刻保持在那样高度的集中可是很累的。”
<br/>　　“没错，比赛和排位不一样的，我估计大魔王就算是在排位中遇到盲僧，恐怕也不想在使用净化了，不用想都知道那种专注肯定很耗费精力。”
<br/>　　“卧槽，对面的下路是日本的小石兄弟，大魔王这一局一定要狠狠的将他们按到地上摩擦啊，老子看他们这个国家就是不爽。”
<br/>　　“对，一定要摩擦他们，我也看他们不爽，本来就是咱们的地方，还敢过来和咱们抢，太不要脸了。”
<br/>　　“这一把我的情绪也被调动起来了，大魔王你要是打不赢他们，你就可以自裁谢罪了，旁边的那个工作人员这一局一定要伺候好大魔王。”
<br/>　　进入排队等待界面的时候，当众人看到对面下路组合的那对id的时候，一个个顿时暴躁了起来，飞快的在那里发送着弹幕。
<br/>　　而一旁本来以为直播间上针对自己的弹幕已经风消云散了，这个时候倒是抽空瞄了一眼，可是这一看，蒋念心里却是再次喷了一口老血。
<br/>　　我去你妹的！你们大魔王排到小石兄弟了，这尼玛居然也能把节奏带到我的身上，你们到底亏不亏心啊。
<br/>　　不过这个时候蒋念倒是也没敢说话，他现在只能装作自己根本没看到，要不然的话那群粉丝不知道又要怎么诋毁自己呢。
<br/>　　而叶帝在看到对方下路组合是小石兄弟的时候，他的眉头倒是也跟着挑了起来。
<br/>　　不过他本人倒是并没有像直播间里的那些粉丝戾气那么重，虽然他看起来确实应该那样才对，只不过许多事情都需要用变向的眼光去看的，对方国家并不一定所有人都是坏的，那里也会有很多好人，很多喜爱和平的人。
<br/>　　同理国内也是如此，并不一定这些义愤填膺的就都是好人，他们在现实里做过多少龌龊事谁也不知道。
<br/>　　所以叶帝看待事情的时候，从来不会那么武断，来到这个世界后，他基本上都是用着一颗包容的心对待别人的。
<br/>　　而他之所以挑眉，完全是因为这还是他第一次和对方在排位中遇到呢。

<br/>　　虽然没有遇到过他们，但是在他们那个十王者群中，叶帝却总是能够听到金相赫他们在谈论着小石这对兄弟。
<br/>　　因为两人是双胞胎，所以在下路的配合上显得要更加的有默契一些，可能是心有灵犀的缘故，两人在下路几乎没有出现过失误，每一次都能在下路打开局势。
<br/>　　既然是领土战的对手，这个时候叶帝自然是想要好好研究一下他们了，这样的话也好知己知彼。
<br/>　　“蒋念你让老秦还有景禹他们观战，ob一下下路视角，研究一下小石兄弟。”叶帝冲着一旁的蒋念说道。
<br/>　　虽然他们之前也都看过小石兄弟的比赛，但是那毕竟不是亲自上场，这一次有着叶帝在场上体会，在加上秦霄他们在第一视角位研究，想必肯定会更加的清晰的。
<br/>　　蒋念那边这一次倒是没有反驳，连忙跑到客厅将正在那里研究东西的两人找了过来。

<br/>　　“其实倒是没有什么好担心的，他们日本也就这么一个下路组合强上一些，咱们中上野可都是要碾压他们的。”香蕉淡淡的说道。
<br/>　　“知己知彼方能百战不殆，这是最后一场领土战了，一点遗漏都不能有，要不然咱们可就是千古罪人了。”李景禹跟着说道。
<br/>　　而就在几人说话的时候，叶帝他们已经是进入到了游戏当中。
