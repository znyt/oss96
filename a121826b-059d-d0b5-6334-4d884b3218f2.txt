　　“哼...来给我送本命金丹的是吧”
<br/>　　涂小安立刻将金丹抓在手中，然后直接放到嘴巴吞了进去。
<br/>　　他的力量消耗太严重，这颗三阶本命金丹蕴含强大的纯粹能量，刚好可以给涂小安填补一下。
<br/>　　“蛇妖，你别高兴的太早了”
<br/>　　涂小安刚吞下金丹，可上方遮天蔽日的昏暗了下来，一张恐怖的鼠口吞噬而来。
<br/>　　“叮...鼠族三阶妖灵，变异吞天鼠，等级三十三级”
<br/>　　一道系统信息传来，涂小安直接被巨大嘴口吞了进去，防不胜防。
<br/>　　“哈哈...蛇妖被我吞天鼠吞进肚子里去了，我立了大功了”
<br/>　　旋即，一头更为庞大的巨型老鼠狰狞可怖的大声咆哮，仿佛中了大奖一样，昭告天下。
<br/>　　这句话还未落下，巨型老鼠的头顶也遮天蔽日的昏暗下来，充满神兽气息的青龙盘踞在上，发出威严不凡的声音：“我家主人，也是你能吞的，快给本神龙吐出来”
<br/>　　青龙的威严气息笼罩，这变异吞天鼠没来由的就心生惊恐起来，跟青龙一比，忽然感觉自己无比的卑贱渺小起来。
<br/>　　“你堂堂神兽青龙，居然认一条蛇为主，简直还不如...”
<br/>　　变异吞天鼠本来想说你还不如我这只老鼠呢，在这里牛逼什么。
<br/>　　但一句话还没说完，它的表情就惊惧了起来，巨型体形更加的暴涨变大，紧接着就痛苦的嚎叫起来，最后难以控制的爆炸了。
<br/>　　爆炸后，一道清瘦身影出现显化，一颗崭新的金丹握在手中：“作死的东西，蛇王你也敢吞，轮回去吧”
<br/>　　“主人，你没事吧”
<br/>　　青龙连忙开口。
<br/>　　“没什么事情，就被恶心了一下”涂小安稍稍大意就被一只老鼠给吞到了肚子里去，可一旦破体而出，这只三十三级的变异吞天鼠也就往生极乐了。
<br/>　　旋即，涂小安表情俨然起来，他这一被干扰，鼠潮肯定再次恢复推进，蜂拥而至。
<br/>　　可下一秒，他傻眼了，先头部队的鼠潮如变形金刚一样的形成了一只巨大的拳头，朝着凌空的涂小安诡异的砸来。
<br/>　　空间顷刻扭曲，激烈的爆响声此起彼伏。
<br/>　　“卧靠，还有这种操作？”
<br/>　　尼玛，这肯定是灰妖王躲在暗中控制鼠潮，如同手脚一样的变化莫测。
<br/>　　“主人，你歇歇，我来”
<br/>　　青龙神兽气息暴涨，一个神龙摆尾，朝着鼠潮形成的巨大拳头抽去。
<br/>　　轰...！
<br/>　　无数鼠兽的尸体簌簌而落，而青龙倒退五六公里之外，大吃一惊，好伟岸的力量啊。
<br/>　　伪妖王的青龙，可谓算是被这一拳打飞出去了，后劲里非常的大。
<br/>　　虽然撞击死了很多的鼠兽，但别人数量源源不绝，杀不尽，死不完。
<br/>　　别看一个拳头，但可是多如几十万的鼠兽组成，集中力量爆发，绝对不是开玩笑的。
<br/>　　鼠潮本来真正的敌人就是涂小安，现在涂小安自己作死的抵挡，更是正中敌人的下怀。
<br/>　　一个十数丈的手掌凭空出现，单手凭空一拍，犹如平地惊雷，轰鸣声不绝而与，从天而降，铺天盖地的朝着涂小安盖去。
<br/>　　“尼玛，这是不是太夸张了”
<br/>　　涂小安为什么有种玄幻视角，十数丈的手掌，其中包含多少鼠兽啊，空气中有无数道吱吱喳喳的尖锐叫声，叫的人头皮发麻。
<br/>　　“火拳”
<br/>　　涂小安震惊归震惊，手下动作也不慢，全身火焰缭绕，一拳轰了出去，漫天火焰横流。
<br/>　　十数丈的手掌直接被轰的溃散开来，天空鼠如雨下，倾盆大雨一样挥洒。
<br/>　　巨大的鼠潮宛如觉醒，形成了十几道同样的巨大手掌，呼啸狰狞的朝着涂小安拍去，每一只手掌都由几十万鼠兽组成，排山倒海而来。
<br/>　　涂小安目露苦笑，一道道十数丈的巨大手掌拍来，空间不断的扭曲轰鸣。
<br/>　　一拳轰散一只，另一道巨大手掌接踵而至。
<br/>　　整个猴儿山变得无比的恐怖，一道清瘦的身影在十几道巨大手掌中穿梭，天空中充斥着鼠兽的尖锐声音。
<br/>　　“倚天剑”
<br/>　　锵！
<br/>　　涂小安手中三尺青锋就地显化，下一刻，一抹惊艳的剑气凭空出世，将一道巨大手掌一分为二，无数鼠兽簌簌而落，支离破碎，血肉模糊。
<br/>　　紧接着，一剑横空，几十丈剑芒仿佛星河倒卷，以一种狂猛无匹的姿态，斩杀而下，这一刻，茫茫剑气，遮天蔽日，让无数星辰为之暗淡。
<br/>　　“来啊，特么来多少，死多少”
<br/>　　涂小安杀的眼睛都红了，空气中布满了血的味道，他现在终于明白灰妖王为什么会引发鼠潮了。
<br/>　　因为鼠潮虽然非常的可怕，但是对于能腾空驾雾的妖灵来说并不会构成太大的威胁，除非来不及反应就被重重包围了。
<br/>　　然，如今鼠潮形成的攻击模式，有徒手摘月之能，那就另当别论了。
<br/>　　这一阵杀戮，硬生生被涂小安拖了二十分钟过去。

<br/>　　涂小安凌空而立，被蛇潮中那条百丈之大的巨蟒给惊呆了。

<br/>　　巍峨惊天巨蟒杀入鼠潮，简直比涂小安还恐怖，血盆大口一张，如鲸吞鱼一样，一口下去不知繁几的鼠兽落入蟒口。
