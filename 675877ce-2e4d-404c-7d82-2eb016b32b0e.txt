　　楚风眠的这一剑，直接砸落在金罗殿上。
<br/>　　这身为天级灵器的金罗殿，如今都是被直接被砸破禁制。
<br/>　　甚至是身处于其中的陆雀，都是被直接震伤。
<br/>　　隔着天级灵器，都将陆雀重伤，陆雀感觉的到，要是这一剑没有金罗殿。
<br/>　　现在的他，只怕都要陨落了，这力量，实在是太恐怖了。
<br/>　　更令陆雀恐怖的是，这金罗殿，如今的禁制阵法，居然都在破碎，好像是在这一剑之下，金罗殿都要散架了。
<br/>　　天级灵器，都要被轰碎，这种力量，居然是一名神力境武者做到的，这说出去，无人能信。
<br/>　　陆雀疯狂的狂吼这，也是将一身的灵力，全然爆发出来，一股股的灵力，全部加持在金罗殿上，来帮助金罗殿，来对抗楚风眠这一剑的力量。
<br/>　　一刹那的时间。
<br/>　　在陆雀看来，却是如同无数年一样漫长。
<br/>　　一直到这金罗殿上的禁制阵法，已经被破的七七八八的时候，楚风眠剑锋的力量，才终于变弱了一些。
<br/>　　“挡下了！”
<br/>　　陆雀脸色大喜，这一剑，只怕也是楚风眠的底牌。
<br/>　　只要能够挡下，楚风眠也就再也伤及不到他了。
<br/>　　但是陆雀的喜悦，还没有持续一秒钟，外面天空之上的楚风眠。
<br/>　　他手中的祖龙至尊剑，已经是再次积蓄完成了力量。
<br/>　　又是一道足有万米的剑气，突然从天而降，这剑锋，再一次的落在了金罗殿上。
<br/>　　这一剑的力量，比起刚刚那一剑，丝毫不弱，甚至是比起刚刚那一剑，都还要强大几分。
<br/>　　“怎么可能！他还有灵力？”
<br/>　　陆雀看到这一幕，面色呆滞。
<br/>　　他本以为，这样的攻击，以楚风眠的境界，能够强行催动一次，便是不易了。
<br/>　　不止是陆雀，就算是在场无数武者，都想不到。
<br/>　　楚风眠居然还可以再度打出一剑。
<br/>　　谁都没有反应过来。
<br/>　　只见这一剑，落在金罗殿上，顿时这已经破碎的七七八八的金罗殿，居然是在一瞬之间，都被摧毁了，全部崩塌毁灭。
<br/>　　这陆家的至宝之一，天级灵器金罗殿，如今就这样的在楚风眠的手中，被直接打碎了。
<br/>　　金罗殿碎的一刻，无数人的眼珠子都要掉了下来。
<br/>　　这可是天级灵器，打碎一件地级灵器，算不得什么本事，因为一件天级灵器的坚韧，只怕是比起一百件地级灵器加起来都要坚韧。
<br/>　　生死境巅峰强者都做不到的事情，楚风眠居然做到了。
<br/>　　“噗！”
<br/>　　一道身影，从金罗殿碎片中，直接飞了出去。
<br/>　　这人影，正是陆雀。
<br/>　　金罗殿被破，其中的余波，都是直接将他打成重伤。
<br/>　　陆雀的口中吐出无数鲜血，身上的灵力都失去的七七八八，简直是濒死状态。
<br/>　　这还是金罗殿刚刚已经是消耗了楚风眠剑锋九成九的力量。
<br/>　　其中的余波，都足以让陆雀重伤，再多一点力量，现在的陆雀，只怕已经是陨落了。
<br/>　　“过来！”
<br/>　　楚风眠大手一挥，这陆雀的身躯，被楚风眠直接握在手中。
<br/>　　任由陆雀如何反抗，都无法脱离楚风眠的手心，这陆雀便是如同一只蝼蚁一样，被楚风眠捏在手中。
<br/>　　生死台周围，鸦雀无声。
<br/>　　看着眼前的这一幕发生，简直还都在如梦如幻之中。
<br/>　　尤其是陆雀，就这样被楚风眠捏在手中，这一副画面，简直是超出了众人的认知。
<br/>　　“这，已经不是人了。”
<br/>　　“天才，妖孽，这些词放在楚风眠的身上，都是一种侮辱了。”
<br/>　　“我们……”
<br/>　　一些种子弟子的内心，都是失落无比。
<br/>　　他们的资质都是极高，能够成为种子弟子的，放在外面北域之中，都是绝世天才，可现在他们的骄傲，在楚风眠的面前都是被击的粉碎。
<br/>　　天才？在楚风眠的面前，没有人敢配称为天才。
<br/>　　就连寒月漓如今的目光，都是充满这震撼。
<br/>　　她以为她先一步度过了生死大劫，在加上拥有玄寒之体，如今就可以比起楚风眠先走一步。
<br/>　　但是现在看来，就算是楚风眠没有度过生死大劫，他的实力就已经不比寒月漓弱了。
<br/>　　让寒月漓出手，这陆雀最惨的结果，也不过如此。
<br/>　　“楚风眠，你速速放开我，再不放开我，就是你灭顶之灾的时候！”
<br/>　　陆雀被楚风眠握在手中，简直是屈辱到了极点。
<br/>　　他这样一尊大人物，北邙圣域的长老，现在像是一只小鸡一样被楚风眠捏在手中，被众人看着，这让他都无法接受。
<br/>　　“死到临头，还敢来威胁我？”
<br/>　　听到陆雀的话，楚风眠眉头一挑道。
<br/>　　“敢来对付我的人，都要死，你也一样！”
<br/>　　“手下留人！”
<br/>　　长治长老听到楚风眠的话，脸色惊变，猛然的说道。
<br/>　　“你要是杀了陆雀，就真的跟陆家撕破脸皮了，陆家的人在北邙学院中，可所谓是一手遮天，你不能杀他。”
<br/>　　“是吗？不杀了他，陆家的人也会对付我，更何况，从来没有人可以威胁我！”
<br/>　　楚风眠冷笑一声，根本便是无视这长治长老的话。
<br/>　　今日放了陆雀跟陆画？
<br/>　　以今日的仇怨，就算是放了他们二人，也不会善终，倒时候这二人，也一样会再度来杀楚风眠的。
<br/>　　陆家也一样会追杀楚风眠。
<br/>　　既然恩怨无法化解，那还在乎什么，能杀一个是一个。
<br/>　　楚风眠还从来没有放过敌人的习惯。
